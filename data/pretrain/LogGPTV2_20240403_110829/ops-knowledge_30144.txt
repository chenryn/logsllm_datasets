User: How to install Java environment and configure environment variables on CentOS?
Assistant: Here are the steps to install the Java environment and configure the environment variables on CentOS: 1. Enter the following command in the terminal to get a list of available Java versions: ```sudo yum search java | grep -i --color JDK``` A list of available Java versions will be listed. 2. Select the desired Java version in the list and install it. For example, enter the following command to install Java 11: ```sudo yum install java-11-openjdk-devel``` You may need to enter the root password during installation, please provide the credentials. 3. After the installation is complete, you can enter the following command in the terminal to check whether Java has been installed correctly: ```java -version``` will list the installed Java version information and other related information. 4. Configure Java environment variables. Open the terminal and enter the following command: ```sudo nano /etc/profile``` Add the following content at the end of the file or other locations (modify according to your Java version and installation path): ``` JAVA_HOME=/usr/lib/ jvm/java-11-openjdk-amd64 export JAVA_HOME PATH=$PATH:$JAVA_HOME/bin export PATH ``` Save and exit the file. 5. Reload the environment variables. Enter the following command in the terminal: ```source /etc/profile``` 6. Check whether the Java environment variables are set correctly. Enter the following command in the terminal: ```echo $JAVA_HOME``` The output should be the path of your Java installation. For example: ```/usr/lib/jvm/java-11-openjdk-amd64``` Enter the following command: ```echo $PATH``` The output should contain the path to your Java installation. For example: ```/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/lib/jvm/java-11-openjdk-amd64/ bin```

User: How to Obtain the CPU Usage of the Current Linux System Through Command Line Operations?
Assistant: You can use the top command to obtain the CPU usage of the current Linux system. The command is as follows: ``` top ``` After executing this command, the process list and system resource usage of the current system will be displayed, including CPU usage, memory usage and more. Among them, the CPU usage is located in the second field of the first line, and the current usage will be displayed in real time. Press the q key to exit the top command.

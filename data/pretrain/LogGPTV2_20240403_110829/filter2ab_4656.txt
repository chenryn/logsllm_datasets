**ATT&CK ID:** G0007  
**ATT&CK Technique Name:** APT28

APT28, a well-known advanced persistent threat (APT) group, has exploited a Windows Server Message Block (SMB) Remote Code Execution (RCE) vulnerability to facilitate lateral movement within targeted networks. This technique allows the threat actor to execute arbitrary code on remote systems, enabling them to move laterally and expand their control. References for this activity can be found in sources [5], [6], and [7].

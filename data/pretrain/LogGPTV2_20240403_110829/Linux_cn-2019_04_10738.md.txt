### 头条新闻

#### 开源永存

Linux Journal 发布了新的头条文章《[Open Source is Eternal](https://www.linuxjournal.com/content/open-source-eternal)》。在这篇文章中，作者 Glyn Moody 回顾了从 1994 年开始的 Linux Journal、计算机周刊和互联网档案馆等一系列服务的历史。通过这些网站的历史变迁，Moody 强调了开源技术的持久生命力。

![](/data/attachment/album/201904/15/110332k14jhr8741hrwx81.png)

> 不知不觉，Linux 已经存在了 28 年。在这 28 年里，Linux 深刻地改变了我们的世界，推动了云计算、虚拟化等一系列技术的发展。Linux 永生！

### 快讯

#### Mozilla 发布 Pyodide 项目

Mozilla 发布了一个名为 Pyodide 的项目，该项目允许在浏览器中运行 Python 数据科学工具。除了支持 Python 之外，Pyodide 还可以与 JavaScript 进行无缝交互。详情请参阅 [Mozilla Hacks](https://hacks.mozilla.org/2019/04/pyodide-bringing-the-scientific-python-stack-to-the-browser/)。

#### Kubernetes 简化 Bet365 软件部署

Bet365 是世界上最大的在线投注平台之一，他们使用 Kubernetes 来简化软件部署过程。了解更多关于 Bet365 如何利用 Kubernetes 的信息，请访问 [ComputerWeekly](https://www.computerweekly.com/news/252461716/How-Kubernetes-has-simplified-bet365s-software-deployments)。

> 来自网上投注平台的 Kubernetes 实践案例。

#### 放弃 Windows 选择 Linux 的原因

用户放弃 Windows 而选择 Linux 的主要原因包括精选的系统和软件应用、高度可定制化的界面、更加轻量级以及更低的数据泄露风险。更多信息请参阅 [SlashGear](https://www.slashgear.com/reasons-to-abandon-windows-for-linux-06572307/)。

> 仍需更多实用的应用程序来进一步增强用户体验。

### 美议员建议对 RISC-V 技术实施出口限制

几位来自美国参众两院的议员以国家安全为由，敦促美国政府对 RISC-V 相关技术的出口实施限制。这些议员认为，中国正利用美国公司之间的开放合作文化来推动其半导体产业的发展，这可能会削弱美国在芯片领域的领先地位。他们指出，中国正在“滥用”RISC-V 技术以规避美国在芯片设计所需知识产权方面的主导地位。总部位于加利福尼亚的 RISC-V 初创公司 SiFive 表示，如果美国政府对美国公司在 RISC-V 方面的活动进行限制，那将是一个“巨大的悲剧”。

> **[消息来源：路透社](https://www.reuters.com/technology/us-china-tech-war-risc-v-chip-technology-emerges-new-battleground-2023-10-06/)**
>
> 老王点评：当年他们曾经考虑过对 Linux 等开源软件进行限制，结果你也知道了，这是不可能做到的。

### 仅有 10% 的人主要将 Python 用于开发系统管理脚本

Python 软件基金会和 JetBrains 最新发布的 2022 年度 Python 开发者调查报告显示，当被问及 Python 最常用于什么用途时，22% 的受访者表示用于 Web 开发，18% 用于数据分析，12% 用于机器学习，而只有 10% 的人将其主要用于 DevOps、系统管理和编写自动化脚本。此外，报告还显示，93% 的受访者采用了 Python 3，但仍有 7% 的人在使用 Python 2（上一年度这一比例为 5%）。在受访者中，51% 的人表示他们在工作和个人/教育用途或副业项目中使用 Python，85% 的受访者表示 Python 是他们的主要编程语言。

> **[消息来源：Info World](https://www.infoworld.com/article/3707798/python-developers-wont-let-go-of-python-2.html)**
>
> 老王点评：Python 难道不是运维人员最爱的语言吗？看起来它的用户群体已经发生了很大变化。所以，即便你不是运维，也该学习一下 Python 了。

### 澳大利亚科学家用《帝国时代》游戏模拟蚂蚁战争

澳大利亚各地已有 50 种不同的入侵蚂蚁，包括电蚁、火蚁和黄疯蚁。为了消灭这些入侵蚂蚁，已花费了数亿美元。为了更好地了解本地蚂蚁和外来入侵蚂蚁之间的斗争，澳大利亚国家科学机构的科学家们利用微软的经典电脑游戏《帝国时代》来模拟蚂蚁战争。研究人员称，“蚂蚁是少数几个在战争规模和死亡率方面与人类战争相似的动物物种群体之一。”他们认为这项研究有助于开发新的栖息地管理方法，例如在城市化环境中增加灌木丛或更复杂的环境，使竞争天平重新倾向于本地蚂蚁。

> **[消息来源：ABC](https://www.abc.net.au/news/rural/2023-10-03/age-of-empires-computer-game-ant-warfare-simulation-csiro-uwa/102923706)**
>
> 老王点评：所以，我们的文明和战争是不是也是“别神”在玩的《帝国时代》？

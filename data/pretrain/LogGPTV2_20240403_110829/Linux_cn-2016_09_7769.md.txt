[QOwnNotes](http://www.qownnotes.org/) 是一款自由且开源的笔记记录和待办事项管理应用，支持 Linux、Windows 和 macOS。该程序将笔记保存为纯文本文件，并支持 Markdown 格式，同时与 ownCloud 云服务紧密集成。

![](/data/attachment/album/201609/14/135853szxxerxksk7orlt9.jpg)

QOwnNotes 的一大亮点在于其对 ownCloud 云服务的集成（可选）。通过在 ownCloud 上使用此应用，用户可以在网络上记录和搜索笔记，并可在移动设备上使用类似 CloudNotes 的软件[2](http://peterandlinda.com/cloudnotes/)。

不久之后，通过连接到你的 ownCloud 账户，你还可以从 ownCloud 服务器上分享笔记，并查看或恢复之前的笔记版本（包括已删除的笔记）。

此外，QOwnNotes 还可以与 ownCloud 任务或 Tasks Plus 应用程序集成。

> 如果你不熟悉 [ownCloud][3]，这是一款替代 Dropbox、Google Drive 等商业性网络服务的自由软件，可以安装在自己的服务器上。它提供了一个网络界面，具备文件管理、日历、照片、音乐、文档浏览等功能，并提供了桌面同步客户端和移动应用。

由于笔记以纯文本形式保存，它们可以通过云存储服务（如 Dropbox、Google Drive 等）在不同设备间同步。然而，这些服务无法完全替代 ownCloud 提供的功能，例如恢复以前的笔记版本（尽管 Dropbox 和其他服务也提供文件恢复功能，但不能直接在 QOwnNotes 中访问）。

QOwnNotes 支持 Markdown 语言（内置预览模式），允许标记笔记并进行搜索，在笔记中插入超链接和图片：

![](/data/attachment/album/201609/14/135441wraix0attti0tb0x.png)

嵌套标签和笔记文件夹同样受到支持。

待办事项管理功能较为基础，有待改进。目前，待办事项在一个单独窗口中打开，不使用与笔记相同的编辑器，也不允许添加图片或使用 Markdown 语言。

![](/data/attachment/album/201609/14/135451vb9p5p19lmqp8q6p.png)

该功能允许你搜索待办事项，设置优先级，添加提醒，并显示已完成事项。此外，待办事项可以添加到笔记中。

软件界面可定制，允许调整字体大小、切换窗格等，并支持无干扰模式。

![](/data/attachment/album/201609/14/135459gk6k66zxxn6bk6xh.png)

在程序设置中，你可以启用黑夜模式（注意：在 Ubuntu 16.04 中存在一些工具栏图标消失的 bug），更改状态栏大小、字体和颜色方案（白天和黑夜）：

![](/data/attachment/album/201609/14/135507cxuab4ozt53u83a4.png)

其他功能包括笔记加密（仅在 QOwnNotes 中有效）、自定义键盘快捷键、导出笔记为 PDF 或 Markdown 格式，以及自定义自动保存间隔等。

访问 [QOwnNotes](http://www.qownnotes.org/) 官方网站了解更多详细信息。

### 下载 QOwnNotes

有关安装方法，请参阅安装页面（支持 Debian、Ubuntu、Linux Mint、openSUSE、Fedora、Arch Linux、KaOS、Gentoo、Slackware、CentOS 以及 macOS 和 Windows）。

QOwnNotes 的 [snap](https://uappexplorer.com/app/qownnotes.pbek) 包也可用，在 Ubuntu 16.04 及更高版本中，可以直接通过 Ubuntu 软件管理器安装。

为了集成 QOwnNotes 到 ownCloud，你需要拥有 [ownCloud 服务器](https://download.owncloud.org/download/repositories/stable/owncloud/)，还需要安装 [Notes](https://github.com/owncloud/notes)、[QOwnNotesAPI](https://github.com/pbek/qownnotesapi)、[Tasks](https://apps.owncloud.com/content/show.php/Tasks?content=164356) 和 [Tasks Plus](https://apps.owncloud.com/content/show.php/Tasks+Plus?content=170561) 等 ownCloud 应用。这些应用可以从 ownCloud 的 Web 界面直接安装，无需手动下载。

请注意，QOwnNotesAPI 和 Notes ownCloud 应用目前仍处于实验阶段，需要在 ownCloud 的 Web 界面上“启用实验程序”才能发现并安装它们。请在 Apps 菜单下的左下角点击设置按钮进行相关设置。

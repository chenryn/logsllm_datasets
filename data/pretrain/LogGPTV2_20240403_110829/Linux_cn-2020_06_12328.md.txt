### Ripple20 漏洞影响数亿设备

在一款广泛使用的底层 TCP/IP 库中发现了一系列零日漏洞，这些漏洞被统称为 Ripple20，影响了数亿台设备，包括大量的物联网设备，如电网设备、医疗系统和工业设备等。该 TCP/IP 库由软件公司 Treck 开发，首次发布于 1997 年，实现了一个轻量级的 TCP/IP 堆栈，并在过去二十多年里被企业广泛用于联网功能。

**来源**: [solidot](https://www.solidot.org/story?sid=64693)

> **硬核老王点评**: 祖传代码有风险，继承需谨慎。

### Parallels 与 Google 合作将 Windows 应用带到 Chrome OS

Parallels 公司此前主要通过虚拟化软件将 Windows 应用程序引入苹果的 Mac 平台。此次合作使得搭载 Chrome OS 的企业级设备能够运行旧版 Windows 应用程序，而许多企业通常依赖这类应用程序。这将使 Chrome OS 设备能够在之前必须使用 Windows 硬件的工作场所中得到应用。

**来源**: [solidot](https://www.solidot.org/story?sid=64692)

> **硬核老王点评**: 可惜，国内无法使用 Chrome OS。

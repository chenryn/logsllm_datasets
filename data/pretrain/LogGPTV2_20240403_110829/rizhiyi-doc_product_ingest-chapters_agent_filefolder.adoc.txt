用户在管理 agent 及字典、仪表盘等其他资源的时候，通常需要对接第三方 CMDB 的信息，对于成百上千条记录以列表形式展示，不利于管理。为了方便展示和管理，日志易引入组织结构标签的概念。
组织标签是用来组织和管理日志易内各种资源和agent分组的一种类似文件夹路径的目录结构。每一层组织目录内可包含若干资源、agent 分组和下一级组织目录，日志易最多支持4个层级的组织目录。了解组织标签的更多内容，请查看《日志易使用手册》资源管理相关章节。
===== 为 agent 分组添加组织标签
为 agent 分组添加组织标签需要首先进入 agent 分组列表页面。在 agent 列表页右上角点击"更多->分组设置"，进入 agent 分组管理页面进行操作。
在新建/编辑 agent 分组时可以选择设置组织标签， 新建界面如下所示。默认不设置组织标签时，新建 agent 分组直接展示在组织目录的最外层。
image::images/agent-group-management.png[]
您可以手动输入 "org...." 格式的组织标签，或者从下拉菜单中直接选择系统中已经存在的组织标签。和普通资源一样，一个 agent 分组也只能有一个组织标签。
===== 为 agent 分组修改组织标签
agent 分组创建完毕后，再修改组织标签有两种方式，一种和新建界面类似，在 agent 分组管理页面左侧的组织目录树上找到具体的 agent 分组，点击选中后，即可在页面右侧修改 agent 分组的名称、组织标签等信息。另一种方式，您可以在页面左侧的组织目录树上，直接鼠标拖拽具体的 agent 分组到某个组织目录里，自动完成该分组的组织标签修改。
===== agent 分组其他管理操作
在页面左侧组织目录树上，鼠标悬停在agent分组位置，点击右键，可以选择编辑，跳转，删除，采集配置，收藏/取消收藏操作。这些操作在左键选中具体某个 agent 分组后，同样可以在页面右侧完成。例如：点击右上角"跳转"图标，会自动打开 agent 列表页，并添加该 agent 分组的过滤条件。
在页面左侧组织目录树顶部，有 agent 分组名称的快速查找输入框。在这里输入想要查找的 agent 分组名称，系统会自动展开相关分组所在的组织目录，并高亮查找命中的 agent 分组。查找可能会命中多个分组，您可以上下拖动查看。
收藏后的 agent 分组，在 agent 列表页左侧，可以切换到"我的收藏"标签区域快速查看和选择。在"我的收藏"区域，只平铺展示 agent 分组名称，不展示组织目录树。

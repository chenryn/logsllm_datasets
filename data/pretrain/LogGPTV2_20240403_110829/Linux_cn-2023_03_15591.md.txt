### OpenAI 推出低价 ChatGPT API
OpenAI 向开发者推出了其 ChatGPT 和 Whisper 模型，允许他们通过 API 将聊天机器人互动和语音转文本功能集成到自己的应用程序中。新的 ChatGPT API 使用与 ChatGPT 相同的人工智能模型，价格为每 1000 个字符（约 750 个单词）0.2 美分，比之前的 GPT 模型便宜了 10 倍。Whisper 是一个自动语音识别系统，能够将英语口语转换为文本，价格为每分钟 0.6 美分。

**消息来源**: [The Register](https://www.theregister.com/2023/03/02/openai_api_chatgpt_whisper/)

**老王点评**: 这种低价策略可能会引发反竞争的诉讼。其他 AI 厂商在技术和价格上都难以与之抗衡，该如何应对？

### Fedora 38 继续支持 X11 和 NIS+
Fedora 38 原计划放弃对一些老旧技术的支持，但由于无法在发布前完成，这些更改被推迟到以后的版本。例如，在 PAM 和用户空间中对 NIS+ 的支持将继续保留。NIS+ 是 Unix 的一个网络目录服务，现在基本上已被 LDAP 取代。此外，尽管 X.org 仍在使用，但原计划在 Fedora 36/37 中删除 VESA 和 fbdev 等老式驱动程序。删除旧的帧缓冲控制台及其驱动程序是合并实时内核补丁的一部分，但现在这些驱动程序仍然存在。

**消息来源**: [Phoronix](https://www.phoronix.com/news/Fedora-38-Incomplete-Changes)

**老王点评**: 即便是旧的驱动程序，也不能简单地一删了之。

### 近 40% 的软件工程师只愿意远程工作
根据 Hired 发布的《2023 年软件工程师状况》报告，近 40% 的软件工程师只喜欢远程工作。如果雇主强制要求回到办公室，21% 的人表示会立即辞职，而另外 49% 的人则表示会开始寻找另一份工作。该报告基于 68,500 名软件工程候选人以及对 1300 多名软件工程师和 120 名人才专业人士的调查数据。

**消息来源**: [Tech Target](https://www.techtarget.com/searchhrsoftware/news/365531979/Nearly-40-of-software-engineers-will-only-work-remotely)

**老王点评**: 除了众所周知的原因外，现在的基础设施也基本满足了远程工作的需求，使得人们更倾向于这种工作方式。

User: 如何在Linux服务器上使用系统资源监控命令？
Assistant: 在Linux服务器上使用系统资源监控命令可以帮助管理员了解系统的负载情况，监控系统资源的使用情况。以下是几个使用系统资源监控命令的示例：

1. top命令

top命令是用来实时监控系统资源使用的命令，可以查看CPU、内存、进程等信息。使用方法如下：

```
top
```

按下“q”键退出命令。

2. vmstat命令

vmstat命令可以用来显示虚拟内存和CPU统计信息，可以了解到系统的运行状况。使用方法如下：

```
vmstat
```

按下“q”键退出命令。

3. free命令

free命令可以用来查看系统内存的使用情况，包括空闲内存和已用内存等信息。使用方法如下：

```
free
```

4. iostat命令

iostat命令可以查看系统的磁盘IO使用情况，包括平均读写速度、读写操作次数等信息。使用方法如下：

```
iostat
```

按下“q”键退出命令。

5. sar命令

sar命令可以用来查看系统资源使用情况的历史记录，包括CPU、磁盘、网络等信息。使用方法如下：

```
sar
```

以上是几个在Linux服务器上使用系统资源监控命令的示例，可以根据需要选择使用。

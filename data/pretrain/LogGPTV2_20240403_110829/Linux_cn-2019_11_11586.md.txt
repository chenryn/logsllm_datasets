> 感谢众多开源开发者的努力，现在日常使用 Linux 比以往任何时候都更加容易。

![](/data/attachment/album/201911/17/203212qes7o7zpeefbpffe.jpg)

自 2004 年开始从事 IT 工作以来，我一直都是 Mac 的忠实粉丝。但几个月前，由于种种原因，我决定将 Linux 作为我的日常操作系统。这不是我第一次尝试完全采用 Linux，但我发现这次比以往更容易。以下是促使我做出这一转变的原因。

### 我在个人电脑上的首次 Linux 体验

我还记得那次经历，当我抬头看着投影机时，我们俩都困惑不已。VGA 线已经接好，针脚也没有弯曲。我尝试了所有可能的按键组合，试图让我的笔记本电脑克服“舞台恐惧症”。

大学期间，我只是出于实验目的运行 Linux。而我在 IT 部门的经理是一位多种发行版的支持者。随着我对桌面支持和编写脚本的信心增强，我想了解更多关于 Linux 的知识。对我来说，IT 比我的计算机科学课程有趣得多，后者显得如此抽象和理论化：“二叉树有什么用？”我当时这样想——而我们的系统管理员团队的工作却是如此真实具体。

最终，我通过登录到 Windows 工作站完成了我的课堂演讲，这标志着我第一次尝试将 Linux 作为日常操作系统的终结。尽管我很欣赏 Linux 的灵活性，但它缺乏兼容性。我偶尔会写一些脚本，通过 SSH 连接到机器上运行其他脚本，但这只是我对 Linux 的有限使用。

### 对 Linux 兼容性的全新印象

几个月前，当我决定再次尝试 Linux 时，我以为会遇到更多的兼容性问题，但实际上并没有。

安装完成后，我立即插入了 USB-C 集线器以测试兼容性。一切工作正常。连接 HDMI 的超宽显示器立即出现在我的笔记本屏幕上，并且轻松调整为第二屏幕。USB 连接的网络摄像头对于我的在家工作方式至关重要，它也能顺利显示视频。甚至从我使用 Mac 以来一直插在集线器中的 Mac 充电器也能为我的非 Mac 硬件充电。

这种积极的体验可能与 USB-C 的改进有关，该技术在 2018 年得到了更多关注，使其能够与其他操作系统的体验相媲美。正如 Phoronix 所解释的那样：

> “USB Type-C 接口为非 USB 信号提供了‘替代模式’扩展，在规范中该替代模式的最大应用场景是支持 DisplayPort。除此之外，另一个替代模式是支持 Thunderbolt 3。DisplayPort 替代模式支持 4K 甚至 8Kx4K 的视频输出，包括多声道音频。
>
> 尽管 USB-C 替代模式和 DisplayPort 已经存在了一段时间，并且在 Windows 上很常见，但主线 Linux 内核之前并不支持此功能。所幸的是，多亏英特尔，这种情况正在改变。”

而在端口之外，快速浏览一下 [笔记本电脑 Linux](https://www.linux-laptop.net/) 的硬件选择，列出了比我 2000 年代初期所经历的更完整的选择集。

与我第一次尝试采用 Linux 相比，现在的体验已经有了巨大的改进，这是我非常欢迎的变化。

### 突破 Apple 的樊篱

使用 Linux 给我的日常工作流程增加了一些新的麻烦，但我喜欢这种挑战。

我的 Mac 工作流程是无缝的：早上打开 iPad，写下关于今天要做的事情的想法，然后在 Safari 中阅读一些文章；接着在 iPhone 上继续阅读；最后登录我的 MacBook，这些设备我已经微调多年，弄清楚了它们之间的连接方式。键盘快捷键已经内化在我的大脑中；用户体验一如既往。简直太舒适了。

这种舒适是有代价的。我几乎忘记了如何运作我的环境，也无法回答我想解答的问题。我是否自定义了一些 PLIST 文件来获得快捷方式？我是否记得将其签入我的 dotfiles 中？为什么我还在依赖 Safari 和 Chrome，即使 Firefox 的功能更好？为什么我不使用基于 Android 的手机代替我的 i-系列产品呢？

我经常考虑改用基于 Android 的手机，但我会失去所有这些设备之间的连接性和生态系统的一些便利。例如，我将无法在 iPhone 上为 Apple TV 输入搜索内容，也无法与其他使用 Apple 设备的朋友通过 AirDrop 共享密码。这些功能是同类设备环境的巨大优势，也是令人赞叹的工程成就。也就是说，这些便利是以被生态系统所困为代价的。

我喜欢了解设备的工作原理。我希望能够解释使我的系统变得有趣或易于使用的环境配置，但也想看看增加一些麻烦会对我的观点产生什么影响。用 Marcel Proust 的话说，“真正的发现之旅不在于寻找新的土地，而在于用新的眼光来看待。”技术的使用如此方便，以至于我不再对它的工作原理感到好奇，而 Linux 让我有机会再次用新的眼光看待。

### 受你的启发

以上所有理由足以让我探索 Linux，但我也受到了你们的启发。尽管所有操作系统都受到开源社区的欢迎，但 Opensource.com 的作者和读者对 Linux 的热爱是极具感染力的。这激发了我重新投入的乐趣，我享受这段旅程。

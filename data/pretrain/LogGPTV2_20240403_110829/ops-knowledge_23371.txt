User: What are the main procedures an organization should follow in order to implement CIS Control 7.6, Perform Automated Vulnerability Scans of Externally-Exposed Enterprise Assets?
Assistant: An organization should Perform automated vulnerability scans of externally-exposed enterprise assets using a SCAP-compliant vulnerability scanning tool. Perform scans on a monthly, or more frequent, basis. CIS Control 7.6 has the asset type Applications, and has the security function Identify.

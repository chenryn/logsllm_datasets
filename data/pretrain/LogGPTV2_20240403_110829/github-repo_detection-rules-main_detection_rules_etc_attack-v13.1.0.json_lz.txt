### Ember Bear 的钓鱼攻击
**Ember Bear**（[MITRE ATT&CK 组织 G1003](https://attack.mitre.org/groups/G1003)）曾尝试通过在鱼叉式网络钓鱼邮件中嵌入恶意链接来诱使用户点击。(引用：[Palo Alto Unit 42 OutSteel SaintBot February 2022](https://unit42.paloaltonetworks.com/ukraine-targeted-outsteel-saintbot/))

### APT37 对受害者的识别
[APT37](https://attack.mitre.org/groups/G0067) 能够识别受害者用户名。(引用：[Talos Group123](https://blog.talosintelligence.com/2018/01/korea-in-crosshairs.html))

### Evilnum 的钓鱼邮件
[Evilnum](https://attack.mitre.org/groups/G0120) 发送了精心设计的钓鱼邮件，诱使收件人打开恶意快捷方式链接，从而下载 .LNK 文件。(引用：[ESET EvilNum July 2020](https://www.welivesecurity.com/2020/07/09/more-evil-deep-look-evilnum-toolset/))

### DDKONG 的单实例运行机制
[DDKONG](https://attack.mitre.org/software/S0255) 利用 Rundll32 来确保每次只有一个实例在运行。(引用：[Rancor Unit42 June 2018](https://researchcenter.paloaltonetworks.com/2018/06/unit42-rancor-targeted-attacks-south-east-asia-using-plaintee-ddkong-malware-families/))

### LookBack 恶意软件的侧加载技术
[LookBack](https://attack.mitre.org/software/S0582) 将其通信模块作为 DLL 侧加载到 libcurl.dll 加载器中。(引用：[Proofpoint LookBack Malware Aug 2019](https://www.proofpoint.com/us/threat-insight/post/lookback-malware-targets-united-states-utilities-sector-phishing-attacks-impersonating-engineering-licensing-boards))

### DCSrv 的休眠与重启功能
[DCSrv](https://attack.mitre.org/software/S1033) 具有休眠两小时后重启系统的能力。(引用：[Checkpoint MosesStaff Nov 2021](https://research.checkpoint.com/2021/mosesstaff-targeting-israeli-companies/))

### Operation Wocao 中的 API 函数使用
在 [Operation Wocao](https://attack.mitre.org/campaigns/C0014) 行动中，威胁行为者利用 `CreateProcessA` 和 `ShellExecute` API 函数在注入选定进程后执行命令。(引用：[FoxIT Wocao December 2019](https://www.fox-it.com/media/kadlze5c/201912_report_operation_wocao.pdf))

### FunnyDream 的文件下载功能
[FunnyDream](https://attack.mitre.org/software/S1044) 可以将额外的文件下载到被攻陷的主机上。(引用：[Bitdefender FunnyDream Campaign November 2020](https://www.bitdefender.com/files/News/CaseStudies/study/379/Bitdefender-Whitepaper-Chinese-APT.pdf))

### CHOPSTICK 的远程命令执行能力
[CHOPSTICK](https://attack.mitre.org/software/S0023) 具备远程命令执行的功能。(引用：[Crowdstrike DNC June 2016](https://www.crowdstrike.com/blog/bears-midst-intrusion-democratic-national-committee/)；[ESET Sednit Part 2](http://www.welivesecurity.com/wp-content/uploads/2016/10/eset-sednit-part-2.pdf))

### APT39 的数据聚合工具
[APT39](https://attack.mitre.org/groups/G0087) 使用工具在数据外泄前进行聚合。(引用：[FBI FLASH APT39 September 2020](https://www.iranwatch.org/sites/default/files/public-intelligence-alert.pdf))

### Mustang Panda 的 mshta.exe 使用
[Mustang Panda](https://attack.mitre.org/groups/G0129) 使用 mshta.exe 来启动收集脚本。(引用：[Secureworks BRONZE PRESIDENT December 2019](https://www.secureworks.com/research/bronze-president-targets-ngos))

### SVCReady 的时区信息收集
[SVCReady](https://attack.mitre.org/software/S1064) 能够收集时区信息。(引用：[HP SVCReady Jun 2022](https://threatresearch.ext.hp.com/svcready-a-new-loader-reveals-itself/))

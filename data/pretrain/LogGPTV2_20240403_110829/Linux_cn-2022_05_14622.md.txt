### 谷歌的人工智能 PaLM 能够理解幽默

笑话、讽刺和幽默需要对语言和人类行为有深刻的理解，而这些通常是通过多年的人际互动习得的。谷歌迄今为止最大的人工智能模型 PaLM，经过 5400 亿个参数的训练，即使没有经过专门的幽默和笑话逻辑训练，也能够理解幽默。在被输入两个笑话后，PaLM 能够解释它们并给出合理的解读。谷歌在其博客中展示了一个 PaLM 如何理解一个新颖笑话的例子，这已经超越了许多科幻小说中的人工智能水平。

**消息来源：[CNET](https://www.cnet.com/tech/services-and-software/no-joke-googles-ai-is-smart-enough-to-understand-your-humor/)**

**老王点评：** 5400 亿个参数，这比 GPT-3 的 1750 亿个还要多！其所能做到的事情也更令人惊讶。以后我们可以和 AI 开玩笑，而不用担心它会误解了。

![](/data/attachment/album/202205/21/182743ctqmbsbux4xqoumm.jpg)

### 英国政府网站 GOV.UK 移除 jQuery 以改进性能

jQuery 自 15 年前诞生以来，一直是使用最广泛的 JavaScript 库之一。数据显示，2021 年有 84% 的移动网页使用了 jQuery。由于其普遍性，许多 jQuery 提供的功能已经成为 Web 平台的一部分。如今，我们几乎可以用原生 JavaScript 实现 jQuery 的所有功能，尽管语法稍长一些。因此，在便利性和性能之间，有人选择了后者。英国政府网站 GOV.UK 在 3 月移除了 jQuery 库，结果显示，网站整体上减少了前端处理时间，95% 的用户等待时间减少了 10%。

**消息来源：[Web Dev](https://web.dev/gov-uk-drops-jquery/)**

**老王点评：** 尽管 jQuery 显示出了一些老化迹象，但 30KB 的大小并不是什么负担。为了这点性能提升而重新改写大量网页，我认为并不值得。

![](/data/attachment/album/202205/21/182759kqznsygxvxzfl49p.jpg)

### Ubuntu 正在努力集成微软的 Azure AD

Canonical 从 Ubuntu 20.10 版本开始集成微软的活动目录（AD）。根据 Ubuntu 桌面团队的更新，多名 Canonical 工程师现在正致力于支持 Azure AD。Azure AD 是微软基于 Azure 的身份管理解决方案，作为内部活动目录的替代品，提供单点登录、多因素认证和各种网络安全保障。这一举措旨在使 Linux 桌面在企业环境中更具吸引力，并更好地融入企业环境。

**消息来源：[Phoronix](https://www.phoronix.com/scan.php?page=news_item&px=Ubuntu-MS-Azure-AD-Integration)**

**老王点评：** 自从 Ubuntu 进入微软商店以来，Canonical 和微软的合作越来越紧密了。

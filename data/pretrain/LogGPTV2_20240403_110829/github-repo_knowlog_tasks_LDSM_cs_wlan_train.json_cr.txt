1. **IOS-3-PEER_CERT_VER_FAIL: "LOCAL-AUTH: Peer cert"**
   - **描述**: 本地认证过程中，对等证书验证失败。
   - **严重性**: 1

2. **Ipsec mutex init failed**
   - **描述**: IPsec互斥锁初始化失败。
   - **严重性**: 1

3. **ciscoLwappSiSensorCrashRev1: "Failed to delete a distribution system interface."**
   - **描述**: 未能删除分配系统接口。
   - **严重性**: 0

4. **SSLVPNMGR-5-SSL_CONNECT_OK: SSL/TLS connection successful with client %s\\n**
   - **描述**: 与客户端的SSL/TLS连接成功。
   - **严重性**: 1

5. **WPS-3-SIG_PTRN_MASK_EQ: MSG_SECURITY**
   - **描述**: 解析令牌模式时发生错误。
   - **严重性**: 1

6. **CMRP-0-MESSAGE_REGISTER: Failed to register with active RP because %s**
   - **描述**: 活动RP注册失败。
   - **严重性**: 1

7. **AAA-6-GUEST_ACCOUNT_DELETE: "Guest user account"**
   - **描述**: 客户端账户已删除。
   - **严重性**: 1

8. **APF-4-PARSE_BSN_PKT_FAILED: "Could not parse Bsn Packet. Reason:%s."**
   - **描述**: 无法解析Bsn数据包。
   - **严重性**: 1

9. **RRM_MGR_TRACE_MESSAGE-3-EWLC_EXEC_ERR: %% Error: %s**
   - **描述**: DTL接口尚未准备好接受IP帧。它尚未初始化。
   - **严重性**: 0

10. **DX-0-DX_NAC_EMERG: "%s"**
    - **描述**: 系统检测到从AP接收到无效的Linktest响应。除非问题持续存在，否则无需采取行动。
    - **严重性**: 0

11. **CMRP-4-SYSTEM_OVER_CURRENT: Overall system over-current has been detected. System will be reset.**
    - **描述**: 检测到系统过流情况，系统将重置。如果此错误再次出现，可能需要更换硬件。
    - **严重性**: 1

12. **AAA-4-AAA_SYNC_FAIL_REQUEST: "Authentication request"**
    - **描述**: 认证请求队列失败。
    - **严重性**: 1

13. **SUBSYS-6-SUBSYS_UNORDERED: "Subsystem class %d (%s) has no order"**
    - **描述**: 任务在初始化时检测到一个未知顺序的类。
    - **严重性**: 1

14. **BONJOUR-3-MTU_ERR: "Incorrect MTU retrieved : %s"**
    - **描述**: 检索到不正确的MTU。
    - **严重性**: 1

15. **EPM-6-AAA:| "POLICY %s|| EVENT %s"|**
    - **描述**: 指定dACL的下载请求已成功发送/下载。
    - **严重性**: 1

16. **LWAPP-1-AP_CONTAINED: "AP %s is being contained on slot %d"**
    - **描述**: LAG处于过渡状态。交换机重启DUE..!!!
    - **严重性**: 0

17. **DOT1X-5-FAIL: "Authentication failed for client (%s) on Interface %s AuditSessionID %s"**
    - **描述**: Dot1x认证未成功。
    - **严重性**: 1

18. **APF-3-UNABLE_TO_VALIDATE_ROGUE_AP_POL: MSG_SECURITY**
    - **描述**: 无法验证恶意AP策略。系统不会自动包含该AP。
    - **严重性**: 1

19. **USMDB-3-MSGTAG046: "Unable to release cfgsync sem Lock Command"**
    - **描述**: 进程因多次失败而重新启动，并被置于hold down状态。
    - **严重性**: 0

20. **WCP-3-SOCK_RECV_FAILED: "Failed to receive a packet from ip address %s: port %d."**
    - **描述**: 在指定端口上建立了DTLS连接。
    - **严重性**: 0

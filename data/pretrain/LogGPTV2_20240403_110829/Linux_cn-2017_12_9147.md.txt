## 有人试图挽救 Ubuntu Unity，将其作为官方分支

Ubuntu Unity Remix 将支持九个月。

![](/data/attachment/album/201712/15/141555ytknhx1k3oze1len.jpg)

Canonical 在七年后突然决定放弃其 Unity 用户界面，这一决定影响了许多 Ubuntu 用户。现在看来，有人正试图将其从死亡中复活，并使其成为官方分支。

长期的 Ubuntu 成员 Dale Beaudoin 上周在官方的 Ubuntu 论坛上发起了一项调查，以了解社区的意见，看看他们是否对随 Ubuntu 18.04 LTS（Bionic Beaver）一起发布的 Ubuntu Unity Remix 感兴趣，该版本将提供九个月或五年的支持。

共有 30 人参与了投票，其中 67% 的人选择了 LTS 版本，而 33% 的人则支持九个月的支持版本。看起来这个即将到来的 Ubuntu Unity Spin 可能会成为官方特色版本，这意味着开发它的承诺。

Dale Beaudoin 表示：“最近的一项民意调查显示，三分之二的人支持 Ubuntu Unity 成为 LTS 发行版。我们应该按照它成为 LTS 和官方特色版的更新周期去努力。我们将尝试使用当前默认的 Ubuntu Bionic Beaver 18.04 的每日构建版本作为平台，每周或每十天发布一次更新的 ISO。”

### Ubuntu Unity 是否会卷土重来？

正常情况下，最后一个带有 Unity 的 Ubuntu 版本应该是 Ubuntu 17.04（Zesty Zapus），它将在 2018 年 1 月终止支持。目前流行的稳定版本 Ubuntu 17.10（Artful Aardvark）是 Canonical CEO 宣布后第一个默认使用 GNOME 桌面环境的版本，Unity 将不再继续开发。

然而，Canonical 仍在官方软件仓库中提供 Unity 桌面环境，因此如果有人想安装它，只需点击一下即可。但坏消息是，它们仅支持到 2018 年 4 月发布的 Ubuntu 18.04 LTS（Bionic Beaver）之前，因此 Ubuntu Unity Remix 的开发者们需要在独立的仓库中继续支持。

另一方面，我们不认为 Canonical 会改变主意，接受这个 Ubuntu Unity Spin 成为官方的特色版，这意味着他们不会继续开发 Unity，现在只有一小部分人可以进行这项工作。最有可能的是，如果对 Ubuntu Unity Remix 的兴趣没有很快消失，那么这可能会成为一个由怀旧社区支持的非官方版本。

问题是，你会对 Ubuntu Unity Spin 感兴趣吗？无论是官方还是非官方？

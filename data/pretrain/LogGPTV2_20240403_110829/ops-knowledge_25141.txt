User: The operation and maintenance personnel encountered a 403 error when processing the fault report of a certain service, and needed to confirm whether the service is normal. Please provide a command for quickly checking service status that works in Linux environment.
Assistant: You can use the curl command to check the service status, for example: curl -I http://example.com This command will return the HTTP response header of the service. If the returned status code is 200, it means the service is normal, and if the returned status code is 403, it means that it has permission question.

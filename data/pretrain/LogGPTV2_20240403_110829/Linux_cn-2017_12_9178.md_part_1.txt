> 您是否关注过开源技术的发展趋势？以下是 10 个预测。

![2018 年开源技术的 10 大发展趋势](/data/attachment/album/201712/27/085930grj6p6jwpwwpewej.png "10 open source technology trends for 2018")

技术在不断变革，诸如 OpenStack、增强型网页应用（PWA）、Rust、R、认知云、人工智能（AI）和物联网等新技术正在颠覆我们对世界的固有认知。以下概述了 2018 年最可能成为主流的开源技术。

### 1. OpenStack 认可度持续高涨

[OpenStack](https://www.openstack.org/) 是一个云操作系统，为管理员提供直观友好的控制面板，以便配置和管理大量的计算、存储和网络资源。

许多企业使用 OpenStack 平台来构建和管理云计算系统。由于其灵活的生态系统、透明度和运行速度，OpenStack 越来越受欢迎。相比其他解决方案，OpenStack 可以以较低的成本支持关键任务应用程序。然而，其复杂的架构以及对虚拟化、服务器和大量网络资源的依赖使一些企业望而却步。此外，要成功使用 OpenStack，需要良好的硬件支持和高水平的技术人员。

OpenStack 基金会一直在努力改进其产品。无论是已经发布的还是正在开发中的功能创新，都将解决许多 OpenStack 的潜在问题。随着复杂性的降低，OpenStack 将获得更大的认可。再加上大型软件开发和托管公司的支持，以及成千上万的会员，OpenStack 在云计算时代前景光明。

### 2. PWA 或将大热

PWA，即增强型网页应用（Progressive Web App），是一种结合了技术、设计和 Web API 的整合，能够在移动浏览器上提供类似原生应用的体验。

传统的网站有许多与生俱来的缺点。虽然原生应用提供了更加个性化和用户参与度更高的体验，但它们占用大量系统资源，并且需要预先下载和安装。PWA 结合了两者的优点，可以通过浏览器访问，被搜索引擎检索，并能响应式适应外部环境，为用户提供应用级体验。PWA 还可以像原生应用一样自我更新，始终显示最新的实时信息，并通过 HTTPS 安全地传递数据。PWA 运行在标准容器中，无需安装，用户只需输入 URL 即可访问。

现代移动用户重视便利性和参与度，PWA 的特性完美契合这一需求，因此 PWA 成为主流是必然趋势。

### 3. Rust 成开发者新宠

大多数编程语言都需要在安全性和控制性之间做出妥协，但 [Rust](https://www.rust-lang.org/) 是一个例外。Rust 通过广泛的编译时检查实现了 100% 的控制而不影响程序安全性。在最近的 [Pwn2Own](https://en.wikipedia.org/wiki/Pwn2Own) 竞赛中，发现了 Firefox C++ 底层实现中的许多严重漏洞。如果 Firefox 是用 Rust 编写的，这些漏洞在产品发布之前的编译阶段就会被发现并解决。

Rust 独特的内置单元测试方法使其成为开发者的首选开源语言。它是 C 和 Python 等其他编程语言的有效替代方案，可以在不牺牲代码可读性的情况下编写安全的代码。总之，Rust 的未来非常光明。

### 4. R 用户群在壮大

[R](https://en.wikipedia.org/wiki/R_(programming_language)) 是一种用于统计计算和图形呈现的编程语言，属于 GNU 项目的一部分。它提供了大量的统计和图形技术，并且具有扩展性。R 是 S 语言的延续，S 语言已成为统计方法学的首选工具。R 为数据操作、计算和图形显示提供了开源选择。R 语言的另一个优势是对细节的关注和对细微差别的处理能力。

与 Rust 一样，R 语言也正处于上升期。

### 5. 广义的 XaaS

XaaS（一切皆服务）的概念涵盖了各种基于云的服务，从基础设施到软件再到平台。这种模式使得企业和个人能够按需访问和使用各种资源和服务，而无需投入大量资金购买和维护硬件或软件。XaaS 的灵活性和成本效益使其成为越来越多企业和组织的选择。

**ATT&CK ID:** G0004  
**ATT&CK Technique Name:** Ke3chang

**Description:**
Ke3chang has been observed conducting local network configuration discovery by utilizing the `ipconfig` command. This technique allows the threat actor to gather information about the target's network settings, including IP addresses, subnet masks, and default gateways, which can be used for further malicious activities. References to this behavior can be found in [116], [117], and [118].

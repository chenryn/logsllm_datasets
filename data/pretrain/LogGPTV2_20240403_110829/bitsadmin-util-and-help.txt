### Bitsadmin Utility and Help

**Applies to:** Windows Server (All supported versions)

**Description:**
The `bitsadmin util` command, when used in conjunction with the `/help` switch, displays detailed help information for the utility switches available in the `bitsadmin` tool.

**Syntax:**
```plaintext
bitsadmin /util /help
```

**Examples:**
To display the command-line help for the `/util` switches, run the following command:
```plaintext
bitsadmin /util /help
```

This will provide you with a list of available utility switches and their usage details.

### Detail Partition

**Description:**
The `detail partition` command displays the properties of a selected partition. Before executing this command, you must first select a partition. Use the `select partition` command to choose and focus on the desired partition.

**Syntax:**
```plaintext
detail partition
```

**Examples:**
To view the properties of the currently selected partition, enter:
```plaintext
detail partition
```

**Steps:**
1. Select a partition using the `select partition` command.
2. Execute the `detail partition` command to display its properties.

This process ensures that the `detail partition` command has a valid target and can provide the required information.

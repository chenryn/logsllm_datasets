User: How to View Windows OS Version Information?
Assistant: 1. Open the Run dialog box. Press the Windows key and the R key simultaneously to open the Run dialog box. 2. Enter "winver". Enter "winver" in the "Run" dialog box, and click the "OK" button. 3. Check the Windows version information. In the pop-up window, you can see information such as the operating system version, operating system version number, installation date, and copyright information. In addition, if you want to use the command line to view the Windows version information, please press the Windows key and the R key at the same time to open the "Run" dialog box, enter "cmd" and click the "OK" button to open the command prompt window. Enter the "ver" command in the command prompt window to display the Windows version number and version information.

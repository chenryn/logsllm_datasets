ATT&CK ID:M1050
ATT&CK Technique Name:Exploit Protection
Microsoft's Enhanced Mitigation Experience Toolkit (EMET) Attack Surface Reduction (ASR) feature can be used to block regsvr32.exe from being used to bypass application control. [44] Identify and block potentially malicious software executed through regsvr32 functionality by using application control [45] tools, like Windows Defender Application Control[46], AppLocker, [47] [48] or Software Restriction Policies [49] where appropriate.

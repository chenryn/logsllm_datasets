## Keyboard Shortcuts

### Actions
| Shortcut | Action |
| --- | --- |
| `N` | Compose a new tweet |
| `L` | Like a tweet |
| `R` | Reply to a tweet |
| `T` | Retweet |
| `M` | Send a direct message |
| `U` | Mute an account |
| `B` | Block an account |
| `Enter` | Open tweet details |
| `O` | Expand photos |
| `/` | Search |
| `Ctrl + Enter` | Send a tweet |

### Navigation
| Shortcut | Action |
| --- | --- |
| `?` | Open the full keyboard menu |
| `J` | Navigate to the next tweet |
| `K` | Navigate to the previous tweet |
| `Space` | Scroll down the page |
| `.` | Load new tweets |

### Timelines
| Shortcut | Action |
| --- | --- |
| `G` `N` | Go to the Home timeline |
| `G` `O` | Go to Moments |
| `G` `N` | Go to the Notifications tab (Note: This appears to be a duplicate of the Home timeline shortcut) |
| `G` `R` | Go to Mentions |
| `G` `P` | Go to your Profile |
| `G` `L` | Go to the Likes tab |
| `G` `I` | Go to the Lists tab |
| `G` `M` | Go to Direct Messages |
| `G` `S` | Go to Settings and Privacy |
| `G` `U` | Go to someone's profile |

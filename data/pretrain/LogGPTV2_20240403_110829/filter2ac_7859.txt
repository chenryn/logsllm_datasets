**ATT&CK ID:** S0448  
**ATT&CK Technique Name:** Rising Sun

Rising Sun employs dynamic API resolution to access various Windows APIs. This is achieved by using the `LoadLibrary()` and `GetProcAddress()` functions, which allow the malware to load libraries and resolve function addresses at runtime.[131]

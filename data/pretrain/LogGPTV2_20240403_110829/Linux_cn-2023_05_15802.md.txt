Firefox 113 版本现已发布，带来了多项新功能、改进和错误修复。以下是具体更新内容：

### 增强的画中画功能
Firefox 113 版本中的画中画（Picture-in-Picture, PIP）功能得到了显著改进。用户现在可以在最受欢迎的视频网站上轻松倒带、查看视频长度并切换到全屏模式。

### 提升的密码安全性
Firefox 的密码生成器增加了特殊字符，进一步增强了密码的安全性。

### 更简单的书签导入
从 Safari 或基于 Chrome 的浏览器导入书签变得更加简单，因为默认会导入这些书签上的网站图标。

### Windows GPU 沙盒加强
Firefox 113 进一步加强了在 Firefox 110 中首次引入的 Windows GPU 沙盒，提供了更强的安全保障。

### 微软 Outlook 文件拖放
此版本还实现了一个 13 年前被请求的功能：允许用户直接从微软 Outlook 拖放文件。

![Firefox 113](/data/attachment/album/202305/11/163549t73ii1a2l7i2iajz.jpg)

### macOS 和 Windows 上的新功能
- **macOS 用户**：现在可以直接从上下文菜单中访问 macOS “服务”。
- **Windows 用户**：默认已启用弹性滚动效果，当滚动超出滚动容器的边缘时提供反弹动画。

### 支持 AV1 图像格式文件
Firefox 113 增加了对包含动画的 AV1 图像格式文件（AVIS）的支持，并允许使用 `window.print()` 的网站在安卓版 Firefox 中打印。安卓版 Firefox 默认支持硬件加速的 AV1 视频解码，并启用了 GPU 加速的 Canvas2D。

### WebRTC 功能增强
开发人员将受益于 Firefox 113 对许多 WebRTC 功能的支持，包括 `RTCMediaSourceStats`、`RTCPeerConnectionState`、`RTCPeerConnectionStats`（“peer-connection” RTCStatsType）、`RTCRtpSender.setStreams()` 和 `RTCSctpTransport`。模块脚本现在可以在 Worklets 中导入其他 ES 模块脚本，Firefox 现在支持四级颜色规范中的颜色函数，包括 `lab()`、`lch()`、`oklab()`、`oklch()` 和 `color()` 函数。

### 无障碍引擎重新设计
Firefox 113 引入了重新设计的无障碍引擎，显著提高了 Firefox 对屏幕阅读器和其他辅助功能软件用户的响应速度、稳定性和整体体验。

### Awesomebar 搜索结果菜单
Awesomebar 搜索结果菜单现在已启用，允许用户删除历史记录结果并关闭赞助的 Firefox Suggest 结果。

### 移除弃用的 API
虽然 Firefox 113 带来了许多令人兴奋的新功能和增强，但也移除了长期弃用的 `mozRTCPeerConnection`、`mozRTCIceCandidate` 和 `mozRTCSessionDescription` 类型。网站应使用其非前缀变体。

### 安卓版改进
安卓版 Firefox 还改进了内置 PDF 阅读器的用户界面，使直接保存 PDF 文件变得更加容易。

### 下载
该版本现已正式发布，并可以从以下 FTP 网站下载：
> 
> **[下载 Firefox 113](https://ftp.mozilla.org/pub/firefox/releases/113.0/)**

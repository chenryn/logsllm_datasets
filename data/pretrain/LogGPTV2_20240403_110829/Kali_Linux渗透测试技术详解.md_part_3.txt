### 1.4 安装 Kali Linux

#### 1.4.1 安装至硬盘

**选择区域**

在安装过程中，系统会根据您所选择的语言生成一个地区列表。如果您的地区未列出，请选择“其他”。

- **国家/地区**
  - 台湾
  - 新加坡
  - 其他/Other

**配置键盘**

请从以下选项中选择键盘布局：

- 比利时
- 波斯尼亚语
- 巴西
- 英国英语
- 保加利亚语（标准布局）
- 保加利亚语（音标布局）
- 加拿大法语
- 加拿大多种语言
- 希腊语
- 汉语
- 克罗地亚语
- 捷克语
- 丹麦语
- 荷兰语
- 德沃夏克布局
- 不丹语
- 世界语
- 爱沙尼亚语
- 埃塞俄比亚语

**设置主机名**

在此界面，您可以设置系统的主机名。默认值为Kali，但用户可以自定义。设置完成后点击“继续”按钮。

- **主机名**: kali

**配置网络**

- **域名**: kali.secureworks.com

如果您当前计算机没有连接到网络，可以跳过此步骤并直接点击“继续”。

**设置用户和密码**

- **Root 用户**: 为了安全起见，建议使用由字母、数字和标点符号组成的强密码，并定期更新。
- **确认密码**: 请再次输入以验证其正确性。

**磁盘分区**

- **分区方法**:
  - 使用整个磁盘
  - 使用整个磁盘并配置LVM
  - 使用整个磁盘并配置加密的LVM
  - 手动分区

**选择要分区的磁盘**

- **磁盘选择**: SCS13 (0, 0, 0) (sda) - 85.9 GB VMware, VMware Virtual S

**分区方案**

- 将所有文件放在同一个分区中（推荐新手使用）
- 将 /home 放在单独的分区
- 将 /home、/usr、var 和 /tmp 都分别放在单独的分区

**分区详细信息**

- **分区表更改**: SCS13 (0, 0, 0) (sda)
- **格式化分区**:
  - 第5分区: 设置为 swap
  - 第1分区: 设置为 ext4

**写入磁盘**

- **确认修改**: 如果您继续，这些修改将被写入磁盘。否则，您可以进一步手动修改。

**安装系统**

- **正在安装系统**: 正在将数据复制到磁盘。

**配置软件包管理器**

- **HTTP 代理设置**: 如果不需要通过 HTTP 代理连接到外部网络，直接点击“继续”。

**选择镜像站点**

- **Debian 镜像所在国家**: 中国
- **镜像站点选择**: mirrors.163.com

**GRUB 引导加载程序**

- **安装 GRUB 到 MBR**: 如果这是计算机上唯一操作系统，建议将 GRUB 安装到主引导记录 (MBR) 上。

**完成安装**

- **安装完成**: 安装过程已经完成，现在可以重新启动系统。

#### 1.4.2 安装至 USB 驱动器

Kali Linux USB 驱动器提供了一种持久存储系统设置、更新及安装软件的能力。以下是安装 Kali Linux 至 USB 驱动器的步骤：

**准备工作**

- FAT32 格式的 USB 驱动器，最小 8GB 空间
- Kali Linux ISO 映像
- Win32 Disk Imager 工具
- 下载 Kali Linux 从 http://www.kali.org/downloads/

**安装步骤**

1. 插入已格式化的 USB 驱动器到 Windows 系统。
2. 启动 Win32 Disk Imager，选择 Kali Linux ISO 文件和目标 USB 设备，点击“Write”按钮。
3. 使用 UNetbootin 工具将 USB 设备做成可引导盘：
   - 选择光盘镜像
   - 选择 ISO 文件位置
   - 设置保留空间为 4096MB
   - 选择 USB 驱动器并点击“确定”
4. 创建完成后，重启计算机并在 BIOS 中选择 USB 启动，开始安装 Kali Linux。

#### 1.4.3 安装至树莓派

树莓派是一款基于 ARM 的微型电脑主板，适合携带。以下是安装 Kali Linux 至树莓派的步骤：

1. 从 http://www.offensive-security.com/kali-linux-vmware-arm-image-download/ 下载树莓派映像文件：kali-linux-1.0.6a-rpi.img.xz。
2. 使用 7-Zip 解压下载的文件，得到 kali-linux-1.0.6a-rpi.img。
3. 使用 Win32 Disk Imager 工具，将解压后的映像文件写入树莓派的 SD 卡中。

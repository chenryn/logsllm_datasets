### 第四步: 安装Ixgbe驱动
当确认驱动已经加载后，就可以安装驱动到系统中了
```
$ sudo make install
```
ixgbe.ko将会安装在下列目录
```
/lib/modules//kernel/drivers/net/ixgbe
```
此时，编译好的驱动将在启动时自动加载，也可以通过运行命令加载它：
```
$ sudo modprobe ixgbe 
```

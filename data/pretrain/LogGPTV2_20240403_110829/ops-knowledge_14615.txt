User: What are the main procedures an organization should follow in order to implement CIS Control 6.3, Require MFA for Externally-Exposed Applications?
Assistant: An organization should Require all externally-exposed enterprise or third-party applications to enforce MFA, where supported. Enforcing MFA through a directory service or SSO provider is a satisfactory implementation of this Safeguard.CIS Control 6.3 has the asset type Users, and has the security function Protect.

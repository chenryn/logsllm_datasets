I used my set of data to plot a streamplot using matplotlib 1.4.3
the data range is from 0 to 1, grid points are equally spaced (except near
corners which is dx/2 & dy/2)  
the generated streamlines are correct but it seems that the whole streamplot
is somehow scaled,, the range is shown to be from 0 to 0.5 instead of 0 to 1.
Here is my python code:
    import numpy as np
    import matplotlib.pyplot as plt
    import linecache
    str = linecache.getline("./result.dat", 2)
    [n_x, n_y] = [int(s) for s in str.split() if s.isdigit()]
    [X, Y, P, U, V] = np.genfromtxt("./result.dat", skip_header=2, unpack=True)
    X = X.reshape(n_x,n_y)
    Y = Y.reshape(n_x,n_y)
    U = U.reshape(n_x,n_y)
    V = V.reshape(n_x,n_y)
    plt.streamplot(X, Y, U, V,color=U, linewidth=.6)
    plt.colorbar()
    plt.show()

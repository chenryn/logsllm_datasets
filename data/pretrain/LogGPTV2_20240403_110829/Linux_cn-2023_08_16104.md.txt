> 想了解 Linux 命令行的基础知识吗？这是一个带有实践方法的教程系列。

Linux 终端可能对初学者来说显得有些令人望而生畏。这个黑色的屏幕只能通过命令进行操作，很容易让人感到迷茫。然而，Linux 命令行是一个强大的工具，可以用来管理整个系统。它在系统管理员、网络工程师以及其他许多工作中都发挥着重要作用。

**本教程系列的目的并不是让你立刻成为专家，而是为你提供一个进入 Linux 命令行世界的起点。** 它将帮助你掌握基本的终端使用技巧，例如读取和编辑文件。由于这只是入门内容，大部分教程都集中在“文件操作”上，这是大多数 Linux 书籍和课程的起点。

> **最好的学习方法是亲自动手实践。** 我以“实践模式”编写了这些教程，以便你可以在 Linux 系统上跟随示例进行操作。每章都包含一些示例练习来帮助你巩固所学知识。通过不断练习，你很快就能熟练掌握 Linux 命令行。

### 第 0 章：熟悉终端和术语

当你第一次接触终端时，即使阅读教程也会感到困惑。这是因为你需要理解一些基本术语。下面的文章将帮助你解决这些问题。虽然我分享的一些技巧对你来说可能有点超前或不太实用，但你会发现很多有用的内容。

- **[19 个基础而重要的 Linux 终端技巧](https://itsfoss.com/basic-terminal-tips-ubuntu/)**

### 第 1 章：更改目录

在第一章中，你将学习如何使用绝对路径和相对路径切换目录（文件夹）。这样，你就可以在 Linux 命令行中自由切换目录。

- **[在 Linux 终端更改目录](https://itsfoss.com/change-directories/)**

### 第 2 章：创建目录

现在你已经掌握了如何切换目录，接下来学习如何创建新目录。

- **[在 Linux 终端创建目录](/article-15595-1.html)**

### 第 3 章：列出目录中的内容

你已经很好地掌握了目录操作。接下来学习如何查看目录内部，并查看它们包含哪些文件和子目录。

- **[在 Linux 终端列出目录](https://itsfoss.com/list-directory-content/)**

### 第 4 章：创建文件

关于目录的操作已经足够了。接下来学习如何在 Linux 命令行中创建新文件。

- **[在 Linux 终端创建文件](/article-15643-1.html)**

### 第 5 章：读取文件

文件里面有什么？在本章中，你将学习如何阅读文本文件。

- **[在 Linux 终端读取文件](https://itsfoss.com/view-file-contents/)**

### 第 6 章：删除文件和目录

现在你已经学会了如何创建新文件和文件夹，是时候学习如何删除它们了。

- **[在 Linux 终端删除文件和目录](/article-15809-1.html)**

### 第 7 章：复制文件和目录

继续进行文件操作，在本章中学习如何在终端中复制文件和目录。

- **[在 Linux 终端复制文件和目录](/article-16053-1.html)**

### 第 8 章：移动文件和目录

移动文件操作类似于剪切和粘贴。你还可以使用相同的方法重命名文件和目录。

- **[在 Linux 终端移动文件和目录](https://itsfoss.com/move-files-linux/)**

### 第 9 章：编辑文件

作为最后一个主要的文件操作，学习如何在命令行中编辑文本文件。

- **[在 Linux 终端编辑文件](https://itsfoss.com/edit-files-linux/)**

### 第 10 章：获取帮助

现在你已经了解了许多基本的 Linux 命令行操作，是时候学习如何在终端本身中获取帮助了。

- **[在 Linux 终端获取帮助](https://itsfoss.com/linux-command-help/)**

### 下一步该怎么做？

现在你已经更熟悉终端并了解了命令行中的基本文件操作，你可能想知道接下来该做什么。我建议你购买一本 Linux 书籍，例如《How Linux Works》。当然，你可以从任何你喜欢的 Linux 书籍开始。这里有一些 [我喜欢的几本 Linux 书籍](https://itsfoss.com/best-linux-books/)。

还不想花钱买书吗？不用担心！这里有一些 [你可以下载的免费 Linux 电子书](https://itsfoss.com/learn-linux-for-free/)。

Bash 脚本也是 Linux 学习中不可或缺的一部分。即使你不必编写 Shell 脚本，如果你了解基础知识，也能更好地理解和使用 Linux 中遇到的脚本。

- **[Bash 基础知识系列](https://itsfoss.com/tag/bash-basics/)**

学无止境。不可能面面俱到。但是，如果你至少了解基础知识，就能帮助你更有效地使用系统。

title:botconf2018 Red Teamer 2.0: Automating the C&C Set up Process
 This talk follows the amazing documentation provided by Steve Borosh (@424f424f) and Jeff Dimmock’s (@bluscreenofjeff) on their dedicated repo.Besides, it follows several experiences of red team operations leveraging the tips issued by these authors.We will describe a new open source tool, whose name will be revealed during the presentation. That tool aims at managing red teams’ operations, and, in particular enables Command and Control infrastructure set up automation.

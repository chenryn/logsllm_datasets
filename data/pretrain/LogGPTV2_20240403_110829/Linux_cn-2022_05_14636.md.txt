### DuckDuckGo 浏览器对微软的跟踪程序网开一面

DuckDuckGo 的搜索主要基于微软的必应搜索。近期，研究人员发现 DuckDuckGo 手机浏览器并未阻止第三方网站上的微软追踪器。DuckDuckGo 解释称，虽然微软无法获取用户在 DuckDuckGo 上的搜索内容，并且 DuckDuckGo 浏览器屏蔽了所有微软的 cookie，但如果用户访问包含微软跟踪程序的网站，其数据将暴露给必应搜索引擎和领英。根据与微软签署的一份保密搜索合作协议，DuckDuckGo 必须这样做才能从必应获取搜索信息。

> **消息来源：[Reviewgeek](https://www.reviewgeek.com/118915/duckduckgo-isnt-as-private-as-you-thought/)**
>
> **老王点评**：原来 DuckDuckGo 已经被微软深深影响了。

### GitHub Copilot 将对学生免费开放

去年 6 月，微软旗下的 GitHub 和 OpenAI 推出了 Copilot，这是一款可下载的扩展工具，支持 VSCode、JetBrains 等集成开发环境（IDE），能够根据现有代码上下文提示代码行和功能。微软宣布将在今年夏天全面发布这一工具，并免费提供给学生和“经过验证的”开源贡献者。在技术预览期间，数万人经常使用该工具，开发者生成的 Java 和 Python 等语言的代码中，约有 35% 是由 Copilot 提示的。

> **消息来源：[Techcrunch](https://techcrunch.com/2022/05/24/copilot-githubs-ai-powered-coding-tool-will-become-generally-available-this-summer/)**
>
> **老王点评**：嗯，对学生免费——这是帮助学生们尽快上手编程实习职位吗？只是不知道用人单位会怎么想。

### 荷兰警方制作被害男孩的深度伪造视频，希望获得新线索

一名少年在鹿特丹地铁站停车场扔雪球时被击毙，近 20 年后，荷兰警方通过深度伪造技术将其“复活”，收到了数十条线索。2003 年发生的这起谋杀案一直困扰着警方。在其家人的同意下，警方制作了一段视频，在视频中，这位 13 岁的少年请求公众帮助解决悬案。荷兰警方认为这可能是世界首例，视频中出现了栩栩如生的少年形象，并请求知情者提供线索。

> **消息来源：[卫报](https://www.theguardian.com/world/2022/may/23/dutch-police-create-deepfake-video-of-murdered-boy-13-in-hope-of-new-leads)**
>
> **老王点评**：虽然深度伪造技术颇受非议，但这种技术也能用于好的方面。

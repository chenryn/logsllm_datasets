User:  How to scan for all open ports on a network using Nmap.
Assistant: To use Nmap to scan all open ports on the network, you can use the following command: nmap -p 1-65535 -T4 -A -v <target IP address> Among them, the parameters are as follows: -p 1-65535: scan all ports; -T4: Set the scanning speed, 4 means faster than the default speed; -A: enable OS identification, service/version information and script scanning; -v: display verbose output. For example, if you want to scan all open ports on the host with IP address 192.168.0.1, you can use the following command: nmap -p 1-65535 -T4 -A -v 192.168.0.1

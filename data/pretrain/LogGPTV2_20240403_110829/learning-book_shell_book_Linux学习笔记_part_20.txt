### 网络统计数据

`sar` 提供四种不同的语法选项来显示网络信息。`-n` 选项使用四个不同的开关：`DEV`、`EDEV`、`SOCK` 和 `FULL`。

- **DEV**: 显示网络接口信息
- **EDEV**: 显示关于网络错误的统计数据
- **SOCK**: 显示套接字信息
- **FULL**: 显示所有三个开关的信息

#### 表 3-4: `sar -n DEV` 字段

| 字段       | 说明                     |
|------------|--------------------------|
| IFACE      | LAN 接口                 |
| rxpck/s    | 每秒钟接收的数据包       |
| txpck/s    | 每秒钟发送的数据包       |
| rxbyt/s    | 每秒钟接收的字节数       |
| txbyt/s    | 每秒钟发送的字节数       |
| rxcmp/s    | 每秒钟接收的压缩数据包   |
| txcmp/s    | 每秒钟发送的压缩数据包   |
| rxmcst/s   | 每秒钟接收的多播数据包   |

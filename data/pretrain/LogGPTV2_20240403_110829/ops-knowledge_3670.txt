User: On a server, how to view all the processes currently running and the resources they use?
Assistant: You can use the following commands to view all currently running processes and the resources they use: 1. top command: You can use the top command to view the resource usage of all currently running processes in real time, including CPU usage, memory usage, etc. 2. ps command: Through the ps command, you can list the process ID, process name, status, PID and other information of all currently running processes. 3. htop command: The htop command is an interactive process viewer, which can view the resource usage of all currently running processes more intuitively. 4. lsof command: The lsof command can be used to view open files, network connections and other information, which can help users better understand the resources used by the process.

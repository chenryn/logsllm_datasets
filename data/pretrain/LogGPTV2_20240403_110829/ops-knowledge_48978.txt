User: What are the main procedures an organization should follow in order to implement CIS Control 15.7, Securely Decommission Service Providers?
Assistant: An organization should Securely decommission service providers. Example considerations include user and service account deactivation, termination of data flows, and secure disposal of enterprise data within service provider systems. CIS Control 15.7 has the asset type Data, and has the security function Protect.

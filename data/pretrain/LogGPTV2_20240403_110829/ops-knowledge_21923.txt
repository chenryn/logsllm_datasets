User: What are the main procedures an organization should follow in order to implement CIS Control 12.4, Establish and Maintain Architecture Diagram(s)?
Assistant: An organization should Establish and maintain architecture diagram(s) and/or other network system documentation. Review and update documentation annually, or when significant enterprise changes occur that could impact this Safeguard.CIS Control 12.4 has the asset type Network, and has the security function Identify.

# JWTPyCrack
JWT 弱口令 Key 爆破以及生成 NONE 加密的无 Key 的 JWTString。
# 环境
> Python >= 3
>
> pip install pyjwt
# 使用
该脚本能实现两种攻击方式：禁用哈希重新生成JWT字符串攻击、批量爆破弱密钥
## 禁用哈希
> python jwtcrack.py -m generate -s {\"admin\":\"True\"}

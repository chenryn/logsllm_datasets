## Safari 将采用与 Chrome 和 Firefox 相同的 WebExtensions API

Chrome 拥有庞大的扩展生态系统。随着主流浏览器逐步采用 WebExtensions API，这一技术已成为事实上的标准，为用户从一个浏览器切换到另一个浏览器提供了便利。Firefox 自 v57 版本起正式支持 WebExtensions API。

来源：[solidot](https://www.solidot.org/story?sid=65376 "Solidot")

> 拍一拍：主要浏览器支持一致的扩展 API 是一件大好事。

## 恶意 npm 软件包试图窃取敏感的 Discord 和浏览器文件

名为 `fallguys` 的 JavaScript 库声称提供《Fall Guys: Ultimate Knockout》游戏 API 接口，但实际却尝试访问五个本地文件并读取其内容，然后将数据发布至 Discord 频道。npm 安全团队已将其从 npm 门户网站上移除。

来源：[zdnet](https://www.zdnet.com/article/malicious-npm-package-caught-trying-to-steal-sensitive-discord-and-browser-files/ "ZDNet")

> 拍一拍：即使是开源软件包也不一定经过严格审查，不能盲目信任。

## 苹果阻止 Facebook 应用更新，要求征收 30% 内购销售税

Facebook 计划通过一次性收费的方式实现在线直播活动货币化，但在提交应用更新时遭到苹果拒绝。苹果认为此次更新包含“不相关”信息，并坚持收取 30% 的内购费用。然而，Facebook 希望所有收入能够归主办方所有，请求苹果免除该费用，但未获同意。

来源：[zdnet](https://www.zdnet.com/article/apple-stops-facebook-update-calling-out-30-sales-tax/ "ZDNet")

> 拍一拍：“在我苹果的地盘上，所有人都必须遵守我的规则。”

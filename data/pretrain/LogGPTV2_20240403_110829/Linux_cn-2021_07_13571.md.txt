### 乌克兰查封加密货币矿场，查获 3800 台索尼 PS4 主机

![](/data/attachment/album/202107/11/221104p47emm34abnxqx71.jpg)

乌克兰安全局在一次执法行动中查封了一个大型非法加密货币挖矿矿场。除了查获超过 500 张显卡外，还发现了 3800 台 PS4 游戏机。初代 PS4 配备了具有 18 个计算单元的 GPU，单精度算力约为 1.84 TFLOPs；而升级后的 PS4 Pro 则拥有 36 个计算单元，算力提升至 4.2 TFLOPs。

> 不仅仅是显卡，连游戏机也被用于挖矿。

### 开发人员逐步从 Java 8 迁移到 Java 11

![](/data/attachment/album/202107/11/221144qv35aikau4ka3ud0.jpg)

过去的研究表明，开发人员主要使用 Java 8。然而，根据 Snyk 的《2021 年 JVM 生态系统报告》，今年有 61.5% 的受访者在部分生产环境中使用 Java 11，近 12% 的人使用最新版本 Java 15。尽管如此，一半的 Java 11 用户仍在其生产环境中继续使用 Java 8。此外，44% 的开发者在生产环境中使用免费的 AdoptOpenJDK 发行版作为他们的 JDK 之一，而在开发过程中这一比例为 48%。

> 尽管逐步迁移到最新版本，但由于旧版本 JDK 上运行的老应用，开发人员仍难以完全放弃和迁移。

### 广告商增加对安卓用户的支出

![](/data/attachment/album/202107/11/221118zbonzzfx33fo5r9o.jpg)

自苹果公司于今年 4 月初强制实施“应用跟踪透明度”政策以来，用户可以选择是否允许跨应用程序跟踪。一些广告公司报告称 iOS 平台上的广告支出有所减少，但 Android 平台的广告支出却增加了 10% 以上。有趣的是，只有约 66% 的用户真正拒绝了追踪。

> 既然 iOS 用户拒绝追踪，那么就转向安卓吧，毕竟谷歌是一家以广告为主的公司。

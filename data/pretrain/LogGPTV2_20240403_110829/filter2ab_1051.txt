**ATT&CK ID:** S1040  
**ATT&CK Technique Name:** Rclone

Rclone is a versatile command-line program that can be used to list files and directories. Specifically, it provides the following commands for this purpose:
- `ls`: Lists all files and directories.
- `lsd`: Lists only directories.
- `lsl`: Provides a long listing format, including detailed information about each file and directory.

For more information, refer to [245].

### 谷歌希望 RISC-V 成为 “一级” 安卓架构

在 RISC-V 峰会上，谷歌表达了希望将 RISC-V 视作安卓“一级平台”的愿景，这将使其与 Arm 架构处于同等地位。尽管在 RISC-V 上实现优化的安卓构建需要大量工作，并可能需要几年时间才能完成，但 AOSP 自去年 9 月以来已开始发布官方 RISC-V 补丁。目前，用户最多只能使用命令行界面。谷歌承诺，在 2023 年初提供初始的模拟器支持，并在第一季度为 Java 工作负载提供安卓运行时（ART）支持。一旦安卓系统在 RISC-V 上成功运行，大量的安卓应用程序生态系统也将随之迁移。

**消息来源**: [Ars Technica](https://arstechnica.com/gadgets/2023/01/google-announces-official-android-support-for-risc-v/)

**老王点评**: RISC-V 显然已成为整个行业的希望。

### 智能电视机的隐藏获利方式

智能电视机之所以比其他电子产品便宜，主要原因在于它们能够通过收集和出售用户数据来获利。这些电视机能播放在线服务的内容，并安装各种应用程序，免费为用户提供服务，但代价是对用户进行监控，并将收集的数据出售给广告商。智能电视收集用户的观影内容、时长等信息，然后打包出售。电视厂商将这一模式称为“购买后货币化”，意味着他们可以以接近成本的价格出售硬件，通过分享客户的观影数据长期获利。

**消息来源**: [The Atlantic](https://www.theatlantic.com/technology/archive/2023/01/smart-tvs-sony-lg-cheap/672614/)

**老王点评**: 正如那句老话所说，“免费服务的代价就是你自己”。

![](/data/attachment/album/202301/04/194530erk4is8c4xarmai4.jpg)

### Debian 移除 Python 2

Debian 已从 FTP 服务器上移除了 Python 2 软件包。Python 2 在 2020 年结束支持，最后一个版本是 2020 年 4 月发布的 Python 2.7.18。这标志着大多数 Linux 发行版开始着手移除 Python 2 包。

**消息来源**: [Debian](https://bugs.debian.org/cgi-bin/bugreport.cgi?bug=1027108)

**老王点评**: 想要淘汰 Python 2 真不容易啊。

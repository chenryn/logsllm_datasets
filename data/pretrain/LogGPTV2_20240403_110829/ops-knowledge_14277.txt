User: 如何在Linux系统中查看磁盘空间占用情况？
Assistant: Linux系统中可以使用以下命令查看磁盘空间占用情况：

1. df命令：查看所有文件系统的空间占用情况

命令格式： `df [选项] [文件或目录]`

常用选项：

- -h：以人类可读的方式显示磁盘空间大小
- -i：显示inode信息，也就是硬盘索引节点的使用情况
- -T：显示文件系统类型

示例：

```bash
df -h
```

2. du命令: 查看指定目录的空间占用情况

命令格式：`du [选项] [目录或文件]`

常用选项：

- -h：以人类可读的方式显示目录或文件占用的空间
- -s：只显示目录或文件的总大小

示例：

```bash
du -h /var/log/
```

3. ls命令：查看文件或目录的大小

命令格式：`ls [选项] [文件或目录]`

常用选项：

- -h：以人类可读的方式显示文件或目录的大小

示例：

```bash
ls -lh /var/log/messages
```

> 酷冷至尊（CoolerMaster）计划推出开源版本的 MasterPlus 软件，这听起来非常不错！

![](/data/attachment/album/202301/11/105414yt7ziqw41tvbwijy.png)

大多数游戏和外设软件套装要么是专有的，要么没有对 Linux 的官方支持。因此，用户通常需要寻找开源工具来配置硬件以获得原生功能。例如，Piper、OpenRGB 和 Solaar 等工具在这些情况下非常有用。然而，有时候这些工具仍然不够用。

幸运的是，酷冷至尊（CoolerMaster）已经决定发布其 MasterPlus 软件的开源版本，旨在为其散热器及第三方散热器提供服务。虽然这并不能保证它可以在 Linux 系统上运行，但我们对此充满期待。此举还应鼓励像雷蛇和罗技这样的公司考虑制作精简的开源工具。

让我们来看看酷冷至尊的具体计划。

### MasterPlus 开源版本：我们目前所知的情况

![酷冷至尊 Masterplus 改版](/data/attachment/album/202301/11/105414btppihwcjnpp6gln.png)

**酷冷至尊在最近的 CES 2023 活动中透露了他们计划发布新的 MasterPlus 开源版本**。感谢来自 Boring Text Reviews 的 Albert 让我们注意到了这一点。

**预期会有什么？** MasterPlus 软件将进行全面重新设计，并引入一个 API 插件系统，允许非酷冷至尊散热器与之整合。酷冷至尊已明确表示，他们的独有功能不能与其他散热器配合使用。例如，检测 AIO 散热器的泄漏或计算 PSU 的效率等功能，不能应用于第三方产品。

相反，该应用程序将仅支持读取基本性能信息（如温度和风扇速度），并能够配置 ARGB 设备。即使如此，这也比没有要好得多。如果你的系统使用了酷冷至尊的组件，这无疑是一个令人兴奋的消息！

酷冷至尊还展示了 API 系统的潜在应用，例如将其与照片应用程序集成，以控制安装在电脑机箱侧面的辅助显示器。此外，他们还介绍了软件的全面云整合，但遗憾的是，这部分不会开源。

**什么时候？** 我们还没有具体的发布日期。但如果让我猜的话，2023 年某个时候可能是最好的选择。

即使该工具尚未确认可在 Linux 上运行，这对于开源工具来说也是一个好的开始，不是吗？你怎么看？

11.18 BFD/4/STACHG_TODWN
日志信息
BFD/4/STACHG_TODWN(l): BFD session changed to Down. (Discriminator=[session-
discriminator], Diagnosis=[diagnosis])
日志含义
某个BFD会话的状态转变为Down。
日志级别
级别名称 级别含义
4 可能存在某种差错
日志参数
参数名称 参数含义
session-discriminator BFD会话的本地标识符。
diagnosis BFD会话Down的诊断字。具体情况如下：
l Admindown：会话管理Down。
l ReceiveAdminDow：接收到对端管理Down。
l DetectDown：会话检测Down。
可能原因
设备在检测时间内没有收到BFD报文，或者收到了AdminDown的BFD报文，或者本地执
行了AdminDown。
处理步骤
步骤1 执行命令display bfd session all，检查会话状态是否自动恢复。
l 是 => 结束。
l 否 => 步骤2。
步骤2 执行命令display current-configuration configuration bfd，检查两端是否有一端未启用
BFD功能。
l 是 => 步骤3。
l 否 => 执行命令bfd，全局启用BFD功能 => 步骤1。

User: 请解答这个和日志有关的问题Allow shell access from one ip only Is it possible to have root shell access from one particular ip only

I am using centOS. i want that only one user can log in via shell from one ip only

Assistant: Yes, add the following to sshd_config to enable key only login and to restrict to a particular IP address.

AllowUsers root@ip.add.re.ss
PermitRootLogin without-password

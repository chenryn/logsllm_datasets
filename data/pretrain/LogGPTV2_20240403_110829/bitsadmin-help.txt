### bitsadmin help

**Applies to:**
- Windows Server (All supported versions)

**Description:**
The `bitsadmin /help` command provides detailed information about the command-line parameters and options available for the `bitsadmin` utility. This is useful for users who need to understand the various functionalities and syntax of the `bitsadmin` tool.

**Syntax:**
```shell
bitsadmin /help | /?
```

**Examples:**
To display the command-line help, use the following command:
```shell
bitsadmin /help
```

This will output a list of all available `bitsadmin` commands along with their respective parameters and options, providing a comprehensive guide for using the utility.

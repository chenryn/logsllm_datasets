### Telnet: Display Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `display` command in Telnet is used to show the current operating parameters of the Telnet session.

**Syntax:**
```
d[isplay]
```

**Examples:**
To display the current operating parameters, enter:
```
d
```

This command will provide information about the current settings and configurations of the Telnet session.

**ATT&CK ID:** S0336  
**ATT&CK Technique Name:** NanoCore

**Description:**
NanoCore is a Remote Access Trojan (RAT) that has the capability to edit the Windows Registry. This functionality allows the malware to modify system settings, enable persistence, and potentially evade detection by security software.

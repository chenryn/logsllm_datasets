  * VSCode Version: 1.1.1
  * OS Version: OSX 10.11.5 (El Cap)
Steps to Reproduce:
  1. Select a code block
  2. Hit backspace fast after completing the selection.
Result:
VSCode goes ahead and carries over the selection to whatever is occupying the
same space now. If you wait a little after selection, the problem does not
happen.

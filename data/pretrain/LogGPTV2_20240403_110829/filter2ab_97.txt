## Keyboard Shortcuts for VLC Media Player

### General I
| Shortcut | Action |
| --- | --- |
| `F` | Toggle Fullscreen |
| `Esc` | Exit Fullscreen |
| `Space` | Play/Pause |
| `]` | Pause Only |
| `[` | Play Only |
| `+` | Increase Playback Speed |
| `-` | Decrease Playback Speed |
| `=` | Reset Playback Speed to Normal |
| `N` | Next Track/Chapter |
| `P` | Previous Track/Chapter |
| `S` | Stop |
| `T` | Show Current Position |
| `Shift + Left Arrow` | Very Short Jump Backwards |
| `Shift + Right Arrow` | Very Short Jump Forwards |
| `Alt + Left Arrow` | Short Jump Backwards |
| `Alt + Right Arrow` | Short Jump Forwards |
| `Ctrl + Left Arrow` | Medium Jump Backwards |
| `Ctrl + Right Arrow` | Medium Jump Forwards |
| `Ctrl + Alt + Left Arrow` | Long Jump Backwards |
| `Ctrl + Alt + Right Arrow` | Long Jump Forwards |
| `E` | Next Frame |
| `Enter` | Activate Selection |
| `Up Arrow` | Navigate Up |
| `Down Arrow` | Navigate Down |
| `Left Arrow` | Navigate Left |
| `Right Arrow` | Navigate Right |
| `Shift + M` | Go to DVD Menu |
| `Shift + O` | Select Previous DVD Title |

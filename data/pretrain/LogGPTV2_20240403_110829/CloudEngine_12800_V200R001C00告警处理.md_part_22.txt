址。）
原因2：The stack system's MAC address is not the manufacture MAC address of any
member device. （堆叠系统的MAC地址非本系统任何成员设备的出厂MAC地址。）
处理步骤
l 原因1：The configured stack MAC address is not the manufacture MAC address of any
member device.
a. 在分裂的堆叠系统上分别执行命令display system mac-address查看配置的堆
叠系统MAC地址及所有成员设备的出厂MAC地址。
n 如果MAC地址没有冲突，可以在堆叠管理视图执行命令set system mac-
address inconsistence-alarm off关闭该告警。
n 如果MAC地址存在冲突，可以先在堆叠管理视图执行命令undo set
system mac-address取消配置的堆叠系统MAC地址，然后执行命令set
system mac-address switch-delay immediately使堆叠系统MAC地址切换
成当前主交换机的MAC地址。
l 原因2：The stack system's MAC address is not the manufacture MAC address of any
member device.
a. 在分裂的堆叠系统上分别执行命令display system mac-address查看堆叠系统
MAC地址及所有成员设备的出厂MAC地址。
n 如果MAC地址没有冲突，可以在堆叠管理视图执行命令set system mac-
address inconsistence-alarm off关闭该告警。
n 如果MAC地址存在冲突，可以在堆叠管理视图执行命令set system mac-

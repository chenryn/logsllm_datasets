### English Version

We were provided with the source code of the server and tasked with playing a game to obtain the flag. Upon reading the source code, I was reminded of an old riddle, which can be found [here](http://robertheaton.com/2014/01/13/mathematicians-hate-civil-liberties-100-prisoners-100-boxes/). The solution to this riddle is also provided on that webpage, and I highly recommend reading it as the solution is quite surprising.

Our solution to the challenge is available in the `doit.py` file.

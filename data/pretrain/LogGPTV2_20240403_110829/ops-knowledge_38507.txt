User: How to configure a security group for an AWS EC2 instance?
Assistant: To configure a security group for an AWS EC2 instance, follow these steps: 1. Sign in to the AWS console and navigate to the EC2 console. 2. Select the instance to configure the security group. 3. On the instance details page, go to the Description tab and review the instance's security group. 4. Click the Security Group link to change the rules of the security group or add new rules. 5. Click the Edit Inbound Rules or Edit Outbound Rules button and add or change rules. 6. You can select specific IP addresses, IP address ranges, or security groups to grant access. 7. Click Save to apply the new security group rules. After completing these steps, your instance will now be protected by the new security group to prevent unwanted access.

### Filesystems

**Description:**
This command displays information about the current file system of the selected volume and lists the file systems that are supported for formatting the volume.

**Prerequisites:**
- A volume must be selected for this operation to succeed. Use the `select volume` command to select a volume and shift the focus to it.

**Syntax:**
```
filesystems
```

**Example:**
1. Select a volume:
   ```
   select volume <volume_number>
   ```
2. Display filesystem information:
   ```
   filesystems
   ```

By following these steps, you can view detailed information about the current file system and the available options for reformatting the volume.

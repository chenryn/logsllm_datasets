User: 请完成操作系统运维领域的监控与告警类任务在Linux环境下，如何查看系统资源使用情况和进程状态？


Assistant: 可以使用以下命令查看系统资源使用情况和进程状态：

1. top命令
top命令可以实时监控系统资源使用情况和进程状态。
例如，可以使用top命令查看CPU的负载情况、进程数、内存使用情况等等：
```
top
```

2. free命令
free命令可以显示系统内存使用情况。
例如，可以使用free命令查看系统内存使用情况的总览：
```
free -h
```
其中，-h参数表示以人类可读的方式显示内存使用情况。

3. df命令
df命令可以显示磁盘空间使用情况。
例如，可以使用df命令查看磁盘空间使用情况的总览：
```
df -h
```
其中，-h参数表示以人类可读的方式显示磁盘空间使用情况。

4. ps命令
ps命令可以显示当前系统运行的进程状态。
例如，可以使用ps命令查看当前运行的所有进程状态：
```
ps aux
```
其中，a表示显示所有进程，u表示显示进程的详细状态，x表示显示无控制终端的进程。

除了上述命令外，还可以使用其他命令如vmstat、sar等来监控系统资源使用情况和进程状态。

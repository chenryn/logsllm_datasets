IntroductionIntroduction
Digital.ai Intelligence Solutions connect to your Xray Cloud database to extract metadata, transaction and reference data. This document provides details about all Xray Cloud and Digital.ai configurations to be completed. These configurations are necessary for successful data extraction.
Note: Currently, we do not support the Xray Server version.

Creating Xray Cloud users for Digital.ai
To provide Digital.ai the ability to connect to your Xray Cloud instance, you first need to create a new user account which provides Digital.ai the privileges to extract the necessary data.
Steps:
1. Create a user for Digital.ai on your Jira Software Cloud instance. 	Refer to  for detailed steps.

Providing credentials to Digital.ai
You must now allow the Digital.ai application to access your Xray instance using the credentials configured for Digital.ai. Digital.ai encrypts your credentials before storing them and Digital.ai personnel will never have direct access to them. These credentials are only used for programmatic connectivity to your system.Steps:
1. Log in to the Digital.ai Intelligence admin Site with the credentials provided by Digital.ai. 2. Ensure that you are on the default Source Instances page when you log in. If not, click Modeling 	> Source Modeling > Source Instances from the navigation menu on the left.
3. On the Source Instances page, select the correct instance name from the list in the selector on 	the left. The format will typically be “Xray_Tenant”.4. Click the PROPERTIES tab.
5. Click the Edit Values button next to Application Connection.
6. In the End Point URL field, provide the URL required to access your Xray instance. For example, 	.
7. Enter the OAuth authentication URL, OAuth Client ID, and OAuth Client Secret and click Save. 8. Click Save.
Testing Connection:
1. Click the X button to exit the dialog box.2. Click the Test Connection button next to Xray_tenant.
3. If successful, the message, “Sample Extraction Completed” is displayed in the Test Connection dialog box. Any other message could indicate failure. It can take up to a few minutes for the final message to display.
4. If the test is not successful or you are not sure, please contact your Customer Success Manager 	or Digital.ai at .

### 微软推出 Pylance，改善 VS Code 中的 Python 体验

微软宣布发布了一款新的 Python 语言服务器——Pylance。它通过语言服务器协议与 Visual Studio Code 进行通信。目前，Pylance 可作为 Visual Studio Marketplace 的扩展供用户下载。其主要功能包括类型信息、自动导入、类型检查诊断以及多根工作区支持。

来源：[开源中国](https://www.oschina.net/news/116917/microsofts-pylance-vs-code-python)

> 拍一拍：VS Code 实在是太好用了……慢慢地，我发现微软推出的开源产品越来越优秀，心情颇为复杂……

### 维基百科新项目 Abstract Wikipedia 推动跨语言内容发展

这是维基媒体基金会七年来首次启动的新项目。Abstract Wikipedia 允许贡献者使用抽象符号创建内容，并通过名为 Wikilambda 的工具托管一组函数，将其翻译成不同的自然语言。这有助于更好地平衡不同语言版本的内容，而无需在自然语言生成或词汇知识表达方面取得重大突破。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/998581.htm)

> 拍一拍：目前英文版维基百科的内容最为丰富详实。如果此项目能够成功，将会对各语言版本的发展起到极大的推动作用。不过，中文维基百科的落后不仅仅是语言问题，你懂的……

### Ubuntu 19.10 将于本月结束生命周期

官方建议用户迁移到 Ubuntu 20.04 版本。从 Ubuntu 19.10 升级到 20.04 非常方便。今年四月发布的 Ubuntu 20.04 是一个长期支持版本，将持续提供支持直至 2025 年。

来源：[开源中国](https://www.oschina.net/news/116915/ubuntu-19-10-will-soon-eol)

> 拍一拍：Ubuntu 的 STS 版本类似于 Fedora 的快速迭代模式，而 LTS 则类似于 CentOS。然而，每当我遇到 STS 版本时，我都不太愿意去使用。

### openSUSE Leap 15.2 发布

最新版本的 openSUSE Leap 15.2 引入了人工智能和机器学习库，如 Tensorflow、PyTorch、ONNX、Grafana 和 Prometheus。此外，该版本还加入了实时内核以管理微处理器的时序。

来源：[solidot](https://www.solidot.org/story?sid=64842)

> 拍一拍：这些新增的功能非常棒！

### 第二次 LibreOffice 7.0 Bug 猎杀会即将开始

随着 LibreOffice 7.0 发布日期的临近，文档基金会正努力准备下一次重大更新。LibreOffice QA 团队将于 2020 年 7 月 6 日（星期一）组织第二次 Bug 猎杀会。预计 LibreOffice 7.0 将于八月初正式发布。

来源：[softpedia](https://news.softpedia.com/news/new-libreoffice-7-0-bug-hunting-ready-to-begin-530449.shtml)

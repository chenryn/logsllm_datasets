User: What are the main procedures an organization should follow in order to implement CIS Control 17.6, Define Mechanisms for Communicating During Incident Response?
Assistant: An organization should Determine which primary and secondary mechanisms will be used to communicate and report during a security incident. Mechanisms can include phone calls, emails, or letters. Keep in mind that certain mechanisms, such as emails, can be affected during a security incident. Review annually, or when significant enterprise changes occur that could impact this Safeguard.CIS Control 17.6 has the asset type N/A, and has the security function Respond.

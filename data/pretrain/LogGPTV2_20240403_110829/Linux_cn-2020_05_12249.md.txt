### Linus Torvalds 升级主力电脑，15 年来首次不用英特尔处理器
Linux 内核创始人 Linus Torvalds 在发布 Linux Kernel 5.7 rc7 的帖子中透露，他将不再使用英特尔 CPU 作为其主要开发平台。这是自 2005 年以来，Torvalds 首次转向非英特尔架构。虽然他尚未转至 ARM 架构，但目前采用的是 AMD Threadripper 3970x 处理器。据称，新配置下进行 allmodconfig 测试的速度提升了三倍之多。尽管短期内效果可能不明显，但在下一个合并窗口期间，性能提升有望显著体现。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/982945.htm)  
**点评**: Linus 作为开源社区的重要人物之一，他的任何举动都会引起广泛关注。此次更换更强大的硬件设备，或许会进一步加速 Linux 内核的发展进程，毕竟更快的编译速度意味着可以节省大量等待时间。

### 世界上使用量最大的数据库引擎 SQLite 3.32.0 发布
SQLite 是一个用 C 语言编写的 SQL 数据库引擎，以其小巧、高效、独立运行以及高度可靠性而闻名。它被嵌入到几乎所有的智能手机和大部分计算机中，同时也是许多应用程序的标准组成部分，堪称全球范围内应用最广泛的数据库系统之一。

**来源**: [开源中国](http://www.oschina.net/)  
**点评**: 尽管 SQLite 相较于其他大型数据库显得十分低调，但凭借其轻量化特性赢得了极其广泛的应用场景，正如 MINIX 操作系统一样令人惊讶地成为世界上部署最多的 OS 之一。

### 民间高手魔改卡西欧计算器遭版权组织警告
印度一位名叫 Neutrino 的技术爱好者最近成功地将一台卡西欧 fx-MS991 计算器上的太阳能板改装成了显示屏，并通过添加 ESP 芯片及 Firebase 编程实现了网络连接功能。随后他在 GitHub 上分享了相关代码供他人参考学习。然而不久后，版权保护组织 REACT 发出警告，指出该项目中包含了部分属于卡西欧专有的软件资产，不应公开传播。尽管 Neutrino 坚持认为所有代码都是自己原创开发而成。

**来源**: [快科技](https://www.cnbeta.com/articles/tech/982723.htm)  
**点评**: 应该鼓励这种创新精神，建议卡西欧与 REACT 对此类情况采取更加宽容的态度。

### OpenCV 开源许可协议拟从 BSD 变更为 Apache 2
OpenCV 团队正在讨论关于更改当前使用的 BSD 开源许可证为 Apache License 2.0 的可能性，并计划于 6 月 29 日进行初步评估。相较于 BSD，Apache License 提供了更强有力的法律保障，在处理专利侵权等问题时能够更好地保护用户权益。

**来源**: [开源中国](https://www.oschina.net/news/115902/opencv-license-apache2-changing)  
**点评**: 尽管 BSD 许可证以宽松著称，但从整体角度来看，对于促进开源软件健康发展而言，选择哪种类型的许可协议还需综合考量各种因素。

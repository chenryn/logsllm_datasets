User: How to install and configure Nginx server in Linux system to host multiple websites on the same server?
Assistant: Following are the steps to install and configure Nginx server in Linux system to host multiple websites: 1. Install Nginx Enter the following command in terminal to install Nginx: sudo apt update sudo apt install nginx 2. Configure Nginx By default, Nginx The configuration file path is /etc/nginx/nginx.conf. Open this file and add the following content: http { server { listen 80 default_server; server_name _; root /var/www/html; location / { try_files $uri $uri/ =404; } } } #Add the following content to http The last include /etc/nginx/sites-enabled/*; This configuration file specifies that Nginx will listen to port 80, and does not specify a server name, so the default server is used by default. The root directory of the website is /var/www/html, if the file cannot be found, a 404 error will be returned. 3. Configure Website Create a new configuration file to specify the website to host. Add the following to the new configuration file, replacing SERVER_NAME with the domain name you want to host and ROOT_DIRECTORY with the path to your website's root directory: server { listen 80; server_name SERVER_NAME; root ROOT_DIRECTORY; index index.html; } 4. Enable the website Create a symbolic link to add the new configuration file to Nginx's sites-enabled directory to enable the site: sudo ln -s /etc/nginx/sites-available/NEW_CONFIG_FILE_NAME /etc/nginx/sites-enabled/ 5. Restart Ngnix Restart Nginx by entering the following command in Terminal for the new configuration to take effect: sudo systemctl restart nginx Your Nginx server is now configured to host multiple websites. You can access a hosted website by typing the domain name of the website into your browser.

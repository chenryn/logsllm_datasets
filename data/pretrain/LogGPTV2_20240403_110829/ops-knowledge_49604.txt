User: How to install the Nginx web server on a Linux server and set it up to listen for HTTP and HTTPS traffic.
Assistant: Here are the steps to install the Nginx web server on a Linux server and set it up to listen for HTTP and HTTPS traffic: 1. Install Nginx: a. Update the system with: sudo apt-get update b. Install Nginx with: sudo apt- get install nginx c. Check whether Nginx is successfully installed: sudo systemctl status nginx 2. Configure Nginx to listen to HTTP and HTTPS: a. Open the Nginx configuration file: sudo nano /etc/nginx/nginx.conf b. Add the following content in the http section : server { listen 80; server_name example.com; location / { root /var/www/example.com; index index.html; } } c. Add the following in the server section: server { listen 443 ssl; server_name example. com; ssl_certificate /path/to/cert; ssl_certificate_key /path/to/key; location / { root /var/www/example.com; index index.html; } } d. Save the file and close. 3. Restart Nginx for the changes to take effect: a. Restart Nginx using the following command: sudo systemctl restart nginx b. Check whether Nginx started successfully: sudo systemctl status nginx Notes: 1. In steps 2b and 2c, replace example.com with for your domain name, and replace /root/to/cert and /root/to/key with the paths to the actual certificate and key on your server. 2. When enabling HTTPS, a valid SSL certificate must be purchased and installed for your domain name.

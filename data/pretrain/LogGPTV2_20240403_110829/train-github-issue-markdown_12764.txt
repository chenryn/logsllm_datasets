##### Issue Type:
Bug Report
##### Ansible Version:
ansible 1.6.3
##### Environment:
Ubuntu 14.04
##### Summary:
When using apache2_module for mpm_prefork a change occurs every playbook run,
even though the module is already enabled.
##### Steps To Reproduce:
    - name: Ensure mpm_prefork is enabled
      apache2_module: name=mpm_prefork
##### Expected Results:
No change during playbook run.
##### Actual Results:
A change occurs every run, even though mpm_prefork is already enabled:
    # a2enmod mpm_prefork
    Considering conflict mpm_event for mpm_prefork:
    Considering conflict mpm_worker for mpm_prefork:
    Considering conflict mpm_itk for mpm_prefork:
    Module mpm_prefork already enabled

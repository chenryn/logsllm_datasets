## 互联网Java进阶面试训练营
### 互联网 Java 工程师面试突击（第二季）

以下是本训练营的课程内容概览，涵盖了从技术广度到深度、系统设计及项目经验等多个方面，旨在帮助您全面提升技能，为大厂面试做好准备。

1. **互联网大厂面试要求：技术广度、技术深度、系统设计以及项目经验**  
   了解顶尖公司对候选人的期望标准。[阅读更多](/docs/distributed-system/distributed-design.md)

2. **Java工程师面试突击第一季总结：你离一次成功的面试还差多少？**  
   回顾上一季的关键知识点，并评估您的准备情况。[查看总结](/docs/distributed-system/java-interview-season-1-summary.md)

3. **《21天互联网Java进阶面试训练营》的课程说明**  
   详细介绍为期三周的培训计划及其目标。[获取详情](/docs/distributed-system/21-day-course-instructions.md)

4. **作业：自我分析 - 距离大厂Offer还有多远？**  
   客观评价个人技术水平与市场需求之间的差距。[开始作业](/docs/distributed-system/homework.md)

5. **体验BAT级别分布式技术面试**  
   模拟真实场景下的密集提问环节。[参与挑战](/docs/distributed-system/BAT-interview-fire.md)

6. **深入探讨Dubbo与Spring Cloud核心架构**  
   对比两种流行框架的设计理念和技术特点。[了解更多](/docs/distributed-system/core-architecture-principle.md) [示例代码下载](https://github.com/shishan100/Java-Interview-Advanced/raw/master/docs/distributed-system/code/code.zip)

7. **基于Dubbo和Spring Cloud搭建电商系统实践**  
   动手操作以加深理解。[立即尝试](/docs/distributed-system/Dubbo-SpringCloud-experience.md) [示例代码下载](https://github.com/shishan100/Java-Interview-Advanced/raw/master/docs/distributed-system/code/code.zip)

8. **作业：服务框架选型理由分析**  
   分析并解释所在团队选择特定服务框架的原因。[提交答案](/docs/distributed-system/distributed-framework-selection.md)

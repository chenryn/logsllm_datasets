> 在 Ubuntu 的混合风味中加入肉桂味，完美！

![Ubuntu Cinnamon](/data/attachment/album/202303/31/091748pq8tq8nluucc5eth.jpg)

您是否在等待下个月即将发布的 Ubuntu 23.04？我们已经提到，在即将到来的 Ubuntu 23.04 版本中，将包括一个新的官方 Cinnamon 风味版本（最初是 Ubuntu Cinnamon Remix）。现在，由于 Ubuntu 技术委员会的批准，它已成为官方版本。

### Ubuntu Cinnamon 23.04 的亮点

![Ubuntu Cinnamon 23.04 的屏幕截图](/data/attachment/album/202303/31/091753g22p86xzx021qm8z.jpg)

如果您之前使用过非官方的 Ubuntu Cinnamon Remix，那么您可能对它有一定的了解。如果您希望有一个基于 Cinnamon 桌面的 Ubuntu 风味版本，并且不希望包含过多的定制化内容，Ubuntu Cinnamon 将是一个不错的选择。

另一方面，Linux Mint 也是一个基于 Ubuntu 的 Cinnamon 桌面发行版，但它带有独特的主题、图标、工具和调整，使其与 Ubuntu Cinnamon 有所不同。此外，Linux Mint 默认禁用 Snap 并优先使用 Flatpak，而 Ubuntu Cinnamon 23.04 则相反。

![Ubuntu Cinnamon 菜单截图](/data/attachment/album/202303/31/091754ef57t6mfu8u4elnm.jpg)

因此，如果您更喜欢使用 Snap 而不是 Flatpak，并且习惯于 Ubuntu 的工作方式，Ubuntu Cinnamon 是一个值得尝试的新选择。这个版本的一些其他值得注意的特点包括：
- 预装的应用程序，如 Synaptic Package Manager 和 LibreOffice
- 使用 Ubuntu 的 Yaru 主题
- 不包含 Linux Mint 中常见的 XApp

根据最新的每日构建情况，Ubuntu Cinnamon 23.04 已经接近发布状态（预计下个月发布）。

虽然能否赢得 Linux Mint 用户的青睐还有待观察，但新的 Ubuntu Cinnamon 版本无疑会吸引更多的用户尝试 Cinnamon 桌面，这是一件好事。Cinnamon 桌面结合了传统的 Windows 风格布局，并提供了更多功能。

您可以在 [Ubuntu Cinnamon 博客](https://ubuntucinnamon.org/ubuntu-cinnamon-flavor-status-announcement/?ref=its-foss-news) 上阅读官方公告。您是否会考虑将 Ubuntu Cinnamon 作为您的首选桌面环境？请在评论中分享您的想法。

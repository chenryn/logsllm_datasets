### 如何使用 Cream 提高 Vim 的用户友好性

大约十年前，我既使用 Emacs 进行文本编辑，也使用 Vim 进行文本编辑。虽然我是 Emacs 的忠实用户，但我也知道 Vim 同样是一款强大的编辑器。

然而，对于那些在技术上不太熟练的人来说，Vim 可能会显得有些难以掌握。多年来，我在与一些 Linux 新手交流时发现，他们希望使用 Vim，但却因它与他们在其他操作系统上使用的编辑器不同而感到失望。

当我向他们介绍了 Cream 之后，情况发生了变化。Cream 是一个基于 Vim 的附加包，它通过提供一个更加友好的界面，使得 Vim 更易于使用。这让许多新手转变成了坚定的 Vim 用户。

让我们一起来看看 Cream 是什么，以及它是如何使 Vim 更加用户友好的。

#### Cream 的安装

在安装 Cream 之前，请确保你的系统中已经安装了 Vim 和 GVim 的图形界面组件。最简单的方法是使用你所使用的 Linux 发行版的包管理器来完成这一过程。

安装好 Vim 后，你可以从 [Cream 的官方网站](http://cream.sourceforge.net/download.html) 下载安装程序，或者再次利用包管理器进行安装。

安装完成后，你可以从应用程序菜单（如 **Applications** -> **Cream**）或通过在启动器中输入 `Cream` 来启动该软件。

![How to make Vim user-friendly with Cream](/data/attachment/album/201705/25/161055s6ewwhospwflorv6.png "How to make Vim user-friendly with Cream")

#### Cream 的使用

如果你之前使用过 Gvim，你会发现 Cream 几乎没有改变其外观和感觉。主要的区别在于 Cream 的菜单栏和工具栏，它们看起来和其他常见的编辑器类似，并且功能分组更为合理。

Cream 的菜单栏隐藏了一些更高级的技术选项，例如指定编译器的能力或运行 `make` 命令的功能。随着你对 Cream 越来越熟悉，你可以通过 **Settings** -> **Preferences** -> **Behavior** 选项轻松访问这些特性，从而体验到兼具 Cream 和传统 Vim 优点的强大编辑器。

尽管 Cream 提供了菜单驱动的操作方式，但它仍然支持常用的键盘快捷键，如 `CTRL-O`（打开文件）和 `CTRL-C`（复制文本）。你不需要切换多种模式，也不需要记住复杂的命令。

启动 Cream 后，你可以直接打开或创建新文件并开始编辑。很多我介绍过 Cream 的人表示，尽管它保留了许多 Vim 的典型风格，但使用起来更加舒适。

![Cream add-on for VIM in action](/data/attachment/album/201705/25/161057fvh3v3vo4kkku4zv.png "Cream add-on for VIM in action")

这并不是说 Cream 是 Vim 的简化版本；相反，它保留了 Vim 的所有功能，并增加了一些有用的特性，包括：
- 标签式界面
- 语法高亮（特别是针对 Markdown、LaTeX 和 HTML）
- 自动拼写纠正
- 字数统计
- 内置文件浏览器

此外，Cream 还提供了许多扩展插件，可以为编辑器添加新的功能，例如文本加密、邮件内容清理，甚至还有教程。虽然我个人并未发现特别有用的插件，但每个人的体验可能会有所不同。

有些 Vi/Vim 的狂热爱好者批评 Cream“降低了”Vi/Vim 的标准。的确，Cream 并不是为他们设计的。它是为那些希望快速上手 Vim，同时保持他们熟悉的编辑器外观和感觉的人准备的。在这种情况下，Cream 是值得称赞的，因为它让 Vim 更容易被广泛接受和使用。

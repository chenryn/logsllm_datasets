**ATT&CK ID:** S0669  
**ATT&CK Technique Name:** KOCTOPUS

**Description:**
KOCTOPUS is a malicious technique that attempts to delete or disable all Registry keys and scheduled tasks associated with Microsoft Security Defender and Microsoft Security Essentials. This action aims to undermine the security measures provided by these antivirus solutions, making the system more vulnerable to further attacks. [48]

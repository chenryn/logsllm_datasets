### Environment
- **Windows Build Number:** 10.0.19041.264
- **PowerToys Version:** 18.2

### Steps to Reproduce
1. Open PowerToys Run.
2. Click inside the input box.
3. Press the Windows key in combination with another key (e.g., Windows + Space).
4. Observe that no key combination is bound.

### Expected Behavior
I expect to be able to bind the Windows key in combination with other keys (e.g., Windows + Space) to launch PowerToys Run.

### Actual Behavior
The Windows key cannot be used in keyboard shortcuts within PowerToys.

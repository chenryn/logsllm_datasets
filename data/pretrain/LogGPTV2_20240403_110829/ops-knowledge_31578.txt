User: How to use Git for version control? Please list at least three commonly used Git commands and what they do.
Assistant: 1. git init: Set the current directory as the Git warehouse and start version control. 2. git add: Add the files in the current workspace to the temporary storage area, ready to be submitted to the warehouse. 3. git commit: Submit the files in the temporary storage area to the warehouse to form a new version and add the submission information. Other commonly used commands include: - git status: View the status of the workspace and temporary storage area. - git log: view the commit history. - git branch: View and manage branches. - git merge: Merge the code of two different branches. - git clone: Copy the remote warehouse to the local. - git pull: Pull the latest changes from the remote warehouse to the local. - git push: Push the latest modification of the local warehouse to the remote warehouse.

### RMS 建议使用 Zoom 的学生们自我洗脑“自由软件很重要”

据消息来源，自由软件基金会创始人 RMS 在一场 90 分钟的演讲中发表了一些观点。他提到使用 Zoom 这样的非自由应用时建议学生进行自我提醒：“如果你不得不使用非自由程序”，每次上课时都说一句：“我为在这节课上使用 Zoom 而感到苦恼。”随着时间推移，这将增强你对自由软件重要性的认识。他还谈到了硬件问题，指出 Macintoshes 正朝着成为监狱的方向发展，就像 iMonsters 一样。“出售一台不让用户从源代码中安装自己的软件的电脑应该是非法的。”对于采用非自由固件的电脑，“我们拥有的自由启动的机器越来越老，越来越少。……因为英特尔和 AMD 都在设计他们的硬件来压制人们。”

> 老王点评：真是活在理想中的人。

### GitHub 暂停受美国制裁的公司的俄罗斯开发者账户

根据报道，GitHub 至少屏蔽了数十个俄罗斯开发者的账户。被屏蔽的个人账户无法访问其存储库，而公司账户则可以在几个小时内访问。尽管 GitHub 曾表示会确保所有人都能获得免费的开源服务，但同时也要遵守美国法律。他们正在彻底审查政府制裁以确保用户不会受到超出法律要求的影响。

> 老王点评：说到底，这是美国的公司。中国开发者们，我们也要未雨绸缪啊。

![](/data/attachment/album/202204/17/135748d070z9vdhjlvhpo0.jpg)

### 攻击者利用偷来的 OAuth 令牌入侵了 GitHub 上的几十个组织

据报道，一名攻击者正利用偷来的 OAuth 用户令牌从 GitHub 私有仓库下载数据。Heroku 和 Travis-CI 通过这些令牌来维护 OAuth 应用程序。已有包括 NPM 在内的几十个受害组织被窃取了私有仓库的代码和数据。然而，GitHub 并不认为攻击者是通过破坏其系统来获取这些令牌的，并已向 Heroku 和 Travis-CI 通报此事件。

> 老王点评：如果你使用 Heroku 和 Travis-CI 来管理你的 OAuth 应用，请核查并保护你的仓库。

User: How to Check CPU and Memory Usage in Linux System?
Assistant: 1. Check the CPU usage: Use the top command to check the CPU usage in the system in real time, including the CPU usage of each process and the overall CPU usage of the system. Command: top In addition, if you want to accurately view the CPU usage of a process, you can use the ps command. Command: ps aux 2. View memory usage: Use the free command to view the memory usage in the system, including total memory, used memory, and free memory. Command: free In addition, if you want to accurately view the memory usage of a process, you can use the ps command. Command: ps aux

### lpq

**适用系统：**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**功能描述：**
`lpq` 命令用于显示运行 Line Printer Daemon (LPD) 的计算机上的打印队列状态。

**语法：**
```
lpq -S <主机名或IP地址> -P <打印机名称> [-l]
```

**参数说明：**

- **-S <主机名或IP地址>**
  - 指定（通过名称或 IP 地址）承载 LPD 打印队列的计算机或打印机共享设备。此参数是必需的，并且必须大写。
  
- **-P <打印机名称>**
  - 指定（通过名称）要显示状态的打印队列所对应的打印机。此参数是必需的，并且必须大写。
  
- **-l**
  - 指定显示打印队列状态的详细信息。
  
- **/?**
  - 在命令提示符下显示帮助信息。

**示例：**
- 要显示位于 10.0.0.45 的 LPD 主机上 Laserprinter1 打印机队列的状态，可以输入以下命令：
  ```
  lpq -S 10.0.0.45 -P Laserprinter1
  ```

### 命令行工具
* [Bastet](https://github.com/fph/bastet) - 在命令行玩俄罗斯方块。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/fph/bastet) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Daily Reddit Wallpaper](https://github.com/ssimunic/Daily-Reddit-Wallpaper) - 开机时将壁纸更换为 /r/wallpapers 或其它子版块的今日最高票壁纸。 [![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/ssimunic/Daily-Reddit-Wallpaper) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Fishfry](https://github.com/LubuntuFu/fishfry) - 将 fish 的历史记录替换成为渗透测试人员量身定制的高效版本以及方便新手学习。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/LubuntuFu/fishfry) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [htop](http://hisham.hm/htop/) - 带有改进的特性和用户体验的 top。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/hishamhm/htop) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Pacman4console](https://launchpad.net/ubuntu/+source/pacman4console) - 在控制台玩吃豆人。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/alexdantas/pacman4console.debian) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Tmux](https://tmux.github.io/) - 它让你在一个终端中在多个程序间方便地切换，分离他们（保持在后台运行）并另一个终端中重新连接上去。以及还有好多事情可以做。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/tmux/tmux) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [十佳命令行工具](http://lifehacker.com/399468/top-10-command-line-tools)
* [wicd-curses](https://wiki.archlinux.org/index.php/wicd#Running_Wicd_in_Text_Mode) - 命令行 WiFi 连接管理器。
### 桌面环境
* [Budgie](https://solus-project.com/budgie/) - Budgie 是一个面向现代用户思想设计的桌面环境，专注于简洁与优雅。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/solus-project/budgie-desktop) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Cinammon](http://cinnamon.linuxmint.com/) - Cinnamon 努力提供一种传统的用户体验。Cinnamon 是一个 GNOME 3 的 fork。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/linuxmint/Cinnamon) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [GNOME Flashback](https://wiki.gnome.org/Projects/GnomeFlashback) - GNOME Flashback 是 GNOME 3 的一个 shell，最初叫 GNOME fallback 模式。桌面的布局和底层技术都与 GNOME 2 相似。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/GNOME/gnome-flashback) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Gnome](https://www.gnome.org/gnome-3/) - GNOME 桌面环境是一个有吸引力且直观的的桌面，同时有现代（GNOME）和经典（GNOME Classic）会话模式。 [![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/GNOME/gnome-desktop) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [KDE Plasma](https://www.kde.org/workspaces/plasmadesktop/) - KDE Plasma 桌面环境是一个熟悉的工作环境。Plasma Desktop 提供现代桌面环境体验所需的所有工具，所以你可以从开始就充满生产力。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/KDE/plasma-desktop) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [LXDE](http://lxde.org/) - Lightweight X11 Desktop Environment （轻量化 X11 桌面环境）是一个快速且节能的桌面环境。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/lxde) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [LXQt](http://lxqt.org/) - LXQt 是 LXDE 的 Qt 移植和即将到来的版本，轻量化的桌面环境。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/lxde/lxqt) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Mate](http://mate-desktop.com/) - Mate 用传统隐喻给 Linux 用户提供一个直观又有吸引力的桌面。MATE 是 GNOME 2 的一个 fork。 [![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/mate-desktop/) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Pantheon](https://elementary.io/) - Pantheon 是最初为发行版 elementary OS 创建的桌面环境，同时也是它的默认桌面环境。 ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Unity](https://unity.ubuntu.com/) - Unity 是 GNOME 的一个 shell，由 Canonical 为 Ubuntu 设计。 ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Xfce](https://www.xfce.org/) - Xfce 体现了模块化和可重用的传统 UNIX 哲学。 [![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/xfce-mirror) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)

### 窗口管理
#### 合成器
* [Compton](https://github.com/chjj/compton/wiki) - Compton 是一款独立的合成管理器，适合同没有原生提供合成功能的窗口管理器一同使用。 [![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://camo.githubusercontent.com/123c212a73b8dd36e355c193788ea3d8baa36f73/68747470733a2f2f63646e2e7261776769742e636f6d2f6943484149542f617765736f6d652d6f73782f6d61737465722f6d656469612f6f73732e737667) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Xcompmgr](https://cgit.freedesktop.org/xorg/app/xcompmgr) - Xcompmgr 是一个简单的合成管理器，能够渲染下拉阴影，使用 transset 工具的话，还可以实现简单的窗口透明。 [![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://cgit.freedesktop.org/xorg/app/xcompmgr) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
#### 叠加式窗口管理器
* [2bwm](https://github.com/venam/2bwm) - 快速的浮动窗口管理，有两个特殊边界，基于 XCB 库，由 mcwm 衍生。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/venam/2bwm) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Blackbox](http://blackboxwm.sourceforge.net/) - 快速，轻量化的 X 窗口系统窗口管理器，没有那些烦人的库依赖。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/bbidulock/blackboxwm) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Fluxbox](http://fluxbox.org/) - 基于 Blackbox 0.61.1 代码的 X 窗口管理器。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/fluxbox/fluxbox) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Openbox](http://openbox.org/) - 高度可配置，带有可扩展标准支持的下一代窗口管理器。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/danakj/openbox) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
#### 平铺式窗口管理器
* [Bspwm](https://github.com/baskerville/bspwm/wiki) - bspwm 是一个平铺式窗口管理器，将窗口以二叉树的叶结点的方式展现。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/baskerville/bspwm) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [Herbstluftwm](https://herbstluftwm.org/) - 使用 Xlib 和 Glib 的手工平铺式窗口管理器。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/herbstluftwm/herbstluftwm) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
#### 动态窗口管理器
* [awesome](https://awesome.naquadah.org/) - 高度可配置，下一代 X 框架窗口管理器。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/awesomeWM/awesome) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [dwm](http://dwm.suckless.org/) - X 动态窗口管理器。它以平铺，单片镜以及浮动布局的方式管理窗口。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/cdown/dwm) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [i3](https://i3wm.org/) - 完全重写的平铺式窗口管理器。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/i3/i3) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [spectrwm](https://github.com/conformal/spectrwm/wiki) - X11 小型动态窗口管理器，很大程度上受了 xmonad 和 dwm 的启发。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/conformal/spectrwm) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)
* [xmonad](http://xmonad.org/) - 动态平铺式 X11 窗口管理器，用 Haskell 写就和配置。[![](/data/attachment/album/201608/24/174953qn9khfxeaxutdknn.png)](https://github.com/xmonad/xmonad) ![](/data/attachment/album/201608/24/174932s63zqzknkuaaiika.png)

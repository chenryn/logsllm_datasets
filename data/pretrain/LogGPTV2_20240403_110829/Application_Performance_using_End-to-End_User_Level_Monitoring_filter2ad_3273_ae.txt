### 优化后的文本

在模型（1）中，未知参数为 \(d, I_1, I_2, \ldots, I_N\) 和频率参数 \(f_1, f_2, \ldots, f_N\)。其中 \(N\) 表示频率参数的数量。假设独立的高斯白噪声具有均值 0 和方差 \(\sigma^2\) [22][23]。模型（1）中的频率表示在时间周期 \(T\) 内完成的完整周期数。以一个例子说明，设 \(N = 1\) 并假设 \(I_1\) 在每个 \(T\) 时间单位内完成一个完整周期。

当 \(I_1\) 每个 \(T\) 时间单位重复一次时，我们首先选择适当的频率 \(f_1, f_2, \ldots, f_N\) 对于给定的时间序列。一旦选定频率，估计模型参数 \(\{d, I_1, I_2, \ldots, I_N\}\) 的问题可以通过正则多重回归分析解决，其中 \(N\) 可以通过向前或向后选择过程确定。

我们在一个由 TruTime™ 应用保障系统收集的 217 小时内的 Web 服务器延迟数据集上展示了我们的方法。每个数据点代表相应小时内的平均 Web 服务器延迟（以毫秒为单位）。假设 Web 服务器延迟模式通常每 168 小时（一周）重复一次，因此我们设置 \(T = 168\) 小时，并通过前向选择过程选择 \(N = 6\)。功率谱分析确定的频率从最重要到最不重要依次为 4、7、8、10、9 和 6。

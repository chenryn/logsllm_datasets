# Feature request
## Is your feature request related to a problem? Please describe.
When I include a Typescript file from another package, I get this error:
`You may need an appropriate loader to handle this file type, currently no
loaders are configured to process this file. See
https://webpack.js.org/concepts#loaders`
## Describe the solution you'd like
I would like to be able to import Typescript files from other packages.

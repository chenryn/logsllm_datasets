Assistant: 1. 200 OK: Successfully returned information 2. 201 Created: Created successfully 3. 204 No Content: The request was successfully processed but no entity body was returned 4. 301 Moved Permanently: The requested resource has been permanently moved to a new location, the returned response The message will contain the new URI 5. 302 Found: The requested resource is temporarily moved to a new location, and the returned response message will contain the new URI 6. 304 Not Modified: The requested resource has not been modified and can be Read the cache directly, omitting the transmission process and improving efficiency. 7. 400 Bad Request: There is a syntax error in the request message or the request parameter is incorrect. 8. 401 Unauthorized: No authentication or authentication failed. 9. 403 Forbidden: The server rejects the Request 10. 404 Not Found: The requested resource does not exist 11. 500 Internal Server Error: The server has an internal error and cannot complete the request 12. 503 Service Unavailable: The server is temporarily unable to process the request, possibly due to maintenance or overload.

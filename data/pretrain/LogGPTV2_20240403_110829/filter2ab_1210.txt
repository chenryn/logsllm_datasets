**ATT&CK ID:** S0529  
**ATT&CK Technique Name:** CarbonSteal

**Description:**
CarbonSteal is a technique used to gather detailed network information from a device. Specifically, it collects the following data:
- 16-bit GSM Cell Identity
- 16-bit Location Area Code
- Mobile Country Code (MCC)
- Mobile Network Code (MNC)

Additionally, CarbonSteal invokes the `netcfg` command to retrieve network statistics [7].

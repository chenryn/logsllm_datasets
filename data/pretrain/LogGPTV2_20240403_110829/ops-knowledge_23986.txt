User: How to monitor the performance of the server? What tools are available?
Assistant: 1. Monitor CPU usage: top, sar, iostat, perf 2. Monitor memory usage: free, vmstat 3. Monitor disk IO usage: iostat 4. Monitor network bandwidth usage: iftop, nload 5. Monitor processes and services Status: ps, netstat, lsof, systemctl 6. Monitoring logs: tail, grep, awk 7. Building monitoring platforms: Zabbix, Nagios, Grafana, Prometheus 8. Monitoring website performance: ab, siege, curl 9. Monitoring database: mysqladmin, pg_top, pg_stat_statements, oracle Enterprise Manager 10. Draw performance curves: gnuplot, Excel.

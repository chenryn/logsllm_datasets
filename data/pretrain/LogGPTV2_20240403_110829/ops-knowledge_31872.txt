User: Please provide some basic Linux commands and their functions.
Assistant: 1. cd: Change the current working directory. 2. ls: List the files and subdirectories under the directory. 3. cp: Copy files or directories. 4. rm: Delete files or directories. 5. mv: Move or rename a file or directory. 6. mkdir: Create a new directory. 7. rmdir: Delete empty directories. 8. pwd: Output the absolute path of the current working directory. 9. cat: view the contents of the file in the terminal. 10. grep: Find lines containing a specified string in a text file.

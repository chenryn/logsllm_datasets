User: On the Linux system, how to check the resource usage, running status and CPU usage of a certain process?
Assistant: You can use the following commands to view: 1. top command: Display the running status of the process in real time, including information such as CPU and memory usage. ``` top -p process id ``` 2. ps command: View the basic information of a process, including process ID, status, CPU usage and memory usage. ``` ps -ef | grep process name``` 3. pidstat command: Check the CPU usage, IO and other information of the process. ``` pidstat -p process id ``` 4. strace command: Track the occurrence of system calls and signals of the process. ``` strace process name``` 5. lsof command: View the files and network connections opened by the process. ``` lsof -p process id ```

## JavaScript 比 Java 和 .NET 缺陷更少，修复更快

一项针对数百万商业应用程序的研究显示，近 75% 的应用程序存在安全缺陷，并且随着时间推移变得更加不安全。其中，82.2% 的 .NET 应用程序存在缺陷，Java 应用程序的缺陷率为 77.7%，而 JavaScript 应用程序的缺陷率仅为 55.8%。此外，仅有 9.5% 的 JavaScript 应用程序存在严重的安全漏洞，相比之下，Java 和 .NET 的比例分别为 19.9% 和 21.9%。在修复速度方面，JavaScript 缺陷的平均修复时间为 116 天，而 .NET 和 Java 分别为 158 天和 243 天。

**消息来源：[Dev Class](https://devclass.com/2023/01/17/application-security-survey-finds-javascript-beats-java-and-net-with-fewer-flaws-faster-fixes/)**

**老王点评：这个结果有些出乎意料。专门为大型企业应用设计的 Java 和 .NET 在安全性方面反而不如 JavaScript。**

## 微软通过补丁检查过期的 Office 套件

微软计划通过更新推送一个补丁，以了解 Windows PC 上安装了多少个已停止支持的 Office 应用副本。该补丁主要检查的是已经停止支持的 Office 2007 和 2010 版本，以及即将在今年 4 月停止支持的 Office 2013。微软表示，此更新将只运行一次，不会在用户的设备上安装任何内容。不过，微软并未透露如何处理从更新中收集的数据。

**消息来源：[The Register](https://www.theregister.com/2023/01/21/microsoft_office_count_update/)**

**老王点评：微软此举似乎有些过分干预用户的行为，难道过了支持期的 Office 就不能使用了吗？**

## Linux 基金会发起新的“开放元宇宙基金会”

Linux 基金会宣布成立开放元宇宙基金会（OMF），其使命是培养一个由开发者、工程师、学者和思想领袖组成的强大社区，致力于开发开源软件和标准，构建一个包容性、全球性、供应商中立且可扩展的元宇宙。他们认为元宇宙可以在数字空间创造新的就业机会和产业，弥合物理世界和数字世界之间的差距，提供一个任何人都可以创造自己机会的世界。未来市场价值可能超过任何单一媒体市场。

**消息来源：[Linux 基金会](https://www.linuxfoundation.org/press/open-metaverse-foundation)**

**老王点评：我认为现在成立开放元宇宙基金会可能还为时过早。**

User: How to create EC2 instance using AWS CLI? Please provide detailed step-by-step instructions.
Assistant: To create an EC2 instance, follow these steps: 1. Install the AWS CLI (if not installed) 2. Configure the AWS CLI Run the following command: ``` aws configure ``` Enter the Access Key ID, Secret Access Key, default region, and output as prompted Format. 3. Create an EC2 instance and run the following command: ``` aws ec2 run-instances --image-id ami-0c55b159cbfafe1f0 --count 1 --instance-type t2.micro --key-name mykey --security-group-ids sg-1234567890 --subnet-id subnet-1234567890 ``` Among them, the parameters are explained as follows: --image-id: Specify the AMI ID to start the instance. --count: Specifies the number of instances to start. --instance-type: Specifies the type of instance. --key-name: Specify the name of an existing key pair to enable access via SSH. --security-group-ids: Specifies the security group IDs to apply to the instance. --subnet-id: Specifies the subnet ID in which to launch the instance. 4. After the instance start command is executed successfully, AWS will return the instance ID and other relevant information. At this point, you need to wait for the instance to start. 5. Access the instance You can access and manage the instance through SSH or RDP (depending on the operating system). The above are the basic steps for creating an EC2 instance. According to different requirements, other parameters can also be added for configuration, such as specifying user data, setting tags, etc. It should be noted that before creating an instance, you need to create the required key pair, security group, and subnet.

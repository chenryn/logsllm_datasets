Assistant: 1. ls - list the files and directories in the current directory 2. cd - enter the specified directory 3. mkdir - create a new directory 4. rm - delete a file or directory 5. cp - copy a file or directory 6. mv - move or Rename a file or directory 7. cat - view file content 8. grep - search for a specified string 9. tar - compress or decompress a file 10. chmod - modify the permissions of a file or directory 11. ssh - log in to the server remotely 12. ps - display System process 13. top - display system resource usage 14. man - view the help manual of the command 15. echo - output the specified text 16. sudo - run the command with super administrator privileges 17. awk - process text data 18. sed - process Text data 19. curl - Get data from URL 20. ifconfig - Display network interface information

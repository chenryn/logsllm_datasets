In pull request titles, you may see indicators such as [WIP] (Work In Progress) and [MRG] (Ready to Merge). Similarly, in commit messages, there are specific tags used to denote the type of changes, including:

- ENH: Enhancement
- FIX: Bug Fix
- TST: Test-related changes
- DOC: Documentation updates
- MAINT: Maintenance tasks

These tags help in quickly identifying the nature and status of the changes being proposed.

User: What are the main procedures an organization should follow in order to implement CIS Control 5.2, Use Unique Passwords?
Assistant: An organization should Use unique passwords for all enterprise assets. Best practice implementation includes, at a minimum, an 8-character password for accounts using MFA and a 14-character password for accounts not using MFA. CIS Control 5.2 has the asset type Users, and has the security function Protect.

##### SUMMARY
When custom callback plugin with same name as built in plugin is defined, the
plugin gets called twice. The plugin called is the one in runtime folder.
##### STEPS TO REPRODUCE
Add a callback plugin to "runtime folder/callback_plugins" and enable it.
Tested with logstash plugin only.

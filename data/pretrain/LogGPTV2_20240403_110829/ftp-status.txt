### FTP Status Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `status` command in FTP displays the current status of active FTP connections, including information such as the current transfer mode, type of data being transferred, and any other relevant details.

**Syntax:**
```plaintext
status
```

**Examples:**
To display the current status of FTP connections, use the following command:
```plaintext
status
```

This command will provide you with a detailed overview of the current FTP session, helping you to monitor and manage your FTP activities effectively.

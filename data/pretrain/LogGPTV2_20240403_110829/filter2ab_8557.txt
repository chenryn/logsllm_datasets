**ATT&CK ID:** M1028  
**ATT&CK Technique Name:** Operating System Configuration

**Mitigation Description:**
To enhance system security, it is recommended to disable the default setting that hides file extensions for known file types in Windows operating systems. This can help users more easily identify potentially malicious files by displaying their full file extensions.

### FTP mkdir Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `mkdir` command in FTP is used to create a directory on the remote computer.

**Syntax:**
```plaintext
mkdir <directory_name>
```

**Parameters:**
- **<directory_name>**: Specifies the name of the new remote directory.

**Examples:**
To create a directory called `dir1` on the remote computer, use the following command:
```plaintext
mkdir dir1
```

This command will create a new directory named `dir1` in the current working directory on the remote server.

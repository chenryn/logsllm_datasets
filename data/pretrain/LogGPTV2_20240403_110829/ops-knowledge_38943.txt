User: What are the main procedures an organization should follow in order to implement CIS Control 4.1, Establish and Maintain a Secure Configuration Process?
Assistant: An organization should Establish and maintain a secure configuration process for enterprise assets (end-user devices, including portable and mobile, non-computing/IoT devices, and servers) and software (operating systems and applications). Review and update documentation annually, or when significant enterprise changes occur that could impact this Safeguard.CIS Control 4.1 has the asset type Applications, and has the security function Protect.

### Geo-IP Blocking Strategies

#### The Shotgun Approach
- **Block Entire RIR Address Space**
  - **Pros:** Easy and quick.
  - **Cons:** May block some countries or areas you want.
  - **Example:** Australian and New Zealand sites use APNIC address space.

#### The Rifle Approach
- **Block by Specific Country Assignments**
  - **Pros:** Less likely to block wanted traffic.
  - **Cons:** Time-consuming, need to keep configurations up to date, some assignments are not country-specific.

#### Limited Geo-IP Blocking
- **Block all protocols but mail (no spam benefit)**
- **Block all except web server and mail**
- **Block on all private LAN segments but not DMZ (no spam or public server exploits benefits)**

#### More Limited Geo-IP Blocking
- **Block only interactive services (Telnet, SSH, VNC, Terminal Services)**
- **Block only on specialized servers (VPN, database)**
- **Block RIR but allow certain ISP and hotel webspace (to allow for VPN access)**

### IP Chains Geo-IP Blocking Example
```sh
ipchains -I input -s 58.0.0.0/8 -j DENY
```

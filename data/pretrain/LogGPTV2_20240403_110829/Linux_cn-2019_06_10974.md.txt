如果你是设计师、开发者或艺术家，你可能一直在使用 Linux 上的实验性 [Unity 编辑器](https://unity3d.com/unity/editor)。然而，实验性版本并不总是理想的——开发者需要一个完整且稳定的开发环境。因此，Unity 最近宣布其在 Linux 上推出了功能完整的编辑器。

虽然这是一个令人振奋的消息，但让我们来看看它具体支持哪些 Linux 发行版和其他细节。

> **非 FOSS 提醒**
>
> 在 Linux 或其他平台上使用的 Unity 编辑器并不是开源软件。我们在此提及是因为：
>

### 官方支持 Ubuntu 和 CentOS 7

![](/data/attachment/album/201906/14/112102oo7l7rjstuvisnus.png)

无论您拥有个人许可还是专业许可，只要安装了 Unity 2019.1 或更高版本，即可使用该编辑器。Unity 主要支持以下发行版：Ubuntu 16.04, Ubuntu 18.04 和 CentOS 7。

根据[官方公告](https://blogs.unity3d.com/2019/05/30/announcing-the-unity-editor-for-linux/)，以下是支持的具体配置：
- x86-64 架构
- Gnome 桌面环境（运行于 X11 窗口系统之上）
- Nvidia 官方专有显卡驱动程序和 AMD Mesa 显卡驱动程序
- 在没有仿真或兼容层的情况下直接运行于桌面计算机上的硬件

尽管你可以尝试其他配置，但为了获得最佳体验，建议遵循官方推荐的要求。

> **关于第三方工具的说明**
>
> 如果你在项目中使用了任何第三方工具，请单独确认这些工具是否得到支持。
>

### 如何在 Linux 上安装 Unity 编辑器

了解了基本要求后，接下来是如何安装？

1. 首先，你需要下载并安装 [Unity Hub](https://forum.unity.com/threads/unity-hub-v-1-6-0-is-now-available.640792/)。
   ![](https://example.com/path/to/unityhub-image.jpg)
   
2. 下载适用于 Linux 的 Unity Hub AppImage 文件。若不熟悉如何处理 AppImage 文件，可以参考[这篇指南](https://itsfoss.com/use-appimage-linux/)。

3. 运行 Unity Hub 并登录你的 Unity ID（如果还没有账号，则需注册）。更多关于许可证激活的信息请访问他们的 [FAQ 页面](https://support.unity3d.com/hc/en-us/categories/201268913-Licenses)。

4. 登录后，进入 "Installs" 选项卡（如上图所示），选择所需的版本或组件进行安装。

就这样！以上就是获取并快速安装 Unity 编辑器的最佳方法。

### 结论

尽管这是一条令人兴奋的消息，但官方支持的配置范围似乎有限。如果您正在 Linux 上使用 Unity 编辑器，请考虑在[官方论坛帖子](https://forum.unity.com/forums/linux-editor.93/)上分享您的反馈和意见。

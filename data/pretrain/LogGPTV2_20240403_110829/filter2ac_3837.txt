**ATT&CK ID:** S1072  
**ATT&CK Technique Name:** Industroyer2

**Description:**
Industroyer2 is a sophisticated malware that can query a target device for various operational statuses and parameters. Specifically, it has the capability to:

- Poll the connection status of the target device.
- Monitor the data transfer status.
- Retrieve the Common Address (CA).
- Access Information Object Addresses (IOAs).
- Obtain IO state values across multiple priority levels.

These capabilities enable Industroyer2 to gather detailed information about the target's operational environment, facilitating further malicious activities. [6][7]

User: What are the main procedures an organization should follow in order to implement CIS Control 9.3, Maintain and Enforce Network-Based URL Filters?
Assistant: An organization should Enforce and update network-based URL filters to limit an enterprise asset from connecting to potentially malicious or unapproved websites. Example implementations include category-based filtering, reputation-based filtering, or through the use of block lists. Enforce filters for all enterprise assets.CIS Control 9.3 has the asset type Network, and has the security function Protect.

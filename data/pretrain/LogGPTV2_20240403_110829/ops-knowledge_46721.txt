User: How to install Docker on a Linux server? Please provide specific steps.
Assistant: The following are the steps to install Docker on Ubuntu: 1. Update system packages ``` sudo apt-get update ``` 2. Install dependent packages ``` sudo apt-get install apt-transport-https ca-certificates curl software-properties-common ``` 3. Add Docker GPG key ``` curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add - ``` 4. Add Docker APT source ``` sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" ``` 5. Update APT source and install Docker ``` sudo apt-get update sudo apt-get install docker-ce ``` 6. Verify that Docker is installed correctly ``` sudo docker run hello-world ``` Through the above steps, you should have successfully installed on Linux Docker is installed on the server.

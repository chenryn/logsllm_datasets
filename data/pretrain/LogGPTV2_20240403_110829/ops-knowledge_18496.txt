User:  How to add and manage Linux users?
Assistant: To add and manage Linux users, you can follow the steps below: 1. Add a user: use the adduser or useradd command to add a new user, for example: sudo adduser newuser 2. Set a password: use the passwd command to set a password for the new user, for example: sudo passwd newuser 3. Manage user groups: You can use the usermod command to add users to existing groups or create new groups, for example: sudo usermod -a -G groupname username 4. Delete users: Use the userdel command to delete users, for example: sudo userdel username 5. Modify user information: You can use the usermod command to modify the userâ€™s user ID, home directory, login Shell and other information, for example: sudo usermod -l newusername oldusername 6. View user information: use the id or whoami command to view the current user information, for example : id uid=1000(username) gid=1000(username) groups=1000(username),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),116(lpadmin) ,126(sambashare) whoami username Through the above operations, you can easily add and manage Linux users.

## Keyboard Shortcuts

### General
| Shortcut | Action |
| --- | --- |
| `Ctrl` + `S` | Save all |
| `Ctrl` + `Alt` + `Y` | Synchronize |
| `Ctrl` + `Shift` + `F12` | Minimize or maximize the editor |
| `Alt` + `Shift` + `F` | Add to favorites |
| `Alt` + `Shift` + `I` | Inspect the current file with the current profile |
| `Ctrl` + `\` | Quick switch the scheme |
| `Ctrl` + `Alt` + `S` | Open the settings dialog |
| `Ctrl` + `Alt` + `Shift` + `S` | Open the project structure dialog |
| `Ctrl` + `Tab` | Switch between tabs and tool windows |

Assistant: This is a reference script, where the file name, compression password and compressed package name can be modified according to the actual situation: #!/bin/bash # file name FILE="example.txt" # compression password PASSWORD="123456" # Compressed package name TAR_FILE="example.tar.gz" ZIP_FILE="example.zip" # Compressed into .tar.gz format tar zcvf $TAR_FILE --password=$PASSWORD $FILE # Compressed into .zip format zip -er $ZIP_FILE $FILE -P $PASSWORD echo "compression complete"

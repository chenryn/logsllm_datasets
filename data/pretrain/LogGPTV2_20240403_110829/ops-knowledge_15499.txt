User: Generate a backup script implemented using a Bash script to regularly back up files in a certain directory.
Assistant: #!/bin/bash # Set the backup directory backup_dir="/home/user/backup" # Set the directory to be backed up source_dir="/home/user/data" # Create a backup directory mkdir -p $backup_dir # Format the current date So as the backup directory name date=$(date +"%Y-%m-%d_%H-%M-%S") # Execute backup tar -czf "$backup_dir/$date.tar.gz" $source_dir # Delete old backups (only last 7 days) find $backup_dir -type f -mtime +7 -exec rm {} \; echo "Backup complete!"

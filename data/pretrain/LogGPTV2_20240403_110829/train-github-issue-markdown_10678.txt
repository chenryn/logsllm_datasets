# Sample
    class Foo {
        something;
        constructor(public bar /*rename bar*/) {
            this.something = bar;
        }
    }
Rename `bar` to `foo` and you **expect** :
    class Foo {
        something;
        constructor(public foo /*rename bar*/) {
            this.something = foo;
        }
    }
But you get:
    class Foo {
        something;
        constructor(public foo /*rename bar*/) {
            this.something = bar; // Error here
        }
    }
Error reproduce-able on playground. Fortunately we get an error on usages ;)

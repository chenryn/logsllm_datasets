#### 标题
**Linux 不需要主宰桌面市场**

#### 正文
Linus Torvalds 最近发布了 Linux 3.17 rc-2，这偏离了他正常的发布时间表，因为8月25日是 Linux 的第23个生日。在 Linux 的第一次发布中，Torvalds 写道：“大家好，你还在使用 Minix 吗？” 这一发布标志着 Linux 诞生的开始。

与此同时，PCMag.com 网站最近声称 Linux 的时间已经不多了。然而，这种无休止地讨论 Linux 是否能在桌面端取得成功其实毫无意义。Linux 已经广泛应用于超级计算机和车载系统，构建了 Android 的基础，并且是流行的云平台（如 OpenStack）的主要运行环境。这些仅仅是 Linux 成就的一部分，而桌面并不是其唯一的战场。

Jon Buys 在他的文章《专业化与 Linux 桌面》中提到：
> “最近，IT 业在追问‘Linus 是否还在执著于 Linux 桌面？’，Tech Republic 的 Matt Asay 也在说‘拜托不要再讨论 Linux 桌面了行吗？’。这两篇文章都对 Linux 在个人计算机方面的发展空间持怀疑态度，并以 Android 的成功故事为例…… 但是它们忽略了 Linux 的灵活性及其开源许可证的开放性，也许这两者正是拯救 Linux 桌面的关键。”

尽管如此，对于大多数桌面用户来说，Linux 可能仍然显得多余。Linux 分享庞大市场的最佳机会可能已经错过。事实上，Linux 已经改变了世界，取得了无可比拟的成功——除了在桌面系统领域。Android 不仅仅是一个基于 Linux 的平台，它已经成为了一个伟大的标志。Linux 在服务器和嵌入式技术领域占据巨大份额，并为平台整合提供了创新动力。Ubuntu 是部署 OpenStack 的最流行平台之一。全球的超级计算机都在运行 Linux，Chrome OS 也是基于 Linux 构建的。

因此，Linux 正在推动整个世界的变革。批评家们应该停止过分关注 Linux 在桌面端的表现。

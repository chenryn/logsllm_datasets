### 黑掉微软定制的 Linux 操作系统，微软将支付最高 10 万美元

去年，微软为其 Azure Sphere 系统开发了一个紧凑型定制版 Linux。该系统专为物联网（IoT）平台设计，可在专用芯片上运行，其基本服务和应用程序在沙盒环境中隔离运行，以确保安全性。目前，微软希望黑客们测试 Azure Sphere 的安全性，并承诺如果 Pluton 安全子系统或 Secure World 沙箱被攻破，将支付最高 10 万美元的奖励。

**来源**：[cnBeta.COM](https://www.cnbeta.com/articles/tech/975693.htm)

**硬核老王点评**：对于 IoT 系统，安全漏洞带来的危害更大，因此必须高度重视。

### GitHub 宣布基于云端托管的 Codespaces 集成开发环境

GitHub Codespaces 是基于微软的 VS Code 编辑器打造的集成开发环境（IDE），改名前称为 VS Code Online。两者采用了相同的概念和技术，但 GitHub Codespaces 直接集成了现有的 GitHub 服务。该服务现正处于免费公测阶段，正式上线后的定价尚未公布，但预计与 GitHub Actions 类似。

**来源**：[cnBeta.COM](https://www.cnbeta.com/articles/tech/975803.htm)

**硬核老王点评**：被微软收购后的 GitHub，在收费服务方面动作频频，对原来的一些核心收费服务进行了免费化处理，同时增加了新的收费服务。

### Ubuntu 20.10 将为 ZFS 提供更多改进

ZFS 将是 Ubuntu 20.10 开发工作的重点之一，可能会推出加密的 Ubuntu ZFS 安装选项。早在 Ubuntu 19.10 中，开发团队就已将 ZFS（OpenZFS/ZFSOnLinux）的实验性根文件系统添加到桌面 Ubiquity 安装程序中。到了 Ubuntu 20.04 LTS，虽然使用 ZFS 作为根文件系统仍然是一个实验性选项，但已经进行了一些改进。

**来源**：[开源中国](https://www.oschina.net/news/115433/ubuntu-20-10-more-zfs-improvement)

**硬核老王点评**：与 ext 文件系统相比，ZFS 近年来获得的关注和技术进步似乎更多。

### 谷歌携手盖茨基金会为发展中国家开发开源数字支付服务 Mojaloop

谷歌与比尔和梅林达·盖茨基金会等组织合作成立了非营利机构 Mojaloop 基金会，致力于为发展中国家开发开源实时数字支付服务。通过让储蓄变得更加容易、小企业更加高效，Mojaloop 旨在帮助更多人摆脱贫困。基于 Mojaloop 的国家支付系统可以连接不同的银行和支付系统，使数字钱包、传统银行账户或汇款服务无缝互动，从而提升现有支付服务的价值。

**来源**：[网易科技](https://www.cnbeta.com/articles/tech/975763.htm)

### Sailfish OS 3.3 “Rokua” 发布

Sailfish OS（中文：旗鱼系统）是一款开源操作系统，其核心基于 Mer 项目。Mer 是一个针对移动设备优化的开源 Linux 发行版，同时也是 MeeGo 的分支。当 MeeGo 计划终止后，社区开发者从中分支出 Mer 项目，诺基亚前员工成立的 Jolla 公司又从 Mer 分支出了 Sailfish OS。

**来源**：[开源中国](https://www.oschina.net/news/115428/sailfish-os-3-3-rokua-released)

**硬核老王点评**：在 iOS 和 Android 二分天下的移动操作系统市场中，旗鱼系统仍然被寄予厚望。

### 成人网站泄露 108 亿条数据

成人网站 CAM4 因错误配置的 ElasticSearch 数据库泄露了 108 亿条数据，包括姓名、性取向、支付记录、电子邮件和聊天记录，总容量达到 7 TB。尽管没有证据表明 CAM4 遭受黑客攻击或数据库被恶意访问，但由于数据库配置错误，这种风险仍然存在。

**来源**：[solidot](https://www.solidot.org/story?sid=64284)

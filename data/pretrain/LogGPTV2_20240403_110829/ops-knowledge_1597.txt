User: In the Linux system, use the command line to set the IP address, subnet mask, gateway, and DNS server.
Assistant: 1. Set the IP address Use the ifconfig command to set the IP address of the network interface. The command format is: ``` sudo ifconfig [network interface name] [IP address to be set] netmask [subnet mask] ``` For example, to set the network The IP address of interface eth0 is 192.168.0.100, and the subnet mask is 255.255.255.0. The command is as follows: ``` sudo ifconfig eth0 192.168.0.100 netmask 255.255.255.0 ``` 2. Set the gateway Use the route command to set the default gateway, command The format is: ``` sudo route add default gw [IP address of gateway] [network interface name] ``` For example, to set the default gateway to 192.168.0.1, the command is as follows: ``` sudo route add default gw 192.168.0.1 eth0 ``` 3. Set the DNS server You can set the DNS server by modifying the /etc/resolv.conf file, the command is as follows: ``` sudo vim /etc/resolv.conf ``` Add the following content to the file: ``` nameserver [IP address of the DNS server] ``` For example, to set the DNS server to 8.8.8.8 and 8.8.4.4, the command is as follows: ``` sudo vim /etc/resolv.conf ``` Add in the file: ``` nameserver 8.8.8.8 nameserver 8.8.4.4 ```

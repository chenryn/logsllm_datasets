### 永远不会被应用
**对策：**
- 系统应当使用原子操作。
- 操作系统可以使用软件锁。

### 跨站脚本攻击 (XSS)
**定义：**
跨站脚本攻击 (Cross-Site Scripting, XSS) 是一种利用 Web 应用程序的漏洞，注入恶意代码（如 JavaScript）的攻击。当不知情的用户访问该网站时，恶意代码会在其浏览器中执行，可能导致以下后果：
- Cookies 被盗
- 会话被劫持
- 恶意代码被执行
- 访问控制被绕过
- 利用浏览器的漏洞

**XSS 攻击类型：**
1. **非永久/非持久性 XSS (Nonpersistent XSS)**
   - 也称为反射型 XSS 或反射脆弱性。
   - 攻击者通过欺骗受害者点击一个包含恶意脚本的 URL 来实施攻击。
   - 原理是利用动态网站上缺乏适当的输入或输出验证。

2. **永久/持久性 XSS (Persistent XSS)**
   - 也称为存储型 XSS 或二阶漏洞。
   - 攻击者将恶意代码存储在数据库或其他地方（如论坛、留言板），其他用户浏览这些内容时会触发恶意代码。

3. **基于文档对象模型 (DOM) 的 XSS**
   - 也称为本地跨站点脚本。
   - 攻击者利用 DOM 环境修改客户端 JavaScript，使受害者的浏览器执行恶意代码。

### SQL 注入攻击
**定义：**
SQL 注入攻击是一种通过向 Web 应用程序输入特殊字符来操纵数据库查询的技术。攻击者可以通过这种方式访问数据库中的敏感信息。

**示例：**
- 正确的密码输入是 "123456"。
- 如果攻击者输入 "123456' OR 'a' = 'a"，数据库可能会执行两条 SQL 语句，从而绕过密码验证。

**防范措施：**
1. **输入验证：**
   - 验证用户输入的数据类型，限制非法输入。
2. **限制用户特权：**
   - 使用权限最小化的数据库账户。
3. **利用数据库存储过程：**
   - 将 SQL 语句封装在存储过程中，仅允许调用存储过程而不是直接执行 SQL 语句。

### 侦察攻击/网络侦察扫描
**目的：**
- 找出 IP 地址、端口和漏洞。
- 渗透测试的基本技能。
- 常用工具包括 Nmap、Nessus 和 OpenVAS。

### 伪装攻击
**类型：**
1. **IP 欺骗 (IP Spoofing)**
   - 防范方法是在网络边缘配置过滤程序，确保数据包满足以下标准：
     - 内部 IP 地址的包不能从外网进入。
     - 外部 IP 地址的包不能从内网出去。
     - 私有 IP 地址的包不能过路由器。

2. **会话劫持 (Session Hijacking)**
   - 攻击者中途拦截已授权用户与资源之间的通信数据，并接管会话。
   - 具体方式包括：
     - 捕获身份验证详细信息并伪装成客户端。
     - 欺骗客户端使其认为攻击者是服务器，并作为中间人断开合法连接。
     - 使用未正常关闭连接的用户的 cookie 数据访问 Web 应用程序。

### 内存重用（客体重用）
**描述：**
- 内存被重新分配后，应清空被覆写的数据以防止信息泄露。

### CISSP 认证考试大纲
**关于 CISSP：**
- CISSP（注册信息系统安全师）是全球广泛认可的信息安全认证。
- 持证者具备设计、构建及管理组织整体安全态势所需的知识和经验。
- CISSP 的公共知识体系 (CBK) 包含八大知识域：
  - 安全与风险管理
  - 资产安全
  - 安全架构与工程
  - 通信与网络安全
  - 身份与访问管理
  - 安全评估与测试
  - 安全运营
  - 软件开发安全

**经验要求：**
- 至少 5 年全职工作经验，涵盖 CBK 中的至少两个领域。
- 拥有 4 年大学本科学历或同等学历以及 (ISC)² 认可的证书可以抵免一年工作经验。
- 不满足工作经验要求的考生可以通过 CISSP 考试成为 (ISC)² 准会员，并在接下来的 6 年内积累所需经验。

**认可：**
- CISSP 符合 ANSI/ISO/IEC 17024 国际标准。

**工作任务分析 (JTA)：**
- (ISC)² 定期进行 JTA 以确定 CISSP 安全专业人士的工作内容，并更新考试内容。

**CISSP 计算机自适应测试 (CAT) 信息：**
- **考试时长：** 3 小时
- **考题数量：** 100 至 150 道
- **考题格式：** 多项选择和高级创新型考题
- **及格线：** 1000 分中得到 700 分
- **考试语言：** 英语
- **测试中心：** (ISC)² 授权的 Pearson VUE 测试中心

**CISSP 线性考试信息：**
- **考试时长：** 6 小时
- **考题数量：** 250 道
- **考题格式：** 多项选择和高级创新型考题
- **及格线：** 1000 分中得到 700 分
- **考试语言：** 法语、德语、巴西葡萄牙语、西班牙语、日语、简体中文、韩语
- **测试中心：** (ISC)² 授权的 Pearson VUE 测试中心

1. 安全与风险管理：15%

**别名**：Naikon  
**创建日期**：2017-05-31T21:31:54.232Z  
**描述**：[Naikon](https://attack.mitre.org/groups/G0019) 被认为是一个由中国人民解放军成都军区第二技术侦察局（军事单位代号78020）支持的网络间谍组织。(引用来源：CameraShy) 自2010年以来，[Naikon](https://attack.mitre.org/groups/G0019) 主要针对东南亚地区的政府、军事和民间组织进行攻击，同时也对国际机构如联合国开发计划署（UNDP）和东南亚国家联盟（ASEAN）进行了攻击。(引用来源：CameraShy, Baumgartner Naikon 2015)

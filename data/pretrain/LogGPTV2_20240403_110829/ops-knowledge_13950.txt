User: What are the main procedures an organization should follow in order to implement CIS Control 17.9, Establish and Maintain Security Incident Thresholds?
Assistant: An organization should Establish and maintain security incident thresholds, including, at a minimum, differentiating between an incident and an event. Examples can include: abnormal activity, security vulnerability, security weakness, data breach, privacy incident, etc. Review annually, or when significant enterprise changes occur that could impact this Safeguard.CIS Control 17.9 has the asset type N/A, and has the security function Recover.

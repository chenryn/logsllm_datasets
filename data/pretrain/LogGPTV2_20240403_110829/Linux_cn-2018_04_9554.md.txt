让我们回顾一下自上一个长期支持版本（LTS）以来，Ubuntu 在这两年间发生了哪些变化。

![](/data/attachment/album/201804/17/221345uv1pcuz51ivg355g.jpg)

Canonical 计划在本月26日发布 Ubuntu 18.04 LTS（Bionic Beaver）操作系统。这是一个长期支持版本，大多数使用 Ubuntu 16.04 LTS（Xenial Xerus）的用户应考虑升级。Ubuntu 16.04 LTS 系列于2016年4月21日发布，已进行了五个计划维护更新中的四个，最后一个更新 Ubuntu 16.04.4 LTS 发布于2018年3月1日。

Ubuntu 16.04.4 LTS 包含了来自 Ubuntu 17.10（Artful Aardvark）的最新内核和显卡驱动程序，相比最初的 Ubuntu 16.04 LTS 有了显著改进。作为 Ubuntu 16.04 LTS 的用户，你可能想知道是否值得升级到即将发布的 Ubuntu 18.04 LTS。

![Ubuntu 18.04 LTS 应用](/data/attachment/album/201804/17/221349nirzqo6iqmze070r.jpg "Ubuntu 18.04 LTS 应用")

### 从 Ubuntu 16.04 LTS 到 Ubuntu 18.04 LTS 的主要变化

从 Ubuntu 16.04 LTS 到 Ubuntu 18.04 LTS 最显著的变化是 Unity 用户界面被 GNOME 桌面环境取代。尽管开发者们尽力使其看起来像 Unity，但 GNOME 桌面环境带来了许多视觉上的变化。例如，窗口控制按钮现在位于右侧，这是自 Ubuntu 9.10 （Karmic Koala）以来首次如此设置。

另一个视觉变化体现在 Nautilus 文件管理器中，它现在具有暗色风格的左侧边栏。此外，LightDM 登录管理器被替换为 GNOME 的 GDM 显示管理器，并且默认虚拟终端从 1 号改为 7 号。

Ubuntu 18.04 LTS 还使用 GNOME 屏幕虚拟键盘替代了默认的 Onboard 键盘。预装在 ISO 镜像中的应用程序也有所不同，其中包括 GNOME To Do。对于那些查看 systemd 日志的用户，“GNOME 日志”替换了原来的系统日志应用；gconf 默认不再安装，取而代之的是 gsettings。从 Ubuntu 16.04 LTS 升级时，Aisleriot 卡牌游戏的记录和设置将丢失。

其他许多 GNOME 家族的应用程序也经历了重大的用户界面重新设计，包括“Ubuntu 软件”、“系统设置”、“磁盘用量分析（Baobab）”、Remmina 和“GNOME 日历”，后者现在支持周视图和重复事件。这些应用程序大多在其菜单中提供了键盘快捷键弹窗。

![Ubuntu 18.04 LTS 应用](/data/attachment/album/201804/17/221350ttty0rey4aae02uq.jpg "Ubuntu 18.04 LTS 应用")

### 其他重要变化

随着 GNOME 成为 Ubuntu 的默认桌面环境，Ubuntu GNOME 分支不再继续开发。Ubuntu GNOME 16.04 用户将自动升级到标准的 Ubuntu 版本。如果你安装了 gnome-session 并在登录屏幕选择了 GNOME，仍然可以使用原生的 GNOME 会话。

值得注意的是，Canonical 放弃了对 Ubuntu 18.04 LTS（Bionic Beaver）32 位桌面的支持。因此，如果你使用的是 32 位 Ubuntu 16.04 LTS 操作系统，建议下载 64 位 Ubuntu 18.04 LTS 镜像并重新安装（前提是你的机器支持 64 位）。

在底层方面，默认安装的是 Python 3，Python 2 不再预装；GPG 二进制文件现在由 GnuPG2 提供；全新安装的 Ubuntu 系统中用交换文件代替了交换分区；默认 DNS 解析器是 systemd-resolved；图形化安装程序中的“加密家目录”选项已被移除。

Ubuntu 18.04 LTS（Bionic Beaver）是 Canonical 的第七个长期支持版本，将获得五年的安全和软件更新支持，直到 2023 年 4 月。最终版的 Ubuntu 18.04 LTS 将很快可供下载，Ubuntu 16.04 LTS 和 Ubuntu 17.10 用户可以在 2018 年 4 月 26 日准备升级。

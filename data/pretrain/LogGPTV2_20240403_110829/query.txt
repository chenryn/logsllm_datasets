### Query Commands

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `query` commands provide information about processes, sessions, and Remote Desktop Session Host (RDSH) servers. For details on new features in the latest version, refer to [What's New in Remote Desktop Services in Windows Server](#).

### Syntax

```plaintext
query process
query session
query termserver
query user
```

### Parameters

| Command          | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| `query process`  | Displays information about processes running on a Remote Desktop Session Host server. |
| `query session`  | Displays information about sessions on a Remote Desktop Session Host server. |
| `query termserver` | Lists all Remote Desktop Session Host servers on the network.               |
| `query user`     | Displays information about user sessions on a Remote Desktop Session Host server. |

This structure provides a clear and organized overview of the `query` commands, their applicability, and the specific information each command retrieves.

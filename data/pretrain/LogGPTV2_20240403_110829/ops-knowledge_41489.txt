User: How to install and configure Nginx as a web server on CentOS 7?
Assistant: The following are the steps to install and configure Nginx on CentOS 7: 1. Install Nginx Execute the following command: ``` sudo yum install nginx ``` 2. Start Nginx Execute the following command: ``` sudo systemctl start nginx ``` 3 . To set Nginx to start at boot, execute the following command: ``` sudo systemctl enable nginx ``` 4. Configure Nginx The main configuration file of Nginx is located in `/etc/nginx/nginx.conf`. You can edit this file to change Nginx's configuration. For example, you can change the port that Nginx listens on: ``` # Modify the default port number http { ... server { listen 8080 default_server; listen [::]:8080 default_server; server_name _; ... } ... } ` `` Run the following command to reload the Nginx configuration: ``` sudo systemctl reload nginx ``` 5. Firewall Settings You need to open the firewall to allow HTTP and HTTPS traffic. Execute the following command: ``` sudo firewall-cmd --permanent --zone=public --add-service=http sudo firewall-cmd --permanent --zone=public --add-service=https sudo firewall-cmd - -reload ``` Now you have successfully installed and configured Nginx as a web server. You can enter your server's IP address and port number, such as `http://server_ip:8080`, in your browser to access your web application.

User: What are the main procedures an organization should follow in order to implement CIS Control 16.7, Use Standard Hardening Configuration Templates for Application Infrastructure?
Assistant: An organization should Use standard, industry-recommended hardening configuration templates for application infrastructure components. This includes underlying servers, databases, and web servers, and applies to cloud containers, Platform as a Service (PaaS) components, and SaaS components. Do not allow in-house developed software to weaken configuration hardening.CIS Control 16.7 has the asset type Applications, and has the security function Protect.

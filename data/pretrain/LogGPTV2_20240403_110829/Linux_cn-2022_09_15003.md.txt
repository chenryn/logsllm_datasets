![Introduce the different Fedora Linux editions](/data/attachment/album/202209/05/224754mqk5p3zqz5epfcf5.png)

Fedora Linux 用户有不同的偏好。例如，一些用户选择 Fedora Workstation，因为它默认使用 GNOME 作为桌面环境。然而，有些用户希望使用其他桌面环境，或者有特定需求但不想被系统配置和应用安装所困扰。因此，Fedora Linux 提供了多个版本以满足这些需求。本文将介绍这些不同的 Fedora Linux 版本。

### 官方 Fedora 版本

首先，我们来看一下 Fedora Linux 的官方版本：Fedora Workstation、Fedora Server 和 Fedora IoT。

- **Fedora Workstation**：适用于笔记本电脑和台式机，预装 GNOME 桌面环境和各种标准应用，适合日常使用。
- **Fedora Server**：专为服务器用途设计，提供邮件服务器、DNS 等功能的安装。
- **Fedora IoT**：用于物联网和边缘设备生态系统。

在 Fedora 项目网站主页上，你还可以找到以下两个版本：

- **Fedora CoreOS**：一个自动更新的操作系统，旨在安全、大规模地运行容器化工作负载。
- **Fedora Silverblue**：一个不可变的桌面操作系统，支持以容器为中心的工作流。

### Fedora 定制版：可选桌面

对于那些非常在意桌面外观的用户，Fedora 定制版（Spins）是一个很好的选择。虽然 Fedora Linux 默认使用 GNOME，但也提供了多种替代桌面环境，如 KDE Plasma、XFCE、LXQt、MATE、Cinnamon、LXDE 和 SoaS。此外，还有 Fedora i3 定制版，预装 i3 窗口管理器和几个标准应用。

### Fedora 实验室：功能包

Fedora 实验室（Labs）是根据特定需求打包的 Fedora Linux 软件包集合。这些版本包含针对特定功能的应用和内容。Fedora 实验室提供多种软件包选择，包括天文学、计算神经学、设计套件、游戏、JAM、Python 教室、安全实验室、机器人套件和科学等。

- **设计套件**：适用于设计工作的工具集。
- **游戏版**：适用于游戏爱好者的工具集。

### 其他 Fedora 下载选项

Fedora 还提供了一些特定目的的可选安装程序，例如测试版或特定架构的安装程序。此外，还提供了网络安装程序、种子下载、可选架构、云基础镜像、所有内容、测试镜像和 Rawhide 等格式。

更多信息请参见 [此处](#)。

### 总结

你可以自由选择适合你偏好的 Fedora Linux 版本。如果你喜欢各种桌面环境，可以选择 Fedora 定制版；如果你需要特定的应用和软件包，可以选择 Fedora 实验室；如果你是专家并且希望更灵活地安装 Fedora Linux，可以考虑其他下载选项。希望本文能帮助你选择合适的 Fedora Linux 版本，并欢迎在评论中分享你的使用经验。

#### 微软在其云服务上打造全新超算

微软与 OpenAI 合作，在 Azure 上构建了一台新的超级计算机，拥有超过 28.5 万个 CPU 核心和 1 万个 GPU。根据世界超级计算机 TOP500 的排名，这台新型超级计算机位列前五名。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/981083.htm)

> **硬核老王点评**：国内的云服务商是否也会基于云服务器打造类似的超级计算机呢？

#### WSL 2 计划支持 Linux GUI 应用，可与 Windows 应用同时使用

微软计划在本月底为 WSL 2 添加完整的 Linux 内核，并进一步支持 Linux 的 GUI 应用，使其能够与常规 Windows 应用一起运行。目前，用户可以通过第三方 X 显示服务器在 Windows 中运行 Linux GUI 应用程序，但性能表现不佳。一旦该功能上线，Windows 用户将能够在不使用 X11 转发的情况下直接启用 Linux GUI 应用程序，与 Windows 应用一同运行。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/981187.htm)

> **硬核老王点评**：未来在 Windows 上可能会出现越来越多的 Linux 应用，无论是命令行还是图形界面的应用。或许有一天，连内核都会换成 Linux 内核，而用户可能不会察觉到这种变化。欢迎 Windows “主题”上线。

#### Windows 10 终于迎来了一款软件包管理器

微软宣布为 Windows 10 引入了一款内置的软件包管理器。这款工具使得用户无需频繁点击窗口即可轻松安装桌面程序。Windows 软件包管理器已支持以下命令：
- `install`：安装应用程序
- `show`：显示有关应用程序的信息
- `source`：管理应用程序的可用来源
- `search`：查找应用程序的基本信息

感兴趣的朋友可以加入 Windows Insider 测试项目，并通过 Microsoft Store 获取 Windows 软件包管理器。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/981255.htm)

> **硬核老王点评**：包管理器在 Linux 系统中已经司空见惯，现在终于在 Windows 中出现了。这应该是微软拥抱开源和 Linux，并在系统中嵌入 WSL 的结果。

# Steps to reproduce
Inconstant behavior, periodically starts and stops with no clear trigger.
# Expected behavior
PTR prompt appears when its key command is typed, currently the default Alt +
Space.
# Actual behavior
Every so often PTR will trigger on any press of the space bar; as if Alt was
already being held down.

User: How to Backup and Restore MySQL Database Using Linux Command Line Tools? Please provide example commands.
Assistant: Backup the MySQL database: 1. Use the mysqldump command to back up the entire database ``` mysqldump -u root -p dbname > backup.sql ``` Among them, the -u parameter specifies the user name, the -p parameter specifies the password, dbname specifies the database name, > Save the backup to the backup.sql file. 2. Back up the specified table ``` mysqldump -u root -p dbname table1 table2 > backup.sql ``` table1 and table2 specify the name of the table to be backed up, and the backup will be saved in the backup.sql file. To restore the MySQL database: 1. Use the mysql command to restore the backup ``` mysql -u root -p dbname < backup.sql ``` Among them, the -u parameter specifies the user name, the -p parameter specifies the password, and dbname specifies the name of the database to be restored ,< Read data from the backup file backup.sql. 2. Restore the specified table ``` mysql -u root -p dbname < backup.sql table1 ``` table1 specifies the name of the table to be restored, and only table1 in the backup.sql file will be restored.

**使用 `mvexpand` 命令扩展多值字段的示例**

假设我们有一个事件数据集，其中包含一个名为 `iplist` 的多值字段，该字段存储了多个 IP 地址。我们可以使用 `mvexpand` 命令将这些多值字段展开为单独的事件。

**示例 SPL 语句：**

```spl
... | mvexpand iplist limit=100
```

**解释：**

- `...` 表示前面的搜索或处理步骤。
- `mvexpand iplist` 将 `iplist` 字段中的每个值拆分成独立的事件。
- `limit=100` 参数限制了每个事件最多展开 100 个值。如果 `iplist` 中的值超过 100 个，则只会展开前 100 个值。

**应用场景：**

- 当您需要对多值字段中的每个值进行单独分析时，`mvexpand` 非常有用。例如，如果您需要统计每个 IP 地址的访问次数，可以先使用 `mvexpand` 展开 `iplist`，然后再进行统计。

**完整示例：**

假设我们有以下原始数据：

```plaintext
_event_
{
  "iplist": ["192.168.1.1", "192.168.1.2", "192.168.1.3", ...]
}
```

使用 `mvexpand` 后，数据将被展开为：

```plaintext
_event_
{
  "iplist": "192.168.1.1"
}

_event_
{
  "iplist": "192.168.1.2"
}

_event_
{
  "iplist": "192.168.1.3"
}

...
```

这样，您可以对每个 IP 地址进行单独的分析和处理。

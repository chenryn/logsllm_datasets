### Python 和 Go 成为年度最受欢迎的黑客工具榜首
网络安全公司 Imperva Cloud WAF 在对数据进行聚类分析后得出结论：按工具分类，Python 依然是大多数黑客首选的语言；其次是快速增长中的 Go 语言；再之后是 Windows 平台上常用的 WinHttp 库（主要用于 .NET 和 C++ 开发）；Shell 工具如 cURL、wget 等也进入了榜单。

*资料来源:* [开源中国](https://www.oschina.net/news/116375/python-and-go-most-popular-hacking-tools)

> **硬核老王点评**: 因此，Python 不仅是一个优秀的编程语言，也是黑客们钟爱的选择。

### 黑客冒充 SpaceX 频道诈骗 两天获利价值 150 万美元的比特币
近期发生了一起涉及三个 YouTube 频道被黑客劫持并冒充 Elon Musk 的 SpaceX 官方频道实施比特币诈骗的事件。这些频道目前正播放着马斯克参与的采访或 SpaceX 会议视频，并宣传一种骗局，诱使人们发送小额比特币以换取双倍回报。截至目前已收到 29 笔交易，总计 4.08 BTC。

*资料来源:* [雷锋网](https://www.cnbeta.com/articles/tech/990151.htm)

> **硬核老王点评**: 总有人愿意为此类明显可疑的提议买单。

### BellSoft 与 VMware 合作改进 OpenJDK
根据 Snyk 在 2020 年的一项调查结果显示，仅有 34% 的 Java 用户继续使用 Oracle JDK，而选择 OpenJDK 的比例达到了 53%，相比前一年增长了 36%。此外，80% 的 Oracle JDK 用户正在考虑转向其他支持选项。作为 OpenJDK 的主要贡献者之一，BellSoft 将为其旗舰产品 Liberica JDK 提供全面的服务支持，特别是针对 VMware Tanzu 的优化。此次合作的重点在于加强 ARM 处理器的支持以及针对云部署和微服务架构的 Java 优化。

*资料来源:* [开源中国](https://www.oschina.net/news/116376/bellsof-vmware-openjdk-evolution)

> **硬核老王点评**: 这标志着开源解决方案再次战胜了专有技术。

### .NET 5.0 不再内置 WinRT 支持，可精简六万行代码
微软宣布将在即将发布的 .NET 5.0 中移除对 WinRT 的直接支持。此举基于以下几点考量：
1. 从 .NET runtime 中分离出 WinRT 互操作性功能以便于独立开发和迭代；
2. 让 WinRT 的互操作性与其他操作系统如 iOS 和 Android 上实现的功能保持一致；
3. 利用更多 .NET 特性（例如 AOT 编译、C# 新特性及 IL 链接）；
4. 简化 .NET runtime 的代码库。开发者仍可通过 C#/WinRT 工具链来获得 WinRT 支持。

*资料来源:* [cnBeta.COM](https://www.cnbeta.com/articles/tech/990023.htm)

> **硬核老王点评**: 的确，.NET 框架长期以来一直显得过于臃肿。

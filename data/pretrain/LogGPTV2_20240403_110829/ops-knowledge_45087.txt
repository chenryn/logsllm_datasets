User: How to install and configure the Apache Tomcat server in the Linux system to run Java Web applications?
Assistant: The steps to install the Apache Tomcat server are as follows: 1. Install Java: Tomcat is developed based on Java, you need to install Java first, you can use the following command to install: ```shell sudo apt-get update # update source sudo apt-get install - y default-jdk # Install JDK ``` After the installation is complete, you can use the following command to verify whether Java is successfully installed: ```shell javac -version # Check the Java version``` 2. Download and unzip Tomcat: go to Tomcat official website ( https://tomcat.apache.org/) to download the latest version of Tomcat, and then extract it to the /usr/local directory, the command is as follows: ```shell cd /usr/local sudo wget https://mirrors.bfsu.edu .cn/apache/tomcat/tomcat-9/v9.0.50/bin/apache-tomcat-9.0.50.tar.gz # Download Tomcat sudo tar -xzvf apache-tomcat-9.0.50.tar.gz # Unzip sudo mv apache-tomcat-9.0.50 tomcat # Rename to tomcat ``` 3. Set environment variables: In order to use Tomcat conveniently, you need to set environment variables, the command is as follows: ```shell echo "export CATALINA_HOME=/usr/local /tomcat" >> ~/.bashrc source ~/.bashrc ``` 4. Start Tomcat: Execute the following command to start Tomcat: ```shell cd /usr/local/tomcat/bin sudo sh startup.sh ``` After the execution is successful, you can visit http://localhost:8080/ through the browser to verify whether the startup is successful. After the above steps are completed, the Java Web application can be deployed to the Tomcat server. The deployment method can be by directly copying the WAR package to the /usr/local/tomcat/webapps directory, or by configuring the conf/server.xml file to specify the deployment directory of the application.

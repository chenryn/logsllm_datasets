Assistant: The following are the detailed steps and operation guide for deploying MySQL database in Linux environment: 1. Install MySQL Enter the following command in the terminal: ``` sudo apt-get update sudo apt-get install mysql-server ``` During the installation process, You will be prompted to create a password for the root user. Please remember this password for future login to the MySQL database. 2. Configure MySQL After the installation is complete, enter the MySQL security configuration and set the root password: ``` sudo mysql_secure_installation ``` Answer the questions according to the prompts and create the root password. 3. Log in to MySQL Use the following command to log in to MySQL: ``` sudo mysql -u root -p ``` Enter the root password you just set. 4. Create a database In MySQL, you need to create a database to store data. Use the following command to create a new database: ``` CREATE DATABASE new_database; ``` Replace "new_database" with your own database name. 5. Create User Create a new user and assign it database privileges: ``` CREATE USER 'new_user'@'localhost' IDENTIFIED BY 'password'; GRANT ALL PRIVILEGES ON new_database.* TO 'new_user'@'localhost'; ` `` Replace "new_user" and "password" with your own username and password, and in the second line replace "new_database" with the database name you created in step 4. 6. Exit MySQL After completing the operation, use the following command to exit MySQL: ``` exit ``` Now you have successfully deployed the MySQL database in the Linux environment and created a new database and user.

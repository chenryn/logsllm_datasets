## 项目构建

- **[Maven](https://maven.apache.org/)**：一个基于项目对象模型 (Project Object Model, POM) 的软件项目管理和构建工具。Maven 可以通过中心信息管理项目的构建、报告和文档生成。更多详情请参阅 [Maven 核心概念总结](https://javaguide.cn/tools/maven/maven-core-concepts.html)。
- **[Gradle](https://gradle.org/)**：一款灵活的开源构建自动化工具，适用于几乎所有类型的软件项目。Gradle 对于要构建的内容或方式没有过多限制，这使其非常灵活。更多详情请参阅 [Gradle 核心概念总结](https://javaguide.cn/tools/gradle/gradle-core-concepts.html)。

## 反编译工具

- **[JADX](https://github.com/skylot/jadx)**：一种命令行及图形界面工具，用于从 Android Dex 和 Apk 文件中提取 Java 源代码。
- **[JD-GUI](https://github.com/java-decompiler/jd-gui)**：独立的图形界面工具，能够显示 CLASS 文件中的 Java 源代码。

## 数据库工具

### 数据库建模

- **[CHINER](https://gitee.com/robergroup/chiner)**：国产免费开源数据库建模工具，旨在丰富数据库生态系统并提供独立于特定数据库的关系模型设计平台。前身是 PDMan，定位为 PowerDesigner 的替代方案。

#### 非开源数据库建模工具（部分需付费）

- **[MySQL Workbench](https://www.mysql.com/products/workbench/)**：MySQL 官方提供的可视化工具，支持数据建模、SQL 开发、服务器配置等功能，适用于 Windows、Linux 和 macOS 平台。
- **[Navicat Data Modeler](https://www.navicat.com.cn/products/navicat-data-modeler)**：强大的数据库设计工具，帮助用户创建高质量的概念、逻辑和物理数据模型，并支持多种功能如逆向工程等。
- **[DbSchema](https://dbschema.com/)**：全面覆盖关系型和 NoSQL 数据库的数据设计与管理可视化工具。
- **[dbdiagram.io](https://dbdiagram.io/home)**：在线 ER 图绘制工具，支持通过编写自定义语言来创建数据模型，并提供版本历史记录、共享以及导出图像等功能。

### 数据库管理

- **[Chat2DB](https://github.com/alibaba/Chat2DB)**：阿里巴巴推出的智能通用数据库工具，具备自然语言生成 SQL 和性能优化能力。
- **[Beekeeper Studio](https://github.com/beekeeper-studio/beekeeper-studio)**：跨平台数据库管理工具，兼容多种数据库类型。
- **[Sequel Pro](https://github.com/sequelpro/sequelpro)**：专为 macOS 设计的 MySQL/MariaDB 管理工具。
- **[DBeaver](https://github.com/dbeaver/dbeaver)**：基于 Java 的开源数据库管理工具，广泛支持各类关系型和非关系型数据库。
- **[Kangaroo](https://gitee.com/dbkangaroo/kangaroo)**：跨平台的数据库管理系统客户端，支持 SQLite、MySQL、PostgreSQL 等。
- **[Arctype](https://arctype.com/)**：桌面版数据库查询工具，可连接多种数据库执行 SQL 查询并以图表形式展示结果。
- **[Mongood](https://github.com/RenzHoly/Mongood)**：MongoDB 的图形化管理工具，采用微软 Fluent UI 设计。

### Redis 管理

- **[Another Redis Desktop Manager](https://github.com/qishibo/AnotherRedisDesktopManager/blob/master/README.zh-CN.md)**：稳定高效的 Redis GUI 客户端。
- **[Tiny RDM](https://github.com/tiny-craft/tiny-rdm)**：现代化的 Redis 管理客户端，使用 Webview2 技术。
- **[Redis Manager](https://github.com/ngbdf/redis-manager)**：一站式 Redis 集群监控与管理平台。

## Docker 管理

- **[Portainer](https://github.com/portainer/portainer)**：易于使用的 Docker 可视化管理工具。
- **[lazydocker](https://github.com/jesseduffield/lazydocker)**：简洁直观的终端界面，适用于 docker 和 docker-compose 的管理。

## ZooKeeper 管理

- **[PrettyZoo](https://github.com/vran-dev/PrettyZoo)**：高颜值的 ZooKeeper 图形化客户端，支持 Mac、Windows 和 Linux。
- **[zktools](https://zktools.readthedocs.io/en/latest/#installing)**：低延迟且美观的 ZooKeeper GUI 客户端。

## Kafka 监控

- **[Kafka UI](https://github.com/provectus/kafka-ui)**：开源的 Kafka 集群管理和监控 Web 界面。
- **[Kafdrop](https://github.com/obsidiandynamics/kafdrop)**：查看 Kafka 主题和消费者组状态的 Web 应用。
- **[EFAK](https://github.com/smartloli/EFAK)**：高性能的 Kafka 集群监控系统。

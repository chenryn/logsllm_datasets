> GitHub Copilot 已正式推出，旨在通过人工智能帮助程序员更高效地编写代码。但是，这项技术是否真的能提升编程体验，还是可能会带来负面效果？

![GitHub](/data/attachment/album/202206/26/115204xzz8uuxcxwtdcxdw.jpg)

在 2021 年，我花了大量时间研究 GitHub Copilot 的文档，以期能够加入其技术预览计划。幸运的是，我的努力得到了回报，最终成功加入了该计划。现在，Copilot 终于向所有用户开放了！

如果你还不了解，[GitHub Copilot](https://copilot.github.com/) 是一种 AI 辅助工具，旨在帮助开发者更快、更高效地编写代码。它的功能类似于手机输入法中的自动完成功能，但不同之处在于，Copilot 可以生成完整的代码段。

### Copilot 现已全面开放

如前所述，Copilot 在过去的一年中一直处于技术预览阶段。这意味着 GitHub 只允许有限数量的开发者免费使用它，并通过监控他们的使用情况来改进程序的最终版本。

现在看来，GitHub 对 Copilot 的表现感到满意，决定将其公之于众。任何拥有 GitHub 账户的用户都可以使用它，不过需要支付一定的费用（下文将详细介绍）。

在 [官方公告](https://github.blog/2022-06-21-github-copilot-is-generally-available-to-all-developers/) 中提到：

> 直到最近，人工智能还未能显著改善代码编写过程。软件开发几乎完全是手动进行的。今天，我们很高兴地宣布，我们将向所有个人开发者提供 [GitHub Copilot](http://copilot.github.com)。你的 AI 编程伙伴已经到来。
>
> —— Thomas Dohmke, GitHub CEO

作为一款免费编辑器扩展，Copilot 已经帮助数百万开发者提高了编程速度。然而，无论是直接还是间接，使用它都是有成本的。

### GitHub Copilot 的定价

和许多令人兴奋的新技术一样，Copilot 对某些人来说可能过于昂贵。每月订阅费用为 10 美元，或每年 100 美元。如果你是开源项目的维护者或经过验证的学生，则可以免费使用它。

### GitHub Copilot 是否道德？

围绕 GitHub Copilot 产品的争议非常大。从技术角度来看，这个 AI 是通过使用托管在 GitHub 上的代码进行训练的。因此，基本上，GitHub 利用用户的代码推出了一个新产品。此外，自由软件基金会 (FSF) 也 [建议](https://www.fsf.org/blogs/licensing/fsf-funded-call-for-white-papers-on-philosophical-and-legal-questions-around-copilot) 不要在 GitHub 上托管代码。

我们知道，企业总是喜欢利用资源，但有些人认为这不应该直接损害托管在 GitHub 上的项目或代码。**真的是这样吗？**

简而言之，在 Copilot 发布后，许多开发者发现它生成了受版权保护的代码。例如，一位开发者在 Twitter 上分享了他的经历：

> 我试用了 GitHub Copilot，这是一项付费服务，看看它是否会使用带有特定许可证的存储库中的代码。我发现它确实包含了我在前雇主那里编写的代码，这些代码有一个仅允许用于免费游戏的许可证。  
> 
> ![来源：Twitter 用户 Chris Green](/data/attachment/album/202206/26/115225zb1g28y82gpcpfso.png)

当然，根据 GitHub Copilot 的常见问题解答（FAQ），其中指出：

> GitHub 不拥有 GitHub Copilot 生成的建议。你在 GitHub Copilot 的帮助下编写的代码属于你自己，由你负责。

那么，你为一项服务付费，最终却为自己的项目增加了不便和更多的工作？

在我看来，就简化开发者的任务而言，这听起来并不令人兴奋。

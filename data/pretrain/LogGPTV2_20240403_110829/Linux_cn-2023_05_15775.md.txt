### 使用游戏机来控制机枪炮塔

乌克兰士兵正在使用 Steam Deck 手持游戏机远程操作一种名为“军刀”的大口径机枪炮塔。这种武器是通过众筹资金建造的，允许士兵在远距离与敌人作战，并能处理从轻机枪到 AK-47 等多种口径的武器。

> **消息来源**: [VICE](https://www.vice.com/en/article/5d9g9z/ukraine-is-now-using-steam-decks-to-control-machine-gun-turrets)

**老王点评**: Valve 可能没想到枪战游戏会变成真人 CS 吧。

### OpenAI 对利用其 API 的开源项目提起诉讼威胁

一个名为 GPT4free 的 GitHub 项目允许用户免费使用 GPT-4 和 GPT-3.5 模型，通过 You.com 和 Quora 等网站传送请求并返回结果。该项目在 GitHub 上获得了 22.6k 星标。OpenAI 要求该项目在五天内关闭，否则将提起诉讼。项目作者认为他并未直接连接到 OpenAI 的 API，而是通过其他付费网站获取数据。如果这些网站对其脚本有意见，应直接联系他。而如果这些网站的 API 不安全，任何人都可以找到使用方法。

> **消息来源**: [Tom's Hardware](https://www.tomshardware.com/news/openai-sends-shutdown-letter-to-gpt4free)

**老王点评**: OpenAI 应该与其 API 客户合作保护 API 调用，而不是直接针对该项目。

### AI 教父离开谷歌并警告未来的危险

Geoffrey Hinton 是人工智能领域的先驱。2012 年，Hinton 博士和他的学生开发的技术成为现代 AI 系统的基础。半个世纪以来，他培育了 ChatGPT 等聊天机器人的核心技术。现在，他担心 AI 可能带来的严重危害。为了自由表达对 AI 风险的看法，他辞去了在谷歌的工作。他表示对自己一生的工作部分感到后悔，认为很难防止坏人利用 AI 做坏事。

> **消息来源**: [纽约时报](https://www.nytimes.com/2023/05/01/technology/ai-google-chatbot-engineer-quits-hinton.html)

**老王点评**: 行业顶尖领袖可能比普通大众更担忧 AI 的潜在风险。

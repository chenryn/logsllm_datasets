 **Symfony version(s) affected** : 4.1.0
**Description**  
When using the new `name_prefix` option, routes will be generate both with
prefix and without prefix.
**How to reproduce**  
In `config/routes.yaml` add a prefix for a controller folder
    admin:
        resource: ../src/Controller
        type: annotation
        name_prefix: admin_
And, by using `debug:router` command, you will see that a router named
`dashboard` will be created 2 times.
    ------------------------------------------------------ -------- -------- ------ -----------------------------------------
      Name                                                   Method   Scheme   Host   Path
     ------------------------------------------------------ -------- -------- ------ -----------------------------------------
      dashboard                                              ANY      ANY      ANY    /
      admin_dashboard_index                                  ANY      ANY      ANY    /

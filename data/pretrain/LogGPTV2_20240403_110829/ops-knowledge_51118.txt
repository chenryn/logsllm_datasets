User: 如何使用命令行监控Linux服务器的CPU、内存和磁盘使用情况？请提供相关指令。
Assistant: 1. 监控CPU： 
使用 top 命令可以查看 Linux 服务器的 CPU 使用情况。 
输入 top 命令后，可以实时查看所有进程的 CPU 使用情况，按 P 键按照 CPU 使用率排序，按 Q 键退出。 
top

2. 监控内存： 
使用 free 命令可以查看 Linux 服务器的内存使用情况。 
free -m 命令可以查看内存使用情况，以 MB 为单位。 
free -h 命令可以查看内存使用情况，以易读的容量单位为单位。 
 
3. 监控磁盘： 
使用 df 命令可以查看 Linux 服务器的磁盘使用情况。 
df -h 命令可以查看磁盘使用情况，以易读的容量单位为单位。 

以上是常见的命令行监控 Linux 服务器的方法，你也可以使用一些工具来进行监控，如 htop、atop 等。

User: How to install and manage Docker engine in Linux system? Please list commonly used commands and parameters, and explain their functions.
Assistant: Install the Docker engine: 1. First, you need to install the software packages that Docker depends on in the Linux system. - For Ubuntu system: sudo apt-get install apt-transport-https ca-certificates curl software-properties-common - For CentOS system: sudo yum install -y yum-utils device-mapper-persistent-data lvm2 2. Add Docker official GPG key: curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add - 3. Add Docker repository: sudo add-apt-repository "deb [arch=amd64] https ://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" 4. Update the repository and install the Docker engine: sudo apt-get update && sudo apt-get install docker-ce To manage the Docker engine: 1. docker version: View Docker version information. 2. docker info: View Docker system information. 3. docker ps: View running containers. 4. docker images: View the list of local images. 5. docker pull: Download the image from the Docker repository. 6. docker run: Start the Docker container. 7. docker stop: Stop the running container. 8. docker rm: delete the specified container. 9. docker rmi: delete the specified image. 10. docker logs: View the log information of the container. 11. docker exec: Execute the specified command in the container. 12. docker cp: Copy the files in the container to the local, or copy the local files to the container. 13. docker build: Build a custom image according to the Dockerfile. 14. docker-compose: Use docker-compose to manage Docker containers.

Red Hat Enterprise Linux 8
在混合云部署环境中提供稳定、安全、持续的基础，并提供更迅速地提供工作负载所需的工具。它可部署为虚拟机监控程序及云供应商环境，也可以部署到物理构架中，以便您的应用程序可在主要硬件构架平台中利用创新。
这部分包含安装 Red Hat Enterprise Linux 支持的构架和系统要求的信息。

-   IBM Power Systems, Little Endian

::: titlepage
# []{#installation-methods_standardinstalling-RHEL.html#installation-methods_standardinstalling-RHEL}第 2 章 RHEL 安装方法 {.title}
:::
您可以使用以下任一方法安装 Red Hat Enterprise Linux：
::: itemizedlist
-   基于 GUI 的安装
-   系统或云镜像安装
-   高级安装
:::
::: {.note style="margin-left: 0.5in; margin-right: 0.5in;"}
### 注意 {.title}
本文档提供有关使用用户界面(GUI)安装 RHEL 的详情。
:::
::: title
**基于 GUI 的安装**
:::
有以下基于 GUI 的安装方法可用：
::: itemizedlist
-   [**使用客户门户网站中的 ISO 镜像安装
    RHEL：通过从客户门户网站下载**]{.strong} [**DVD ISO**]{.strong}
    镜像文件安装 Red Hat Enterprise Linux。系统注册会在 GUI
    安装完成后进行。Kickstart 支持这个安装方法。
-   [**从 Content Delivery Network 注册并安装
    RHEL：**]{.strong}注册您的系统、附加订阅并从 Content Delivery
    Network（CDN）安装 Red Hat Enterprise Linux。[**引导 ISO 和 [**DVD
    ISO**]{.strong}** ]{.strong} 镜像文件支持这个安装方法，但建议您使用
    [**Boot ISO 镜像文件作为引导 ISO**]{.strong} 镜像文件的安装源默认为
    CDN。注册是在从 CDN 下载并安装软件包前执行的。Kickstart
    支持这个安装方法。
    ::: {.important style="margin-left: 0.5in; margin-right: 0.5in;"}
    ### 重要 {.title}
    您可以使用 GUI 根据特定要求自定义 RHEL
    安装。您可以选择其他选项来满足特定环境要求，例如连接到红帽、软件选择、分区、安全性等。如需更多信息，[请参阅自定义安装](#graphical-installation_graphical-installation.html "第 10 章 自定义安装"){.link}。
    :::
:::
::: title
**系统或云镜像安装**
:::
您只能在虚拟和云环境中使用系统或云镜像安装方法。
要执行系统或基于云镜像的安装，请使用 Red Hat Image
Builder（红帽镜像构建器）。镜像构建器创建 Red Hat Enterprise Linux
的自定义系统镜像，包括用于云部署的系统镜像。
有关使用镜像构建器安装 RHEL [的更多信息，请参阅[*生成自定义的 RHEL
系统镜像*]{.emphasis}文档](https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/8/html/composing_a_customized_rhel_system_image/creating-system-images-with-composer-command-line-interface_composing-a-customized-rhel-system-image){.link}。
::: title
**高级安装**
:::
可用的高级安装方法如下：
::: itemizedlist
-   [**使用 Kickstart 执行自动 RHEL 安装：**]{.strong} 使用 Kickstart
    安装 Red Hat Enterprise Linux。Kickstart
    是一个自动安装，可让您执行无人值守操作系统安装任务。
-   [**使用 VNC 执行远程 RHEL 安装：**]{.strong} RHEL 安装程序提供两种
    VNC 安装模式： Direct 和
    Connect。建立连接后，这两种模式不会有所不同。您选择的模式取决于您的环境。
-   [**使用 PXE 从网络安装 RHEL:** ]{.strong} 网络安装可让您将 Red Hat
    Enterprise Linux
    安装到可访问安装服务器的系统。网络安装至少需要两个系统。
:::
::: itemizedlist
**其它资源**
-   有关高级安装方法的详情,请查看 [[*执行高级 RHEL
    安装文档*]{.emphasis}](https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/8/html-single/performing_an_advanced_rhel_installation/index/){.link}。
:::

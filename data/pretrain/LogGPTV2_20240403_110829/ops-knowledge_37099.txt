User:  How to retrieve HTTP server header information using command line tool?
Assistant: You can use the curl command to retrieve the header information of the HTTP server. The method is as follows: 1. Open a terminal or a command line tool, and enter the following command: ``` curl -I http://www.example.com ``` Among them, http ://www.example.com is the address of the website you want to retrieve. 2. Press the Enter key and wait for the server to return the header information. 3. You will see the header information returned by the server, for example: ``` HTTP/1.1 200 OK Content-Type: text/html; charset=UTF-8 Date: Mon, 01 Jan 2020 12:00:00 GMT . .. ``` Among them, HTTP/1.1 200 OK indicates the HTTP protocol version and the status code returned by the server. Content-Type: text/html; charset=UTF-8 indicates the content type and charset returned by the server. Date: Mon, 01 Jan 2020 12:00:00 GMT represents the timestamp returned by the server. Through these header information, you can understand the specific information returned by the server, so as to better understand the website you want to retrieve.

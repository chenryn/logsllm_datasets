User: 请选择ConfigMgmt领域相关问题的正确答案及解释问题：在puppet中，我们如何通过编写manifests 判断系统类型，以实现在系统为RedHat的情况下安装mysql-server和mysql，系统为SLES的情况下安装mysql和mysql-client，其他系统安装mysql-server和mysql包，并确保安装的包一直处于安装状态？

A: ``ruby/nclass mysql {/n    package { 'mysql-server':/n        ensure =&gt; installed,/n        alias  =&gt; 'mysql',/n        provider =&gt; redhat,/n      }/n    package { 'mysql':/n        ensure =&gt; installed,/n        alias  =&gt; 'mysql',/n        provider =&gt; redhat,/n      }/n    package { 'mysql':/n        ensure =&gt; installed,/n        alias  =&gt; 'mysql',/n        provider =&gt; sles,/n      }/n    package { 'mysql-client':/n        ensure =&gt; installed,/n        alias  =&gt; 'mysql',/n        provider =&gt; sles,/n      }/n    package { 'mysql-server':/n        ensure =&gt; installed,/n        alias  =&gt; 'mysql',/n        provider =&gt; default,/n      }/n    package { 'mysql':/n        ensure =&gt; installed,/n        alias  =&gt; 'mysql',/n        provider =&gt; default,/n      }/n}/n```
B: ``ruby/nclass mysql {/n    if $facts['os']['family'] == 'RedHat' {/n        package { ['mysql-server', 'mysql']:/n            ensure =&gt; installed,/n        }/n    } elseif $facts['os']['family'] == 'SLES' {/n        package { ['mysql', 'mysql-client']:/n            ensure =&gt; installed,/n        }/n    } else {/n        package { ['mysql-server', 'mysql']:/n            ensure =&gt; installed,/n        }/n    }/n}/n```
C: ``ruby/nclass mysql {/n    case $facts['os']['family'] {/n        'RedHat': {/n            package { ['mysql-server', 'mysql']:/n                ensure =&gt; installed,/n            }/n        }/n        'SLES': {/n            package { ['mysql', 'mysql-client']:/n                ensure =&gt; installed,/n            }/n        }/n        default: {/n            package { ['mysql-server', 'mysql']:/n                ensure =&gt; installed,/n            }/n        }/n    }/n}/n```
D: ``ruby/nclass mysql {/n    if $facts['os']['family'] == 'RedHat' {/n        package { 'mysql-server':/n            ensure =&gt; installed,/n        }/n    } /n    package { 'mysql':/n        ensure =&gt; installed,/n    } /n}

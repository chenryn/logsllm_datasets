**ATT&CK ID:** S0088  
**ATT&CK Technique Name:** Kasidet

**Description:**
Kasidet establishes persistence by creating a Registry Run key. This technique allows the malware to automatically execute each time the system starts, ensuring continued presence on the infected machine. References for this behavior can be found in sources [128] and [129].

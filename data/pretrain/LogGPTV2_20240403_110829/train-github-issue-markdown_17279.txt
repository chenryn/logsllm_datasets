# Environment
- **Windows Build Number:** Microsoft Windows [Version 10.0.18362.418]
- **Windows Terminal Version:** 0.5.2762.0 (Preview from Microsoft Store)
- **Device:** Huawei MateBook X Pro

# Steps to Reproduce
1. Ensure that your device is in full Power Saving mode.
2. Verify that "Max Battery Efficiency" is not enabled.

# Expected Behavior
The terminal should display a normal acrylic background, consistent with other acrylic effects in the operating system (e.g., the taskbar).

# Actual Behavior
In Power Saving mode, the terminal's background turns completely black, while other acrylic effects in Windows continue to function as expected (e.g., the taskbar).

# Environment
- **Windows Build Number:** Microsoft Windows [Version 10.0.19613.1005]
- **PowerToys Version:** 0.19.1
- **PowerToy Module:** FancyZones

# Steps to Reproduce
1. Enable the SHIFT feature in FancyZones.
2. Position a window in any zone that is adjacent to the taskbar.

# Expected Behavior
The window should not overlap or clip into the taskbar area, remaining fully accessible and visible.

# Actual Behavior
The window extends below the taskbar, making it unclickable and inaccessible.

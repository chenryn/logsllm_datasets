### Hugging Face 发布免费的代码生成 AI

人工智能可以显著降低开发成本，同时让程序员专注于更具创造性的任务。研究表明，开发人员至少有一半的工作时间用于调试，而非实际编程。近期，人工智能初创公司 Hugging Face 与 ServiceNow 联合发布了 StarCoder，这是一款免费的代码生成 AI，类似于 GitHub Copilot。StarCoder 在超过 80 种编程语言以及 GitHub 仓库中的文本上进行训练，包括文档和编程笔记本。该工具可集成到 VSCode 中，并且像 ChatGPT 一样，能够遵循基本指令（例如，“创建一个应用程序的用户界面”）并回答关于代码的问题。

> **[消息来源：Tech Crunch](https://techcrunch.com/2023/05/04/hugging-face-and-servicenow-release-a-free-code-generating-model/)**
>
> **老王点评**: 但 AI 真的能让程序员从事更多创造性工作吗？还是只会让他们变得更懒惰？

![](/data/attachment/album/202305/06/181104rb0i2os8chtxceoi.jpg)

### Gmail 开始在收件箱中间显示广告

在移动设备上，Gmail 已经开始在“更新”过滤器中显示广告。最近，桌面版的 Gmail 也开始在“推广”标签下的邮件列表中插入广告，这些广告混杂在你收到的推广邮件中，但至少没有出现在“主要”标签中。

> **[消息来源：9to5google](https://9to5google.com/2023/05/05/gmail-ads-increase-2023/)**
>
> **老王点评**: 尽管很多人反对，但我认为这还可以接受，毕竟“推广”标签本来就是各种软硬广告的聚集地。

![](/data/attachment/album/202305/06/181120qj6j164e6mj86u6j.jpg)

### OpenAI 不再依赖 API 客户的数据来训练

OpenAI 表示，该公司已不再使用付费客户的数据来训练其人工智能大语言模型，“已经有一段时间没有训练了”。OpenAI 的服务条款已于 3 月 1 日悄然更新。新的隐私和数据保护政策仅适用于使用该公司 API 服务的客户，如微软、Salesforce 等，但可能会使用来自 API 以外的服务的内容，例如输入到聊天工具 ChatGPT 的文本。

> **[消息来源：CNBC](https://www.cnbc.com/2023/05/05/sam-altman-openai-wont-tap-into-customer-apis.html)**
>
> **老王点评**: 免费的代价就是被用来训练。

### NoteLab

* 支持多种常用的图片格式导出，如SVG、PNG、JPG和BMP等。
* 提供了可配置的笔和纸张自定义选项。
* 内置内存管理器，允许用户指定NoteLab所使用的内存量。
* 多种纸张格式可供选择，包括宽格式、大学格式和图形纸。
* 包含所有标准绘图工具。
* 可对笔记部分进行调整大小、移动、删除、改变颜色等多种操作。

![NoteLab](/data/attachment/album/202309/22/234740a6e6ce55e483w5dk.jpg)

NoteLab 是一个使用 Java 编写的 .jar 文件应用。因此，运行该应用需要 Java 运行环境。你可以参考以下指南在 Linux 系统中安装 Java 或 JRE：

* [在基于 Ubuntu 的系统中安装 Java/JRE 的方法](https://www.debugpoint.com/2016/05/how-to-install-java-jre-jdk-on-ubuntu-linux-mint/)
* [在 Arch Linux 中安装 Java/JRE 的方法](https://www.debugpoint.com/2021/02/install-java-arch/)

#### 5. Rnote

接下来介绍的是 [Rnote](https://github.com/flxzt/rnote)。这是一款专为触屏设备设计的手写笔记应用程序。它基于矢量图像技术，不仅便于绘图和注释图片，还支持 PDF 文件的标注。Rnote 支持本地 .rnote 文件格式，并提供了 PNG、JPEG、SVG 和 PDF 的导入/导出选项。

Rnote 最引人注目的特性之一是支持 Xournal++ 的文件格式（即本列表中的第一款应用程序），使其成为必备工具。该应用使用 GTK4 和 Rust 开发，非常适合在 GNOME 桌面及各种 Linux 系统上运行。请注意，该应用程序目前仍在开发中。

![基于 GTK4 和 Rust 的 Linux 白板应用 Rnote](/data/attachment/album/202309/22/234752ui5ol5igvo0f4oor.jpg)

#### 6. Lorien

[Lorien](https://github.com/mbrlabs/Lorien) 是一款理想的数字笔记本软件，适用于利用多种工具创建个性化笔记。Lorien 是一个基于 Godot 游戏引擎的跨平台、自由开源的“无限画布绘图/笔记”应用，非常适合快速记录头脑风暴会议中的笔记。

其工具箱包括自由手绘刷、橡皮擦、直线工具和选择工具等。你可以移动或删除笔划部分，这些笔划在运行时被视为点的集合进行渲染。

![Lorien 白板应用程序 for Linux](/data/attachment/album/202309/22/234801ecc0jze3zz41nknf.jpg)

> 
> **[下载 Lorien](https://github.com/mbrlabs/Lorien/releases)**

#### 7. Rainbow Board

Rainbow Board 是一款基于 Electron 和 React 的自由开源白板应用。尽管 Electron 应用通常因性能问题而不受欢迎，但考虑到我们正在列出此类应用，我认为它值得一提。

它提供了一个标准的画布，支持触控和手写输入。工具箱包括刷子大小、颜色、填充颜色、字体以及撤销和重做操作。你可以将绘制结果导出为 PNG 或 SVG 文件。

### 概述
在这篇回顾历史的文章中，我们将探讨一些最早的 Linux 发行版是如何演变的，并形成我们今天所熟知的发行版。

![](/data/attachment/album/201910/04/100019oc79ftxhuw91x7rc.png)

本文将探讨在第一个 Linux 内核发布后，诸如 Red Hat、Debian、Slackware、SUSE 和 Ubuntu 等流行发行版是如何逐步发展起来的。自 1991 年 Linux 最初以内核形式发布以来，在全球众多贡献者的帮助下，通过创建 shell、库、编译器和相关软件包，Linux 逐渐成为一个完整的操作系统。

### 1. 第一个已知的“发行版”由 HJ Lu 创建

追溯到 1992 年，HJ Lu 发布了第一个可用于访问 Linux 的类似发行版的工具。它由两个 5.25 英寸软盘组成：

- **LINUX 0.12 BOOT DISK**: 用于启动系统的“启动”磁盘。
- **LINUX 0.12 ROOT DISK**: 用于在系统启动后获取命令提示符以访问 Linux 文件系统的“根”磁盘。

要在硬盘上安装 LINUX 0.12，必须使用十六进制编辑器来编辑主引导记录（MBR），这在当时是一个非常复杂的过程。

> 感觉太怀旧了吗？你可以[安装 cool-retro-term 应用程序](https://itsfoss.com/cool-retro-term/)，它能为你提供 90 年代计算机复古外观的 Linux 终端。

### 2. MCC Interim Linux

![MCC Linux 0.99.14, 1993 | 图片来源](/data/attachment/album/201910/04/100028sbt9ujbjx29xuxpg.jpg)

MCC Interim Linux 由英格兰曼彻斯特计算中心的 Owen Le Blanc 于 1992 年与 LINUX 0.12 同时发布。它是面向普通用户的第一个 Linux 发行版，具有菜单驱动的安装程序和用户/编程工具。它也是以软盘集的形式发布，可以安装在系统上以提供基于文本的基本环境。

MCC Interim Linux 比 LINUX 0.12 更易于使用，并且在硬盘上的安装过程更加简单和现代化，不需要使用十六进制编辑器来编辑 MBR。尽管它于 1992 年 2 月首次发布，但自当年 11 月以来也可以通过 FTP 下载。

### 3. TAMU Linux

![TAMU Linux | 图片来源](/data/attachment/album/201910/04/100029kz397iwlonxoyxr9.jpg)

TAMU Linux 由 Texas A&M University 的 Aggies 与 Texas A&M Unix & Linux 用户组于 1992 年 5 月开发，被称为 TAMU 1.0A。它是第一个提供 X Window System 的 Linux 发行版，而不仅仅是基于文本的操作系统。

### 4. Softlanding Linux System (SLS)

![SLS Linux 1.05, 1994 | 图片来源](/data/attachment/album/201910/04/100031cromfglvzipyyrqg.jpg)

Softlanding Linux System (SLS) 由 Peter McDonald 于 1992 年 5 月发布，口号是“DOS 伞降的温柔救援”。SLS 在其时代得到了广泛的使用和推广，但由于开发人员决定更改发行版中的可执行格式，用户逐渐停止使用它。

许多当今流行的发行版都是从 SLS 演变而来的，例如：
- **Slackware**: 由 Patrick Volkerding 于 1993 年创建，是最早的 Linux 发行版之一。
- **Debian**: 由 Ian Murdock 发起，Debian 在从 SLS 模型继续发展之后于 1993 年发布。我们今天熟知的 Ubuntu 发行版就是基于 Debian。

### 5. Yggdrasil

![LGX Yggdrasil Fall 1993 | 图片来源](/data/attachment/album/201910/04/100034o21yhz60az10dg88.jpg)

Yggdrasil 于 1992 年 12 月发布，是第一个产生 Live Linux CD 想法的发行版。它由位于加利福尼亚州伯克利的 Adam J. Richter 创立的 Yggdrasil 计算公司开发。它可以在系统硬件上自动配置自身，即“即插即用”功能，这是当今非常普遍的功能。Yggdrasil 后来的版本还包括一个用于在 Linux 中运行任何专有 MS-DOS CD-ROM 驱动程序的技术。

![Yggdrasil’s Plug-and-Play Promo | 图片来源](/data/attachment/album/201910/04/100035rf14o70x5jt9l590.jpg)

他们的座右铭是“ 我们其余人的自由软件”。

### 6. Mandriva

在 90 年代后期，有一个非常受欢迎的发行版 [Mandriva](https://en.wikipedia.org/wiki/Mandriva_Linux)，该发行版于 1998 年首次发布，是通过将法国的 Mandrake Linux 发行版与巴西的 Conectiva Linux 发行版合并形成的。它的发布周期为 18 个月，会对 Linux 和系统软件进行更新，并且每年都会发布基于桌面的更新。它还有带有 5 年支持的服务器版本。现在是 [Open Mandriva](https://www.openmandriva.org/)。

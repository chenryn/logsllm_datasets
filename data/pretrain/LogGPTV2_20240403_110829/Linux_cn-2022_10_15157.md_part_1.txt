很高兴为大家介绍 Opensource.com 的 2022 年暑期阅读清单。今年的榜单包含来自社区成员推荐的 7 本精彩读物。这个列表涵盖了各种类型的书籍，从有趣的悬疑小说到引人深思的非小说类作品。希望你能在这个榜单中找到感兴趣的书。

![](/data/attachment/album/202210/20/115515jsppwzz8s1ssle7p.jpg)

### 《每个 Java 程序员都应该知道的 97 件事：专家的集体智慧》

![Book title 97 Things Every Java Programmer Should Know](/data/attachment/album/202210/20/115713gmeqa3ds3aqw1qmd.jpg)

**《每个 Java 程序员都应该知道的 97 件事：专家的集体智慧》**

编辑：Kevlin Henney 和 Trisha Gee  
*[由 Seth Kenlon 推荐](https://opensource.com/users/seth)*

这本书由 73 位在软件行业工作的不同作者共同撰写。尽管书名提到的是 Java 编程，但其中很多内容适用于所有编程语言。例如，书中讨论了容器环境、软件交付的最佳实践以及开发工具的重要性等主题。此外，还有一些章节提供了适用于生活中的建议，如将问题分解成小部分来解决，建立多样化的团队，以及如何从混乱中找到秩序。

每章只有几页，总共有 97 个章节，你可以轻松跳过不适用的部分。无论你是经验丰富的 Java 开发者，还是刚刚开始学习编程，这都是一本值得一读的好书。

### 《城市不是计算机：其他的城市智能》

![Book title A City is Not a Computer](/data/attachment/album/202210/20/115713x2oocpky6ni1626o.jpg)

**《城市不是计算机：其他的城市智能》**

作者：Shannon Mattern  
*[由 Scott Nesbitt 推荐](https://opensource.com/users/scottnesbitt)*

当前，“智能化”已成为一种潮流，我们的手机、家用电器、手表、汽车，甚至是城市都在变得“智能”。然而，这种智能化是否真的使城市变得更加智能呢？Shannon Mattern 在《城市不是计算机》一书中对此提出了质疑。

Mattern 认为，智慧城市的目标是提供更好的公共服务和增强市民参与感。但实际上，许多智慧城市项目只是将技术管理与公共服务相结合，将公民重新定义为“消费者”和“用户”，而非鼓励他们积极参与城市治理。此外，智慧城市收集的大量数据往往无法得到有效利用，导致市政工作人员不堪重负，只能处理短期任务而忽视了更深层次的问题。

《城市不是计算机》是一本经过深入研究的著作，对智慧城市的理念进行了批判性的分析。它引发了我们对智慧城市真正目的的思考，并促使我们反思：发展智慧城市是否真的必要？

### cURL 发布 8.0，庆祝其 25 岁生日

cURL 是一个用于通过 URL 传输数据的命令行工具和库。几乎所有从事网络交互代码开发的程序员都可能使用过 cURL。据估计，cURL 已经被安装了超过 100 亿次。它的第一个版本是 4.0，发布于 1998 年，当时仅有 2200 行代码，由 httpget 和 urlget 项目改编而成。最初，cURL 只支持三种协议和 24 个命令行选项，而现在它可以支持 28 种协议和 249 个选项。

**[消息来源：The Register](https://www.theregister.com/2023/03/21/curl_project_25)**

**老王点评**：原本只是一个个人项目，现在却发展成了互联网基础设施的一部分。

![](/data/attachment/album/202303/23/083442yva4ldjek41a7jev.jpg)

### GitHub Copilot 将迁移到 GPT 上

GitHub 对其 Copilot 编程助手进行了升级，并更名为 GitHub Copilot X。新的 Copilot 将依赖一系列不断发展的大型语言模型，包括 OpenAI 的 GPT-3.5-turbo 和 GPT-4，而不再使用 Codex 的定制版本。OpenAI 将在 2023 年 3 月 23 日关闭 Codex 的公共 API。根据 GitHub 的数据，截至上个月，Copilot 参与了微软云代码库中 46% 的代码创建，并帮助开发者提高了 55% 的编程速度。

**[消息来源：The Register](https://www.theregister.com/2023/03/22/github_copilot_learns_new_tricks/)**

**老王点评**：通过 VSCode、GitHub 和 Copilot，微软似乎已经掌握了开发者所需的所有方面。

![](/data/attachment/album/202303/23/083454n4gree427e7ccoog.jpg)

### Mozilla 将建立 Mozilla.ai

Mozilla 宣布将投资 3000 万美元成立一个新的初创公司 [Mozilla.ai](http://mozilla.ai/)，专注于“建立一个值得信赖的、独立的、开源的人工智能生态系统”。Mozilla 强调，“人工智能的核心是代理权、问责制、透明度和开放性。[Mozilla.ai](http://mozilla.ai/) 将成为一个独立于大科技公司和学术界的空间，供志同道合的创始人、开发者、科学家、产品经理和建设者聚集。”

**[消息来源：Phoronix](https://www.phoronix.com/news/Mozilla.ai)**

**老王点评**：虽然这又是一个看似“不务正业”的项目，但希望它能为 AI 打造一条可信且独立的道路。

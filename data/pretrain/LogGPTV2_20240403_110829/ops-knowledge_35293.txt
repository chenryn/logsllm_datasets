User: How to use the system audit tool Auditing to record user operations and manage user activities in the Linux operating system?
Assistant: In the Linux operating system, to use the system audit tool Auditing to record user operations and manage user activities, you can follow the steps below: 1. Install Auditing In Ubuntu, Debian and other similar Linux distributions, you can run the following commands to install : sudo apt-get install auditd In RedHat, CentOS and other similar Linux distributions, you can run the following command to install: sudo yum install audit 2. Enable Auditing Before enabling Auditing, you need to ensure that the security settings are enabled, by editing / etc/audit/auditd.conf file to configure Auditing. For example: sudo nano /etc/audit/auditd.conf Here is a simple configuration example: # general configuration local6,info write_logs = yes log_group = root log_file = /var/log/audit/audit.log log_format = RAW flush = INCREMENTAL_ASYNC max_log_file_action = ROTATE max_log_file = 5 num_logs = 4 admin_space_left_action = SUSPEND space_left_action = SUSPEND disk_full_action = SUSPEND disk_error_action = SUSPEND #Rule Configuration#Record the operation of each user -a always,exit -F arch=b32 -S exe cve -k USER_CMD -a always ,exit -F arch=b64 -S execve -k USER_CMD -w /etc/group -p wa -k GROUP_FILE -w /etc/passwd -p wa -k USER_FILE -w /var/log/faillog -p wa -k LOG_AUTH 3. Restart the Auditing service and test Restart the Auditing service with the following command: sudo systemctl restart auditd For testing, you can use a command operating system, such as: sudo apt-get update Then, you can use the following command to view the recorded operations: sudo aureport --tty or use the following command to view the system log to view recorded operations: sudo tail -f /var/log/messages 4. Use with third-party tools In addition to the above-mentioned system logs, third-party tools can also be used , such as log collector and report generator to use Auditing. These tools help you better analyze, monitor and report on the results of user actions and administrative activities.

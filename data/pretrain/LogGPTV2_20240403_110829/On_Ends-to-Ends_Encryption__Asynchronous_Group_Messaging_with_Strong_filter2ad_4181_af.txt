**算法2：辅助函数**

```plaintext
函数 LeftSubtreeSize(x)
    返回 2 * floor(log2(x)) - 1 // 高度为x元素的左子树
end function

函数 CreateTree(λ0, λ1, ..., λn) // 创建n个叶子的树
    如果 n = 0 则返回 (leaf, λ0)
    h := LeftSubtreeSize(n)
    (L, lk) := CreateTree(λ0, ..., λ(h-1)) // 完全左子树

    k := ι(g(lk)(rk))
    返回 (node((L, lk), (R, rk)), k)
end function
```

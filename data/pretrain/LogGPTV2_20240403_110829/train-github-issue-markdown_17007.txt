# Environment
- **Windows Build Number:** Microsoft Windows [Version 10.0.18363.836]
- **PowerToys Version:** v0.18.2
- **PowerToy Module:** PowerToys Run

# Steps to Reproduce
1. Run PowerToys as an administrator.
2. Attempt to launch PowerToys Run using the `Alt + Space` shortcut.

# Expected Behavior
- PowerToys Run should trigger and display its user interface.

# Actual Behavior
- PowerToys Run does not trigger, and no UI is displayed.

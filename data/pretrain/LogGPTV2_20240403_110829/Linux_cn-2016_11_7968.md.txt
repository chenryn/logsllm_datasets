### Ubuntu 17.04 将提供增强的 Unity 8 会话及更多 Snaps 应用

Canonical 在 11 月 15 日至 16 日举办了 Ubuntu 线上峰会 (UOS)，期间社区成员可以了解 Ubuntu 17.04 Linux 的最新进展。会上透露，Ubuntu 17.04（Zesty Zapus）将于 2017 年 4 月发布，并将带来一个显著增强的 Unity 8 会话。此外，Snappy Store 将完全支持运行 snaps，因为 Snappy 正逐渐成为 Ubuntu 操作系统的默认选择。

![](/data/attachment/album/201611/18/203056yqdqps4qiq5ep25p.jpg)

### 微软作为白金会员加入 Linux 基金会

微软公司于 11 月 16 日宣布，将以白金会员的身份加入 Linux 基金会。这一举动距离前 CEO 史蒂夫·鲍尔默称开源平台为“癌症”不到 15 年。同一天，Google 宣布将在 Connect() 开发者大会上加入 .NET 基金会的领导团队。

![](/data/attachment/album/201611/17/163813zkykjl4y1jl50wki.png)

具体报道请参阅我们的[文章](/article-7966-1.html)。

### Canonical 宣布在 Ubuntu Linux 上推出 Microsoft SQL Server 公众预览版

Canonical 于 11 月 16 日宣布，在 Ubuntu 系统上推出 Microsoft SQL Server 的公众预览版。用户可以在 Microsoft's SQL Server on Linux 网站、Microsoft Azure 和 [Docker Hub](http://hub.docker.com/r/microsoft/mssql-server-linux) 下载公众预览版镜像。更多信息，请访问 [www.sqlserveronlinux.com](https://www.microsoft.com/en-us/sql-server/sql-server-vnext-including-Linux)。

![](/data/attachment/album/201611/17/213226blpr367yaaafzaaw.jpg)

具体信息请参阅我们的[报道](/article-7967-1.html)。

### X.Org Server 1.19.0 官方发布，Linux 版本即将推出

X Window 系统开发者 Keith Packard 于 11 月 15 日宣布了 X.Org Server 1.19.0 的正式发布。不久后，用户将能够下载 GNU/Linux 版本的 X.Org Server 1.19.0 显示服务器。要了解具体的改动细节，建议阅读 [xorg-server 1.18.99.2](https://lists.x.org/archives/xorg-announce/2016-September/002709.html)、[xorg-server 1.18.99.901](https://lists.x.org/archives/xorg-announce/2016-September/002711.html) 和 [xorg-server 1.18.99.902](https://lists.x.org/archives/xorg-announce/2016-October/002734.html) 的更改日志。

### Firefox 50.0 for Linux、Mac 和 Windows 发布

Mozilla 于 11 月 15 日发布了 Firefox 50.0 浏览器，支持 GNU/Linux、macOS 和 Microsoft Windows 系统。尽管不是一次重大更新，但该版本包含了许多有趣的变化。有关详细信息，请参阅 Mozilla 发布的[发行说明](https://www.mozilla.org/en-US/firefox/50.0/releasenotes/)。

### 《堡垒之夜》暂无计划移植到 Linux

根据统计数据，仅有不到 1% 的 Steam 用户使用 Linux 系统，而 macOS 用户的比例也仅略高于 3%，相比之下，Windows 用户占据了 96% 的市场份额。尽管许多游戏已能在 Linux 平台上运行，但大多数热门 3A 大作仍无法在该系统下顺利游玩，即使通过 Proton 兼容层也无法实现。Epic Games 表示，《堡垒之夜》目前并无移植至 Linux 的计划。据 Epic 首席执行官 Tim Sweeney 解释，这是因为公司“缺乏信心能够在各种内核配置（包括定制内核）下大规模打击作弊行为”。然而值得注意的是，Epic 已经将其反作弊软件移植到了 Mac 和 Linux 设备上，包括即将发布的 Steam Deck。

> 老王点评：归根结底，Linux 用户基数太小，不足以让开发者投入大量资源进行开发和维护。

![](/data/attachment/album/202202/13/201822kz2s1shcql3jq6wp.jpg)

### Valve 开源 Steam Deck 外壳 CAD 文件

Valve 即将在月底发售其基于 Linux 的掌机 Steam Deck。为鼓励用户探索与改造设备，Valve 宣布将在知识共享许可协议下于 GitHub 开源 Steam Deck 外壳（曲面拓扑）的 CAD 文件。不过需要注意的是，自行拆解或修改可能会导致保修失效（若在此过程中造成损坏的话）。

> 老王点评：这种开放的态度值得点赞！

![](/data/attachment/album/202202/13/201839ec0zsiybt6spmy4t.jpg)

### 英特尔 Alder Lake CPU 在 Linux 上性能大幅提升

今年春季发布的 Linux 5.18 内核将增加对英特尔增强硬件反馈接口 (EHFI) 的支持，以优化工作负载在不同 CPU 核心/线程间的分配。作为首款采用混合架构设计的 x86-64 处理器，Alder Lake 在同一芯片上集成了两种不同的 CPU 架构，各具优势。低优先级任务和轻量级工作负载可以在效率核心上运行以节省能耗，而延迟敏感型及计算密集型任务则可由性能更强的核心处理。

> 老王点评：虽然 Windows 已经实现了类似的功能，但此次更新标志着 Linux 在这方面迈出了重要一步。

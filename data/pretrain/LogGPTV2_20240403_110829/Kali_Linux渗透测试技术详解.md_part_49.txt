## 使用Arpspoof进行ARP欺骗攻击

### Arpspoof工具简介
Arpspoof是一个用于ARP欺骗的源代码程序，通过替换传输中的数据达到欺骗目标的目的。本节将介绍Arpspoof工具的使用。

### URL流量操纵攻击
URL流量操纵类似于中间人攻击，通过ARP注入实现。具体步骤如下：
1. 开启路由转发功能。
   ```bash
   root@kali:~# echo 1 > /proc/sys/net/ipv4/ip_forward
   ```
2. 欺骗目标系统。
   ```bash
   root@kali:~# arpspoof -i eth0 -t 192.168.6.101 192.168.6.1
   ```
3. 欺骗网关。
   ```bash
   root@kali:~# arpspoof -i eth0 -t 192.168.6.1 192.168.6.101
   ```

### 验证攻击
1. 启动Wireshark抓包工具。
2. 在Kali Linux上选择捕获接口并开始抓包。
3. 在目标系统上ping网关，查看Wireshark抓取的数据包。

### 端口重定向攻击
端口重定向又称为端口转发或端口映射。具体步骤如下：
1. 开启路由转发功能。
2. 使用Arpspoof工具注入流量到默认网络。
3. 添加防火墙规则进行端口重定向。

### 捕获并监视无线网络数据
1. 开启路由器转发功能。
2. 使用Arpspoof命令攻击主机。
3. 查看目标主机访问URL地址的信息。
4. 使用Driftnet工具捕获目标系统浏览过的图片。

通过这些步骤，可以有效地实施ARP欺骗攻击，并监控和捕获目标系统的网络流量。

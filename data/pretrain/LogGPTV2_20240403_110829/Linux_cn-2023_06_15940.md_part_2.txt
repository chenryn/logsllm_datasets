### Xubuntu

Xubuntu 是一款将广受欢迎的 Xfce 桌面环境与 Ubuntu 操作系统相结合的 Linux 发行版。它为用户提供了轻量级且高效的计算体验，同时融入了熟悉的 Ubuntu 生态系统。Xubuntu 专注于 Xfce 桌面环境，以实现快速响应的用户界面并节省系统资源。

![Xubuntu 23.04 桌面](/data/attachment/album/202306/25/164843dz2ln2btlyac1v1j.jpg)

Xubuntu 基于 Ubuntu 的基础软件包，包括 Snap 和其他 Ubuntu 桌面应用程序。如果你能接受这些特性，那么 Xubuntu 将是一个稳定且基于 Xfce 的优秀选择。

**下载链接：**
> **[下载 Xubuntu](https://xubuntu.org/download/)**

### EndeavourOS

EndeavourOS 是一个基于 Arch Linux 的 Xfce 发行版。三年前，一个小团队的贡献者们开始开发 EndeavourOS，以继承已停止的 Antergos 项目。自那时起，EndeavourOS 因其简便的安装、用户体验和功能而受到欢迎。

尽管 EndeavourOS 提供了多种桌面环境，但默认配备的是 Xfce 桌面环境，并经过轻微定制，以提供几乎原汁原味的 Xfce 体验。

![EndeavourOS with Xfce desktop](/data/attachment/album/202306/25/164903ub09mjklmedc9vbb.jpg)

如果你曾经尝试过 EndeavourOS，你一定会感受到作为 Arch Linux 发行版，对于终端用户来说，执行任务有多么轻松。这个发行版的座右铭是成为一款“面向大众”的通用目的 Arch Linux 发行版，消除了新用户对 Arch Linux 安装的恐惧感和使用 Arch 的优越感。

**下载链接：**
> **[下载 EndeavourOS](https://endeavouros.com/latest-release/)**

### Linux Lite

Linux Lite 6.0 是一个基于 Ubuntu 并遵循其 LTS（长期支持）生命周期的发行版。这意味着在 Ubuntu 发布后的五年内，你将获得类似的发布时间表和安全更新。Xfce 是其主要且唯一的桌面环境。Linux Lite 主要针对希望开始 Linux 之旅的 Windows 用户，可以被视为一个“桥梁” Linux 发行版。

![Linux Lite 6 Xfce 桌面](/data/attachment/album/202306/25/164930s85yvr5x15r59lxz.jpg)

它提供的 Xfce 桌面环境经过轻微的主题、壁纸和字体定制。不过，它是另一个基于 Ubuntu 的带有 Xfce 的发行版。你可以试试看。

**更多信息：**
> **[Linux Lite 评论](https://www.debugpoint.com:443/?s=linux+lite+review)**

**下载链接：**
> **[下载 Linux Lite](https://www.linuxliteos.com/download.php)**

### Manjaro Xfce

Manjaro Linux 是一款著名且经受时间考验的发行版，旨在将 Arch Linux 带给广大用户。它经过精心设计，打破了“Arch Linux 很难”的迷思。一旦开始使用，你不会感觉到它是一个 Arch Linux 发行版，反而更像是基于 Debian 或 Fedora 的发行版。

尽管 Manjaro Linux 也提供了其他桌面环境（如 KDE Plasma），但它的 Xfce 变体也是基于 Arch Linux 核心设计的。如果你需要另一款基于 Arch Linux 的带有 Xfce 的发行版，可以尝试它。

**更多信息：**
> **[多篇点评](https://www.debugpoint.com:443/?s=endeavouros)**

**下载链接：**
> **[下载 Manjaro Linux](https://manjaro.org/download/)**

### Peppermint OS

Peppermint OS 是一个基于 Debian 稳定版的发行版，非常适合日常使用。它预装了一些应用程序，帮助你快速上手，并带有定制的 Xfce 桌面环境。之前，Peppermint OS 使用的是 LXDE，但最近切换到了 Xfce。

![Peppermint 2022-02-02 桌面](/data/attachment/album/202306/25/164947evm4lu38v448o8z6.jpg)

除了基于 Debian 之外，Peppermint OS 还有两个关键应用程序：Peppermint Welcome 应用程序为新用户提供有关发行版的基本概述；Peppermint Hub 应用程序提供了一些快捷方式，用于管理你的系统，比如更改主题和从软件仓库下载软件。如果你是 Peppermint OS 的新用户，Peppermint Hub 可以从一个便捷位置处理大多数系统管理任务。

**下载链接：**
> **[下载 Peppermint OS](https://peppermintos.com/)**

### Kali Linux

Kali Linux 是网络安全行业中广为人知的发行版，基于 Debian 的测试分支。尽管它是针对特定领域的，但默认的 Xfce 桌面环境使其成为本列表中的另一个候选项。

Kali Linux 配备了各种专门用于渗透测试、漏洞评估和数字取证的工具。这些工具包括密码破解、网络扫描和无线攻击，使其成为评估和保护系统的理想解决方案。

![Kali Linux with Xfce Desktop Environment](/data/attachment/album/202306/25/164956gk24535vbuztullz.jpg)

它的 Xfce 桌面环境经过大量定制，使其成为外观出色的发行版之一。然而，它面向的是专门的用例。因此，你应该只将 Kali 用于那些特定功能。

**下载链接：**
> **[下载 Kali Linux](https://www.kali.org/get-kali/)**

### 哪个最适合你？

这取决于你的个人喜好和使用场景。如果你不确定，我建议你尝试 Linux Mint Xfce 或 MX Linux。它们适用于各种用途。

如果你需要 Fedora 或 Arch Linux 基础，可以尝试 Fedora Xfce 或 EndeavourOS。

其余的发行版可能适用于需要 Ubuntu 或 Debian 基础的特定用例。

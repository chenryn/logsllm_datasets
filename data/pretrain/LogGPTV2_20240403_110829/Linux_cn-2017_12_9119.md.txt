我们最近采访了 Mark McIntyre，探讨了他如何使用 Fedora 系统。这是 Fedora 杂志[系列文章的一部分](https://fedoramagazine.org/tag/how-do-you-fedora/)，该系列简要介绍了 Fedora 用户及其使用经验。如果您希望成为采访对象，请通过[反馈表](https://fedoramagazine.org/submit-an-idea-or-tip/)联系我们。

### Mark McIntyre 是谁？

Mark McIntyre 是一位天生的极客，热爱 Linux。他在 13 岁时开始自学 BASIC 编程，并逐渐成长为一名专业的程序员。Mark 和他的侄女都是披萨爱好者，他们去年秋天开始了尝试诺克斯维尔多家披萨店的任务。您可以点击[这里](https://knox-pizza-quest.blogspot.com/)了解他们的进展。此外，Mark 还是一位业余摄影师，在 Flickr 上发布自己的作品 [链接](https://www.flickr.com/photos/mockgeek/)。

![](/data/attachment/album/201712/07/093534jo5455vg0069759u.jpg)

作为一名开发者，Mark 拥有丰富的编程背景，曾使用过 Visual Basic、LotusScript、PL/SQL（Oracle）、Tcl/TK 以及基于 Python 的 Django 框架。目前，他作为系统工程师主要使用 Python 进行工作。“我经常使用 Python，尤其是在自动化任务中。”

尽管自称为书呆子并喜爱科幻电影，但 Mark 最喜欢的电影并不是科幻片。“虽然我喜欢《星际迷航》和《星球大战》，但我最喜欢的电影是《光荣战役》。”他还提到，《冲出宁静号》是他认为一部著名电视剧（指《萤火虫》）的精彩后续。

Mark 认为谦逊、知识和友善是非常重要的品质。他欣赏能够设身处地为他人着想的人。“如果你决定为他人服务，你会选择一个愿意亲近的人，而不是让自己备受折磨的人。”

McIntyre 目前在 [Scripps Networks Interactive](http://www.scrippsnetworksinteractive.com/) 工作，该公司是 HGTV、Food Network、Travel Channel、DIY、GAC 等多个电视频道的母公司。他是一名系统工程师，负责非线性视频内容的管理，支持开发团队将线性视频发布到线上平台，如亚马逊和葫芦网。这些系统既包括预置系统也包括云系统。Mark 还开发了一些自动化工具，将应用程序部署到云基础设施中。

### Fedora 社区

Mark 将 Fedora 社区描述为充满活力的社区，充满了像他一样热爱生活的人。“从设计师到封包者，这个团体非常活跃且充满生机。”他说，“这让我对操作系统充满信心。”

大约在 2002 年，Mark 开始频繁使用 IRC 上的 #fedora 频道：“那时 Wi-Fi 在启用适配器和配置模块功能时，许多步骤仍需手动完成。”为了使 Wi-Fi 正常工作，他不得不重新编译 Fedora 内核。

McIntyre 鼓励其他人参与 Fedora 社区。“这里有各种各样的机会，从前端设计到测试部署，再到开发、应用程序打包和新技术实现。”他建议选择一个感兴趣的领域，然后向相关群体提出问题。“有很多机会可以贡献自己的一份力量。”

对于帮助他入门的社区成员，Mark 表示感谢：“Ben Williams 非常乐于助人。在我第一次接触 Fedora 时，他帮我解决了 #fedora 支持频道中的安装补丁问题。” Ben 还鼓励 Mark 成为 Fedora [大使](https://fedoraproject.org/wiki/Ambassadors)。

### 使用的硬件和软件

McIntyre 在笔记本电脑和台式机上使用 Fedora Linux 系统。服务器上则选择了 CentOS，因为它具有更长的支持周期。他的台式机是自己组装的，配备了 Intel 酷睿 i5 处理器、32GB 内存和 2TB 硬盘。“我装了一个 4K 显示屏，有足够的空间同时查看所有应用。”他目前使用的笔记本电脑是戴尔灵越二合一，配备 13 英寸屏幕、16GB 内存和 525GB m.2 固态硬盘。

![](/data/attachment/album/201712/07/093535u7orwbnqwj7oeb66.png)

Mark 目前在其所有机器上运行 Fedora 26。当新版本正式发布时，他通常会避开高峰期。“除非我的工作站正在运行下一代测试版，否则我会等到它成熟后再升级。”他经常使用 dnf 系统升级插件进行就地更新，效果非常好。

为了摄影，McIntyre 使用 GIMP、Darktable 以及其他一些照片查看和编辑软件包。当他不使用 Web 电子邮件时，他会使用 Geary 和 GNOME Calendar。Mark 选用 HexChat 作为 IRC 客户端，与在 Fedora 服务器实例上运行的 ZNC bouncer 联机。他的部门通过 Slack 进行沟通。

“我从来都不是 IDE 的粉丝，所以我大多数的编辑任务都在 vim 上完成。”Mark 有时也会打开简单的文本编辑器，如 gedit 或 xed。他用 GPaste 进行复制和粘贴操作。“对于终端的选择，我已经成为了 Tilix 的忠实用户。”McIntyre 通过 Rhythmbox 管理他喜欢的播客，并用 Epiphany 进行快速网络查询。

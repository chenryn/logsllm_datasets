[Indicator Weather](https://launchpad.net/weather-indicator) 是一款用于桌面显示天气信息的工具。最新发布的 13.06 版本新增了对 Kelvin 温度单位的支持。用户可以通过以下路径进行设定：`Indicator Weather appindicator --> Preferences --> Units`，在此处可以选择 Kelvin (K) 作为温度单位。

![](/data/attachment/album/201308/15/2301521t69w96tmcobex9w.png)

安装 Indicator Weather 13.06 非常简单。只需打开终端并输入以下命令（适用于 Ubuntu 12.04, Ubuntu 12.10, Ubuntu 13.04, 和 Ubuntu 13.10 系统）：

```bash
sudo add-apt-repository ppa:weather-indicator-team/ppa
sudo apt-get update
sudo apt-get install indicator-weather
```

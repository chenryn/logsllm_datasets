### 散点层配置说明

散点层是3D平面中国地图的一个子组件，支持独立的样式和数据配置。它可以以散点的形式展示地理位置上的点数据信息。本文将详细介绍散点层的各项配置及其含义。

#### 配置项

- **气泡高度**：设置散点相对于地图平面的高度，取值范围为0到20。
- **呼吸范围**：定义散点在半径基础上增加的内外呼吸半径大小，取值范围为0到3像素（px）。
- **呼吸速度**：控制散点呼吸效果的变化速率，取值范围为0到0.2。
- **最大半径**：设定散点的最大尺寸，与数据源中的`value`字段关联映射，取值区间为1至100。
- **最小半径**：设定散点的最小尺寸，同样基于数据源中`value`字段进行调整，取值区间为1至50。

##### 气泡样式

- **内圈颜色**：指定散点内部的颜色，可通过颜色选择器自定义。
- **外圈颜色**：定义散点外部边缘的颜色。
- **渐变数量**：设置从内圈到外圈颜色变化的层数，数值越高，过渡越平滑，取值范围为1至40。
- **渐变类型**：提供四种选项供用户选择，分别是线性、S型、外疏和内疏。

#### 数据相关配置

下表展示了用于表示地理坐标及权重的数据字段定义：

| 字段 | 说明          |
| ---- | ------------- |
| lng  | 指定气泡点的经度 |
| lat  | 指定气泡点的纬度 |
| value | 表示该位置点的重要性或权重 |

此外，关于数据源的管理还包括以下两个关键参数：

| 配置项       | 描述                                      |
| ------------ | ----------------------------------------- |
| 数据源类型   | 可选静态数据或SPL动态数据两种形式，并支持实时修改。|
| 数据响应结果 | 显示当前所选数据源下的实际请求反馈内容，便于监控数据状态变化。|

通过上述配置，您可以灵活地定制出符合需求的地图散点可视化效果。希望这份指南能够帮助您更好地理解和使用散点层功能。

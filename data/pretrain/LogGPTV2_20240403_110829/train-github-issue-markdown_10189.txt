### Bug report
When creating a legend with a pie graph using the `GTK3Cairo` backend a
TypeError is raised. The issue affects versions 3.0.0rc2, 3.0.0, 3.0.1, 3.0.2
but not 2.2.3. Likewise this exception is not raised with the backend is
`GTK3Agg`.
**Code for reproduction**
    import matplotlib
    matplotlib.rcParams['backend'] = 'GTK3Cairo'
    import matplotlib.pyplot as plt
    fig1, ax1 = plt.subplots()
    ax1.pie(
    	(83, 14, 2),
    	colors=(
    		(0.75, 0.86, 1.0),
    		(0.49, 0.61, 0.73),
    		(0.24, 0.36, 0.47)
    	)
    )
    legend_bbox = fig1.legend(['Sample Text'])
    legend_bbox.legendPatch.set_linewidth(0)
    plt.show()

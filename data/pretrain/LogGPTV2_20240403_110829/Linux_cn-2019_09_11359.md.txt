# 如何在 Linux Mint 中更换主题

作者：It's Foss  
分类：桌面应用  
发布日期：2019-09-19 10:03:56  
来源链接：[https://itsfoss.com/install-themes-linux-mint/](https://itsfoss.com/install-themes-linux-mint/)  

一直以来，使用 Cinnamon 桌面环境的 Linux Mint 都是一种卓越的体验。这也是我喜爱 Linux Mint 的主要原因之一。自从 Mint 开发团队开始更认真地对待设计以来，“桌面主题”应用成为了更换新主题、图标、按钮样式、窗口边框和鼠标指针的重要工具。你还可以通过它安装新的主题。感兴趣吗？让我们开始吧。

## 如何在 Linux Mint 中更换主题

1. 在菜单中搜索“主题”并打开主题应用。
   ![Theme Applet 提供了安装和更换主题的简便方式](/data/attachment/album/201909/19/100358jrpo6nt5k9trhrhn.jpg)

2. 在应用中，你会看到一个“添加/删除”按钮。点击它，可以看到按流行程度排序的 Cinnamon Spices（Cinnamon 的官方插件库）的主题。
   ![在 Linux Mint Cinnamon 中安装新主题](/data/attachment/album/201909/19/100401w51vuff01z8bb8t0.jpg)

3. 要安装主题，只需点击你喜欢的主题并等待下载完成。之后，主题将在应用的“Desktop”选项卡中显示为可用。双击已安装的主题即可开始使用。
   ![在 Linux Mint Cinnamon 中更换主题](/data/attachment/album/201909/19/100404dogcebhher0b7zb4.jpg)

## 在 Linux Mint 中安装第三方主题

如果你在其他网站上看到了一个非常棒的主题，但 Cinnamon Spices 网站上没有收录，可以尝试以下步骤：

1. 前往 Cinnamon Look 浏览并下载你喜欢的主题。
   * [在 Cinnamon Look 中获取更多主题](https://www.cinnamon-look.org/)

2. 下载后，你会得到一个包含所有必要文件的压缩包。解压并将文件夹保存到 `~/.themes` 目录中。`~` 表示你的家目录，即 `/home/{YOURUSER}/.themes`。

3. 打开家目录并按 `Ctrl+H` 显示隐藏文件。如果没有看到 `.themes` 文件夹，请创建一个新文件夹并命名为 `.themes`。记住，文件夹名称开头的点非常重要。

4. 将解压后的主题文件夹从下载目录复制到家目录中的 `.themes` 文件夹中。

5. 最后，在上述主题应用中查找并选择已安装的主题。

> **注记**：请记住，主题必须是针对 Cinnamon 设计的，即使是从 GNOME 复刻的系统也不一定适用，并非所有的 GNOME 主题都适用于 Cinnamon。

除了更换主题，你还可以通过更改图标来进一步定制 Linux Mint 的外观。
* [如何在 Linux Mint 中安装图标](https://itsfoss.com/install-icon-linux-mint/)

希望你现在知道如何在 Linux Mint 中更换主题了。快去挑选你喜欢的主题吧！

When an input group with an input group button is used with ,.navbar-right, it
causes the input field and the button to break onto a new line when viewed in
Chrome.
It displays correctly in Firefox.
See the following fiddle, but make sure you view it at a size of tablet or
greater:  
http://jsfiddle.net/binvisions/Ygz2n/

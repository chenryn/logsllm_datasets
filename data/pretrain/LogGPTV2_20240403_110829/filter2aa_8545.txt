# Keyboard Shortcuts for KanbanMail

## Global Shortcuts
| Shortcut | Action |
| --- | --- |
| `Ctrl` `\` | Toggle split view |

## Board Shortcuts
| Shortcut | Action |
| --- | --- |
| `C` | Open composer |
| `Ctrl` `N` | Create a new note in the to-do column |
| `/` | Focus on the search bar |
| `Esc` | Clear search, if active |
| `1-5` | Select the first thread in the specified column |

## Email Shortcuts
| Shortcut | Action |
| --- | --- |
| `S` | Star or unstar email |
| `I` | Mark as important or not important |
| `U` | Mark as read or unread |
| `Shift` `P` | Pin email |
| `R` | Reply to email |
| `A` | Reply all |
| `F` | Forward email |
| `E` | Move to archived |
| `1-5` | Move to specified column |
| `Shift` `3` | Move to trash |
| `P` | Go to the previous message in the thread |
| `N` | Go to the next message in the thread |
| `Shift` `U` | Unsubscribe from the thread |
| `Enter` | Open selected thread |
| `Esc` | Close thread viewer, if open, or clear selected threads |

## Composer Shortcuts
| Shortcut | Action |
| --- | --- |
| `Ctrl` `Enter` | Send email |
| `Ctrl` `S` | Save as draft |
| `Esc` | Close composer |

## Editor Shortcuts
| Shortcut | Action |
| --- | --- |
| `Ctrl` `B` | Apply bold formatting |
| `Ctrl` `I` | Apply italic formatting |
| `Ctrl` `U` | Apply underline formatting |
| `Ctrl` `K` | Add hyperlink |

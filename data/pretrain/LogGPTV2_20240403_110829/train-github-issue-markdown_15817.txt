Summary:
using with_sequence will skip the play when the sequence is only a single item
long.
Steps To Reproduce:
      tasks:
        - name: print thing
          debug: var={{ item }}
          with_sequence: count=1
Expected Results:
    ok: [localhost] => (item=1) => {
        "item": "1", 
        "var": {
            "1": "1"
        }
    }
Actual Results:
    skipping: [localhost]

User: How to use Python scripts to monitor network conditions and send out alarm messages in time? Please provide a script for monitoring TCP ports, and explain how to set up a regular detection and alarm mechanism.
Assistant: The following is an example of using Python to monitor TCP ports and send alarm information: ``` import socket import smtplib import time SERVER = 'smtp.gmail.com' PORT = 587 USER = 'youremail@gmail.com' PASSWORD = 'yourpassword' ALARM_THRESHOLD = 3 CHECK_INTERVAL = 60 def send_email(to, subject, body): message = f"From: {USER}\nTo: {to}\nSubject: {subject}\n\n{body}" server = smtplib.SMTP (SERVER, PORT) server.starttls() server.login(USER, PASSWORD) server.sendmail(USER, to, message) server.quit() def check_port(host, port): try: sock = socket.socket(socket .AF_INET, socket.SOCK_STREAM) sock.settimeout(5) sock.connect((host, port)) sock.close() return True except: return False def main(): alarm_count = 0 while True: if check_port('localhost ', 8080): print('Port 8080 is open') alarm_count = 0 else: print('Port 8080 is closed') alarm_count += 1 if alarm_count >= ALARM_THRESHOLD: send_email('youremail@gmail.com', 'Port 8080 is down', 'The TCP port 8080 is not responding') alarm_count = 0 time.sleep(CHECK_INTERVAL) if __name__ == '__main__': main() ``` The above example monitors the local port 8080, when the port is not available , send an email alarm. If the number of consecutive unavailable ports exceeds ALARM_THRESHOLD, an email alarm will be sent. The script checks every CHECK_INTERVAL seconds by default. To set up a periodic detection and alarm mechanism, you can use Python's scheduled task library: `sched` or `schedule`. Just call the scheduled task function in the function `main()`, for example: ``` import sched import time ... scheduler = sched.scheduler(time.time, time.sleep) def check_periodically(sc): main() scheduler.enter(CHECK_INTERVAL, 1, check_periodically, (sc,)) if __name__ == '__main__': scheduler.enter(CHECK_INTERVAL, 1, check_periodically, (scheduler,)) scheduler.run() ``` The above code uses` The sched` library regularly executes the function `main()` every CHECK_INTERVAL seconds, and the alarm mechanism can be implemented in the `main()` function. The usage of the `schedule` library is similar, and you can choose to use it according to your needs.

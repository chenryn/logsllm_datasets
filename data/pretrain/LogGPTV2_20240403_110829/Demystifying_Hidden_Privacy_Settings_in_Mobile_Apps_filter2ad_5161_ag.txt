**Table X:** Calculation of Features

- **F1(p):** 
  \[
  F1(p) = 
  \begin{cases} 
   1 & \text{if } (\forall x, x \in T_p, x \neq \text{"Privacy Settings"}) \text{ and } (\exists v, v \in A, H_v = \text{"Privacy Settings"}) \\
   0 & \text{otherwise}
  \end{cases}
  \]
- **F2(p):**
  \[
  F2(p) = \max_{t \in T} \left\{ \text{Jaccard}(N_t, N_p) \right\}, \text{ where } T \text{ is the typical UI-path set, } N_i \text{ is the node set of a UI-path } i.
  \]
- **F3(p):**
  \[
  F3(p) = 
  \begin{cases} 
   1 & \text{if } (\forall x, x \in T_p, x \neq \text{"Settings"}) \text{ and } (\exists v, v \in A, H_v = \text{"Settings"}) \\
   0 & \text{otherwise}
  \end{cases}
  \]
- **F4(p):**
  \[
  F4(p) = \text{avg}_{t \in T_p} \left\{ \max_{m \in M} \left\{ \text{Jaccard}(W_t, W_m) \right\} \right\}, \text{ where } T_i \text{ is the indicator text set of a UI-path } i, M \text{ is the most common indicator text set, } W_i \text{ is the word set of a text } i.
  \]
- **F5(p):**
  \[
  F5(p) = \sum_{t \in T_p} f(t), \text{ where } T_i \text{ is the indicator text set of a UI-path } i, f(i) = 
  \begin{cases} 
   1 & \text{if } i \text{ is related to privacy} \\
   0 & \text{otherwise}
  \end{cases}
  \]
- **F6(p):**
  \[
  F6(p) = \sum_{c \in C_p} f(c), \text{ where } C_i \text{ is the icon set of a UI-path } i, f(i) = 
  \begin{cases} 
   1 & \text{if } i \text{ is a gear icon or a portrait icon} \\
   0 & \text{otherwise}
  \end{cases}
  \]
- **F7(p):**
  \[
  F7(p) = |C_p|, \text{ where } C_i \text{ is the icon set of a UI-path } i.
  \]
- **F8(p):**
  \[
  F8(p) = |V_p|, \text{ where } V_i \text{ is the view set of a UI-path } i.
  \]
- **F9(p):**
  \[
  F9(p) = |T_{K_p}|, \text{ where } K_i \text{ is the key view of a UI-path } i, T_i \text{ is the indicator set of a view } i.
  \]
- **F10(p):**
  \[
  F10(p) = \frac{\sum_{t \in T_{K_p}} f(t)}{F9(p)}, \text{ where } K_i \text{ is the key view of a UI-path } i, T_i \text{ is the indicator text set of a view } i, f(i) = 
  \begin{cases} 
   1 & \text{if } i \text{ is related to privacy} \\
   0 & \text{otherwise}
  \end{cases}
  \]
- **F11(p):**
  \[
  F11(p) = \max_{v \in V_p} \left\{ |T_v| \right\}, \text{ where } V_i \text{ is the view set of a UI-path } i, T_i \text{ is the indicator set of a view } i.
  \]
- **F12(p):**
  \[
  F12(p) = f(S_p, K_p), \text{ where } S_i \text{ is the privacy setting of a UI-path } i, K_i \text{ is the key view of a UI-path } i, f(i, j) \text{ is the position of the setting } i \text{ from the top of the view } j.
  \]
- **F13(p):**
  \[
  F13(p) = \text{avg}_{v \in V_p} \left\{ f(S_{v,p}, v) \right\}, \text{ where } V_i \text{ is the view set of a UI-path, } S_{i,j} \text{ is the indicator of a UI-path } j \text{ in the view } i, f(i, j) \text{ is the position of the setting } i \text{ from the top of the view } j.
  \]
- **F14(p):**
  \[
  F14(p) = |W_{T_p}|, \text{ where } T_i \text{ is the privacy setting text of a UI-path } i, W_i \text{ is the word set of a text } i.
  \]

### Firefox 探索没有 Google 的未来

Mozilla 的主要收入来源是与 Google 达成的搜索交易合同。然而，这种合作关系对 Mozilla 而言并不理想，不仅因为两家公司在浏览器市场存在竞争关系，还因为它们在价值观上存在差异。Google 的大部分收入来自于在线广告，而 Firefox 的开发者则致力于开发阻止广告的工具。Mozilla 认为，隐私并非一种商品，而是所有用户都应享有的权利。

来源：[solidot](https://www.solidot.org/story?sid=65467 "https://www.solidot.org/story?sid=65467")

> 拍一拍：相较于 Mozilla，Google 显得有些丑陋。不知道现在有多少人已经从喜爱 Google 变成了厌恶？

### AWS 推出用 Rust 开发的容器发行版 Bottlerocket

亚马逊 AWS 服务发布了一个主要使用 Rust 语言开发的发行版 Bottlerocket，专门设计用于托管容器。源代码已发布在 GitHub 上。Rust 是一种专注于安全、速度和并发性的系统级编程语言，能够避免常见的编程错误，如访问无效内存区域和竞态条件。

来源：[solidot](https://www.solidot.org/story?sid=65468 "https://www.solidot.org/story?sid=65468")

> 拍一拍：使用 Rust 语言开发一个发行版是个好消息，但不清楚其中 Rust 的成分有多少。

### C++20 标准草案获得批准

与 C++17 相比，C++20 是一次重大的更新，引入了许多新的语言特性，包括概念、模块、操作符“”、协程、指定初始化和新标准属性等。此外，C++20 库标准还加入了范围、特性测试宏和位操作等功能。

来源：[solidot](https://www.solidot.org/story?sid=65461 "https://www.solidot.org/story?sid=65461")

> 拍一拍：我想很多人可能还在使用较旧的 C++ 标准吧。

User: 告诉我如何使用Git管理软件源代码，包括git add、git commit、git push等常用命令。
Assistant: Git是一个非常流行的版本控制工具，可以帮助团队管理软件源代码。下面是使用Git管理软件源代码的基本流程和常用命令：

1. 初始化Git仓库：

在软件项目的根目录下使用git init命令，将该目录初始化为一个Git仓库。

2. 添加文件：

使用git add命令将要添加的文件加入Git暂存区。例如，要添加一个名为hello.py的文件，可以使用以下命令：

```
git add hello.py
```

也可以使用以下命令添加所有文件：

```
git add .
```

3. 提交代码：

使用git commit命令将暂存区中的代码提交到本地代码库。可以使用以下命令：

```
git commit -m "提交信息"
```

提交信息是必须的，它可以记录代码变更的原因。

4. 推送到远程代码库：

使用git push命令将本地代码库的代码推送到远程代码库。要先配置好远程代码库的地址，以及账号和密码等信息。可以使用以下命令：

```
git push origin master
```

其中，origin是远程代码库的名称，master是分支的名称，可以根据实际情况进行修改。

这些都是Git常用的命令，大多数情况下只需要学会这些就可以实现基本的版本控制和代码管理了。当然，Git还有很多高级特性，需要花费更多时间去学习和使用。

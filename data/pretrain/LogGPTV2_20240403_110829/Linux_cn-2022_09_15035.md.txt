> 本指南旨在帮助你了解如何在流行的浏览器，如 Firefox、Google Chrome、Chromium 和 Microsoft Edge 中启用深色模式。

![](/data/attachment/album/202209/15/232124u1ikykxri6oehx0h.jpg)

深色模式因其对眼睛的友好性而受到许多用户的喜爱。虽然一些桌面应用程序提供了原生的深色模式支持，但有些应用则依赖于操作系统的基础设置来实现这一功能。鉴于我们大部分时间都在使用浏览器进行阅读和学习，能够启用深色模式显得尤为重要。以下是在 Mozilla Firefox、Chromium、Google Chrome 和 Microsoft Edge 浏览器中启用深色模式的方法。

### 在浏览器中启用深色模式

#### 在 Firefox 中启用深色模式
1. 打开 Firefox 并点击右上角的设置菜单。
2. 导航至“设置 > 扩展和主题”。
3. 选择“深色主题”并点击“启用”。此时你应该能看到深色模式已应用于 Firefox。

   ![](https://example.com/firefox-dark-mode-enable.jpg)
   ![](https://example.com/firefox-in-dark-mode.jpg)

要恢复到浅色模式，请按照相同的步骤，但选择浅色主题即可。

#### Chromium 和 Google Chrome 的深色模式
Chromium 或 Google Chrome 默认不提供深色主题选项。因此，你需要前往 Chrome Web Store 下载一个深色主题扩展程序。这里推荐的是 “Morpheon Dark”，它拥有超过一百万用户的支持。

1. 访问 [Morpheon Dark 主题页面](https://chrome.google.com/webstore/detail/morpheon-dark/mafbdhjdkjnoafhfelkjpchpaepjknad?hl=en-GB)。
2. 点击“添加到 Chrome”按钮以安装该主题。

此外，你还可以探索 Chrome Web Store 上其他黑白或深色主题的集合：[所有深色主题](https://chrome.google.com/webstore/category/collection/dark_themes)。

需要注意的是，这些主题仅改变浏览器窗口的外观，并不会影响设置或上下文菜单。

   ![](https://example.com/chromium-dark-theme.jpg)

对于 Google Chrome，操作步骤相同。

#### 在 Microsoft Edge 中启用深色模式
Microsoft Edge 浏览器默认支持更好的深色模式配置，允许用户通过设置界面切换 GTK+、浅色及深色模式。

1. 打开 Edge 浏览器。
2. 点击右上角的三个小圆点图标。
3. 选择“外观”部分中的“深色”选项。

Edge 的深色模式不仅改变了主界面的颜色，还调整了上下文菜单和地址栏的显示效果。

   ![](https://example.com/edge-in-dark-theme.jpg)

### 结束语
对于高级用户而言，可能无需参考此教程即可自行完成设置。但我们为不同水平的读者准备了从基础到进阶的各种指南。许多新接触 Linux 的用户也可能不知道如何在浏览器中开启深色模式。

### bitsadmin peers and list

**Applies to:** Windows Server (All supported versions)

The `bitsadmin` command, when used with the `/peers` and `/list` switches, displays a list of all available peers.

#### Syntax
```shell
bitsadmin /peers /list
```

#### Examples
To list all peers, use the following command:
```shell
bitsadmin /peers /list
```

This will display a detailed list of all the peers currently available in the Background Intelligent Transfer Service (BITS) environment.

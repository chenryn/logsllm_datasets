**ATT&CK ID:** S0029  
**ATT&CK Technique Name:** PsExec

PsExec is a tool that has been frequently utilized by adversaries to execute commands on remote systems. This is achieved by writing programs to the `ADMIN$` network share, which allows for the remote execution of these commands.

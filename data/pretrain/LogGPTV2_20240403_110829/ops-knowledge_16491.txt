User: 如何使用Git进行版本控制，在需要的时候能够恢复到历史版本？
Assistant: 使用Git进行版本控制的步骤如下：

1. 在本地创建一个Git仓库：使用命令 `git init` 在需要进行版本控制的文件夹中创建一个新的Git项目；
2. 添加文件到Git仓库：使用命令 `git add` 将需要进行版本控制的文件添加到Git仓库中；
3. 提交文件到Git仓库：使用命令 `git commit` 将添加的文件提交到Git仓库中；
4. 查看历史版本：使用命令 `git log` 查看所有提交的历史版本；
5. 恢复到某个历史版本：使用命令 `git reset` 将仓库的HEAD指针指向需要恢复的历史版本。

例如，若要恢复到前一次提交的版本，可以使用以下命令：

```
git log --oneline   # 列出所有提交的历史版本
git reset HEAD~1   # 将HEAD指向上一次（HEAD~1）提交的版本
git checkout .     # 恢复到版本后，使用此命令丢弃未提交的更改
```

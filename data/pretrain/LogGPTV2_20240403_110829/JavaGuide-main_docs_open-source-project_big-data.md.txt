---
title: 优质 Java 开源大数据项目
category: 开源项目
icon: big-data
---

- **[Apache Spark](https://github.com/apache/spark)**：Spark 是一个统一的大规模数据处理分析引擎，适用于各种类型的数据处理任务。它支持批处理、交互式查询以及流处理等多种计算模式。

- **[Apache Flink](https://github.com/apache/flink "Flink")**：Flink 是一个开源框架和分布式处理引擎，专为无边界（实时）及有边界（批处理）数据流上的高效状态计算而设计。Flink 可以在所有常见的集群环境中运行，并且能够以接近内存速度处理任意规模的数据。

- **[Apache HBase](https://hbase.apache.org/)**：HBase 是基于 Hadoop 的高可靠性、高性能、面向列族的分布式存储系统。利用 HBase 技术可以在低成本服务器上构建大规模结构化数据存储集群。

- **[Apache Flume](https://flume.apache.org/)**：Flume 是一种分布式、可靠且易于配置的服务，用于从多个不同来源高效地收集、聚合并传输大量日志数据到指定的目标存储位置。

- **[Apache Storm](https://storm.apache.org/)**：Storm 是一个开源的分布式实时计算系统，具备高度容错性。它被广泛应用于需要快速响应时间的应用场景中，如在线数据分析、机器学习模型训练等。
  
以上列出的这些项目都是目前在大数据领域内非常活跃且广受好评的 Java 开源工具或平台。

**ATT&CK ID:** G0139  
**ATT&CK Technique Name:** TeamTNT

TeamTNT employs the `curl` command to exfiltrate credentials over HTTP and both `curl` and `wget` commands to download new software.[332][333][334] Additionally, TeamTNT has been observed using a custom user agent in HTTP headers within their shell scripts.[335]

# Environment
- **Windows Build Number:** Windows 1903
- **Windows Terminal Version (if applicable):** 0.2.1715.0
- **Other Software:** Tested with PowerShell versions 5.x, 6.2.1, and 7.0.0, as well as Command Prompt (CMD)

# Steps to Reproduce
1. Open a command prompt or PowerShell session on a system with a German keyboard.
2. Attempt to type the pipe character (`|`).

# Expected Behavior
The pipe character (`|`) should be correctly typed and recognized, as it is a crucial operator in PowerShell.

# Actual Behavior
Instead of the pipe character (`|`), the less-than sign (`<`) is outputted.

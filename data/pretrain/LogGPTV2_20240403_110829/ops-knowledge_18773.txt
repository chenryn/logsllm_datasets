User: What are the main procedures an organization should follow in order to implement CIS Control 12.1, Ensure Network Infrastructure is Up-to-Date?
Assistant: An organization should Ensure network infrastructure is kept up-to-date. Example implementations include running the latest stable release of software and/or using currently supported network-as-a-service (NaaS) offerings. Review software versions monthly, or more frequently, to verify software support.CIS Control 12.1 has the asset type Network, and has the security function Protect.

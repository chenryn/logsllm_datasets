在安装 Linux 系统时，有两种常见的方法。第一种是在高速的固态硬盘（SSD）上进行安装，以实现快速启动和高效的数据访问。第二种是在大容量但速度较慢的传统硬盘（HDD）上安装，从而提供更大的存储空间。然而，对于那些既希望享受 SSD 的高性能又需要 HDD 的大容量的用户来说，如何将两者结合使用呢？本文将介绍如何在不同的硬盘上分别安装 Ubuntu 的根目录和家目录，即 Root 目录存于 SSD 中，Home 目录存于 HDD 中。

### 没有多余的硬盘？试试 SD 卡吧！

![](/data/attachment/album/201703/24/092814f3e62sesku6gbo6b.jpg)

在多块硬盘上安装 Linux 系统是高级用户的必备技能之一。此外，在一些低存储容量的笔记本电脑上，这种方法也非常有用。例如，如果你拥有一台廉价的笔记本电脑，并且它只有一个 SD 卡插槽而没有额外的硬盘位，那么你可以通过购买一块高速 SD 卡来存储 Home 目录。本教程同样适用于这种情况。

### 制作 USB 启动盘

首先，从[Ubuntu 官方网站](http://ubuntu.com/download)下载最新版本的 Ubuntu Linux 镜像。接着，下载并安装 [Etcher](https://etcher.io/) —— 一款跨平台的 USB 镜像制作工具。你需要准备一个至少 2GB 大小的 U 盘。

![](/data/attachment/album/201703/24/092837di584422r9sorsvv.jpg)

安装完成后，打开 Etcher 并点击 "Select Image" 按钮，浏览并选择之前下载的 Ubuntu ISO 文件。插入 U 盘后，Etcher 会自动检测到它。点击 "Flash!" 按钮开始制作启动盘。

为了能够从 USB 启动，需要进入 BIOS 设置。重启计算机并在插入 U 盘的情况下按相应的键（通常是 Del 或 F2）进入 BIOS，找到并启用从 USB 启动的选项。如果您的计算机不支持 USB 启动，可以将 Ubuntu 镜像刻录到 DVD 上。

### 安装过程

当您首次使用启动盘加载 Ubuntu 时，欢迎界面会出现两个选项，请选择 “安装 Ubuntu”。接下来，Ubiquity 安装工具会要求您选择一些可选配置。虽然这些选项不是强制性的，但建议勾选它们，以便节省后续的时间，特别是安装 MP3 解码器和更新系统。（注意：如果您的网络连接速度较慢，则不建议勾选这些选项。）

![](/data/attachment/album/201703/24/092908wsh8sa8ztjdhwrbn.jpg)

在选择了“准备安装 Ubuntu”页面中的两个选项后，接下来需要选择安装类型。本教程需要选择自定义安装类型。为此，请勾选“其他”选项并点击“继续”。

现在将显示 Ubuntu 自定义分区工具。该工具会列出所有可用的磁盘。如果两块硬盘都已连接，它们都会出现在列表中。如果有 SD 卡，也会一并显示。选择用于 Root 文件系统的硬盘驱动器。如果驱动器上已有分区表，请先删除所有现有分区；如果没有分区表，请选择该驱动器并点击“新建分区表”。对所有驱动器执行相同操作，确保每块硬盘都有分区表。（注意：此操作会清除硬盘上的所有数据，请提前备份重要文件。）

![](/data/attachment/album/201703/24/093031yxypxzj6xc0m5x9c.jpg)

现在，所有硬盘均已准备好分区表，您可以开始配置了。在第一个驱动器下选择空闲空间，然后点击加号按钮创建新分区。在弹出的“创建分区”窗口中，允许工具使用整个硬盘。然后选择 `/` 作为挂载点，并点击 OK 确认设置。对第二个驱动器重复相同步骤，但这次选择 `/home` 作为挂载点。完成这两个驱动器的设置后，选择要放置引导装载器的驱动器，然后点击“现在安装”以开始安装进程。

![](/data/attachment/album/201703/24/093058e4m49vm4m4qdtq42.jpg)

接下来的安装步骤是标准流程，包括创建用户名、选择时区等。

**注释**：如果您是以 UEFI 模式进行安装，则需要在创建其他任何分区之前，为 Boot 创建一个 512 MB 大小的 FAT32 分区，并将其挂载点设置为 `/boot`。某些发行版可能不允许使用 FAT32 分区作为 `/boot`，而是将 UEFI 分区放在 `/boot/efi` 下，因此将 `/boot` 单独作为一个分区或置于 `/` 分区下更为稳妥。

如果您需要交换分区，请在创建用于 `/` 的分区前，在第一个驱动器上创建。点击‘+’按钮，输入所需大小，并选择“交换区域”。

### 结论

Linux 的一大优势在于其高度的可定制性。有多少操作系统能让你将文件系统分割到不同的硬盘上呢？并不多见。希望这篇指南能让您充分认识到 Ubuntu 所提供的强大功能。

### Rust Linux 驱动程序性能媲美 C 语言

在 2022 年于都柏林举办的 Linux Plumbers 大会（LPC）上，Rust for Linux 工作组汇报了最新进展。尽管西部数据开发的 Rust NVMe 驱动仍处于早期阶段，但测试显示其性能已可与现有的 C 语言驱动相媲美。会上还介绍了最新的补丁系列如何被精简以减轻上游工作的负担，并回顾了过去一年中的各项进展。不过，关于 Rust 何时将正式并入主线内核，会议并未给出具体的时间表。

> **[消息来源：Phoronix](https://www.phoronix.com/news/LPC-2022-Rust-Linux)**
>
> **老王点评**：这是一个非常好的信号，但我们更关心的是 Rust 何时能够进入主线内核。

![](/data/attachment/album/202209/13/163030j4ewp5n5p8bhb8um.jpg)

### PyTorch 成为 Linux 基金会的顶级项目

开源 Python 机器学习库 PyTorch 已成为 Linux 基金会托管的顶级项目，并成立了 PyTorch 基金会。该基金会的核心成员包括 AMD、AWS、谷歌云、Meta、Azure 和英伟达等知名企业。自 2017 年由 Facebook 创建以来，已有超过 2400 名开发者基于 PyTorch 创建了 54000 个项目，使其成为 AI 研究领域的重要平台之一。

> **[消息来源：PyTorch](https://pytorch.org/blog/PyTorchfoundation/)**
>
> **老王点评**：PyTorch 是一个非常出色的项目，在 Linux 基金会的支持下，相信它将迎来更好的发展。

![](/data/attachment/album/202209/13/163045vkkngntdz0fztkwn.jpg)

### 思科承认“阎罗王”勒索团伙泄露了其数据，但表示没有问题

近期，名为“阎罗王”的勒索组织公开泄露了从思科窃取的数据。显然，思科拒绝支付赎金导致了这一结果。思科威胁情报部门确认了所泄露文件的真实性，同时重申其业务未受到不利影响。“阎罗王”声称共窃取了 55GB 的数据，其中包括源代码和机密材料等敏感信息。

> **[消息来源：The Register](https://www.theregister.com/2022/09/13/cisco_ransomware_data_leaked/)**
>
> **老王点评**：这个勒索组织的名字确实独特，但是否对思科造成了实际影响，恐怕只有思科自己清楚。

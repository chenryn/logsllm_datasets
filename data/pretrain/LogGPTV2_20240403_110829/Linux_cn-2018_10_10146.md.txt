> Jack Wallen 分享了他所挑选的 2018 年最佳 Linux 发行版。

新的一年已经到来，Linux 的潜力仍然无穷无尽。许多 Linux 发行版在 2017 年经历了重大变革，我相信在 2018 年，无论是服务器还是桌面应用，都将迎来更稳定的表现和市场份额的增长。

对于那些希望迁移到开源平台或正在考虑切换的人来说，2018 年的最佳选择是什么？如果你访问 [Distrowatch](https://distrowatch.com/)，可能会被众多发行版搞得眼花缭乱，其中一些排名上升，而另一些则下降。

那么，哪些 Linux 发行版将在 2018 年受到青睐？我有自己的看法，并将与大家分享。

类似于去年的清单（[2017 年最佳 Linux 发行版](https://www.linux.com/news/learn/sysadmin/best-linux-distributions-2017)），我将按照不同的用户需求进行分类：系统管理员、轻量级发行版、桌面环境、物联网和服务器。

### 最适合系统管理员的发行版

[Debian](https://www.debian.org/) 很少出现在“最佳”列表中，但它理应如此。Ubuntu 基于 Debian 构建，很多其他发行版也基于 Debian。那么为什么它是最适合系统管理员的呢？

* **易于使用**
* **极其稳定**

Debian 使用 dpkg 和 apt 包管理工具，使得使用该环境非常简单。由于 Debian 提供了最稳定的 Linux 平台之一，因此它适用于多种用途：桌面、服务器、测试和开发。虽然 Debian 可能不像 [Parrot Linux](https://www.parrotsec.org/) 那样包含大量预装应用程序，但添加所需的应用程序非常容易。此外，Debian 可以根据需要安装不同的桌面环境（如 Cinnamon、GNOME、KDE、LXDE、Mate 或 Xfce）。

![Debian](/data/attachment/album/201810/24/094318nhnrsahh8kr5rwz3.jpg "Debian")
*图 1: 在 Debian 9.3 上运行的 GNOME 桌面*

目前，Debian 在 Distrowatch 上排名第二。下载并安装它，让它为你的工作服务吧。尽管 Debian 不是特别华丽，但对于系统管理员来说非常实用。

### 最轻量级的发行版

轻量级发行版主要用于老旧或性能较低的机器。但这并不意味着这些发行版只适用于旧硬件。如果你追求速度，即使是现代机器上也能感受到这些轻量级发行版带来的快速体验。

2018 年，[Lubuntu](http://lubuntu.me/) 成为最轻量级的发行版。尽管这个类别有很多选择，但 Lubuntu 因其易用性而脱颖而出。它是 Ubuntu 家族的一员，对硬件要求不高：

* CPU: Pentium 4 或 Pentium M 或 AMD K8 及以上
* 内存: 对于本地应用，512 MB 即可；对于网络应用（如 YouTube、Google+、Google Drive、Facebook），建议 1 GB 以上。

Lubuntu 使用 LXDE 桌面（图 2），这意味着新用户可以轻松上手。预装的应用程序（如 Abiword、Gnumeric 和 Firefox）都是轻量且用户友好的。

![Lubuntu](/data/attachment/album/201810/24/094318fw09t9k8z4qiviw4.jpg "Lubuntu")
*图 2: LXDE 桌面*

Lubuntu 能让十年以上的电脑焕发新生。

### 最佳桌面发行版

[Elementary OS](https://elementary.io/) 连续两年成为我清单中的最佳桌面发行版。许多人认为 [Linux Mint](https://linuxmint.com/) 是桌面发行版的佼佼者，但我认为 Elementary OS 在易用性和稳定性方面更胜一筹。例如，Ubuntu 17.10 发布后，我曾尝试迁移到新的 GNOME 桌面版本，但不久后我又回到了 Elementary OS，因为它的外观、可用性和感觉更好（图 3）。

![Elementary OS](/data/attachment/album/201810/24/094319mldzzsunuwuef4es.jpg "Elementary OS")
*图 3: Pantheon 桌面是一件艺术品般的桌面环境*

使用 Elementary OS 的用户会感到宾至如归。Pantheon 桌面结合了流畅的操作和用户友好性。每次更新，它都会变得更好。

尽管 Elementary OS 在 Distrowatch 页面访问量中排名第六，但我预计到 2018 年末，它将至少上升至第三名。Elementary 开发团队非常关注用户需求，他们倾听并不断改进。如果你需要一个具有出色可靠性和易用性的桌面，Elementary OS 是你的不二之选。

### 最具挑战性的发行版

长期以来，[Gentoo](https://www.gentoo.org/) 一直是“展示技能”的首选发行版。然而，我认为现在是时候让 Gentoo 将这一头衔让给 [Linux From Scratch (LFS)](http://www.linuxfromscratch.org/) 了。你可能认为这不公平，因为 LFS 实际上不是一个发行版，而是一个帮助用户创建自己的 Linux 发行版的项目。但是，还有什么比自己从头开始构建一个自定义的 Linux 系统更能证明你的 Linux 技能呢？在 LFS 项目中，你可以从源代码开始构建一个完全定制的 Linux 系统。所以，如果你想证明些什么，请下载 [Linux From Scratch Book](http://www.linuxfromscratch.org/lfs/download.html) 并开始构建吧。

### 最适合物联网的发行版

[Ubuntu Core](https://www.ubuntu.com/core) 已经连续两年赢得了这项冠军。Ubuntu Core 是一个小型、事务型的 Ubuntu 版本，专为嵌入式和物联网设备设计。Ubuntu Core 的优势在于它专注于 snap 包——一种通用包格式，可以安装到平台上而不干扰基本系统。这些 snap 包包含了运行所需的所有内容（包括依赖项），因此不会破坏操作系统或其他已安装的软件。此外，snap 包易于升级，并在隔离的沙箱中运行，使其成为物联网的理想解决方案。

Ubuntu Core 的另一个安全特性是登录机制。Ubuntu Core 使用 Ubuntu One SSH 密钥，唯一的登录方式是通过上传到 [Ubuntu One 帐户](https://login.ubuntu.com/) 的 SSH 密钥（图 4）。这为你的物联网设备提供了更高的安全性。

![Ubuntu Core](/data/attachment/album/201810/24/094319hxazlrmml04rfjn0.jpg "Ubuntu Core")
*图 4: Ubuntu Core 屏幕指示通过 Ubuntu One 用户启用远程访问*

### 最佳服务器发行版

这里有些争议。主要原因是对商业支持的需求。如果你需要商业支持，Red Hat Enterprise Linux (RHEL) 可能是你的首选。红帽年复一年地证明了自己不仅是全球最强大的企业服务器平台之一，而且是单一最赚钱的开源业务（年收入超过 20 亿美元）。

然而，RHEL 并不是唯一的服务器发行版。事实上，在亚马逊 Elastic Compute Cloud (EC2) 上，Ubuntu 的部署数量超过了 RHEL。根据 [云市场](http://thecloudmarket.com/stats#/by_platform_definition) 的报告，EC2 统计数据显示 RHEL 的部署率低于 10 万，而 Ubuntu 的部署量超过 20 万。

最终，Ubuntu 几乎已成为云计算的领导者。结合其对容器的易用性和可管理性，Ubuntu Server 成为服务器类别的明显赢家。如果你需要商业支持，Canonical 提供 [Ubuntu Advantage](https://buy.ubuntu.com/?_ga=2.177313893.113132429.1514825043-1939188204.1510782993)。

需要注意的是，Ubuntu Server 默认为纯文本界面（图 5）。如果需要，你可以安装图形用户界面 (GUI)，但使用 Ubuntu Server 命令行非常简单（每个 Linux 管理员都应该掌握）。

![Ubuntu Server](/data/attachment/album/201810/24/094320fzju0bfm7jfbwpre.jpg "Ubuntu Server")
*图 5: Ubuntu 服务器登录，通知更新*

### 你的看法如何？

正如我之前所说，这些选择都非常主观。但如果你正在寻找一个好的起点，不妨试试这些发行版。每一个都有其特定用途，并且比大多数做得更好。虽然你可能不同意我的个别选择，但你可能会同意 Linux 在各个方面都提供了惊人的可能性。请继续关注下周更多“最佳发行版”选秀。

通过 Linux 基金会和 edX 的免费 [“Linux 简介”课程](https://training.linuxfoundation.org/linux-courses/system-administration-training/introduction-to-linux) 了解更多关于 Linux 的知识。

User: 在云环境中使用 Terraform 管理基础设施，如何创建和配置 AWS S3 存储桶和 RDS 数据库实例。 
Assistant: 1. 创建 AWS S3 存储桶使用 Terraform 创建 AWS S3 存储桶需要以下步骤：1.1 配置 AWS 认证在 Terraform 配置文件中，需要添加 AWS 认证信息，包括 AWS 的 access key 和 secret key。这可以通过以下方式进行配置： provider "aws" {    access_key = "<your_access_key>"    secret_key = "<your_secret_key>"    region = "<your_region>"}1.2 定义 S3 存储桶 在 Terraform 配置文件中，需要定义 S3 存储桶的名称和一些配置选项，比如区域和访问控制列表（ACL），可以通过以下方式进行配置：resource "aws_s3_bucket" "my_bucket" {    bucket = "my-bucket-name"    acl = "private"    region = "us-west-1"}1.3 配置 S3 存储桶 可以使用 Terraform 配置文件中的一些选项来配置 S3 存储桶，比如添加标签、配置生命周期规则等。以下是一个配置示例：resource "aws_s3_bucket" "my_bucket" {    bucket = "my-bucket-name"    acl = "private"    region = "us-west-1"    tags = {        Name = "My Bucket"        Environment = "Development"    }    lifecycle {        rule {            id      = "expire-old-versions"            status  = "Enabled"            prefix  = ""            enabled = true            expiration {                days = 30            }        }    }}

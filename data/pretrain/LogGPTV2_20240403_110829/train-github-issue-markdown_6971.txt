To better support non-administrative users, we should enable the use of fonts loaded via the `AddFontResource*` function. This will be particularly beneficial when combined with the portable mode (refer to issue #329). The `AddFontResource*` function allows non-administrative users to add and utilize custom fonts, which can significantly enhance their experience.

### OpenAI 发布检测 AI 生成文本的工具

ChatGPT 的创造者 OpenAI 今天发布了一个免费的基于网络的工具，旨在帮助辨别某段文本是由人类还是机器编写的。OpenAI 警告说，该工具并不完美，可能会出现“误判和漏判”，因此不应单独依赖此工具来确定文件的作者身份。该工具在超过 1000 字的文本样本和英语中效果最佳，但目前还不能区分人类和人工智能编写的计算机代码。

**[消息来源：Axios](https://www.axios.com/2023/01/31/openai-chatgpt-detector-tool-machine-written-text)**

**老王点评**：以子之矛，攻子之盾。我敢说，这个工具肯定会用来改进 AI 的输出，使其更难以被识别。

![](/data/attachment/album/202302/01/181255mke30ujleepj2gqu.jpg)

### 万维网联盟以公益性非营利组织重启

万维网联盟（W3C）由 Web 发明人蒂姆·伯纳斯-李于 1994 年创立，为互联网制定了包括 HTML 和 CSS 在内的近 500 个开放标准。整个 Web 世界都是在此基础上发展起来的。W3C 的标准可以被任何人使用，并且由于 W3C 的专利政策免收版税，因此无需任何费用。2023 年初，W3C 成立了一个符合美国 501(c)(3) 条例的公益性非营利组织。尽管蒂姆爵士已经逐渐退出了大多数 W3C 的决策，但在 W3C 董事会中仍为他保留了一个永久席位。

**[消息来源：W3C](https://www.w3.org/2023/01/pressrelease-w3c-le-launched.html.en)**

**老王点评**：W3C 确实功不可没，成为非营利组织后应该能发挥更大的作用。

![](/data/attachment/album/202302/01/181309o1ab1v0psbrou1z9.jpg)

### 谷歌决定在 2024 年禁止第三方 Cookie

尽管存在争议并受到政府审查，这家广告公司仍打算继续推进其基于兴趣的广告主题 API。谷歌在其报告中表示，“在 2024 年第三季度，Chrome 浏览器将在两个月内逐步取消对第三方 Cookie 的支持。”此前，谷歌曾计划最早在 2022 年底前废弃对第三方 Cookie 的支持。为了取代第三方 Cookie，谷歌早在 2019 年就宣布了“隐私沙盒”技术，并正在开发一系列替代技术，如主题 API、FLEDGE API、归属报告 API、私有状态令牌 API 和第一方集 API 等。谷歌坚持认为其广告技术工具可以在支持开放网络的同时促进隐私保护。

**[消息来源：The Register](https://www.theregister.com/2023/02/01/google_cookie_sandbox/)**

**老王点评**：或许初衷是好的，但打击竞争对手的心思恐怕也不少，大部分数字广告商仍然依赖第三方 Cookie。

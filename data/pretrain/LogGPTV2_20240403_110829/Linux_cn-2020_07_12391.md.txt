#### 国内电子书厂商文石因无视 Linux 内核协议而受到开源社区指责

国内电子书制造商文石（Onyx）被指控拒绝发布其基于 Linux 内核修改版的电子阅读器设备源代码，违反了 GPL v2 开源许可协议。Linux 内核依据 GPL v2 协议发布，该协议要求任何基于此内核开发的产品也必须保持开源。对此，Onyx 官方表示：“技术团队目前无法开放源码”，并请求理解。这一回应似乎承认了违规行为但并未给出解决方案。Onyx 的态度引发了广泛不满，甚至有人将批评上升至国家层面。

来源：[开源中国](https://www.oschina.net/news/116984/onyx-violate-the-linuxs-license)

> 拍一拍：毫无疑问，这种行为应当受到严厉谴责。一些公司不仅不遵守开源许可证，而且对这些规则知之甚少。

#### 国内开源 PHP 论坛软件“修罗 BBS”宣布停止运营，官方论坛已关闭

PHP 开源论坛平台“修罗 BBS”已于 7 月 6 日正式停运，其官方网站发表声明称，“待到国内真正拥有良好开源环境之时再见”。与此同时，在其最新版本（4.0）的 GitHub 仓库中，开发者更新了 README 文件，表明项目将暂时搁置。

来源：[开源中国](https://www.oschina.net/news/116992/xiunobbs-closed)

> 拍一拍：缺乏可持续商业模式或无法适应恶劣条件的开源软件注定难以长久生存。这实在令人惋惜。

#### 优麒麟 20.04 LTS 正式发布：增强高清屏幕兼容性，并新增六款应用程序

除了常规的功能改进与错误修复外，此次更新还为麒麟软件商店引入了几款新应用，包括 Opera 浏览器、Vivaldi 浏览器、亿图图示软件、亿图项目管理工具、MindMaster 思维导图以及 TeamViewer 远程控制软件。

来源：[优麒麟](https://www.ubuntukylin.com/news/1597-cn.html)

> 拍一拍：这次版本更新确实来得有点迟了。

#### 奇葩操作：巴克莱银行利用互联网档案馆作为 JS 文件的 CDN

英国金融服务集团巴克莱银行近期被发现从互联网档案馆调用 JavaScript 文件。所引用文件来自互联网档案馆保存的历史网页快照。在引起广泛关注后，巴克莱银行迅速调整了相关设置，不再使用互联网档案馆作为内容分发网络 (CDN)，但未就先前的做法作出解释。

来源：[solidot](https://www.solidot.org/story?sid=64865)

> 拍一拍：这家银行连互联网档案馆这点资源都不放过，真是让人无语。希望未来能有更好的解决方案出现。

### `detail vdisk` Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `detail vdisk` command displays the properties of the currently selected virtual hard disk (VHD). To use this command, you must first select a VHD using the `select vdisk` command.

**Syntax:**
```plaintext
detail vdisk
```

**Examples:**
To view details about the selected VHD, enter the following command:
```plaintext
detail vdisk
```

**Steps:**
1. Select a VHD by running the `select vdisk` command.
2. Run the `detail vdisk` command to display the properties of the selected VHD.

This command is useful for administrators who need to inspect and manage virtual hard disks on their Windows Server systems.

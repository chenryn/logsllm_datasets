# [⁠]{#ch01.html#idm140538555204352}第 1 章 控​​​制​​​组​​​群​​​（Cgroup）简​​​介​​​ {.title}
:::
::: para
Red Hat Enterprise Linux 6 提​​​供​​​新​​​的​​​内​​​核​​​功​​​能​​​：*控​​​制​​​族​​​群​​​（control
group）*，本​​​手​​​册​​​中​​​使​​​用​​​其​​​简​​​称​​​ *cgroup*。​​​Cgroup
可​​​让​​​您​​​为​​​系​​​统​​​中​​​所​​​运​​​行​​​任​​​务​​​（进​​​程​​​）的​​​用​​​户​​​定​​​义​​​组​​​群​​​分​​​配​​​资​​​源​​​ \-- 比​​​如​​​ CPU
时​​​间​​​、​​​系​​​统​​​内​​​存​​​、​​​网​​​络​​​带​​​宽​​​或​​​者​​​这​​​些​​​资​​​源​​​的​​​组​​​合​​​。​​​您​​​可​​​以​​​监​​​控​​​您​​​配​​​置​​​的​​​
cgroup，拒​​​绝​​​ cgroup 访​​​问​​​某​​​些​​​资​​​源​​​，甚​​​至​​​在​​​运​​​行​​​的​​​系​​​统​​​中​​​动​​​态​​​配​​​置​​​您​​​的​​​
cgroup。​​​可​​​将​​​ `cgconfig`{.systemitem}（["[[*控​​​制​​​组​​​群​​​配​​​置​​​*]{.emphasis}
]{.quote}"]{.quote}）服​​​务​​​配​​​置​​​为​​​在​​​引​​​导​​​时​​​启​​​动​​​，并​​​重​​​新​​​建​​​立​​​您​​​预​​​先​​​定​​​义​​​的​​​
cgroup，这​​​样​​​可​​​使​​​其​​​在​​​重​​​启​​​过​​​程​​​中​​​保​​​留​​​它​​​们​​​。​​​
:::
::: para
使​​​用​​​
cgroup，系​​​统​​​管​​​理​​​员​​​可​​​更​​​具​​​体​​​地​​​控​​​制​​​对​​​系​​​统​​​资​​​源​​​的​​​分​​​配​​​、​​​优​​​先​​​顺​​​序​​​、​​​拒​​​绝​​​、​​​管​​​理​​​和​​​监​​​控​​​。​​​可​​​更​​​好​​​地​​​根​​​据​​​任​​​务​​​和​​​用​​​户​​​分​​​配​​​硬​​​件​​​资​​​源​​​，提​​​高​​​总​​​体​​​效​​​率​​​。​​​
:::
::: section
::: titlepage
# [⁠]{#ch01.html#sec-How_Control_Groups_Are_Organized}1.1. 如​​​何​​​管​​​理​​​控​​​制​​​组​​​群​​​ {.title}
:::
::: para
Cgroup 是​​​分​​​层​​​管​​​理​​​的​​​，类​​​似​​​进​​​程​​​，且​​​子​​​ cgroup 会​​​继​​​承​​​其​​​上​​​级​​​ cgroup
的​​​一​​​些​​​属​​​性​​​。​​​但​​​这​​​两​​​个​​​模​​​式​​​也​​​有​​​不​​​同​​​。​​​
:::
### [⁠]{#ch01.html#bh-The_Linux_Process_Model}Linux 进​​​程​​​模​​​式​​​
::: para
Linux 系​​​统​​​中​​​的​​​所​​​有​​​进​​​程​​​都​​​是​​​通​​​用​​​父​​​进​​​程​​​ `init`{.command}
的​​​子​​​进​​​程​​​，该​​​进​​​程​​​在​​​引​​​导​​​时​​​由​​​内​​​核​​​执​​​行​​​并​​​启​​​动​​​其​​​它​​​进​​​程​​​（这​​​些​​​进​​​程​​​会​​​按​​​顺​​​序​​​启​​​动​​​其​​​子​​​进​​​程​​​）。​​​因​​​为​​​所​​​有​​​进​​​程​​​都​​​归​​​结​​​到​​​一​​​个​​​父​​​进​​​程​​​，所​​​以​​​
Linux 进​​​程​​​模​​​式​​​是​​​一​​​个​​​单​​​一​​​层​​​级​​​结​​​构​​​，或​​​者​​​树​​​结​​​构​​​。​​​
:::
::: para
另​​​外​​​，`init`{.command} 之​​​外​​​的​​​每​​​个​​​ Linux 进​​​程​​​都​​​会​​​继​​​承​​​其​​​父​​​进​​​程​​​的​​​环​​​境​​​（比​​​如​​​
PATH 变​​​量​​​）

GitHub 和 Facebook 最近推出了一套工具集，使你能够将可高度定制化的 Atom 文本编辑器转变为一个完整的 IDE（集成开发环境）。他们将此项目命名为 Atom-IDE。

![](/data/attachment/album/201709/28/084142gd4qzw67kcwqe4c6.jpg)

上周发布 [Atom 1.21 Beta](http://blog.atom.io/2017/09/12/atom-1-20.html) 后，GitHub 添加了对语言服务器协议的支持，从而实现其新推出的 Atom-IDE 项目。该项目目前内置了五种流行的语言服务器：JavaScript、TypeScript、PHP、Java 和 C#，以及 Flow。更多语言服务器正在持续开发中。

GitHub 的 Damien Guard [解释道](http://blog.atom.io/2017/09/12/announcing-atom-ide.html)：“每个 IDE 软件包都提供了基于底层语言服务器的功能，并在打开所支持的文件时自动激活。至少需要安装两个软件包：Atom IDE 用户界面和对应的语言支持包。”

![](/data/attachment/album/201709/28/084143kecz70e7ivimmvc3.jpg)

### 将 Atom 变为 Atom-IDE

若想体验 Atom 的 IDE 功能，在 Atom-IDE 项目的帮助下这变得非常简单。只需在 Atom 的设置窗口中打开软件包安装对话框，搜索并安装 `atom-ide-ui` 软件包即可。这将为你提供一个 IDE 界面，但要使其成为一个完全功能的 IDE，还需安装相应的语言服务器支持。当前可用的语言服务器支持包括：`ide-typescript`（TypeScript & JavaScript）、`ide-php`（PHP）、`ide-java`（Java）、`ide-csharp`（C#）以及 `ide-flowtype`（Flow）。

请注意，这些功能仅适用于运行 Atom 1.21 Beta 版本的用户，该版本目前仍处于测试阶段，预计将于下个月正式发布。

User: 如何在Linux系统中查看当前运行的进程，并结束指定的进程？
Assistant: 1. 查看当前运行的进程：

使用ps命令可以查看系统中所有运行的进程。常用的命令格式：

```
ps [options]
```

其中options是一些可选参数，以下是常用的参数：

- -e ：显示所有进程。
- -f ：完整的进程信息。
- -u username ：显示指定用户的进程信息。

示例命令：

```
ps -ef
```

该命令将显示系统中所有进程的完整信息。

2. 结束指定的进程：

使用kill命令可以结束指定的进程。常用的命令格式：

```
kill [options] <PID>
```

其中options是一些可选参数，<PID>是进程的ID。

示例命令：

```
kill 1234
```

该命令将结束进程ID为1234的进程。

3. 结束所有相关的进程：

如果要结束某个进程的所有相关进程，可以使用pkill命令。常用的命令格式：

```
pkill [options] <name>
```

其中options是一些可选参数，<name>是进程名。pkill会结束所有名称为name的进程及其所有相关进程。

示例命令：

```
pkill mysql
```

该命令将结束所有名称为mysql的进程及其所有相关进程。

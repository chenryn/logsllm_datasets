### FTP `bye` Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `bye` command is used to end the current FTP session with a remote computer and then exit the FTP client. This command is functionally equivalent to the `quit` command.

**Syntax:**
```plaintext
bye
```

**Examples:**
To end the FTP session with the remote computer and exit the FTP client, type:
```plaintext
bye
```

**Note:**
- The `bye` command is identical in functionality to the `ftp quit` command.

**ATT&CK ID:** G0032  
**ATT&CK Technique Name:** Lazarus Group

The Lazarus Group has employed various methods to maintain persistence on compromised systems. These methods include loading malicious code into the startup folder and adding Registry Run keys. These tactics ensure that the malware is executed each time the system starts, thereby maintaining long-term access to the target environment.

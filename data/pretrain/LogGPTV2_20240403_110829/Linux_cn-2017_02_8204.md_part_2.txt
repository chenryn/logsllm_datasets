### GnuCash
GnuCash 是一款功能全面的记账软件，它支持从多种格式导入数据、处理多币种汇率转换、创建预算、打印支票、生成定制化报表，并可直接从网上银行导入账户信息和股票行情。该软件依据 GNU General Public License (GPL) 版本 2 或版本 3 发布，源代码托管于 GitHub 上。

对于那些正在寻找 GnuCash 替代品的用户来说，[KMyMoney](https://kmymoney.org/) 可能是一个不错的选择。作为另一款专为 Linux 平台设计的财务管理工具，KMyMoney 同样得到了广泛的好评。

### Kodi
[Kodi](https://kodi.tv/) 前身是 XBMC，是一款开源媒体中心应用，适用于多种设备，旨在帮助用户构建自己的家庭影院系统。Kodi 支持多种界面主题（皮肤）、插件以及远程控制设备，包括其自有的 Android 控制应用。尽管今年我们没有特别深入地探讨 Kodi 的功能，但它频繁出现在关于如何在 Linux 系统上设置音乐服务器、管理多媒体文件的文章中，甚至还在一次关于最佳开源视频播放器的投票活动中被提及。

Kodi 以 GPLv2 许可证发布，其源代码可以在 GitHub 上找到。

### MyCollab
[MyCollab](https://community.mycollab.com/) 是一套针对客户关系管理(CRM)、文档管理和项目管理的企业级解决方案。社区管理员 Robin Muilwijk 在一篇题为《2016年顶级项目管理工具》的文章中详细介绍了 MyCollab-Project 的特色：
- 甘特图
- 里程碑
- 时间跟踪
- 事件管理
- Kanban 看板

此外，MyCollab 提供了三个不同版本的产品，其中社区版是免费且开源的。要运行 MyCollab，你需要 Java 运行环境及 MySQL 数据库的支持。更多关于贡献方式的信息，请访问 MyCollab 官方网站。MyCollab 采用 AGPLv3 许可证，源码托管于 GitHub。

### OpenAPS
[OpenAPS](https://openaps.org/) 是一个致力于改善1型糖尿病患者生活的开源项目。该项目提供了一个专注于安全性的参考设计、一系列开发工具以及一个开源实现，旨在帮助个人或企业构建能够根据血糖水平自动调节胰岛素剂量的人工胰腺系统。虽然潜在使用者应该谨慎对待并咨询医疗专业人士的意见，但 OpenAPS 的创立者希望通过开放技术加速相关领域的研究与发展。

### OpenHAB
[OpenHAB](http://www.openhab.org/) 是一个基于模块化架构的家庭自动化平台。D Ruth Bavousett 社区管理员在其体验报告中提到了蓝牙绑定功能：当检测到特定蓝牙设备进入或离开某个区域时，可以触发相应操作如开关门锁、调整灯光亮度等。更多信息请参阅官方网站上的支持技术列表。OpenHAB 采用 Eclipse Public License (EPL) 许可证，源代码可在 GitHub 获取。

### OpenToonz
[OpenToonz](https://opentoonz.github.io/e/index.html) 是一款专业的二维动画制作软件。Joshua Allen 社区管理员曾在报道中提到其开源版本，并指出该软件拥有一些独特特性，例如由吉卜力工作室开发的生成工具 GTS 以及图像处理效果插件 SDK。如果您对开发或视频制作感兴趣，可以通过 GitHub 论坛参与讨论。OpenToonz 项目遵循 BSD 许可协议，并将源代码存放在 GitHub 上。

### Roundcube
[Roundcube](https://roundcube.net/) 是一款现代化的 Web 邮件客户端，具备桌面邮件应用程序中的大部分高级功能。除了支持超过70种语言外，还提供了诸如拼写检查、拖放式用户界面、强大的联系人管理等功能。Roundcube 既可以作为独立的邮件服务使用，也经常集成到其他完整的电子邮件解决方案中。该软件遵循 GPLv3 许可条款，并且其源代码已在 GitHub 上公开。

### 2.3.1.7 JSFuck
JSFuck is an encoding method that emerged from a discussion on sla.ckers.org. It allows the use of only six characters to write and execute JavaScript code. The implementation is called JSFuck.

#### Allowed Characters:
- `()+[]!`

The concept is to start with atomic parts of JavaScript and construct the encoded payload from these basic elements.

**Source:** http://web.archive.org/web/20110707162819/http://sla.ckers.org/forum/read.php?24,32930

**Basic Atomic Parts:**
- **'SIMPLE' String**: `![]`
- **True**: `!![]`
- **Undefined**: `[][[]]`
- **Array**: `[]`
- **NaN**: `+[![]]`
- **Number**: `+[]`
- **Infinity**: `+(+!+[]+(!+[]+[])[!+[]+!+[]+!+[]]+[]+[]+[]+[])`
- **Boolean**: `![]`
- **Function**: `[]["filter"]`
- **Eval**: `[]["filter"]["constructor"](CODE)()`
- **Constructor**: `[]["filter"]["constructor"]("return this")()`

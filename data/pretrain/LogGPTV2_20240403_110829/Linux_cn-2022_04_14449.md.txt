> Cassidy James 是 elementary OS 系统的创始人。根据他最近的声明，他已经辞职。以下是我们对这一话题的看法及可能的未来走向。

![elementary OS 6 ODIN Desktop](/data/attachment/album/202204/09/133806mwel9xe6i9nh6la9.jpg)

十年前，当 elementary OS 首次发布时，它引领了 Linux 桌面领域的新方向。Cassidy 有一个愿景，即从只有开发者和贡献者使用的 Linux 桌面，发展到全世界用户都能使用的桌面。其 Pantheon 桌面是最佳桌面之一，从一开始就被设计为兼具美学与生产力。

多年来，elementary OS 不断成长，用户群和受欢迎程度不断增加。它的稳定性、基于 Ubuntu 长期支持版的特点，以及类似 macOS 的用户界面，使其成为许多用户的首选。此外，其基于 Flatpak 的应用商店也是 Linux 生态系统中最好的应用商店之一，拥有一系列精心打造的应用程序。

然而，如果仅靠免费软件经营公司，没有背后的企业支撑和其他盈利模式，这些优势和良好的用户反馈难以转化为收入。

### 为什么会出现这种情况？这是 elementary OS 的终点吗？

elementary OS 采用“付费下载”的收入模式，但这完全是自愿的。由于疫情持续了两年，销售收入下降，公司领导层不得不削减全职员工的工资和医疗福利以维持运营。

> “自从我全职投入 elementary 公司以来，每个版本的销售业绩都比上一个版本好，直到 OS 6 和 6.1 的表现远低于预期，部分原因是全球疫情的影响。人们在能够免费下载操作系统时，不再愿意支付可选的金额。显然，我们需要重新确定公司的财务优先事项，同时坚持我们的开源、隐私保护和道德筹资原则。”Cassidy 在他的告别信中写道。

全球经济受到疫情冲击，所有人都无法幸免。[elementary OS 6 Odin](https://www.debugpoint.com/2021/08/elementary-os-6-odin-review/) 的发布并不顺利，存在一些错误、NVIDIA 显卡问题等，这些问题加上全球疫情和贡献减少，最终影响了自愿付费下载的收入。

自愿付费下载模式和 GitHub 赞助不足以维持大型项目的运作。其他主流开源项目如 Fedora Linux、GNOME 和 KDE Plasma 都获得了来自 Red Hat、IBM、Google 等大企业的大量赞助。这是因为这些大企业从这些开源项目中获得了商业利益。但 elementary OS 的情况不同。

> “结果，Dani 要求我辞职并完全离开 elementary。在我寻找另一个职位时，这并不是我的本意，但 Dani 一直坚持要我离开。最后，我决定最好的行动方案确实是继续前行；我放弃了我对 elementary 十多年的热情，接受了 Dani 的提议，让她成为 elementary 公司唯一的、100% 的所有者。我已经签署了辞呈。从今天起，她拥有公司的所有股份和责任。我祝愿她在延续公司遗产方面取得成功。”Cassidy 写道。

### 展望未来

我能从 Cassidy 的告别信中感受到他的情感。放弃多年建立的项目和激情并不容易。那里包含了无数个日夜的努力和愿景。放弃并不容易。

没有人能预测未来。我们不知道 elementary OS 作为项目会发生什么。当情况发生变化时，开源公司最终可能会成为一个拥有更多受众和贡献者的社区项目。新的领导人需要审视 elementary OS 在 Linux 发行版或桌面领域的未来路线图。

带有 GTK4/libadwaita 的 GNOME 42+ 看起来很有前景，elementary OS 可能会因此失去用户群。在这种情况下，唯一的卖点就是 Pantheon 桌面，这需要更多的打磨和市场化能力，同时新的领导层也需要寻找更好的收入模式。

我们希望，作为一个社区，elementary OS 能够继续推出新版本，并在必要时成为一个没有 GitHub 赞助墙的社区项目。我认为寻找企业基金和捐赠也可以成为维持项目运行的长期选择之一。

任何开源项目都不应该停止。我希望新的领导层能够找到更好的收入模式来维持这个项目。只要思想不滑坡，方法总比困难多。

关于这整个情况，你有什么看法？请在下方评论让我知道。

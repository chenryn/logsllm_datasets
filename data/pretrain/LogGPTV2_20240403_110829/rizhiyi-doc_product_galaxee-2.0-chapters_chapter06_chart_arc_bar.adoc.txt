环形柱图是柱状图的一种，相比基本柱图，环形柱图可以在有限的可视化应用空间内更加直观地展示多种数据的差异。本文介绍环形柱图各配置项的含义。
**配置**
**图表尺寸**、**图表位置**、**旋转角度**和**透明度**可以参考组件指南的配置项说明。
* **图形属性**
** **内半径**：环形柱图的内半径，范围0-1
** **外半径**：环形柱图的外半径，范围0-1
** **自定义柱宽**：环形柱的宽度，单位为px
* **扇形配置**
** **色值**：环形柱的颜色
* **文本样式**
+
[cols="1, 4"]
|===
|参数 |说明
|字体
|文本的字体样式
|文字粗细
|文本的文字粗细
|文字大小
|文本的文字大小
|文字颜色
|文本的文字颜色
|===
* **动画效果** 柱状图的动画效果样式，image:images/galaxee/eye_icon.png[eye_icon, 32, 32]图标可开启或关闭动画效果。
+
image::images/galaxee/animation.png[animation， 150]
+
[cols="1, 4"]
|===
|参数 |说明
|初始动画时长
|组件第一次渲染动画的时长，单位为ms。
|初始动画效果
|动画的缓动效果，系统提供多种常见的动画缓动效果供您选择。
|更新动画时长
|组件数据更新时动画持续的时间，单位为ms。
|更新动画效果
|动画的缓动效果，系统提供多种常见的动画缓动效果供您选择。
|===
**数据**
image:images/galaxee/arc_bar/data.png[arc_bar_data, 250]
上图中的示例JSON代码如下。
----
[
    {
        "label": "北京",
        "value": 78
    },
    {
        "label": "上海",
        "value": 35
    },
    {
        "label": "广州",
        "value": 85
    },
    {
        "label": "深圳",
        "value": 72
    },
    {
        "label": "成都",
        "value": 60
    }
]
----
***表1.字段说明***
[cols="1, 4"]
|===
|字段 |说明
|label
|柱图中每个柱子的类目。
|value
|柱图中每个柱子的值。
|===
***表2.配置项说明***
[cols="1, 4"]
|===
|配置项 |说明
|数据源类型
| 分为静态数据和SPL数据。可在配置数据中实时更改静态数据或动态SPL配置。
|数据响应结果
| 展示数据请求的响应结果。当数据源发生改变时，可实时查看数据响应结果。
|===

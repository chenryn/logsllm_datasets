### JUC (Java并发编程)

**JUC** 是 **Java** 并发编程的核心内容。鉴于我之前一直从事 **Java** 开发，对 **JavaSE** 有一定的熟悉度，我首先选择了学习 Java 并发包及相关大厂面试题。
- **来源**: Bilibili XX谷周阳老师的课程
- **链接**: [Java并发编程教程](https://www.bilibili.com/video/BV15J4112785)
- **主要内容**:
  - Volatile 关键字
  - CAS 操作
  - JAVA 锁机制
  - 阻塞队列
  - 线程池
- **笔记地址**: [JUC学习笔记](https://gitee.com/moxi159753/LearningNotes)

### 中间件

中间件的学习主要包括 **MQ**（消息队列）、**Redis** 和 **MySQL** 等。虽然调用 API 相对简单，但其背后的原理却相当复杂。
- **来源**: Bilibili 视频教程
- **链接**: [中间件教程](https://www.bilibili.com/video/BV1UJ411X7M1)
- **主要内容**:
  - 中间件在服务开发中的实际应用
  - 解决疑难问题的方法
- **笔记地址**: [中间件学习笔记](./images/image-20201216230006976.png)

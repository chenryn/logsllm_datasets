User: How to use Linux command line tools for file operations? Please list at least three commonly used commands and their corresponding operations.
Assistant: 1. ls command: display all files or folders in the current directory - ls - ls -l (display file details) - ls -a (display all files, including hidden files) 2. cd command: change the current directory - cd Directory name (enter directory) - cd .. (return to parent directory) - cd ~ (return to user root directory) 3. cp command: copy file or folder - cp source file target file (copy file) - cp -r source Folder target folder (copy folder) - cp -i source file target file (prompt whether to overwrite existing files when copying)

### CVE-2022-22978

**漏洞概述：**
CVE-2022-22978 是一个与 Spring Security 5.6.3 相关的安全漏洞。该漏洞允许攻击者绕过某些安全检查，从而访问受保护的资源。

**演示环境：**
- **受影响版本：** Spring Security 5.6.3
- **管理页面路径：** `/admin/`

**攻击载荷：**
```
/admin/index%0a
```

**详细说明：**
通过在请求路径中使用特定的字符序列（如 `%0a`），攻击者可以绕过 Spring Security 的某些过滤机制，进而访问到原本应被保护的管理页面。这一漏洞的存在使得系统面临未经授权的访问风险，建议用户尽快升级至修复后的版本以确保安全。

**防范措施：**
- 尽快更新 Spring Security 至最新版本。
- 定期进行安全审计和漏洞扫描。
- 配置 Web 应用防火墙 (WAF) 以增强防护能力。

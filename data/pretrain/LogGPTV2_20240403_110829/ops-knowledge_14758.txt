User: 如何使用Git进行版本控制？请列出至少三个常用的Git命令以及它们的作用。
Assistant: 1. git init：将当前目录设置为Git仓库，开始进行版本控制。
2. git add：将当前工作区的文件添加到暂存区，准备提交到仓库。
3. git commit：将暂存区的文件提交到仓库，形成一个新版本，并添加提交信息。

其他常用命令包括：

- git status：查看工作区和暂存区的状态。
- git log：查看提交历史。
- git branch：查看和管理分支。
- git merge：合并两个不同分支的代码。
- git clone：将远程仓库复制到本地。
- git pull：将远程仓库的最新修改拉取到本地。
- git push：将本地仓库的最新修改推送到远程仓库。

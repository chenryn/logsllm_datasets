User: How to install and configure Nginx web server on CentOS server?
Assistant: Following are the steps to install and configure Nginx web server on CentOS server: 1. Install EPEL repository ```sh sudo yum -y install epel-release``` 2. Install Nginx ```sh sudo yum -y install nginx` `` 3. Start Nginx ```sh sudo systemctl start nginx ``` 4. Configure Nginx - Open the main configuration file for Nginx, which is located at /etc/nginx/nginx.conf. ```sh sudo vi /etc/nginx/nginx.conf ``` - Add the following configuration inside the http {} block to listen on port 80 and specify the website root directory. ```nginx server { listen 80; server_name example.com; root /var/www/html; location / { index index.html index.htm; } } ``` - Restart Nginx ```sh sudo systemctl restart nginx ` `` 5. Firewall Rule Settings - Add Rule ```sh sudo firewall-cmd --add-service=http --permanent sudo firewall-cmd --reload ``` Now you should be able to access your CentOS server using your browser IP address or public DNS. If all goes well, Nginx should display a welcome page in the browser. The above are the steps to install and configure the Nginx web server on the CentOS server.

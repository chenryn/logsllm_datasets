> Firefox 112 版本引入了显示密码的新选项，并改进了标签管理等功能。

Mozilla 于 2023 年 4 月 11 日发布了 Firefox 112，该版本包含多项新功能和改进，旨在提升用户体验和性能。

![在 Ubuntu 上运行 Firefox 112](/data/attachment/album/202304/18/152415jmlgv3yrmvvwrmsz.jpg)

### 显示密码的新选项

Firefox 112 引入了一个实用的功能：用户只需右键单击密码字段即可显示密码。这对于经常忘记登录凭据的用户来说是一个非常方便的工具。

![Firefox 112 引入了显示密码选项](/data/attachment/album/202304/18/152425au9241xurb1nc3qg.jpg)

### 数据导入功能

现在，Firefox 允许 Ubuntu Linux 用户从 Chromium Snap 包中导入浏览器数据。不过，此功能仅在 Firefox 未以 Snap 包形式安装时有效。Mozilla 正在努力解决这一问题。

### 标签管理改进

Firefox 112 在标签管理方面也进行了改进。用户可以通过在标签列表面板中使用鼠标中键点击来关闭标签。此外，快捷键 `(Cmd/Ctrl)-Shift-T` 用于恢复最近关闭的标签页，当同一会话中没有更多已关闭的标签需要重新打开时，它现在会恢复之前的会话。

### 跟踪参数保护

Firefox 扩展了已知跟踪参数列表，对于启用 ETP Strict 模式的用户，这些参数将从 URL 中删除，从而进一步保护用户免受跨站点跟踪。

### 视频解码优化

新的更新还支持在 Windows 系统上的英特尔 GPU 上叠加软件解码视频，这不仅提高了缩小的视频质量，还减少了 GPU 的使用。

### U2F API 变更

已弃用的 U2F JavaScript API 默认处于禁用状态，但 U2F 协议仍可通过 WebAuthn API 使用。如果需要，用户可以使用 `security.webauth.u2f` 首选项重新启用 U2F API。

### 清除日期选择器

最后，日期选择器面板中添加了一个新的清除按钮，允许用户快速清除类型为“日期”或“本地日期-时间”的输入，从而提供一致的跨浏览器用户体验。

### 下载与更新

Firefox 112 已在 Windows、Mac 和 Linux 平台上发布。我们鼓励用户更新到最新版本，以获得更流畅、更安全的浏览体验。

你可以立即从以下链接下载最新版本：
> **[下载 Firefox 112](https://ftp.mozilla.org/pub/firefox/releases/112.0/)**

或者，你也可以等待几天，Linux 发行版会在测试后通过常规更新推送 Firefox 112。

参考来源：[changelog](https://www.mozilla.org/en-US/firefox/112.0/releasenotes/)

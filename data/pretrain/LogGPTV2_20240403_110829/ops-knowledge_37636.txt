User: How to configure scheduled tasks in the Linux system to periodically perform some operations that need to be automated?
Assistant: In the Linux system, cron can be used to configure scheduled tasks. To configure a scheduled task, you can follow the steps below: 1. Open the terminal and enter the following command to edit the cron table: ```sh crontab -e ``` 2. Add the scheduled task to be executed in the editor, the format is as follows : ``` * * * * * command-to-be-executed ``` Among them, the five * represent the value ranges of minutes, hours, days, months, and weeks, for example: - 0-59: minute values Range - 0-23: hour value range - 1-31: day value range - 1-12: month value range - 0-6: week value range (0 means Sunday, 1 means Monday, and so on ) command-to-be-executed indicates the command or script that needs to be executed. For example: ``` * * * * * /usr/local/bin/my-script.sh ``` means to execute the script named my-script.sh every minute. 3. Save and exit the editor. After the configuration is complete, cron will execute the corresponding command or script regularly at the set time. To view the existing crontab, you can use the following command: ```sh crontab -l ```

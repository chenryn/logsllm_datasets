**EuroLinux**

你应该尝试一下 EuroLinux，因为它提供了全面的软件包，能够同时满足 Linux、Windows 和 macOS 用户的需求。该系统现已发布稳定版本，并计划在 2023 年进行更新。

- **官方网站**: [EuroLinux](https://en.euro-linux.com/eurolinux/desktop/)

### 6. Zinc

![Zinc](/data/attachment/album/202301/21/115620c2e65ho5ot1jsey5.png)

[Zinc](https://teejeetech.com/tag/zinc/) 是一个基于 Ubuntu 的发行版，经过调整后提供独特的用户体验。现有的 Ubuntu 用户可能会对其功能感到惊喜。它基于 Xubuntu 的最新 LTS 版本，使用 XFCE 桌面环境，并进行了多项改进，例如集成的 Linux AppImage 支持、deb-get 包安装程序以及 BTRFS 作为默认文件系统等。正确配置后，它可以成为你的日常操作系统替代品。Zinc 遵循稳定的发布模式，因此你可以期待 2023 年的重大更新。

- **官方网站**: [Zinc](https://teejeetech.com/tag/zinc/)

### 7. CachyOS

![CachyOS](/data/attachment/album/202301/21/115620mx3t8ssx1xn5tn3t.jpg)

[CachyOS](https://cachyos.org) 致力于使 Arch Linux 更加适合初学者，让任何人都可以轻松上手。由于其高度可定制性和最新的软件支持，它受到了广泛欢迎。该操作系统旨在为你提供快速、安全且易于使用的体验，特别适用于希望探索和尝试新事物的用户。作为一个滚动发布的发行版，CachyOS 在 2023 年将会有大量更新。

- **官方网站**: [CachyOS](https://cachyos.org)

### 8. risiOS

![risiOS](/data/attachment/album/202301/21/115621xoqeov3cnwecc009.png)

在众多基于 Arch 和 Ubuntu 的 Linux 发行版中，[risiOS](https://risi.io) 显得尤为独特。该项目基于 Fedora Linux，起源于美国西雅图。它使用 GNOME 桌面环境，为用户提供高度可定制的体验，并提供自定义的 ZSH 版本。如果你有兴趣尝试基于 Fedora 的发行版，这可能是一个不错的选择。risiOS 会在稳定版本之间推送一些小更新，预计在 2023 年会带来更多内容。

- **官方网站**: [risiOS](https://risi.io)

### 9. Exodia OS

![Exodia OS](/data/attachment/album/202301/21/115623x7epu77n6b7v7dwn.jpg)

又是一个基于 Arch 的 Linux 发行版？是的，今年我们已经看到了很多基于 Arch 的发行版，但这并不是坏事。认识一下 [Exodia OS](https://exodia-os.github.io/exodia-website/)，这是一个面向网络安全领域的高度可定制的 Arch 基础发行版。其功能包括预装的各种网络安全工具、命令行界面应用、ElKowars wacky widgets (EWW) 以及 Zsh 等。如果你是网络安全专家或爱好者，不妨一试！他们为三种不同的使用场景提供了三个版本，并将在 2023 年继续推送必要的更新和功能添加。

- **官方网站**: [Exodia OS](https://exodia-os.github.io/exodia-website/)

### 10. Kumandar Linux

![Kumandar Linux](/data/attachment/album/202301/21/115623szcepzuo1wkzsykx.jpg)

乍一看，你可能会误以为这是 Windows 7，但仔细观察就会发现它是 [Kumandar Linux](https://www.kumander.org)。该系统基于 Debian 11，并使用自定义的 XFce 版本。其名称在中文中的含义为“指挥官”，以向开发者的第一台电脑 Commodore VIC20 致敬。如果你喜欢 Windows 7 的外观，但想在 Linux 上获得类似体验，那么可以试试这个系统。目前，该系统仅发布了早期候选版本，但预计 2023 年将推出稳定版本。

- **官方网站**: [Kumandar Linux](https://www.kumander.org)

### 11. Ubuntu Unity

![Ubuntu Unity](/data/attachment/album/202301/21/115624mgn2cgbdo0700h3u.jpg)

今年早些时候，Ubuntu Unity 被宣布为 Ubuntu 的官方版本之一。它使用了 2010 至 2017 年间 Ubuntu 中的 Unity 桌面界面，后来 Canonical 将其替换为 GNOME。开发工作正在积极进行中，年轻的首席开发人员正推动更新并增加新功能。想要尝试不同风格的 Ubuntu 用户可以考虑这个系统。它提供了 LTS 和非 LTS 版本。

- **官方网站**: [Ubuntu Unity](https://ubuntuunity.org/)

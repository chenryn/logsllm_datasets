User: Please list commonly used Linux commands and their functions, and give usage examples.
Assistant: 1. cd function: switch the current directory. Example: cd /home/user/Desktop 2. ls Function: List the files and directories under the current directory. Example: ls 3. pwd Function: Display the current directory. Example: pwd 4. mkdir Function: Create a directory. Example: mkdir new_dir 5. rm Function: delete files or directories. Example: rm file.txt or rm -r folder 6. cp function: copy files or directories. Example: cp file.txt /home/user/Documents or cp -r folder /home/user/Documents 7. mv function: move files or rename files or directories. Example: mv file.txt /home/user/Documents or mv old_name new_name 8. touch function: create an empty file or update the file timestamp. Example: touch file.txt 9. chmod Function: Modify the permissions of a file or directory. Example: chmod 755 file.txt 10. chown function: modify the owner of a file or directory. Example: chown user file.txt 11. ps Function: Display current process information. Example: ps aux 12. top Function: Display system resource usage. Example: top 13. ping Function: Test network connection. Example: ping google.com 14. scp function: copy files remotely. Example: scp file.txt user@remote_host:/home/user/Documents 15. ssh function: remote login. Example: ssh user@remote_host

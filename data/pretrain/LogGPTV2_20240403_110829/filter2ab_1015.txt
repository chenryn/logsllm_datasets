**ATT&CK ID:** T1059.005  
**ATT&CK Technique Name:** Visual Basic

Adversaries may exploit Visual Basic (VB) for execution. VB is a programming language developed by Microsoft, designed to integrate seamlessly with various Windows technologies, including the Component Object Model (COM) and the Native API through the Windows API. Despite being considered a legacy language with no planned future updates, VB remains integrated and supported within the .NET Framework and the cross-platform .NET Core.

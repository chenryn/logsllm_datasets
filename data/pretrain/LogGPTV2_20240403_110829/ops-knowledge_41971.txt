User: How to install and upgrade Docker via command line?
Assistant: To install Docker: 1. First you need to uninstall the old version of Docker, use the following command: ``` sudo apt-get remove docker docker-engine docker.io containerd runc ``` 2. Add the official Docker GPG key, use the following command: ``` curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add - ``` 3. To add the Docker software source, use the following command: ``` sudo add-apt- repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" ``` 4. Update the local package index, use the following command: ``` sudo apt- get update ``` 5. To install Docker, use the following command: ``` sudo apt-get install docker-ce docker-ce-cli containerd.io ``` To upgrade Docker: 1. Update the local package index, use the following command : ``` sudo apt-get update ``` 2. To upgrade Docker, use the following command: ``` sudo apt-get upgrade docker-ce docker-ce-cli containerd.io ```

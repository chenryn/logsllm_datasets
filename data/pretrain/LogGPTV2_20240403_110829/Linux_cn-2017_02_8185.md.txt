了解如何解决 Linux 桌面用户遇到的最常见问题。

![](/data/attachment/album/201702/08/183900gwiazaw7474b7v8x.gif)

尽管大多数用户能够顺利安装和操作 Linux，但仍有少数人会遇到一些技术难题。作为今年任务清单的一部分，我认为总结一下人们在 2016 年遇到的常见 Linux 技术问题非常有意义。我在 LinuxQuestions.org 和社交媒体上发布了这个问题，并分析了 LQ 的帖子情况，以下是主要发现：

### 1. Wi-Fi 驱动程序（特别是 Broadcom 芯片）
Wi-Fi 驱动程序，尤其是 Broadcom 无线网卡，仍然是 Linux 最常见的技术问题之一。在 LQ 上，2016 年有数百个关于此话题的帖子。市场上存在多种 Broadcom 无线网卡，但每个发行版的具体排错步骤有所不同。基本步骤包括：
- 使用 `lspci` 命令确定具体的 Broadcom 卡并找到其 PCI ID。
- 确认所使用的发行版是否支持该卡。
- 如果支持，则找到正确的方法使网卡正常工作。

例如，如果您的卡的 PCI ID 为 `14e4:4315` 且系统是 Ubuntu，那么您可以通过安装 `firmware-b43-installer` 包来驱动 BCM4312 卡。另一种选择是在购买前研究可用的 Wi-Fi 卡以确保它们被您的发行版完全支持。

### 2. 打印机驱动程序（特别是佳能和 Lexmark）
打印机也是常见的问题源，尤其是佳能和 Lexmark 品牌。如果您计划购买新打印机，请提前检查兼容性。[OpenPrinting](http://www.openprinting.org/printers) 数据库和您所用发行版的支持渠道是两个很好的起点。请注意，应确保设备的所有功能都完全兼容，特别是对于多功能产品。佳能打印机的一个常见抱怨是，驱动程序通常只能在非英语甚至难以找到的网站上获得。

### 3. 显卡
显卡在 Linux 上通常可以很好地开箱即用，但当涉及到 3D 加速、最新的显卡技术（如 NVIDIA Optimus 和 ATI 动态 GPU 切换）、专有驱动程序的安装与稳定性、能效管理以及挂起和恢复功能时，可能会出现问题。如果您不是游戏玩家或不需要高端图形功能，可能无需担心这些。但是，如果您需要高端图形性能，建议在购买前进行详细的研究。幸运的是，随着 Wayland 的发展，这些问题有望在 2017 年得到改善。

### 4. 声卡
对于简单的配置，声卡在 Linux 上很容易设置并且运行可靠。然而，一旦涉及专业级功能，如回声消除、音频路由、统一混合等复杂配置，问题可能会迅速增加。我的建议是，如果您需要高端实时音频处理，请考虑使用专门针对音频优化的发行版。

### 5. 安装
虽然本文涵盖了广泛的分类，但公平地说，Linux 并没有普遍存在的安装问题。绝大多数安装都能按预期完成。然而，由于 Linux 支持各种硬件组合，因此偶尔会出现例外情况。最终用户很少像安装 Mac OS 或 Windows 一样从头开始安装 Linux，因为这些操作系统通常是预装的。

### 前景光明
其他提到的问题还包括蓝牙、挂起/恢复、HiDPI 和触摸屏。本文中提到的大部分问题集中在桌面用户身上，这是有道理的。Linux 桌面使用率相对较低，导致用于发现和解决问题的测试资源较少。随着桌面使用量的增加，预计这些问题将逐渐减少。

在这方面，一个很好的例子是字体问题。几年前，获取高质量抗锯齿字体经常是个问题。但现在，随着现代发行版的改进，这已成为常规。

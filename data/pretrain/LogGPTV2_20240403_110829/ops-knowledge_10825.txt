从日志中提取出的模式如下：

```
<日期> <时间>,<毫秒> [类名]-[错误级别]-[线程名称] <方法名>消耗:<耗时>
```

具体实例：
```
2021-10-21 15:58:54,623 [com.asiainfo.boss.inter.cboss.ejbclient.CbossMiscSmsImpl]-[ERROR]-[WebContainer : 8] UPSS_CusInfo消耗:82
```

提取后的模式：
```
<日期> <时间>,<毫秒> [类名]-[ERROR]-[WebContainer : 线程编号] 方法名消耗:耗时
```

其中，`<日期>`、`<时间>`、`<毫秒>`、`[类名]`、`[线程编号]`、`方法名` 和 `耗时` 是具体的变量。

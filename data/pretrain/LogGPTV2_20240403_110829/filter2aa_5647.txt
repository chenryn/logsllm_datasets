# Advertisement
## Category: Ad
### Description
We are interrupting this program for a commercial break. Please watch the following promotional video:

[Watch the Video](https://www.youtube.com/watch?v=QzFuwljOj8Y)

### Solution
The attached YouTube video is a promotional clip related to the CTF (Capture The Flag) event. At 00:17 in the video, a long string of text briefly appears on the screen. This string is actually the flag for the challenge.

**Flag:**
```
CTF{9e796ca74932912c216a1cd00c25c84fae00e139}
```

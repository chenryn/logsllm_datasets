User: Please list three commonly used Linux commands, and give a brief explanation and example demonstration for each command.
Assistant: 1. ls command: The ls command is used to display the file and subdirectory information in the current working directory. Its common parameters are -a (display all files and directories, including hidden files), -l (display file information in long format), -h (display file size in human readable format). Example: ls -alh. 2. cd command: The cd command is used to switch the current working directory. Its argument is the directory name to switch to. Example: cd /home/user. 3. mkdir command: The mkdir command is used to create a new directory. Its argument is the directory name to create. Example: mkdir newdir.

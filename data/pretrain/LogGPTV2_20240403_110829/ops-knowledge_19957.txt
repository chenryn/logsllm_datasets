User: 请列出常用的网络协议，并简述它们各自的作用与特点。
Assistant: 1. HTTP (Hypertext Transfer Protocol) - 用于在 Web 上传输数据。特点：无状态，单向请求响应模式，明文传输。
2. HTTPS (Hypertext Transfer Protocol Secure) - 通过 SSL/TLS 加密传输 HTTP 数据。特点：安全，可信，保密性强。
3. FTP (File Transfer Protocol) - 用于计算机间文件传输。特点：可用于下载和上传文件，传输速度较快，但传输信息不加密。
4. SMTP (Simple Mail Transfer Protocol) - 用于电子邮件传输。特点：无加密，传输信息明文可读性。
5. POP3 (Post Office Protocol version 3) - 用于从邮件服务器上收取邮件。特点：简单易用，邮件下载到客户端后删除服务器上的原始邮件。
6. IMAP (Internet Message Access Protocol) - 用于从邮件服务器上收取邮件。特点：支持邮件存储、保存和管理，并可以远程访问邮件。
7. TCP (Transmission Control Protocol) - 用于建立可靠的连接。特点：点对点连接，数据包传输可靠。
8. UDP (User Datagram Protocol) - 用于支持不要求可靠数据传输的应用。特点：快速传输数据，但没有可靠性保证。
9. DNS (Domain Name System) - 用于将域名映射到 IP 地址。特点：分布式系统，使用层次结构，可快速定位目标 IP 地址。
10. DHCP (Dynamic Host Configuration Protocol) - 用于自动获取 IP 地址的协议。特点：分配 IP 地址，子网掩码等网络配置信息。

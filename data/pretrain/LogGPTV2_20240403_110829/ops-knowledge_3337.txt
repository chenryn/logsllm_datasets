User: How to manage virtual machines and storage resources when using the OpenStack cloud platform? Please provide the relevant commands and steps.
Assistant: To manage virtual machines: 1. Check the virtual machine list: openstack server list 2. Create a virtual machine: openstack server create --flavor <flavor_name> --image <image_name> --network <network_name> <vm_name> 3. Start the virtual machine: openstack server start <vm_name> 4. Shut down the virtual machine: openstack server stop <vm_name> 5. Delete the virtual machine: openstack server delete <vm_name> 6. Change the virtual machine configuration: openstack server set <vm_name> --property <property_name>= <property_value> Manage storage resources: 1. View storage resources: openstack volume list 2. Create storage resources: openstack volume create --size <size> <volume_name> 3. Delete storage resources: openstack volume delete <volume_name> 4. Change storage Resource configuration: openstack volume set <volume_name> --property <property_name>=<property_value> 5. Mount storage resources to virtual machines: openstack server add volume <vm_name> <volume_name> 6. Uninstall storage resources: openstack server remove volume < vm_name> <attachment_id>

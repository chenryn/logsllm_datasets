> 击败了 Arch 和 Fedora，Clear Linux 的最新版本提供了 GNOME 42 的稳定版本。

Clear Linux 是一个开源的滚动发布发行版，专为云计算和边缘计算开发者量身定制。你可能不知道，Clear Linux 是英特尔的一个项目，针对英特尔处理器进行了优化，并提供了一个快速部署容器的环境。它同时支持桌面和服务器（如 AWS、Google Compute Engine 和 Azure Hyper-V）。

**注意：** 与即将发布的 Fedora 36 工作站版和 Ubuntu 22.04 长期支持版相比，Clear Linux 并不是一个完美的替代桌面。如果你对这个发行版感兴趣，建议先在虚拟机上进行测试。

令人惊讶的是，Clear Linux 的最新版本已经包含了 GNOME 42（稳定版）。更不用说，Clear Linux 比 Arch 和 Fedora 更新得更快，提供了最新的 GNOME 稳定版本。

### Clear Linux 上的 GNOME 42

![](/data/attachment/album/202203/26/222217uncfttr8itqdavau.jpg)

你可以期待在 Clear Linux 上看到 [GNOME 42 的最佳特性](https://news.itsfoss.com/gnome-42-features/)。根据我的简单测试，大部分功能都符合 GNOME 42 的预期。

![](/data/attachment/album/202203/26/222218t5v40g8m4bgmy44v.png)

文件管理器、日历、照片、天气以及许多其他应用程序都反映了最新的 GNOME 42 升级到 GTK 4 的变化。当然，在 Clear Linux 上，默认的桌面壁纸会有所不同，但你可以获取同一套壁纸集的浅色和深色版本。

![](/data/attachment/album/202203/26/222219gkepwp6luyper8p3.jpg)

我在虚拟机上进行了测试，运行情况符合预期，没有显著的缺陷。如果你已经在使用 Clear Linux 或对其感兴趣，可以在其 [版本发布说明](https://download.clearlinux.org/releases/36060/clear/RELEASENOTES-36030-to-36060) 中找到更多细节，包括更新的软件包和其他变化。

### 下载附带 GNOME 42 的 Clear Linux

或许你在官网下载页面找不到最新的发布版，但可以在 [官方下载仓库](https://download.clearlinux.org/releases/36060/clear/) 中找到它。你可以根据自身需求（服务器或桌面）下载相应的 ISO 文件，现在就开始体验吧！

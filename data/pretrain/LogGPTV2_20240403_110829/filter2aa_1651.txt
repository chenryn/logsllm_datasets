### 操作指南

#### 1. 系统准备
- **操作系统**: Windows Server 2012 R2 x64
- **硬件要求**:
  - 内存: 8 GB RAM
  - 硬盘: 500 GB HDD

#### 2. 全局代理配置
- 在系统上安装以下软件：
  - .NET Framework 3.5 SP1
  - Microsoft Visual C++ Redistributable for Visual Studio 2019-2015
- 配置全局代理以实现网络访问（翻墙）。

#### 3. 启动 CORE IMPACT
- 打开 CORE IMPACT 19.1。

#### 4. 结束相关进程和服务
- 终止所有与 CORE IMPACT 相关的进程和服务。这可以通过任务管理器或服务管理工具来完成。

#### 5. 复制破解文件
- 将破解包中的 `Program Files` 和 `ProgramData` 目录下的所有文件复制到系统C盘的相应目录中。

#### 6. 复制序列号文件
- 将破解包中的三个序列号文件复制到以下路径：
  ```
  C:\Program Files (x86)\Core Security\Impact\bin\5372-8801-2329-7643-5724
  ```

#### 7. 重启并验证
- 重启计算机。
- 重新启动 CORE IMPACT，确认其正常运行。

请确保在执行上述步骤时遵循所有安全和法律规范。

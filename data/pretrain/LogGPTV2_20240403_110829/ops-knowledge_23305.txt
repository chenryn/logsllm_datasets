User: 请完成其他领域的自动化脚本类任务[15, 23, 5, 12, 8, 3, 27, 16, 5, 9, 1]

Assistant: 可以使用以下代码来查找一组数字中的最大值和最小值：

```
numbers = [15, 23, 5, 12, 8, 3, 27, 16, 5, 9, 1]
max_num = numbers[0] # 初始化最大值为第一个数字
min_num = numbers[0] # 初始化最小值为第一个数字

for num in numbers:
 if num > max_num:
 max_num = num # 更新最大值
 elif num < min_num:
 min_num = num # 更新最小值

print("最大值：", max_num)
print("最小值：", min_num)
```

上述代码中，我们初始化最大值和最小值为第一个数字，并遍历列表中的所有数字。如果当前数字比最大值大，则更新最大值；如果比最小值小，则更新最小值。最后输出最大值和最小值。 

输出结果为：

```
最大值： 27
最小值： 1
```

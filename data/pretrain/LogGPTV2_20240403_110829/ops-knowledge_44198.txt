User: What are the main procedures an organization should follow in order to implement CIS Control 16.5, Use Up-to-Date and Trusted Third-Party Software Components?
Assistant: An organization should Use up-to-date and trusted third-party software components. When possible, choose established and proven frameworks and libraries that provide adequate security. Acquire these components from trusted sources or evaluate the software for vulnerabilities before use.CIS Control 16.5 has the asset type Applications, and has the security function Protect.

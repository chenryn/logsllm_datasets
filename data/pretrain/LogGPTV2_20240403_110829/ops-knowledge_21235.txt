User: How to set timed tasks on Linux system? Please provide commonly used command line tools and operation steps.
Assistant: The crontab command is generally used to set up scheduled tasks on the Linux system. The specific steps are as follows: 1. Open the terminal (Terminal). 2. Enter the following command to edit the scheduled task: crontab -e If it is the first time to edit, the system will prompt you to choose an editor, you can choose a commonly used editor such as vi or nano. 3. Add the commands you want to execute at the end of the opened file, each command occupies one line. The crontab format is as follows: ``` * * * * * command to be executed - - - - - | | | | | | | | | | +----- day of the week (0 - 6) (Sunday=0) | | | +------- month (1 - 12) | | +--------- day of the month (1 - 31) | +---------- - hour (0 - 23) +------------- min (0 - 59) ``` 4. Fill in the time and command according to the above format, for example: ``` 0 1 * * * /home/user/script.sh ``` indicates that /home/user/script.sh will be executed at 1 am every day. 5. When finished, save the file and close the editor. 6. Enter the following command to view the crontab of the current user: crontab -l 7. If you need to delete a crontab, you can enter the following command: crontab -r or use the following command to edit an existing crontab: crontab -e Note Items: - Use of the crontab command requires administrator privileges (sudo). - The commands executed by the scheduled task need to have executable permission (chmod +x). - The time zone may affect the execution time of scheduled tasks.

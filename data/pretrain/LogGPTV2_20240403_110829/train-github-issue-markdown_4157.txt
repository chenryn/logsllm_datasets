# Summary of the New Feature/Enhancement
This feature will enable users with multiple keyboards to easily set up macros without the need for third-party software. It will provide a more streamlined and integrated solution for managing key mappings across different input devices.

# Proposed Technical Implementation Details (Optional)
The proposed implementation will allow users to remap all the keys on more than one keyboard, or even on all connected keyboards. This will provide greater flexibility and customization, enabling a more efficient and personalized user experience.

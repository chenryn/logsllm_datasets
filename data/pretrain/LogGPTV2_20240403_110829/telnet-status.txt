### Telnet: Status

**Applies to:**  
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**  
Displays the current status information for the Telnet client or server.

**Syntax:**
```
status
```

**Example:**
To display the status information, type:
```
status
```

**Note:**
- The `status` command can be abbreviated as `st`.
- Ensure you have the necessary permissions to run this command.

User: How to Check CPU, Memory and Disk Usage in Linux System?
Assistant: 1. Check the CPU usage: Use the top command, enter "top" in the terminal, you can see the current system process and CPU usage, and press "q" to exit. 2. Check the memory usage: use the free command, enter "free -m" in the terminal, you can see the current system memory usage, including used memory, available memory and cache, and press "q" to exit. 3. View the disk usage: use the df command, enter "df -h" in the terminal, you can see the disk usage of the current system, including disk partitions, used space, free space and usage rate, press "q" to exit .

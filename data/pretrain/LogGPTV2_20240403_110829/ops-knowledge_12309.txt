User: 如何查看Linux服务器上当前运行的进程列表？
Assistant: 使用命令"ps"可以查看当前在Linux服务器上运行的进程列表。具体用法可以参考下面的示例：

1. 查看所有进程列表：

```
$ ps -ef
```

2. 查看当前用户的进程列表：

```
$ ps -u username
```

3. 查看某个进程的详细信息：

```
$ ps -p pid -o pid,user,cmd,%cpu,%mem,start,time
```

其中，"pid"为进程号，可以通过第一步或其他方式获取。"%cpu"和"%mem"为进程占用的CPU和内存使用比例。"start"和"time"分别为进程启动时间和运行时间。 

4. 使用top命令查看进程实时占用资源情况：

```
$ top
```

该命令会实时显示进程的CPU、内存和IO资源占用情况。按下键盘上的“q”键退出。

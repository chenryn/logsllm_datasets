### chdir

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `chdir` command is used to display the name of the current directory or to change the current directory. If used with only a drive letter (e.g., `chdir C:`), it displays the names of the current directories on the specified drive. If used without any parameters, `chdir` displays the current drive and directory.

**Example:**
```shell
chdir C:
```

**Note:**
This command has been replaced by the `cd` command. For more information, including syntax and parameter details, see the [cd command documentation](#).

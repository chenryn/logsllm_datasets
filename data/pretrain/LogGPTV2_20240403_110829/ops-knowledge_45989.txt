User: 如何设置SSH密钥免密码登录远程服务器
Assistant: 您可以按照以下步骤来设置SSH密钥免密码登录远程服务器：

1. 生成密钥对。

在本地计算机上打开终端，输入以下命令：

```
ssh-keygen
```

按照提示输入文件名和密码。不输入密码则会省略此项，但建议输入一个强密码来保护您的私钥。

2. 将公钥添加到远程服务器的授权文件中。

使用以下命令将公钥添加到远程服务器的授权文件中：

```
ssh-copy-id username@remote_host
```

输入远程服务器的密码即可。如果您使用的是非标准端口号，请添加 `-p` 选项来指定端口号，例如：

```
ssh-copy-id -p 2222 username@remote_host
```

3. 测试SSH连接。

使用以下命令测试SSH连接：

```
ssh username@remote_host
```

如果您没有输入密码直接连接成功，则表示您已成功设置SSH密钥免密码登录远程服务器。

注意：以上的命令针对Linux系统和macOS系统。如果您使用的是Windows系统，请使用PuTTY来设置SSH密钥。

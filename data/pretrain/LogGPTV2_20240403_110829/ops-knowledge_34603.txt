User: How to set firewall rules in Linux? Please list common firewall commands and parameters with examples.
Assistant: Common firewall commands and parameters are: 1. iptables: A tool for managing IPv4 packet filtering and NAT tables in the Linux kernel. 2. ip6tables: A tool for managing IPv6 packet filtering and NAT tables in the Linux kernel. 3. firewall-cmd: It is a Red Hat Linux firewall management tool. Here are some examples of common firewall rules: 1. Allow local connections: iptables -I INPUT -i lo -j ACCEPT 2. Allow SSH connections: iptables -A INPUT -p tcp --dport 22 -m state --state NEW -j ACCEPT 3. Open the web server port: iptables -A INPUT -p tcp --dport 80 -m state --state NEW -j ACCEPT iptables -A INPUT -p tcp --dport 443 -m state --state NEW - j ACCEPT 4. Allow outbound data: iptables -I OUTPUT -j ACCEPT 5. Allow PING: iptables -A INPUT -p icmp --icmp-type 8 -j ACCEPT 6. Deny all access: iptables -P INPUT DROP iptables -P FORWARD DROP iptables -P OUTPUT DROP These are some common examples of firewall rules. You can learn more firewall rules by consulting manuals and online resources.

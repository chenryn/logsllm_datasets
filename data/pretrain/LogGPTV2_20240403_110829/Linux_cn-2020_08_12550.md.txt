## 微软庆祝 Windows 95 发布 25 周年

Windows 95 对桌面体验产生了深远的影响。它引入了开始菜单和任务栏的概念，这些设计至今仍在使用，并且全球有数十亿用户受益于该系统。在过去的 25 年里，Windows 系统不断演进，已经成为许多人日常工作和生活中不可或缺的一部分。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/1019859.htm "https://www.cnbeta.com/articles/tech/1019859.htm")

> 拍一拍：Windows 95 是民用计算机的一个重要里程碑。

## Chrome 团队正探索 Rust 与 C++ 的互操作性

目前，Chrome 工程师认为 C++ 仍然是主要的编程语言，并将在可预见的未来继续主导代码库。他们正在研究如何使 Rust 能够调用 C++ 代码，而不是让 C++ 适应 Rust。这项工作旨在评估 Rust 在 Chrome 项目中的适用性和集成方式。

来源：[开源中国](https://www.oschina.net/news/118148/chrome-rust-and-c-interoperability "https://www.oschina.net/news/118148/chrome-rust-and-c-interoperability")

> 拍一拍：随着 Rust 的热度上升，你是否考虑学习这门语言呢？

## WordPress 创始人称苹果阻止其更新 iOS 版本并要求三成提成

WordPress 的 iOS 应用是一款免费软件，主要用于网站构建和管理，并未提供任何应用内购买选项。然而，苹果公司依据其 App Store 审核指南，要求跨平台应用必须允许用户通过应用内购买访问其他平台或网站上的内容、订阅或功能。这一规定导致 WordPress 更新受阻。事件曝光后，苹果向 WordPress 道歉，并表示不会强制免费应用增加付费选项。

来源：[solidot](https://www.solidot.org/story?sid=65323 "https://www.solidot.org/story?sid=65323")

> 拍一拍：苹果的行为显得有些过分。

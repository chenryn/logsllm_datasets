**ATT&CK ID:** S0390  
**ATT&CK Technique Name:** SQLRat

**Description:**
SQLRat has been observed to use SQL to execute JavaScript and VBScript code on the host system. This technique allows the attacker to leverage the database's capabilities to run malicious scripts, thereby compromising the target environment.

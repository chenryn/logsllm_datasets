User: What are the main procedures an organization should follow in order to implement CIS Control 16.1, Establish and Maintain a Secure Application Development Process?
Assistant: An organization should Establish and maintain a secure application development process. In the process, address such items as: secure application design standards, secure coding practices, developer training, vulnerability management, security of third-party code, and application security testing procedures. Review and update documentation annually, or when significant enterprise changes occur that could impact this Safeguard.CIS Control 16.1 has the asset type Applications, and has the security function Protect.

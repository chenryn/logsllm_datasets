User: 如何进行分流日志采集
Assistant: 在配置日志易 Agent 的文件及目录采集时，“文件路径白名单”参数可以通过正则表达式捕获 Module 和 Seq 两个值。日志易 Agent 将自动根据 Module 值分流，相同 Module 值内，根据 Seq 值排序，依次进行文件采集。
默认情况下，Agent 对文件路径白名单匹配的每个文件，都会独立启动一个 Golang 协程进行并发读取。当 Agent 从文件路径白名单中捕获到 Module 组后，同一个 Module 组内的文件将共享一个 Golang 协程。
当新部署环境中，存在大量历史 logrotate 回滚日志文件时，您可以尝试使用该方案，保证有序读取，并减轻磁盘 IO 压力。针对 logrotate 日志，一个可能的文件路径白名单正则是：`(?P<Module>[^/]+)/.*\.log\.?(?P<Seq>\d*)`

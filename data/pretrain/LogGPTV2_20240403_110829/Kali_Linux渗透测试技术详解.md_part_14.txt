#### 指定扫描端口范围
当目标主机上开放的端口较多时，用户查看起来可能较为困难。此时，可以使用Nmap指定扫描的端口范围。例如，扫描端口号在1～1000之间的端口，执行以下命令：
```bash
root@kali:~# nmap -p 1-1000 192.168.41.136
```

**输出信息**
- 主机: www.benet.com (192.168.41.136)
- 在线状态: 是 (延迟0.00020秒)
- 开放端口:
  - 21/tcp: FTP
  - 22/tcp: SSH
  - 23/tcp: Telnet
  - 25/tcp: SMTP
  - 53/tcp: DNS
  - 80/tcp: HTTP
  - 111/tcp: RPCbind
  - 139/tcp: NetBIOS-SSN
  - 445/tcp: Microsoft-DS
  - 512/tcp: Login
  - 513/tcp: Shell
- MAC地址: 00:0C:29:31:02:17 (VMware)

Nmap扫描完成：扫描了1个IP地址（1台主机在线），耗时0.35秒。

#### 图形化TCP端口扫描工具Zenmap
Zenmap是Nmap官方推出的一款基于Nmap的安全扫描图形用户界面。它适用于快速扫描大型网络或单个主机。使用步骤如下：
1. 在Target文本框中输入目标主机地址。
2. 在Profile文本框中选择扫描类型。
3. 单击Scan按钮，查看扫描结果。

### AI 在编程竞赛中表现优异，成绩超越了半数程序员

DeepMind 最近推出了其最新研发的 [AlphaCode](https://www.theverge.com/2022/2/2/22914085/alphacode-ai-coding-program-automatic-deepmind-codeforce) 编程 AI。这款 AI 参加了 Codeforces 的编程挑战赛，这是一个极具挑战性的编程平台，通过类似于国际象棋的 Elo 评级系统来分享每周问题和相关排名。在针对 Codeforces 网站上 5000 名用户解决的十个挑战项目进行测试后，AlphaCode 取得了平均名次位于前 54.3% 的成绩。DeepMind 估计该系统的 Codeforces Elo 评分为 1238，位居过去 6 个月在该网站上发起挑战的用户榜单的前 28%。

> **老王点评**: 看来至少有一半的程序员可以考虑改行了。

### 树莓派操作系统正式推出 64 位版本

尽管自 2016 年发布的树莓派 3 及后续产品已经配备了 64 位 CPU，甚至树莓派 Zero 2 也支持 64 位功能，但官方提供的树莓派操作系统一直是 32 位的。然而，64 位的操作系统可以支持更多内存，并且较新的指令集具有性能优势。此外，一些基于 Arm 架构的商业/二进制分发软件仅支持 AArch64。因此，树莓派基金会决定提供 [64 位的树莓派操作系统](https://www.raspberrypi.com/news/raspberry-pi-os-64-bit/)。

> **老王点评**: 终于升级到 64 位了，虽然之前可以使用其他 64 位操作系统。

![](/data/attachment/album/202202/03/184059hy6hnhhwze5kwjpn.jpg)

### Oracle Linux 登陆 Windows 商店

Oracle Linux 8.5 是 CentOS 8.5 的衍生发行版，现已以 WSL（Windows Subsystem for Linux）的形式出现在 Windows 商店。这是红帽家族的发行版首次出现在 Windows 商店，尽管 Oracle Linux 并不属于红帽系。在 Windows 商店中，如果你搜索 Linux，你会发现 Ubuntu、SUSE/openSUSE、Debian、Kali 和 Alpine 等发行版，这些都是为 WSL 开发的。然而，你不会看到 Fedora、CentOS 或 RHEL 等红帽系的 Linux，尽管有人为 WSL 制作了 Fedora remix。

> **老王点评**: 我就说 Oracle Linux 是叛徒，对吧。:D

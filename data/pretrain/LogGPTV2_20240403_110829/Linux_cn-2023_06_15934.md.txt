[Openbox](http://openbox.org/wiki/Main_Page) 是一款适用于 Linux 的轻量级、可配置的窗口堆叠式管理器，支持多种标准，并兼容各种桌面环境。值得注意的是，LXDE 和 LXQT 桌面环境都是基于 Openbox 构建的。你可以用它替换现有的桌面环境窗口管理器。

虽然几乎任何 Linux 发行版都可以安装 Openbox，但配置过程可能需要时间和精力。一个更简便的方法是使用预装了 Openbox 变种的发行版。本文将介绍一些让你轻松体验 Openbox 的发行版。请注意，列表按字母顺序排列，并不代表排名。

### 1. Archcraft

![archcraft live media with neofetch open in terminal](/data/attachment/album/202306/23/093358f0z08b0c8apgkksk.png)

Archcraft 是一个以 Openbox 为默认桌面环境的发行版，提供了一个精简且轻量的环境，只需不到 500MB 的内存即可运行。其界面风格统一，支持通过点击切换主题，甚至提供了类似 Windows 的用户界面。对于资深用户，它内置了对 AUR 和 Chaotic-AUR 的支持，提供了出色的开箱即用体验。

> **[Archcraft](https://archcraft.io/)**

### 2. ArcolinuxB Openbox

![arcolinuxb openbox live media with neofetch open in terminal](/data/attachment/album/202306/23/093358cylo7ys5lrsnjkmr.png)

ArcolinuxB Openbox 是一个适合学习 Arch 的发行版。它是 ArcolinuxB 项目的多个变种之一，可能会有一些学习曲线和不完善之处。与 Archcraft 不同，该发行版的界面元素并不完全统一，可能需要进行一些调整来获得良好的体验。

> **[ArcolinuxB](https://arcolinuxb.com/)**

### 3. AV Linux MX Edition

![av linux live media with neofetch open in terminal](/data/attachment/album/202306/23/093359lwbpk08g3lgux8l0.png)

AV Linux MX Edition 基于 MX Linux，使用 Openbox 作为窗口管理器。它采用了高性能的 [Liquorix 内核](https://liquorix.net/#features)，提供了音频爱好者所需的低延迟，并通过 Wine-staging 支持 Windows 音频。如果你是一位音频专业人士并使用 Linux，这个发行版值得一试。不过，它预装了许多应用程序，可能会显得有些臃肿。

> **[AV Linux](http://www.bandshed.net/avlinux/)**

### 4. Bunsenlabs Linux

![bunsenlabs live media with neofetch open in terminal](/data/attachment/album/202306/23/093359qxupx57td944p3x6.png)

BunsenLabs Linux 是一个基于 Debian 的发行版，提供了一个轻量且易于定制的 Openbox 桌面环境。该项目源于 [CrunchBang Linux](https://en.wikipedia.org/wiki/CrunchBang_Linux)。尽管它仍然基于 Debian 10，软件仓库中的应用程序版本较旧，但由于包含了硬件和多媒体支持，它具有很好的开箱即用体验。它的界面类似于 Archcraft，还提供了丰富的 Conky 配置选项。

> **[BunsenLabs Linux](https://www.bunsenlabs.org/)**

### 5. Crunchbang++

![crunchbangplusplus live media with neofetch open in terminal](/data/attachment/album/202306/23/093400dijwfizuc9wpew29.png)

Crunchbang++ 是 Crunchbang 的一个分支，尽量保持与原版接近。Crunchbang 是一个十年前停止开发的流行 Openbox 发行版。Crunchbang++ 是一个极简且轻量级的发行版，基于 Debian 11，相比 Bunsenlabs 可以提供更新的软件包。

> **[Crunchbang++](https://crunchbangplusplus.org/)**

### 6. Mabox Linux

![mabox linux live media with neofetch open in terminal](/data/attachment/album/202306/23/093400cekshe0aesb3qkek.png)

Mabox Linux 是一个基于 Manjaro 的现代发行版，专注于定制化或美化。由于使用了轻量级组件，它是一个极简且快速的系统，并采用滚动更新机制，提供最新的软件版本。该发行版的独特功能包括 Colorizer（根据壁纸更改强调颜色）、Quicktiling（用于轻松进行窗口平铺）和可定制的菜单/面板。这些定制化功能可能让一些极简主义者感到有些复杂。

> **[Mabox Linux](https://maboxlinux.org/)**

### 7. Sparky Linux Openbox

![sparky linux openbox live media with neofetch open in terminal](/data/attachment/album/202306/23/093401agmc5xmx08880666.png)

Sparky Linux 是一个基于 Debian 的 Linux 发行版，也提供 Openbox 作为另一种桌面环境选择。它有一个基于 Debian Testing 的版本，对于需要最新应用程序的用户非常有用。Sparky Linux 致力于提供开箱即用的 Debian 体验，同时留给用户足够的定制空间，因此界面效果相对简单。

> **[Sparky Linux](https://sparkylinux.org/)**

### 总结

还有其他几个 Linux 发行版可以安装 Openbox，但在本列表中，我们介绍了那些在 Live 介质中提供 Openbox 并将其作为默认桌面环境的发行版。你最喜欢的 Openbox 发行版是哪个？你喜欢预先定制好的还是更喜欢自己进行定制？欢迎分享你的建议。

User: How to set the program opened by default under the Windows operating system?
Assistant: 1. Locate the icon of the file whose default opening method you want to change, and right-click it. 2. Select the Properties option. 3. In the pop-up "Properties" dialog box, find the "Open With" option in the "General" tab. 4. Click the "Change" button and select the app you want to set as the default open program. 5. If you want this program to open all files of the same type by default, check the "Always use the selected program to open this type of file" option. 6. Click the "OK" button to save the changes.

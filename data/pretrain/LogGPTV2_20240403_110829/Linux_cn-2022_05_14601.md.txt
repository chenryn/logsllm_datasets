### 微软：“IE 被黑了这么多年”

微软宣布，Internet Explorer (IE) 浏览器将于 2022 年 6 月 16 日正式退役。Edge 浏览器的官方微博发布了一条告别信息：“终于，被黑了这么多年，IE 要正式跟大家说再见啦。IE 27 年的旅程，从今以后，这条路由 Edge 继续走下去。”IE 浏览器是微软在 1995 年 8 月随 Windows 95 推出的。推出三年后，IE 成为全球市场份额第一的浏览器，并在 2003 年占据了高达 95% 的市场份额。

尽管 IE 即将退役，但对于那些仍依赖 IE 的网站，Edge 浏览器将继续提供 IE 模式支持，该模式至少会持续到 2029 年。此外，出于企业需求，Windows 10 LTSC（长期服务版）也将继续保留 IE 浏览器。

**消息来源**: [微博](https://weibo.com/7732075754/Ltb3e5PQM)

**老王点评**: 我认为，真不是大家黑你，是你本身就有问题。不过，大家也别高兴得太早，还有很多公共网站仍然依赖于 IE。

### Python 的速度将提升一倍

去年，微软资助了 Python 软件基金会的 Faster CPython 项目，旨在使 Python 的速度比目前稳定的 3.10 版本快一倍。现在，该项目公布了更多结果，将 3.11 beta 预览版与 3.10 版本的几十项性能指标进行了比较，结果显示 3.11 总体上比 3.10 快 1.25 倍。该项目负责人表示，“Python 一直被认为较慢。虽然 Python 永远无法达到像 C、Fortran 或 Java 这样的低级语言的性能，但我们希望它能与脚本语言的快速实现相竞争，比如 JavaScript 的 V8 或 LuaJIT。”

**消息来源**: [ZDNet](https://www.zdnet.com/article/programming-languages-python-is-slow-but-its-about-to-get-faster/)

**老王点评**: 微软真是做了一件大好事，久被诟病的 Python 速度问题，居然就这样提高了一倍！

### 微软还有第二个内部使用的 Linux 发行版

我们知道，微软有一个内部使用的 Linux 发行版 CBL-Mariner，最近刚刚发布了 2.0 版本。现在有人发现，微软在同一时期还发布了一个名为 CBL-Delridge（或 CBL-D）的 Linux 发行版。两者之间的主要区别在于：Delridge 是一个定制的 Debian 衍生版，而 Mariner 是一个定制的 LFS 风格的发行版。Azure Cloud Shell 就是基于 CBL-D 构建的，它提供了一套打包在容器中的云管理工具。

**消息来源**: [ZDNet](https://www.zdnet.com/article/surprise-theres-yet-another-microsoft-linux-distro-cbl-delridge/)

**老王点评**: 微软现在真正践行了“拿来主义”，凡是好的东西都拿来使用。

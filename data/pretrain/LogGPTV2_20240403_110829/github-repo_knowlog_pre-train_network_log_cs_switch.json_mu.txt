- **建议操作**: 调查应用程序中的错误。原因类型 - API

- **建议操作**: 确保源节点或其网络节点服务器上的任何 mode-to-COS 映射表不使用未识别的 COS。原因类型 - SNA

### 10. 无法生成会话路由：从源节点找不到合适的 TG
- **日志代码**: SNASW-4-TS_LOG_1
- **警告级别**: 4
- **描述**: 无法生成会话路由：从源端节点到骨干网络找不到适合指定类服务的 TG [chars]。这可能是由于源节点和网络节点之间的临时链路故障引起的。会话激活将以指定的感觉代码失败。
- **建议操作**: 首先检查源节点是否存在解释路由失败的链路故障。如果没有找到链路故障，则：
  - 更改源节点或其网络节点服务器上的 mode-to-COS 映射，以使用不同的类服务。
  - 重新配置网络，以确保从源节点到骨干网络有可用的路由，例如通过定义和/或激活从源节点到网络节点的另一条链路。原因类型 - SNA

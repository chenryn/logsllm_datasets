User: In the UNIX system, how to check the current CPU load?
Assistant: You can use the following commands to view the current CPU load: 1. top: This command can view system processes in real time, including information such as CPU usage, memory usage, and process status. After entering the top command on the command line, press the "1" key to view the load of each CPU core. 2. uptime: This command can view information such as the running time of the system, the number of currently logged-in users, and the average load. After entering the uptime command on the command line, you can see information similar to the following: ``` 13:38:41 up 35 days, 22:52, 2 users, load average: 0.45, 0.26, 0.18 ``` Among them, load average indicates the average load of the system, representing the average load of the last 1 minute, the last 5 minutes, and the last 15 minutes respectively. 3. mpstat: This command can view the usage of each CPU core. After entering the mpstat command in the command line, you can see information similar to the following: ``` Linux 4.15.0-45-generic (hostname) 08/31/2021 _x86_64_ (4 CPU) 05:58:58 PM CPU %usr %nice %sys %iowait %irq %soft %steal %guest %gnice %idle 05:58:58 PM all 0.06 0.00 0.03 0.13 0.00 0.01 0.00 0.00 0.00 99.78 ``` Among them, %usr indicates the user mode CPU usage, % sys indicates the CPU usage rate in the kernel mode, and %idle indicates the CPU idle rate. 4. sar: This command can check the resource utilization of the system, including CPU, memory, disk, network, etc. After entering the sar command on the command line, you can use the parameters -c, -u, -n, etc. to view information about the CPU and network. For example, sar -u can view CPU usage.

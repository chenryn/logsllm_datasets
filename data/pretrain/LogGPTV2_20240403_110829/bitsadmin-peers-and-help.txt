### Bitsadmin Peers Command

The `bitsadmin peers` command, when used with the `/help` switch, displays the command-line usage for the `/peers` switches.

#### Syntax
```sh
bitsadmin /peers /help
```

#### Examples
To display the command-line usage for the `/peers` switches, run the following command:
```sh
bitsadmin /peers /help
```

This will provide you with detailed information on how to use the `/peers` switches in the `bitsadmin` utility.

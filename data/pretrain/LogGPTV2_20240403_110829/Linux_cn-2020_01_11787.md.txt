华为推出了一款基于 CentOS 的企业级 Linux 发行版 EulerOS。最近，华为发布了 EulerOS 的社区版——openEuler。

### 关于 openEuler

[openEuler](https://openeuler.org/en/) 的源代码也已公开发布。你可以在 Gitee（一个中文版的 GitHub 替代品）上找到它，而不是在微软旗下的 GitHub 上。Gitee 上有两个独立的存储库：
- 一个用于存储 [源代码](https://gitee.com/openeuler)。
- 另一个作为 [软件包的源代码](https://gitee.com/src-openeuler)，包含构建该操作系统所需的软件包。

![](/data/attachment/album/202001/16/105741tdqgzjqg2o2s7mlo.jpg)

openEuler 基础架构团队分享了他们使源代码可用的经验：
> 我们现在很兴奋。很难想象我们会管理成千上万的仓库。为了确保它们能被成功地编译，我们要感谢所有参与贡献的人。

### openEuler 是基于 CentOS 的 Linux 发行版

与 EulerOS 一样，openEuler 也是基于 [CentOS](https://www.centos.org/)，但华为技术有限公司进一步开发了该操作系统以满足企业应用的需求。它是专为 ARM64 架构服务器设计的，并且华为声称已经进行了一些改进以提高其性能。更多详情请参阅 [华为开发博客](https://developer.huaweicloud.com/en-us/euleros/euleros-introduction.html)。

![](/data/attachment/album/202001/16/105745hamtndgugldeniil.jpg)

目前，根据 openEuler 的官方声明，已有超过 50 名贡献者为 openEuler 提交了近 600 个提交。这些贡献者的努力使得源代码对社区开放成为可能。

值得注意的是，存储库还包括两个相关的子项目：
- [iSulad](https://gitee.com/openeuler/iSulad)：一个轻量级的容器运行时守护进程，专为物联网和云基础设施设计。
- A-Tune：一个基于 AI 的操作系统调优软件。

另外，官方公告提到，这些系统是在华为云上通过脚本自动构建的，这确实非常有趣。

### 你对华为的 openEuler 有何看法？

据 cnTechPost 报道，华为曾宣布 EulerOS 将以新名字 openEuler 成为开源软件。目前还不清楚 openEuler 是否会取代 EulerOS，或者两者是否会像 CentOS（社区版）和 Red Hat（商业版）那样同时存在。

我还没有测试过它，因此不能说 openEuler 是否适合英文用户。你愿意试一试吗？如果你已经尝试过了，请在下面的评论中告诉我你的体验。

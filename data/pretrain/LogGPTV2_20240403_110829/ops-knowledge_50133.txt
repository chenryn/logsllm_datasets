User: How to configure the IP address of the Ethernet network interface card (NIC) under the Linux system?
Assistant: The IP address of an Ethernet network interface card (NIC) can be configured using the ifconfig or ip command. Use the ifconfig command: 1. Open the terminal and enter the following command: $ ifconfig eth0 2. Then, enter the following command: $ sudo ifconfig eth0 192.168.1.100 netmask 255.255.255.0 This will configure the IP address for the eth0 network card as 192.168.1.100, subnet The mask is 255.255.255.0. Use the ip command: 1. Open the terminal and enter the following command: $ ip addr show 2. Then, enter the following command: $ sudo ip addr add 192.168.1.100/24 dev eth0 This will configure the IP address for the eth0 network card as 192.168.1.100, The subnet mask is 255.255.255.0. In both methods, you need to replace eth0 in the command with the corresponding network card name, such as eth1 or enp0s3.

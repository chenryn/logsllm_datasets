### Oracle 调研如何避免让 Java 开发者投奔 Rust 和 Kotlin

Oracle 委托分析公司 Omdia 评估其 Java 的 6 个月发布策略，以及这一策略是否足以使数百万 Java 开发者远离内存安全的替代方案，如谷歌的 Kotlin 和 Mozilla 的 Rust。报告指出，“6 个月的更新周期，并引入新的模块化水平，使 Java 在约 1200 万开发人员中处于有利地位。Oracle 和 Java 编程语言需要持续不断的创新，以确保现有 Java 开发者的满意度，并吸引潜在的 Java 开发者，使其不转向 Rust 和 Kotlin 等新语言。”

来源：[zdnet](https://www.zdnet.com/article/oracles-java-15-new-features-aim-to-keep-millions-away-from-languages-like-rust-kotlin/)

> **拍一拍**：尽管 Java 已有 25 年历史并仍占据主导地位，但这些新兴编程语言也不容忽视。

### 英特尔对迟迟未被 Linux 主线接受的 SGX Enclave 进行了第 38 次修订

多年来，英特尔的 Linux 开发人员一直致力于将软件卫士扩展（SGX）支持及其新的 SGX Enclave 驱动整合到 Linux 内核中。自 Skylake 核心推出以来，尽管一直在进行相关工作，但由于安全问题和技术障碍，SGX 支持尚未被主线内核接受。此外，非英特尔上游内核开发者对 SGX 的热情也相对较低。

来源：[cnbeta](https://www.cnbeta.com/articles/tech/1031405.htm)

> **拍一拍**：英特尔已进行了 38 次修订，但要获得上游社区的认可，还需要更广泛的共识。

### ARM 支持开源的 Panfrost Gallium3D 驱动

Mesa 的 Panfrost Gallium3D 开源驱动为 ARM 的 Mali Midgard 和 Bifrost GPU 提供了支持。该项目最初基于对私有 Mali 图形驱动的逆向工程，进展缓慢。然而，随着主要开发者 Alyssa Rosenzweig 受雇于咨询公司 Collabora，项目取得了显著进展。在 XDC2020 大会上，Alyssa 透露，ARM 正与 Collabora 合作，支持 Panfrost Gallium3D 作为开源的 Mali 驱动。

来源：[solidot](https://www.solidot.org/story?sid=65592)

> **拍一拍**：从最初的逆向工程到现在得到厂家的主动支持，企业越来越重视开源项目。

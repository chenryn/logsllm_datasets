# [⁠]{#ap-admin-options.html#ap-admin-options}第 28 章 引导选项 {.title}
:::
::: para
Red Hat Enterprise Linux
安装系统包括一系列管理员使用的功能和选项。要使用引导选项，请在
`boot:`{.prompt} 提示符后输入 **`linux option`**。
:::
::: para
对于显示形化引导界面的系统来说，当图形化引导界面出现时，按
[**Esc**]{.keycap} 键可以访问 `boot:`{.prompt} 提示符。
:::
::: para
如果需要指定多个选项，请使用一个空格将其分开。例如：
:::
``` screen
linux option1 option2 option3
```
::: {.note xmlns:d="http://docbook.org/ns/docbook"}
::: admonition_header
**注意**
:::
::: admonition
::: para
Red Hat Enterprise Linux
安装和*救援光盘*可以使用*救援模式*引导，或者载入安装系统。有关救援光盘和救援模式的更多信息，请参考
[第 28.6.2 节
"使用救援模式引导计算机"](#ap-admin-options.html#sn-mode-rescue){.xref}。
:::

## [⁠]{#ap-admin-options.html#sn-boot-options-language}28.1.1. 指定语言 {.title}
:::
::: para
要设置安装过程和最终系统使用的语言，请用 `lang`{.option} 选项指定语言的
ISO 代码，用 `keymap`{.option} 选项配置正确的键盘布局。
:::
::: para
例如， ISO 代码 **`el_GR`** 和 **`gr`** 分别是希腊语和希腊键盘布局:
:::
``` screen
linux lang=el_GR keymap=gr
```
:::
::: section
::: titlepage
## [⁠]{#ap-admin-options.html#sn-boot-options-interface}28.1.2. 配置界面 {.title}
:::
::: para
要使用特定显示分辨率，请输入 `resolution=setting`{.option}
作为引导选项。例如：要将显示分辨率设定为 1024x768，请输入：
:::
``` screen
linux resolution=1024x768
```
::: para
要在[]{#ap-admin-options.html#idm139817112773232 .indexterm}
`text`{.option} 模式中运行安装进程，请输入：
:::
``` screen
linux text
```
::: para
要启用对[]{#ap-admin-options.html#idm139817169287520
.indexterm}串口控制台的支持，请输入 `serial`{.option} 作为附加选项。
:::

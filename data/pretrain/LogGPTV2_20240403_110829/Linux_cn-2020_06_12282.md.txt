### 2020 年 5 月数据：Windows 10 占比达 57.8%，Linux 系统增长明显

根据 NetMarketShare 的最新数据，Windows 10 在全球桌面操作系统中的市场份额进一步上升了 1.8 个百分点，达到了 57.8%。与此同时，GNU/Linux（包括 Ubuntu 等发行版）的市场份额也从 2020 年 3 月的 1.36% 上升至 3.17%，创下历史新高。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/986851.htm)

**硬核老王点评**: 据说 Linux 桌面份额的增长与全球疫情导致的居家办公需求增加有关。

### 一张墙纸导致部分 Android 手机变砖

近日，多名 Android 用户在社交媒体上报告称，一张风景照片被设置为壁纸后会导致系统崩溃，屏幕反复重启，甚至需要恢复出厂设置才能恢复正常。受影响的手机品牌包括三星和 Google 的 Pixel。据分析，该图像并非标准 RGB 格式，可能包含了一些设备无法处理的色彩信息。

**来源**: [solidot](https://www.solidot.org/story?sid=64566)

**硬核老王点评**: 据悉，修复措施已在进行中，但用户应避免尝试以防止出现类似问题。

### Firefox 77.0 发布

Mozilla 正式发布了 Firefox 77.0 版本。主要的新功能包括：
- 向更多 Windows 用户推送 WebRender 技术；
- 通过新的 `about:certificate` 页面管理 Web 证书；
- 改进 JavaScript 调试工具；
- 支持 JavaScript API String.prototype.replaceAll()；
- 多项 Bug 修复等。

**来源**: [solidot](https://www.solidot.org/story?sid=64559)

### Devuan Beowulf 3.0.0 发布

2014 年，由于对初始化系统 systemd 的争议，一群不满的开发者创建了不使用 systemd 的 Debian 分支 Devuan。2017 年，Devuan 开发者发布了基于 Debian 8 Jessie 的 Devuan Jessie 1.0 正式版。2018 年 6 月，他们又发布了 Devuan 2.0 ASCII。现在，他们发布了 Devuan Beowulf 3.0.0，基于 Debian Buster (10.4) 和 Linux kernel 4.19。下一个大版本代号为 Chimaera。

**来源**: [solidot](https://www.solidot.org/story?sid=64558)

**硬核老王点评**: 开源的一大优势在于，如果对某个项目不满，可以复刻一个新分支。

### Linux Mint 表示不会违背用户意愿安装 Snap 封装应用

Ubuntu 20.04 发布后，Canonical 被发现强制推广 Snap 应用，即使用户是通过 apt 安装的应用。基于 Ubuntu 的流行发行版 Linux Mint 明确表示不会违背用户意愿安装 Snap 应用。即将发布的 Linux Mint 20 不会预装任何 Snap 应用，默认情况下 apt 也不会安装 Snap 应用，用户如需使用可自行安装。

**来源**: [solidot](https://www.solidot.org/story?sid=64554)

**硬核老王点评**: 开源的另一个好处是，各发行版可以根据自己的理念独立发展。

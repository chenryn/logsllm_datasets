User: Using Python and the Requests library, write a script that can get the IP address of a specified website.
Assistant: import requests # Enter the website address url = input("Please enter the website address to be queried:") # Send the request and get the response response = requests.get(url) # Extract the IP address ip_address = response.json()['origin' ] # Output IP address print(f"The IP address of {url} is: {ip_address}")

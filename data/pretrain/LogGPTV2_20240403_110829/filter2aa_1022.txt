## 中国贡献者在微软漏洞赏金计划中的表现

2019年，中国的安全研究人员成为了我们漏洞赏金计划中最活跃且最具影响力的贡献者之一。以下是一些关键数据点：

- **参与度增长**：来自中国的参与者增加了57%。
- **提交成果**：来自中国的提交占总提交量的46%。
- **影响力**：中国提交的安全问题影响了全球48%的用户。
- **奖金分配**：中国研究者获得了全球漏洞赏金总额的22%。

### 漏洞类型分析
根据统计，在2019年所提交的漏洞中：
- 内存安全问题占56%
- 信息泄露问题占15%

### 微软漏洞赏金计划覆盖范围
该计划涵盖了多个产品领域，包括但不限于：
- Microsoft Office Insider
- .NET Core 和 ASP.NET Core
- Microsoft Edge
- Azure DevOps
- Windows Defender Application Guard
- Windows Insider Preview
- Microsoft Identity Mitigation Bypass and Defense
- Microsoft Hyper-V

特别值得注意的是，针对Microsoft Edge、Windows Insider Preview以及Microsoft Hyper-V等项目的提交数量分别增长了75%、40%和50%。

## 回顾过去
自上世纪90年代起，微软就一直致力于提升产品的安全性。从MSRC（微软安全响应中心）于1998年的成立到近年来推出的一系列创新技术如控制流防护(CFG)、高级威胁防护(ATP)等，无不体现了公司对于构建更加安全计算环境的决心。

随着数字化转型步伐加快，预计到2025年将有超过60%的计算能力通过云端提供。为了应对这一变化趋势，每个组织都需要成为软件驱动型企业。而Azure作为全球领先的云计算平台之一，正助力成千上万家企业实现业务创新与发展。

User: How to install and configure Nginx server on Linux system?
Assistant: Install Nginx server: 1. Execute the following command in the terminal to install Nginx: ``` sudo apt-get update sudo apt-get install nginx ``` During the installation process, the system will prompt you to enter the sudo password and confirm whether to install it. 2. After the installation is complete, use the following command to check whether Nginx is successfully installed: ``` nginx -v ``` If the installation is successful, you will see a message similar to the following: ``` nginx version: nginx/1.10.0 (Ubuntu) ``` To configure the Nginx server: 1. Open the /etc/nginx/conf.d/default.conf file: ``` sudo vi /etc/nginx/conf.d/default.conf ``` This is the default configuration of the Nginx server Configuration file, you can modify it according to your needs. 2. Configure the virtual host: Add the following line of code in the default.conf file to create a new virtual host: ``` server { listen 80 default_server; listen [::]:80 default_server; root /var/www/ html; index index.html index.htm index.nginx-debian.html; server_name _; location / { try_files $uri $uri/ =404; } } ``` In this example, you create a file named "_ " virtual host. This virtual host will listen on port 80 and use the files in the /var/www/html folder as the root directory. 3. Customize the Nginx server according to your needs: If you need specific configuration, such as setting up Nginx as a reverse proxy, using an SSL certificate, or modifying the default Nginx port, you can edit the default configuration file /etc/nginx/nginx. conf.

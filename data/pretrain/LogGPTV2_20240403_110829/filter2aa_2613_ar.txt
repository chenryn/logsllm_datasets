### 积极与消极、自由派与保守派

积极的人被称为“自由派”，而消极的人则被称为“保守派”。宗教作为一种正式的组织，拥有自己的政策、计划、活动和成员，但它本身并不能赋予人们持久的安全感或内在价值。只有遵循教堂所倡导的原则，个人才能获得这种安全感。

宗教也无法长期为人指引人生方向。以宗教为中心的人在礼拜日和工作日的行为方式往往截然不同，这种人格上的不一致性会威胁到他们的安全感，并促使他们进一步给别人贴标签和为自己辩护。将宗教视为目标而非实现目标的手段，会削弱智慧和平衡感。尽管宗教声称能通过教导赋予人们力量，但其本质上只是上帝传授神圣力量的工具，并非力量本身。

### 以自我为中心

当前社会中，以自我为中心的人最为常见。这些人最明显的特征是自私自利，这与多数人的价值观相悖。然而，市面上盛行的各种个人成功术无一不是以个人为中心，鼓吹只索取不付出。这种狭隘的自我中心观会导致缺乏安全感和人生方向，也不会有智慧及行动力。就像以色列的死海，只有流入没有流出，最终变成一潭死水。唯有以造福人群和无私奉献为目的追求自我成长，才能在这些方面不断进步。

### 识别生活中心

你目前的生活状况如何？什么是你的生活中心？这个问题并不容易回答。一个有效的方法是详细考察支撑自己人生的因素。下表列出了一些常见的行为特征，通过这些特征可以追踪到导致这些行为的生活中心——一种限制效能的生活中心。

一般而言，我们的生活中心往往是几种因素的混合体，且随环境变化而变化。大多数人的生活受到多种因素的影响，可能今天以朋友为中心，明天又变为以配偶为中心。

#### 表 4-1 各种重心的特征

| 生活中心 | 特征 |
| --- | --- |
| 家庭 | 优先考虑家庭成员的感受和需求 |
| 金钱 | 重视经济利益和个人财富积累 |
| 工作 | 专注于职业发展和工作表现 |
| 名利 | 追求名声和社会地位 |
| 享乐 | 注重个人享受和娱乐 |
| 朋友 | 依赖于社交关系和朋友的支持 |
| 敌人 | 以对抗和竞争为动力 |
| 宗教 | 遵循宗教信仰和教义 |
| 自我 | 只关心个人利益 |

生活中心的摇摆不定会导致情绪起伏不定，一会儿意气风发，一会儿颓唐沮丧；一会儿斗志昂扬，一会儿又落魄消沉。缺乏固定的人生方向，没有持久的智慧，也没有稳定的力量或自我评价。因此，建立清晰明确的生活中心至关重要，这样才能产生高度的安全感、人生方向、智慧和力量，使人生更加积极和谐。

### 以原则为中心

以正确原则为生活中心可以为发展上述四个支撑人生的因素奠定坚实的基础。认识到这一点，我们就有了安全感。原则是恒久不变且历久弥新的，不像其他中心那样多变，因此值得信赖，可以给我们高度的安全感。原则是理性的而非感性的，不会因环境、他人行为或流行趋势而改变。即使某时某地某人无视原则的存在，我们也无须忧心，因为原则超越时空的限制，几千年的历史一次又一次地见证了原则的胜利。

理解成长的原则可以让我们在寻找正确原则的时候充满自信，相信学得越多，就越能以正确的角度更清楚地观察世界。如果以原则为生活中心，智慧和人生方向的来源就是正确的地图，反映事物的真实历史和现状。正确的地图让我们能够清晰了解自己的目标以及实现途径，基于正确的资料做出更有意义且易执行的决策。

力量来自自我意识、知识和积极的心态，因而能够摆脱环境及他人态度和行为的制约。唯一能制约力量的是原则本身的必然后果。我们可以自由选择行动，但无法选择行动的后果。任何原则都有必然后果，遵从原则，后果就是积极的；忽视原则，后果就是消极的。越了解正确的原则，明智行动的自由度就越大。

以永恒不变的原则作为生活中心，就能建立高效能的思维定式，从而正确审视所有其他的生活中心。一个人的思维定式能决定他的态度和行为，就像透镜影响一个人对世界的观察一样。生活中心不同，产生的观念也就各异。

### 实例分析

假设你已经买好票，准备晚上与配偶一起听音乐会，对方兴奋不已，满怀期待。下午四点钟，老板突然来电话要你晚上加班，理由是第二天上午九点钟有一个重要会议。以下是不同生活中心的人可能的反应：

- **以家庭或配偶为中心的人**：优先考虑配偶的感觉，可能会委婉地拒绝老板。
- **以金钱为中心的人**：看重加班费或加班对调薪的影响，理直气壮地告诉配偶自己要加班。
- **以工作为中心的人**：觉得正中下怀，认为加班有利于晋升，无论是否需要都会自动延长加班时间。
- **以名利为中心的人**：算计加班费能买到什么，或考虑加班对个人形象有何助益。
- **以享乐为中心的人**：即使配偶不介意，也会撇下工作赴约，因为需要犒劳自己。
- **以朋友为中心的人**：根据是否有朋友同行或其他工作伙伴是否也加班来做决定。
- **以敌人为中心的人**：乐于留下，认为这是一个打击对手的良机。
- **以宗教为中心的人**：考虑其他教友的计划，办公室是否有其他教友，或者音乐会是否与宗教相关。
- **以自我为中心的人**：只关心哪一样对个人的好处更大，听音乐会还是让老板增加好感。

以原则为中心的人会保持冷静和客观，不受情绪或其他因素的干扰，综观全局——工作需要、家庭需要、其他相关因素以及不同决定的可能后果，深思熟虑后才做出正确的选择。这种选择具有以下特征：

1. **主动的选择**：不受环境或他人的影响，是通盘考虑后的最佳方案。
2. **有效的选择**：基于原则，其长期后果可以预料。
3. **有意义的选择**：提高自身价值，有助于从整体上提高生活质量。
4. **可解释的选择**：若平时已与配偶和老板建立了良好的相互依赖关系，此时就不难向他们解释如此决定的理由，并得到体谅。

通过这种方式，以原则为中心的人能够更好地应对生活中的各种挑战，实现更加积极和谐的人生。

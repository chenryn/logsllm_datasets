**ATT&CK ID:** S0284  
**ATT&CK Technique Name:** More_eggs

**Description:**
The More_eggs malware has been observed using `regsvr32.exe` to execute a malicious DLL.[26] 

This technique leverages the legitimate Windows utility `regsvr32.exe` to load and execute the malicious code, thereby bypassing some security controls that might otherwise detect or block the execution of the malware.

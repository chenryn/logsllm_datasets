##### SUMMARY
The pip module verifies that python libraries with a particular version are
installed on the target host. However, it does not support range versions.
##### STEPS TO REPRODUCE
With support for a range version, the module could be used in a task as
follows-
    - name: Verify dependencies
      pip:
        name: docker-py
        version: > 1.7.0

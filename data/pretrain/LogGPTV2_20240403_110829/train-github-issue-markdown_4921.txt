### Bug summary
Not sure if this is a Matplotlib bug or not. I am aware that this method is
deprecated, but it is currently used in IPython and I assume that it shouldn't
restart the terminal.

Matplotlib 3.5.1 a restart happens on my Windows 10 laptop. Same thing if
executed in a standard python prompt (although only the deprecation warnings
are shown).
On a Linux machine with Python 3.9.7 and IPython 7.29.0 it works (as in not
crashing).

1. **ATMPA-3-TXFREEZE**
   - **错误代码**: 3
   - **描述**: 发送冻结。此消息表示PCI-ATM接口在等待数据时停止了发送。
   - **建议操作**: LOG_STD_RECUR_ACTION

2. **ATMPA-3-RXFREEZE**
   - **错误代码**: 3
   - **描述**: 接收冻结。此消息表示ATM接口上的流量拥塞可能导致了信元丢失。当信元到达接口时，PCI-ATM接口的信元缓冲区已满，导致接收冻结。
   - **建议操作**: LOG_STD_RECUR_ACTION

3. **ATMPA-2-INVDEVICE**
   - **错误代码**: 2
   - **描述**: 槽位[int]设备ID无效[ID: hex]。ATM Deluxe PCI驱动程序在初始化过程中发现了一个不正确的PCI设备ID。这可能意味着硬件故障导致无法读取ID，或者软件需要升级以支持新硬件。
   - **建议操作**: LOG_STD_ACTION

4. **ATMPA-3-NOTXDESCSPACE**

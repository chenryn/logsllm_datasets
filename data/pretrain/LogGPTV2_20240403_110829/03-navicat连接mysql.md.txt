# 使用 Navicat 连接 MySQL

以下步骤将指导您如何使用 Navicat 工具连接到 MySQL 数据库：

1. **启动 Navicat**  
   打开您的 Navicat 应用程序。  
   ![](media/image1.png){width="5.768055555555556in" height="2.95625in"}

2. **创建新连接**  
   在 Navicat 的主界面左侧空白区域，点击右键并选择“新建连接”选项。  
   ![](media/image2.png){width="2.327366579177603in" height="4.077863079615048in"}

3. **输入连接信息**  
   在弹出的对话框中填写必要的连接信息，包括但不限于服务器地址、端口号、用户名和密码等详细数据。请确保这些信息与您的 MySQL 服务器配置相匹配。  
   ![](media/image3.png){width="3.6313331146106735in" height="3.159822834645669in"}

4. **测试连接有效性**  
   填写完毕后，请点击“测试连接”按钮来验证所输入的信息是否正确且能够成功建立链接至目标数据库。如果一切正常，您会收到确认消息；反之，则需要检查错误提示并进行相应调整。  
   ![](media/image4.png){width="3.637089895013123in" height="4.084732064741908in"}

5. **访问 MySQL 数据库**  
   成功完成上述所有步骤后，您可以通过双击列表中的 MySQL 项来浏览或操作已连接的数据库了。  
   ![](media/image5.png){width="2.134682852143482in" height="2.7767563429571305in"}

通过遵循以上指南，您应该能够顺利地在 Navicat 中设置并使用 MySQL 数据库连接。

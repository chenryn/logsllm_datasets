### Facebook 将 Windows 版 PyTorch 的开发维护权移交给微软

PyTorch 是一个基于 Torch 的开源 Python 机器学习库，由 Facebook 的人工智能研究团队开发。特斯拉的自动驾驶系统和 Uber 的 Pyro 深度学习项目均构建于 PyTorch 之上。此前，Facebook 的工程师们已经做了大量工作，以在 Windows 上支持 PyTorch。但由于资源有限，Windows 对 PyTorch 的支持相对滞后。如今，微软已扩大了对 PyTorch 社区的参与，并接管了 Windows 平台上 PyTorch 的开发和维护。

> **拍一拍**: 大家似乎已经习惯了微软在开源领域的贡献，因此看到这一消息并不感到惊讶。
>
> **来源**: [开源中国](https://www.oschina.net/news/117544/microsoft-becomes-maintainer-of-the-windows-version-of-pytorch)

### 微软首次公布 GitHub 产品路线图

微软近日首次公布了 GitHub 的产品路线图（[GitHub Roadmap](https://github.com/github/roadmap)）。这个新的公共信息库允许任何人访问。除了为未来几个季度的新功能做好准备外，开发者还可以积极向开发团队提供反馈。路线图上的每个项目都可以作为一个讨论板块使用。

> **拍一拍**: 如果大家对 GitHub 有什么期望，不妨去反馈一下。
>
> **来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/1009037.htm)

### 华为将 HMS 生态优先级提升至最高，目前已有 2500 个应用

华为轮值 CEO 徐直军表示，“打造 HMS 生态是解决华为手机在海外不可用状态的最快途径，这是当前最紧迫的任务。尽管 GMS 不再可用，但 AOSP 作为开源软件仍然可以使用。我们计划基于 AOSP 构建一个替代 GMS 的 HMS，让开发者能够迅速加入全球 HMS 生态。”

> **拍一拍**: 尽管当前面临困难，但这一关必须闯过。
>
> **来源**: [搜狐IT](https://www.cnbeta.com/articles/tech/1009119.htm)

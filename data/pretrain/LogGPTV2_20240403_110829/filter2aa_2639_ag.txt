#### 0.7.56 - LCDMsg Functions
**LCD message management functions.** These functions must be registered with a call to `UI.CreatePublic()`.
- **Create:**
  ```python
  var = LCDMsg Create()
  ```
- **Set String:**
  ```python
  LCDMsg SetStr(var, STRING)
  ```
- **Move:**
  ```python
  LCDMsg Move(var, Xpos, Ypos)  # (pixel coordinates)
  ```
- **Change Color:**
  ```python
  LCDMsg ChangeColor(var, COLOR)
  ```
  **COLOR varies with model. For example, in G10: 0=black, 1=gray, 3=white, 5=green, >5=red.**

**Example:**
```python
UI.CreatePublic()
a = LCDMsg Create()
LCDMsg SetStr(a, "HELLO WORLD")
LCDMsg Move(a, 10, 30)
```

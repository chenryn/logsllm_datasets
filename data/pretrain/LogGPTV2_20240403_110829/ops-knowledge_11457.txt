User: Please use Ansible to write a simple automated script to deploy the Apache server.
Assistant: Here is a simple Ansible playbook for installing and configuring Apache server on Ubuntu: ``` --- - name: Install and configure Apache server hosts: webserver become: true tasks: - name: Update APT cache apt: update_cache : yes - name: install Apache apt: name: apache2 state: present - name: start Apache service service: name: apache2 state: started enabled: yes - name: disable default virtual host apache2_module: state: absent name: "default" - name: Copy Apache configuration file copy: src: /path/to/your/apache_config_file.conf dest: /etc/apache2/sites-available/your_website.conf mode: '0644' - name: Create symbolic link file: src: / etc/apache2/sites-available/your_website.conf dest: /etc/apache2/sites-enabled/ state: link mode: '0644' - name: Apache service restarted systemd: name: apache2 state: restarted ``` To use For this playbook, make sure you have defined the appropriate webserver host in your hosts file and replace "your_website.conf" above with the name of your own Apache virtual host configuration file. Then, run the following command: ```bash ansible-playbook apache.yaml``` This will execute the above playbook and automatically install and configure the Apache server.

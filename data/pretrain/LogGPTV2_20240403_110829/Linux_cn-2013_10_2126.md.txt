### Ubuntu Tweak 0.8.6 发布，支持 Ubuntu 13.10 并有大量改进

**Ubuntu Tweak** 是一款功能强大且先进的配置工具，通过简洁直观的界面提供了大量的可调整选项，帮助用户对桌面系统进行深度定制。无论是修改字体大小、GTK+主题、Unity 启动器还是侧边栏启动列表等，Ubuntu Tweak 都能满足您的需求，甚至还可以屏蔽某些内核选项。其功能覆盖了整个强大的 Ubuntu 桌面环境。

[Ubuntu Tweak](https://launchpad.net/ubuntu-tweak) **0.8.6** 已经 [发布](http://blog.ubuntu-tweak.com/2013/10/13/ubuntu-tweak-0-8-6-released-for-13-10-saucy.html)，支持了 Ubuntu **13.10**。因此，用户可以在即将发布的 Ubuntu 13.10 中使用这一方便而强大的应用程序。

在新版本 0.8.6 中，改进了对旧内核的处理（可以更精准地从用户的系统中移除多余的旧内核），并禁用了“应用中心”和“资源中心”。在选择 `Ubuntu Tweak -> Admins` 选项卡时，可以看到“应用中心”和“资源中心”已被移除。

以上提到的功能，连同修复的 bug 和移除的错误，都已包含在新的 Ubuntu Tweak 0.8.6 版本中。

![](/data/attachment/album/201310/18/09065487jkq0klze6oojq0.png)

### 如何安装 Ubuntu Tweak 0.8.6？

对于 Ubuntu 12.04, 12.10, 13.04 和 13.10 用户，可以通过添加官方 PPA 来安装：

```bash
sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak
```

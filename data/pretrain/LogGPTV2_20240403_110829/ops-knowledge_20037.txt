User: How to configure firewall rules for Linux server using iptables command?
Assistant: iptables is a tool for configuring firewall rules on Linux systems, which can be operated through the command line. The following are some commonly used iptables commands: 1. View iptables rules sudo iptables -L 2. Clear iptables rules sudo iptables -F 3. Enable iptables sudo systemctl enable iptables 4. Stop iptables sudo systemctl stop iptables 5. Open specified ports (such as 80 port) sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT 6. Close the specified port (such as port 80) sudo iptables -A INPUT -p tcp --dport 80 -j DROP 7. Allow all external connections sudo iptables -P INPUT ACCEPT sudo iptables -P OUTPUT ACCEPT 8. Forbid all external connections sudo iptables -P INPUT DROP sudo iptables -P OUTPUT DROP 9. Configure to allow the internal network to access the external network through the firewall sudo iptables -A FORWARD -i eth1 -o eth0 -j ACCEPT 10. Configure port forwarding to the internal network sudo iptables -t nat -A PREROUTING -p tcp --dport 80 -j DNAT --to-destination 192.168.1.2:80 The above are some commonly used iptables commands, you can Use it flexibly. Due to the complexity of iptables rules, it is recommended to carefully read the relevant documents and back up the original rules before operating to avoid system problems caused by misoperation.

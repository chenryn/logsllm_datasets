# Linux Mint 网站遭黑客攻击，下载链接被篡改指向带有后门的 ISO 镜像

### 事件概述
2016年2月20日，Linux Mint 官方网站遭到黑客攻击。如果你在当天下载了该操作系统的 ISO 镜像文件，那么你的系统可能已经感染了恶意软件。

Linux Mint 是当前最受欢迎和广泛使用的 Linux 发行版之一。如果你最近安装了该操作系统，很遗憾地通知你，你可能已经下载并安装了一个包含恶意程序的 ISO 镜像。

### 事件详情
20日晚上，未知的黑客或组织入侵了 Linux Mint 的官方网站，并将下载链接替换为指向一个托管着恶意 ISO 镜像（Linux Mint 17.3 Cinnamon 版本）的服务器。

Linux Mint 的负责人 Clement Lefebvre 对此评论道：
> “黑客制作了一个修改过的 Linux Mint ISO 文件，在其中植入了后门程序，然后设法入侵我们的网站，将下载链接指向了这个被篡改的版本。”

### 受影响范围
目前，仅发现 Linux Mint 17.3 Cinnamon 版本受到了此次攻击的影响。由于入侵发生在20日晚上，因此只有在当天晚上下载了该特定版本的用户才会受到影响。如果你是在20日之前下载的，或者下载的是其他版本的 Linux Mint（包括通过 Torrent 或 HTTP 下载），则不会受到此次攻击的影响。

### 攻击细节
黑客利用团队的 WordPress 博客作为切入点，获取了对底层服务器的访问权限。随后，他们篡改了 Linux Mint 的下载页面，将下载链接指向了一个位于保加利亚（IP: 5.104.175.212）的恶意 FTP 服务器。受感染的 Linux ISO 包含了一个完整的操作系统，其中包括一个名为 Tsunami 的 IRC 后门，允许黑客通过 IRC 服务器远程控制这些系统。

**注释**：Tsunami 是一种知名的 Linux ELF 木马，常用于发起 DDoS 攻击。

### 应对措施
尽管 Linux Mint 团队迅速响应并移除了网站上的恶意链接，但黑客不久之后再次入侵了下载页面。为了防止进一步的损害，Linux Mint 决定暂时关闭整个 `linuxmint.com` 域名。

Clement Lefebvre 表示：“我们不清楚这次攻击背后的动机是什么。如果攻击者继续对我们进行攻击，并且他们的目标是造成伤害，我们将寻求法律援助以及安全公司的帮助来找出幕后黑手。”

### 黑客行为
据报道，黑客在网上出售 Linux Mint 的数据，标价仅为85美元。从其选择使用 IRC bot 来攻击最新的 Linux 发行版这一行为来看，这些黑客可能是缺乏经验的新手。

### 如何保护你的 Linux 设备
对于已下载 ISO 镜像的用户，建议先验证文件签名的有效性，确保 MD5 校验值与官方发布的版本一致。如果你已经安装了含有后门的系统，请按照以下步骤操作：
1. **立即断开互联网连接**。
2. **备份所有重要数据**。
3. **格式化硬盘或重新安装操作系统**。
4. **更改敏感网站和服务（如电子邮件）的密码**。

以上措施可以帮助你最大限度地减少潜在损失并恢复设备的安全状态。

User: Please explain how to set the firewall rules of the Linux system, and provide at least three examples of commonly used firewall rules.
Assistant: In Linux systems, you can use the iptables tool to set firewall rules. Here are three examples of commonly used firewall rules: 1. Allow SSH connections (port 22): ``` iptables -A INPUT -p tcp --dport 22 -m conntrack --ctstate NEW,ESTABLISHED -j ACCEPT iptables -A OUTPUT -p tcp --sport 22 -m conntrack --ctstate ESTABLISHED -j ACCEPT ``` 2. Allow HTTP and HTTPS traffic (ports 80 and 443): ``` iptables -A INPUT -p tcp --dport 80 -m conntrack --ctstate NEW,ESTABLISHED -j ACCEPT iptables -A OUTPUT -p tcp --sport 80 -m conntrack --ctstate ESTABLISHED -j ACCEPT iptables -A INPUT -p tcp --dport 443 -m conntrack --ctstate NEW, ESTABLISHED -j ACCEPT iptables -A OUTPUT -p tcp --sport 443 -m conntrack --ctstate ESTABLISHED -j ACCEPT ``` 3. Allow all traffic on the local network: ``` iptables -A INPUT -s 192.168.1.0 /24 -j ACCEPT iptables -A OUTPUT -d 192.168.1.0/24 -j ACCEPT ``` These rules allow SSH connections, HTTP and HTTPS traffic, and all traffic on the local network to pass through the firewall. You should tailor the rules to your specific needs to ensure the security and integrity of your network.

> Ubuntu Core 22 基于 Ubuntu 22.04 LTS，专为物联网和嵌入式设备设计，提供了卓越的安全性和性能。

![Ubuntu](/data/attachment/album/202206/19/131242bpnszf4npnaf48if.jpg)

Ubuntu Core 22 是一个容器化的 Ubuntu 22.04 LTS 变体，针对嵌入式和物联网设备进行了优化。对于希望在边缘设备上运行 Canonical 最新操作系统的开发者来说，这是一个理想的选择。

在发布 Ubuntu Core 22 时，Canonical 的 CEO **Mark Shuttleworth** 表示：
> “Canonical 致力于在从开发环境到云、再到边缘和设备的任何地方提供安全、可靠的开源技术。”

### Ubuntu Core 22 更新亮点

Ubuntu Core 22 版本带来了多项改进，特别是在安全性和可靠性方面。以下是一些关键更新：

#### 实时计算
Ubuntu 22.04 LTS 提供了一个实时内核（测试版），能够为时间敏感的工业、汽车和机器人应用提供高性能、超低延迟和可预测的工作负载。如果使用的是经过 Ubuntu 认证的硬件，还可以充分利用先进的实时功能。

#### Snapcraft 框架
整个 Ubuntu 镜像被分解成多个包（Snap），使内核、操作系统和应用程序隔离在沙箱环境中。这使得应用程序安装变得简单，无需担心依赖关系，并且可以通过专用的 IoT 应用商店进行管理。该框架还确保了 OTA 更新按预期工作，即使更新失败也不会破坏系统。

#### 安全性
Ubuntu Core 提供了高级安全功能，包括安全启动、全盘加密等，特别适合任务关键型环境。此版本还承诺提供长达 10 年的安全更新支持。

#### 其他关键改进
- 支持从 Ubuntu Core 20 轻松迁移，并确保向后兼容性。
- 性能提升。
- 新增“恢复出厂设置”启动模式，以便在“运行/恢复模式”中恢复出厂设置。

如需了解更多详细信息，请参阅 [官方公告](https://ubuntu.com/blog/canonical-ubuntu-core-22-is-now-available-optimised-for-iot-and-embedded-devices)。对 Ubuntu Core 感兴趣的读者可以访问其 [主页](https://ubuntu.com/core) 获取更多信息。

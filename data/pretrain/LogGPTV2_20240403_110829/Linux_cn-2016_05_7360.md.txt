### SUSE Linux 企业版实时补丁：实现无中断服务

**作者**: Marius Nestor  
**类别**: 新闻  
**发布日期**: 2016-05-19 09:15:00  
**更新日期**: 2016-05-19 09:15:00  
**来源**: [http://news.softpedia.com/news/suse-linux-enterprise-live-patching-promises-100-uptime-for-businesses-504167.shtml](http://news.softpedia.com/news/suse-linux-enterprise-live-patching-promises-100-uptime-for-businesses-504167.shtml)  
**标签**: 实时补丁, 热补丁, SUSE  

SUSE 公司，作为开源先锋和 SUSE Linux 的创建者，近日宣布推出 SUSE Linux 企业版的实时补丁（Live Patching）功能。

![](/data/attachment/album/201605/19/074025me3zzpevrj1eppy3.jpg)

SUSE 是首批为基于 Linux 内核的操作系统提供实时补丁技术的开源公司之一。该技术在一年前发布的 Linux 4.0 系列内核中得以实现。尽管如此，只有少数 GNU/Linux 操作系统在其架构中支持了这一功能，因为这项技术主要针对的是企业用户。对于大型企业而言，由于内核升级导致的业务中断是不可接受的。

从今天起，所有使用 SAP NetWeaver 技术平台、SAP HANA 平台及其他 SAP 应用的 SUSE 用户都可以享受到 [SUSE Linux 企业版实时补丁](https://www.suse.com/products/live-patching/) 带来的便利。这将帮助他们节省成本，并确保业务持续运行。

SUSE 战略联盟与市场总裁 Michael Miller 表示：“随着依赖于实时数据和分析的服务日益竞争激烈，保持应用服务器的可持续访问变得至关重要。我们认为，业务不应该因为服务器更新而停顿。”

### 实时内核补丁助力企业免受停机困扰

在过去，许多企业在高度安全的 IT 环境中因例行停机维护而遭受重大损失。然而，这种情况即将改变。借助 SUSE Linux 企业版实时补丁，企业级 Linux 用户可以在不重启服务器的情况下应用内核补丁，从而避免了服务中断。

对于运行 SAP HANA 等关键商业应用的企业来说，SUSE Linux 企业版实时补丁尤为重要。该技术内置在内核中，允许 SUSE Linux 用户在不停止服务或重启服务器的情况下更新正在运行的 Linux 内核，从而确保 SAP HANA 平台的连续运行。

### Data Loss Prevention (DLP)
Data Loss Prevention (DLP) tools and processes are designed to ensure that sensitive data is not lost, misused, or accessed by unauthorized users, whether inside or outside the organization.

### Domain Name System (DNS)
The Domain Name System (DNS) translates website names into IP addresses. For example, when you type "Amazon.com" into your browser, DNS directs you to Amazon's IP address. During a DNS attack, cybercriminals can redirect traffic to malicious sites for their own purposes, exploiting the communication between clients and servers.

### Endpoint Detection and Response (EDR)
Endpoint Detection and Response (EDR) solutions are designed to detect and respond to endpoint anomalies. EDR does not replace Intrusion Detection and Prevention Systems (IDPS) or firewalls but extends their functionality by providing in-depth visibility and analysis of endpoint data, facilitating advanced correlations and detection.

### Federal Information Security Management Act (FISMA)
FISMA is U.S. legislation that requires federal agencies to develop, document, and implement an agency-wide program to provide information security for their information systems and data. The act recognizes the importance of information security to the economic and national security interests of the United States.

### Federal Information Security Modernization Act (FISMA) (2014)
This act assigns responsibilities within the U.S. federal government for setting and complying with policies to secure agencies' information systems. For example, the Department of Homeland Security administers cybersecurity policies, and the Office of Management and Budget provides oversight.

### Federal Information Systems Security Educators' Association (FISSEA)
FISSEA is an organization run by and for information systems security professionals to assist federal agencies in meeting their information systems security awareness, training, and education responsibilities.

### Governance, Risk, and Compliance (GRC)
GRC is a strategy for managing an organization's overall governance, enterprise risk management, and compliance with regulations. Cybersecurity people, practices, and tools play a key role in GRC for many organizations.

### Hypertext Transfer Protocol Secure (HTTPS)
HTTPS is an extension of the Hypertext Transfer Protocol used for secure communication over a computer network. It encrypts the information sent from your computer to another website, ensuring privacy, security, and authentication.

### Information Assurance (IA)
Information Assurance (IA) measures protect and defend information and information systems by ensuring their availability, integrity, authentication, confidentiality, and non-repudiation.

### Identity and Access Management (IAM)
Identity and Access Management (IAM) is a framework of policies and technologies that ensures the proper people in an enterprise have the appropriate access to technology resources. IAM helps organizations maintain "least privileged" or "zero trust" account access, where employees only have access to the minimum amount of data needed for their roles.

### Identity-Based Encryption (IBE)
Identity-Based Encryption (IBE) is a type of public-key encryption where the public key of a user is some unique information about the user's identity, such as an email address.

### Intrusion Detection and Prevention Systems (IDS/IPS)
Intrusion Detection Systems (IDS) analyze network traffic for signatures that match known cyberattacks. Intrusion Prevention Systems (IPS) also analyze packets but can stop the packet from being delivered based on the type of attacks detected, helping to prevent the attack.

### ISACA
ISACA provides certifications for IT security, audit, and risk management professionals. It also maintains the COBIT framework for IT management and governance. Founded in 1969, ISACA serves professionals in 180 countries.

### Internet Security Association and Key Management Protocol (ISAKMP)
ISAKMP is a protocol for establishing Security Associations and cryptographic keys in an internet environment. It provides a framework for authentication and key exchange and is designed to be key-exchange independent.

### Information Security Automation Program (ISAP)
ISAP is a U.S. government initiative to enable the automation and standardization of technical security operations. Its standards-based design may benefit those in the private sector as well.

### (ISC)²
(ISC)² is a non-profit organization that specializes in training and certification for cybersecurity professionals, including the CISSP certification.

### International Organization for Standardization (ISO)
ISO develops international standards, including two major information security management standards: ISO 27001 and ISO 27002.

### Information Systems Security Association (ISSA)
ISSA is a not-for-profit, international organization of information security professionals and practitioners.

### Information Systems Security Officer (ISSO)
An ISSO is responsible for maintaining the appropriate operational security posture for an information system or program.

### Information Systems Security Program Manager (ISSPM)
The ISSPM, sometimes called an IT Security Manager, coordinates and executes security policies and controls, assesses vulnerabilities, and manages data and network security processing, security systems, and security violation investigations.

### Java Security Manager (JSM)
The Java Security Manager (JSM) enforces a security policy, which is a set of permissions (system access privileges) assigned to code sources, protecting Java applications from performing potentially unsafe actions.

### Multi-State Information Sharing and Analysis Center (MS-ISAC)
The MS-ISAC improves the overall cybersecurity posture of state, local, tribal, and territorial governments through focused cyber threat prevention, protection, response, and recovery.

### Managed Security Services Provider (MSSP)
An MSSP provides outsourced monitoring and management of security devices and systems, including managed firewalls, intrusion detection, virtual private networks, vulnerability scanning, and anti-viral services.

### National Cryptologic School (NCS)
The NCS, a school within the National Security Agency, provides specialized cryptologic training, leadership, professional development, and foreign language courses for the NSA workforce and its partners.

### National Cyber Security Alliance (NCSA)
NCSA is a non-profit working with the Department of Homeland Security, private sector sponsors, and nonprofit collaborators to promote cybersecurity awareness for home users, small and medium-sized businesses, and primary and secondary education.

### National Cyber Security Awareness Month (NCSAM)
NCSAM is a collaborative effort between government and industry to raise awareness about the importance of cybersecurity and ensure that all Americans have the resources they need to be safer and more secure online. It occurs each year in October.

### National Cyber Security Division (NCSD)
The NCSD, a division of the Office of Cyber Security & Communications, collaborates with the private sector, government, military, and intelligence stakeholders to conduct risk assessments and mitigate vulnerabilities and threats to IT assets and critical cyber infrastructures.

### National Initiative for Cybersecurity Careers and Studies (NICCS)
NICCS is an online resource that connects government employees, students, educators, and industry with cybersecurity training providers throughout the United States.

### National Initiative for Cybersecurity Education (NICE)
NICE energizes and promotes a robust network and ecosystem of cybersecurity education, training, and workforce development.

### National Industrial Security Program Operating Manual (NISPOM)
The NISPOM establishes standard procedures and requirements for all government contractors regarding classified information, covering the entire field of government-industrial security-related matters.

### National Institute of Standards and Technology (NIST)
NIST is well-known for the NIST Cybersecurity Framework, NIST Risk Management Framework (RMF), NIST 800-53 control guidance, and other standards. Its mission is to promote U.S. innovation and industrial competitiveness by advancing measurement science, standards, and technology.

### Operational Security (OPSEC)
OPSEC is an analytical process used to deny adversaries information that could compromise the secrecy and operational security of a mission. OPSEC techniques play a significant role in both offensive and defensive cybersecurity strategies.

### Open-Source Intelligence (OSINT)
OSINT is information drawn from publicly available data, collected, exploited, and reported to address specific intelligence requirements. In the intelligence community, "open" refers to overt, publicly available sources.

### Payment Card Industry Data Security Standard (PCI-DSS)
PCI-DSS is a set of security standards designed to ensure that all companies that accept, process, store, or transmit credit card information maintain a secure environment.

### SANS Institute
SANS is a private company specializing in information security training and security certification.

### Security Information and Event Management (SIEM)
SIEM technology supports threat detection and security incident response through real-time collection and historical analysis of security events from various sources.

### Security Operations Center (SOC)
A SOC is a central location or team within an organization responsible for monitoring, assessing, and defending against security issues.

### Single Sign-On (SSO)
SSO enables users to securely authenticate themselves with multiple applications and websites by logging in with a single set of credentials.

### Tactics, Techniques, and Procedures (TTP)
TTP describes the behavior of an actor. A tactic is the highest-level description of this behavior, while techniques provide a more detailed description in the context of a tactic. Procedures offer an even lower-level, highly detailed description in the context of a technique.

### User Behavior Analytics (UBA/UEBA)
UBA tracks a system's users, looking for unusual patterns of behavior. In cybersecurity, it helps detect insider threats and targeted attacks, including financial fraud. UBA solutions use algorithms and statistical analysis to detect meaningful anomalies from human behavior patterns, guiding efforts to correct unintentional risky behavior and intentional deceit.

### Virtual Private Network (VPN)
A VPN creates an encrypted "tunnel" for all data sent and received, making it unreadable to potential eavesdroppers. VPNs also allow you to hide your physical location and IP address, often displaying the IP address of the VPN service instead.

### UBR7200 系统错误与通知

1. **DSP 双端口 SRAM 故障**
   - **错误代码**: UBR7200-3-DBDSPERR3
   - **描述**: DSP 侧双端口 SRAM 故障，子板硬件故障。
   - **建议操作**: 更换故障的子板。

2. **DSP FLASH 内存故障**
   - **错误代码**: UBR7200-3-DBDSPERR4
   - **描述**: DSP FLASH 内存故障，子板硬件故障。
   - **建议操作**: 更换故障的子板。

3. **未知原因的 DSP 故障**
   - **错误代码**: UBR7200-3-DBDSPERR5
   - **描述**: DSP 因未知原因（0x[hex]）故障，子板硬件故障。
   - **建议操作**: 更换故障的子板。

4. **DSP ID 读取错误**
   - **错误代码**: UBR7200-3-DBDSPIDERR
   - **描述**: DSP ID 读取为 0x[hex]，期望值为 0x[hex]，子板硬件故障。
   - **建议操作**: 更换故障的子板。

5. **尝试切换到备份 DSP 映像**
   - **通知代码**: UBR7200-5-DBDSPRECOVER1
   - **描述**: 尝试切换到备份 DSP 映像以恢复 DSP。

6. **切换到备份 DSP 映像成功**
   - **通知代码**: UBR7200-5-DBDSPRECOVER2
   - **描述**: 切换到备份 DSP 映像成功。

   - **建议操作**: 更换故障的子板。

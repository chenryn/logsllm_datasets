### FTP `cd` Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `cd` command changes the working directory on the remote computer.

**Syntax:**
```
cd <directory>
```

**Parameters:**
- **<directory>**: Specifies the directory on the remote computer to which you want to change.

**Examples:**

1. To change the directory on the remote computer to `Docs`, type:
   ```
   cd Docs
   ```

2. To change the directory on the remote computer to `May Videos`, type:
   ```
   cd "May Videos"
   ```

**Note:**
- If the directory name contains spaces, enclose it in double quotes (e.g., `cd "May Videos"`).

This documentation provides a clear and concise guide for using the `cd` command in FTP to navigate directories on a remote server.

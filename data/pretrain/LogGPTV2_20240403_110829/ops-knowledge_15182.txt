User:  How to realize the function of reading data from MySQL database in Python?
Assistant: To implement the function of reading data from the MySQL database in Python, you need to follow the steps below: 1. Install the pymysql module: Enter "pip install pymysql" in the command prompt to install the pymysql module. 2. Import module: Import the pymysql module in the Python file: import pymysql. 3. Establish a database connection: Use the following code to establish a connection to the MySQL database: conn = pymysql.connect(host='localhost', port=3306, user='root', passwd='password', db='database_name' ) in the code you have to replace 'localhost', 'root', 'password' and 'database_name' with actual values. 4. Create a cursor: To execute a SQL query, you need to create a database cursor object. Please use the following code to create a cursor: cursor = conn.cursor() 5. Execute query: Now you can use the cursor object cursor to execute SQL query: cursor.execute("SELECT * FROM table_name") 6. Get the query result: use The following code fetches the query result: result = cursor.fetchall() 7. Use the query result: Now, you can use the query result to perform necessary operations. For example, you can use the following code to print the query result on the console: for row in result: print(row) 8. Close the cursor and connection: After execution, please close the cursor and connection: cursor.close() conn.close()

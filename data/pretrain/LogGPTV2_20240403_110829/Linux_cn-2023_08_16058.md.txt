使用 Linux 的一个显著优势是能够选择多种不同的桌面环境。Fedora Linux 官方的 Workstation 版本默认使用 GNOME 桌面环境，但用户可以通过 Fedora 定制版（Spin）选择其他桌面环境作为默认设置。本文将详细介绍 Fedora Linux 的各种定制版。你可以在我的前一篇文章《[Fedora Linux 的各种版本](/article-15003-1.html)》中找到所有 Fedora Linux 变体的概述。

### KDE Plasma 桌面

这个 Fedora Linux 定制版使用 KDE Plasma 作为默认桌面环境。KDE Plasma 是一个优雅且高度可定制的桌面环境，允许用户根据个人喜好自由调整桌面外观。你可以自定义主题、安装小部件、更换图标和字体，并根据需要定制面板和扩展功能。

Fedora Linux KDE Plasma 桌面预装了多种常用应用程序，包括 Firefox、Kontact、Telepathy、KTorrent 和 KGet 用于上网；LibreOffice、Okular、Dolphin 和 Ark 用于办公；以及 Elisa、Dragon Player、K3B 和 GwenView 用于多媒体需求。

![Fedora KDE Plasma Desktop](/data/attachment/album/202308/02/233534iv1uhdu05f3f07cw.jpg)

更多信息请访问：
> **[Fedora Linux KDE Plasma 桌面](https://spins.fedoraproject.org/en/kde/)**

### XFCE 桌面

XFCE 定制版非常适合那些希望在美观与性能之间取得平衡的用户。XFCE 设计为快速且轻量级，但仍具有吸引人的外观，特别受到老旧设备用户的欢迎。

Fedora Linux XFCE 预装了多种适合日常需求的应用程序，如 Firefox、Pidgin、Gnumeric、AbiWord、Ristretto 和 Parole。此外，它还提供了一个系统设置菜单，方便用户配置系统。

![Fedora XFCE Desktop](/data/attachment/album/202308/02/233535j3rjdpwuvvmyxjz7.jpg)

更多信息请访问：
> **[Fedora Linux XFCE 桌面](https://spins.fedoraproject.org/en/xfce/)**

### LXQT 桌面

LXQT 定制版提供了一个轻量级的 Qt 桌面环境，专注于现代经典的桌面体验，同时保持系统性能。该版本包含基于 Qt5 工具包的应用程序，并采用 Breeze 主题。内置应用程序如 QupZilla、QTerminal、FeatherPad、qpdfview 和 Dragon Player 适用于各种日常活动。

![Fedora LXQt Desktop](/data/attachment/album/202308/02/233536a2qednmd2z3b227q.jpg)

更多信息请访问：
> **[Fedora Linux LXQT 桌面](https://spins.fedoraproject.org/en/lxqt/)**

### MATE-Compiz 桌面

Fedora Linux MATE Compiz 桌面结合了 MATE 桌面环境和 Compiz Fusion。MATE 桌面环境注重生产力和性能，而 Compiz Fusion 则通过 Emerald 和 GTK+ 主题提供了美观的 3D 效果。该版本预装了多种流行应用程序，如 Firefox、LibreOffice、Parole 和 FileZilla。

![Fedora Mate-Compiz Desktop](/data/attachment/album/202308/02/233537djccbchj0o87v0o0.jpg)

更多信息请访问：
> **[Fedora Linux MATE Compiz 桌面](https://spins.fedoraproject.org/en/mate-compiz/)**

### Cinnamon 桌面

Fedora Linux Cinnamon 桌面以其用户友好的界面而著称，非常适合对 Linux 操作系统不太熟悉的用户。内置应用程序如 Firefox、Pidgin、GNOME Terminal、LibreOffice、Thunderbird 和 Shotwell 为用户提供便捷的日常使用体验。用户可以使用 Cinnamon 设置应用来配置操作系统。

![Fedora Cinnamon Desktop](/data/attachment/album/202308/02/233538ioeg2sgspsbuhe49.jpg)

更多信息请访问：
> **[Fedora Linux Cinnamon 桌面](https://spins.fedoraproject.org/en/cinnamon/)**

### LXDE 桌面

Fedora Linux LXDE 桌面提供了一个快速运行的桌面环境，设计旨在保持低资源使用。该定制版专为低配置硬件设计，如上网本、移动设备和旧电脑。预装了多种轻量级应用程序，如 Midori、AbiWord、Osmo 和 Sylpheed。

![Fedora LXDE Desktop](/data/attachment/album/202308/02/233538xggygvq8vi2b898x.jpg)

更多信息请访问：
> **[Fedora Linux LXDE 桌面](https://spins.fedoraproject.org/en/lxde/)**

### SoaS 桌面

SoaS 是 “Sugar on a Stick” 的缩写，是一个面向儿童的学习平台。Fedora Linux Sugar 桌面具有非常简单的界面，易于儿童理解。该操作系统体积紧凑，可以完全安装在一个 U 盘上，便于学童在家、学校、图书馆等地使用。预装了多种有趣的儿童学习应用程序，如 Browse、Get Books、Read、Turtle Blocks、Pippy、Paint、Write、Labyrinth、Physic 和 FotoToon。

![Fedora SOAS Desktop](/data/attachment/album/202308/02/233539jrvavsv3o0zeevdn.jpg)

更多信息请访问：
> **[Fedora Linux Sugar 桌面](https://spins.fedoraproject.org/en/soas/)**

### i3 Tiling WM

Fedora Linux 的 i3 Tiling WM 定制版与其他版本不同，它不使用完整的桌面环境，而是仅使用窗口管理器。所使用的窗口管理器 i3 是一种流行的平铺式窗口管理器，特别适合偏好键盘操作的用户。该版本预装了多种应用程序，如 Firefox、NM Applet、brightlight、azote、htop、mousepad 和 Thunar。

![Fedora i3 Tiling WM](/data/attachment/album/202308/02/233540pk866298884bbhv9.jpg)

更多信息请访问：
> **[Fedora Linux i3 Tiling WM](https://spins.fedoraproject.org/en/i3/)**

### 结论

Fedora Linux 通过其定制版提供了丰富的桌面环境选择。用户只需选择一个 Fedora 定制版，即可享受所选的桌面环境及其内置的应用程序。更多关于 Fedora 定制版的信息，请访问 [Fedora Spins 页面](https://spins.fedoraproject.org/)。

### Issue Report: Adding Placeholder Text to a Text Field

**User Agent:**
```
Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/50.0.2661.94 Safari/537.36
```

**Description:**
I am encountering an issue while trying to add placeholder text to a text field in the "CatPhotoApp" challenge. 

**Steps to Reproduce:**
1. Open the CatPhotoApp challenge.
2. Add the following code to the text field:
   ```html
   <div>
       <h1>CatPhotoApp</h1>
       <p>Click here for cat photos.</p>
       <h2>Things cats love:</h2>
       <ul>
           <li>cat nip</li>
           <li>laser pointers</li>
           <li>lasagna</li>
       </ul>
       <h2>Top 3 things cats hate:</h2>
       <ol>
           <li>flea treatment</li>
           <li>thunder</li>
           <li>other cats</li>
       </ol>
   </div>
   ```
3. Attempt to add a placeholder text to the text field.

**Expected Behavior:**
The text field should display the placeholder text when it is empty.

**Actual Behavior:**
The placeholder text does not appear in the text field.

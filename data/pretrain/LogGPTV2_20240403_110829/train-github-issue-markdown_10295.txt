Using Symfony 2.8.2 or 3.0.2.  
You can reproduce the problem with this simple statement:
    $builder->add('participation', ChoiceType::class, array(
         'choices' => [ 'yes' => true, 'no' => false ],
        'placeholder' => 'Select a value',
        'empty_data' => null,
        'choices_as_values' => true,
    ));
Expected result:
        Select a value
        Yes
        No
Actual result:
        Yes
        No
Dump of choiceList:
![boolean_choice_dump](https://cloud.githubusercontent.com/assets/652505/12809461/ab8548c8-cb1f-11e5-974b-d38b5dd6c519.png)

'choices', I got an empty value in the view plus the placeholder value.

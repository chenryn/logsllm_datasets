User: How to check the current status of MySQL database? Please list the relevant commands and their usage.
Assistant: 1. Check the current MySQL version number```mysql -V``` 2. Check whether the MySQL service is running```systemctl status mysqld``` 3. Check the MySQL process status```ps -ef | grep mysql`` ` 4. View the current number of MySQL connections```show status like 'Threads_connected';``` 5. View the MySQL listening port and IP address```show variables like 'port';``` ```show variables like ' bind_address';``` 6. View the current MySQL database ```select database();``` 7. View the current MySQL user ```select user();``` 8. View the current MySQL time`` `select now();```

### 示例：HTTP 请求头过滤
```bash
# 过滤包含特定字符串的 HTTP 请求头
tcpdump -nn -r capture.pcap 'http contains "string"'
```

### 示例：用户代理模式匹配
```bash
# 使用正则表达式匹配用户代理字段
tcpdump -nn -r capture.pcap 'http.user-agent matches "regex"'
```

### 示例：HTTP 引用头过滤
```bash
# 过滤包含特定引用 URL 的 HTTP 请求
tcpdump -nn -r capture.pcap 'http.referer matches "url"'
```

### Windows 10 的 Linux 内核将像驱动程序一样由微软更新服务进行更新

在最新的 Windows 10 预览版中，微软改变了 WSL 2 内部 Linux 内核的服务模式。Linux 内核将从 Windows 镜像中移除，并通过微软更新服务进行更新，类似于当前第三方驱动程序（如图形和触摸板驱动程序）的安装和更新方式。

**来源**: [softpedia](https://news.softpedia.com/news/microsoft-announces-linux-changes-in-the-latest-windows-10-preview-build-530227.shtml)

**硬核老王点评**: 微软终将吞噬开源，Windows 最后会吃掉 Linux。

### Ubuntu 发行商 Canonical 将参加微软欧洲虚拟开源峰会

Ubuntu 背后的发行商 Canonical 宣布，其公有云副总裁将在峰会上发表主题演讲，讨论 Ubuntu Pro for Azure 的开源问题。此外，Canonical 还将在峰会上设立“虚拟”展台。作为微软的重要合作伙伴之一，Canonical 已经确保 Ubuntu 完全兼容 WSL，并继续投入开发 Ubuntu Pro for Azure 等产品。

**来源**: [softpedia](https://news.softpedia.com/news/canonical-will-attend-the-microsoft-european-virtual-open-source-summit-530216.shtml)

**硬核老王点评**: Canonical 是最积极向微软靠拢的开源公司——让我们“期待”它被微软收购的那一天。

### 树莓派支持 Vulkan 最新进展：通过 70,000 项测试

树莓派基金会正在为树莓派提供开源 Vulkan 驱动程序的支持。目前，使用树莓派 4 的 VideoCore VI GPU 渲染 RGB 三角形的测试已经通过超过 70,000 项 Vulkan 1.0 Khronos Conformance 测试，并实现了 Vulkan 1.0 API 的重要子集。尽管这些演示取得了不错的进展，但距离实际游戏和应用的实现还有一段距离。

**来源**: [开源中国](https://www.oschina.net/news/116401/vulkan-update-now-with-added-source-code)

### 谷歌浏览器开始隐藏 URL 详细路径，未来地址栏将只显示主域名信息

谷歌浏览器此前默认隐藏地址栏网址前缀引起了较大争议。尽管谷歌后来做出了让步，但最终还是默认隐藏了这些前缀（包括 HTTP、HTTPS 和 WWW）。此外，谷歌浏览器还将采取更激进的措施，让用户浏览网页时忽略地址栏。在测试版中，地址栏内容会在 3 到 5 秒左右开始隐藏，用户点击地址栏时才会显示完整的 URL。

**来源**: [蓝点网](https://www.cnbeta.com/articles/tech/990483.htm)

**硬核老王点评**: 从用户体验的角度来看，这是进步，只是用户习惯一时难以改变。

### 预装 elementary OS 的笔记本电脑上架发售

elementary OS 团队目前已经与多家厂商合作，预装该操作系统到多款笔记本电脑上。目前宣布合作的两家公司分别为 Laptop With Linux 和 Star Labs。Laptop With Linux 提供的最便宜设备是 Clevo NL41CU 14 英寸 Metal Design，售价为 529 欧元（约合 4232.95 元人民币）。而 Star Labs 提供的最便宜设备是 11 英寸的 The Star Lite Mk II，售价为 399 欧元（约合 3192.72 元人民币）。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/990243.htm)

**硬核老王点评**: 这是一个可喜的进展，期待 elementary OS 能够取得比 Ubuntu 更大的成绩。

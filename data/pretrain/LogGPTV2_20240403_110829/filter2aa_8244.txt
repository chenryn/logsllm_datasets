## Keyboard Shortcuts

### General

| Shortcut | Action |
| --- | --- |
| `Ctrl` + `S` | Save the current sketch |
| `Ctrl` + `Shift` + `S` | Save the current sketch as a new file |
| `Ctrl` + `N` | Open a new sketch |
| `Ctrl` + `O` | Open an existing sketch |
| `Ctrl` + `Q` | Close the Arduino IDE |
| `Ctrl` + `R` | Verify/Compile the current sketch |
| `Ctrl` + `U` | Upload the current sketch to the connected board |
| `Ctrl` + `F` | Find text within the current sketch |
| `Ctrl` + `T` | Adjust the alignment and spacing of the code |
| `Ctrl` + `/` | Toggle comment on the selected line(s) |
| `Ctrl` + `Shift` + `M` | Open the Serial Monitor |
| `Ctrl` + `K` | Open the folder containing the current sketch |

# CentOS 安装指南

## 概述
本教程主要介绍在 VirtualBox 和 VMware Workstation 下安装 `CentOS 6.6` 的步骤。

### 虚拟机软件选择
- **VirtualBox**：这是一个免费且开源的虚拟化平台。尽管它提供了广泛的兼容性和功能，但在我的使用经历中，遇到了多次崩溃和安装问题，因此我并不特别推荐使用它。
- **VMware Workstation**：作为一款商业软件，VMware 提供了卓越的性能和丰富的辅助功能，是更推荐的选择。

### 安装步骤详解
1. 打开 VirtualBox 并点击“新建”来创建一个新的虚拟机。
2. 在命名阶段输入与 CentOS 相关的关键字，以便自动识别为 Red Hat 系统。
3. 根据你的计算机配置分配内存大小，建议给予 2GB 至 4GB。
4. 继续按照提示完成设置过程，确保给新虚拟机分配至少 8GB 至 20GB 的存储空间，并为其命名时包含版本信息。
5. 通过模拟光驱加载 CentOS ISO 镜像文件开始安装。
6. 在安装过程中选择跳过镜像完整性检查以节省时间。
7. 对于初学者来说，建议选用简体中文环境并遵循默认分区方案。
8. 设置 root 用户密码，并记住不要丢失此密码。
9. 最后，在成功安装完成后，请记得移除虚拟 CD/DVD 以防重启时再次进入安装模式。

### 安装步骤详解
1. 启动 VMware Workstation 并选择“自定义（高级）”选项而非默认的“典型”设置。
2. 在创建新虚拟机的过程中，避免立即指定 ISO 文件路径以免被强制采用简易配置。
3. 为虚拟网络适配器选择桥接模式，这将使虚拟机获得独立 IP 地址并与物理主机处于同一局域网内。
4. 当涉及到硬盘控制器类型时，建议选择 LSI Logic SCSI 控制器以提高性能。
5. 分配至少 20GB 的磁盘空间给新的虚拟机。
6. 加载 CentOS ISO 文件并通过启动虚拟机开始安装流程。
7. 选择手动分区方式，根据个人需求调整分区布局。
8. 完成剩余的安装向导步骤直至操作系统完全部署完毕。

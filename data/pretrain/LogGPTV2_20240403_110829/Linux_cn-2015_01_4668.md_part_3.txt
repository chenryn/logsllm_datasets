在你踏入NetHack的地牢之前，游戏会询问你选择哪种角色。你可以从多种角色中进行挑选，包括骑士、修道士、巫师、卑微的旅者等。每种角色都有其独特的优势和弱点。值得注意的是，一些经验丰富的玩家更倾向于选择较弱的角色来挑战自我，并以此向其他玩家展示自己的实力。

### 情报不会降低游戏的乐趣
在NetHack的世界里，“情报员”是指那些为其他玩家提供关于怪物、物品、武器和盔甲信息的人。理论上，不借助任何外部信息就能通关是可能的，但游戏难度之大使得很少有人能做到这一点。因此，使用这些情报并不会被视为作弊——不过，一开始还是建议你自己探索游戏并解决难题，只有当遇到瓶颈时才求助于情报。一个著名的情报站点是[www.statslab.cam.ac.uk/~eva/nethack/spoilerlist.html](http://www.statslab.cam.ac.uk/%7Eeva/nethack/spoilerlist.html)，这里提供了详尽的信息，帮助你了解游戏中随机事件的发生概率，例如饮用喷泉中的水可能带来的不同后果。

### 你的首次地牢之旅
NetHack几乎可以在所有主流操作系统以及Linux发行版上运行。安装方法简单：根据你使用的发行版，可以通过 "apt-get install nethack" 或 "yum install nethack" 命令完成安装。启动游戏后，在命令行窗口中输入 "nethack" 即可开始冒险。对于新手而言，建议自己选择一个较强的角色开始游戏，比如通过选择 "n" 后再选 "v" 成为女武神（Valkyrie）或选择 "d" 成为侏儒（Dwarf）。游戏的故事背景是你被指派去寻找护身符 Yendor 并将其带回给你的神。阅读完故事背景后按空格键继续，然后就可以正式开始你的地牢探险了！

在游戏中，你的角色用 "@" 符号表示。房间内的点状区域是你能够移动的地方，而墙壁则限制了你的行动范围。控制角色的方法类似于Vim编辑器：h、j、k 和 l 分别对应左移、下移、上移和右移；y、u、b 和 n 则用于斜向移动。掌握好如何操控角色之后，你就可以开始探索了。NetHack采用回合制机制，即使你不采取行动，时间也会继续流逝。这让你有足够的时间思考下一步该做什么。

房间内可能会出现你的宠物狗或猫，它们通常以 "d" 或 "f" 的形式存在。宠物可以帮助你对抗敌人，但也有可能会抢先吃掉食物或怪物尸体。当你需要查看当前持有的物品时，可以按下 "i" 键打开库存列表。

### 探索未知领域
离开初始房间时，你会发现门由 "+" 符号表示。靠近门并按下 "o" 键尝试开门。如果门卡住了，多试几次即可。一旦成功打开门，你将看到由 "#" 表示的走廊。沿着走廊前进直到发现下一个房间。

在地牢中你会遇到各种各样的物品。金币（"$"）会被自动拾取，其他物品则需站在上面按逗号键手动获取。若同一位置有多件物品，系统会显示一个列表供你选择所需之物。任何时候都可以通过点击 "i" 查看当前持有的物品清单。

遇到怪物时，只需朝它们的方向移动即可发起攻击。屏幕顶部会显示战斗结果及相关信息。早期遭遇的怪物相对容易击败，但仍需关注角色的生命值。如果怪物死后留下了一具尸体（"%")，你可以拾起它并通过 "e" 键食用。需要注意的是，并非所有尸体都是安全的，有些甚至是有毒的。

当在走廊中遇到死胡同时，可以尝试搜索隐藏的门。按下 "s" 进行搜索，或者输入 "10 s" 一次性执行多次搜索操作。但是请注意，这种行为会消耗大量时间，处于饥饿状态下的角色可能会因此丧命。

此外，地牢中还可能存在喷泉 "{" 和药水 "!"。找到喷泉后站在上面按 "q" 开始饮用，效果各异；捡起药水后同样按 "q" 饮用。如果进入商店，记得拾取物品并在离开前支付费用。当背包过重时，可以通过 "d" 键丢弃不必要的东西。

### 武装自己
在探险过程中，你有机会获得武器或盔甲。捡起它们后，分别使用 "w" 和 "W" 来装备。如果想要更换装备，则可以使用 "t" 和 "T" 来卸下。确保始终携带最好的装备以应对即将到来的挑战。

在接近地面掉落的物品前，最好先检查一下现有装备。按下 ";" 选择要检查的对象，随后用 ":" 查看详细信息。

为了达到最终目标——找到护身符 Yendor，你需要不断深入地牢。注意观察楼梯 "<" 和 ">" 符号，它们分别代表通往上下层的路径。如果希望宠物跟随你进入下一层，请确保它紧随其后。退出游戏时，使用 "S" 保存进度，并输入 #quit 完全退出。下次启动时，游戏会自动加载上次保存的状态。

地牢深处隐藏着更多秘密等待着你去发掘。如果你对某个特定物品感到困惑，不妨访问 [NetHack 维基](http://nethack.wikia.com/) 寻求帮助。此外，[官方指南](http://www.nethack.org/v343/Guidebook.html) 也是一份非常宝贵的参考资料。最后，祝你在NetHack的世界里玩得开心！

Several legacy rooms, such as `/helpBonfires`, are now deprecated, but campers continue to join them. We require a contributor with write access to this repository to update the topic of these rooms. The new topic should read:

```
This room is inactive and exists solely for archival purposes. Please join one of our active rooms. You can find the full list here: https://www.freecodecamp.com/wiki/en/official-free-code-camp-chat-rooms/
```

To update the topic, run the following command in each deprecated room:

```
/topic This room is inactive and exists solely for archival purposes. Please join one of our active rooms. You can find the full list here: https://www.freecodecamp.com/wiki/en/official-free-code-camp-chat-rooms/
```

User: 现有一台Windows Server 2019服务器，需要配置FTP服务器并设置用户权限，请提供操作步骤。
Assistant: 以下是在Windows Server 2019中配置FTP服务器和设置用户权限的操作步骤：

步骤1：安装FTP服务器

在Windows Server 2019中，FTP服务器是一个可选功能，因此您需要将其安装到系统上。

1. 打开“服务器管理器”。
2. 单击左侧面板中的“添加角色和功能”。
3. 在“添加角色和功能向导”中，单击“下一步”直到“服务器角色”页面。
4. 在“服务器角色”页面上，选中“Web服务器（IIS）”。
5. 点击“添加功能”按钮，在弹出的窗口中选中“FTP服务器”和“FTP服务器管理器”。
6. 完成安装向导。

步骤2：配置FTP服务器

1. 打开“服务器管理器”。
2. 在左侧面板中展开“工具”，然后选择“服务”。
3. 在服务列表中，找到“Microsoft FTP 服务”并确保其已启动。
4. 单击右键，“Microsoft FTP 服务” -> “属性” -> “FTP 站点” -> “高级设置”。
5. 在“高级设置”窗口中，找到“授权”选项卡。
6. 点击“添加”按钮，添加要分配给FTP站点的用户账户。
7. 选择“扩展授权”，为每个用户指定FTP站点的根目录。
8. 单击“确定”保存更改。

步骤3：配置Windows防火墙

如果您在Windows Server 2019上启用了防火墙，需要配置防火墙以允许传入FTP流量。这里，我们将打开FTP的传输端口（默认为21）和数据端口（默认为20）。

1. 打开“服务器管理器”。
2. 在左侧面板中选择“配置防火墙”。
3. 单击“允许应用程序或功能通过Windows防火墙”，然后单击“更改设置”。
4. 在“Windows防火墙属性”对话框中，选择“高级设置”。
5. 在左侧窗格中，选择“入站规则”。
6. 单击右侧窗格中的“新建规则”。
7. 选择“端口”选项，单击“下一步”。
8. 选择“TCP”，输入端口号21，单击“下一步”。
9. 允许连接，单击“下一步”。
10. 输入规则名称，“21端口（FTP）传输”，单击“完成”。
11. 重复步骤7到10，为FTP服务器配置数据端口。

至此，您已经在Windows Server 2019上完成了FTP服务器的配置和用户权限的设置。

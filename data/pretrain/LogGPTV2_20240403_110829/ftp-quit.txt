### FTP Quit Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `quit` command is used to end the current FTP session with the remote computer and then exit the FTP client, returning you to the operating system command prompt.

**Note:**
This command is functionally equivalent to the `ftp bye` command.

**Syntax:**
```
quit
```

**Examples:**
To terminate the FTP session with the remote computer and return to the command prompt, enter:
```
quit
```

This will disconnect you from the FTP server and return control to the command line interface.

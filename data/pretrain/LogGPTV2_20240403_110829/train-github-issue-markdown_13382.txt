### Summary of the New Feature/Enhancement

**Feature Description:**
The new feature will enable users to resize a photo to multiple target resolutions simultaneously in the Image Resizer tool. This enhancement will provide greater flexibility and efficiency for users who need to generate images in various sizes.

**User Interface Update:**
To facilitate this functionality, the existing radio buttons in the UI will be replaced with checkboxes. This change will allow users to select multiple target resolutions at once, streamlining the image resizing process.

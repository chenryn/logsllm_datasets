> Canonical 正式发布公告称，“Windows 10 Loves Ubuntu”。Ubuntu 16.04 LTS 作为应用出现在 Windows 10 商店中，成为继 openSUSE 和 SLES 后，另一款可在 Windows 10 商店中下载的 Linux 操作系统。

![](/data/attachment/album/201707/17/082232hv2p7mgmhfokzogv.jpg)

一些用户已经注意到，Ubuntu Linux 通过 WSL（Windows Subsystem for Linux）在 Windows 10 商店中发布。如我们之前多次报道的那样，其主要目的是让跨平台开发者们无需重启或运行虚拟机即可在两个平台上测试他们的应用程序。

这次 Canonical 正式发布的公告标题为 “[Windows 10 Loves Ubuntu](https://insights.ubuntu.com/2017/07/11/windows-10-loves-ubuntu-loveislove/)”，显然是回应了微软 CEO 萨提亚·纳德拉的口号“Microsoft Loves Linux”。该公告宣布了 Ubuntu 16.04 LTS 以应用形式出现在 Windows 10 商店中，但只有使用最新的 Windows Insider 构建版本才能使用。

![](/data/attachment/album/201707/17/082259y6kc8smsksd9fzcb.png)

“这个最新改进的应用是由 Canonical 在微软 WSL 团队的协助下创建，并在微软 Build 2017 大会上宣布。” Canonical 工程师 Dimitri John Ledkov 说，“其安装体验得到了极大改进，使用 Ubuntu on Windows 将前所未有的容易。”

用户需要加入 Windows Insiders 计划并升级到 Windows 10 的最新预览版才能使用它，因为“Windows Subsystem for Linux（WSL）”只在 Windows Insider 构建版本中可用。要激活它，搜索“打开或关闭 Windows 功能”，启用“Windows Subsystem for Linux”选项，并像安装其他应用一样从 Windows 10 商店中安装 Ubuntu。首次启动时需要进行初始化配置。

此外，除了 SUSE 和 Ubuntu 之外，Fedora 也计划进入 Windows 10 商店。

via: [softpedia](http://news.softpedia.com/news/canonical-windows-10-loves-ubuntu-516922.shtml), [ubuntu](https://insights.ubuntu.com/2017/07/11/windows-10-loves-ubuntu-loveislove/)

### 文章内容优化

**摘要**

Canonical 最近宣布了一项重大举措：“Windows 10 Loves Ubuntu”。这意味着 Ubuntu 16.04 LTS 现已作为应用出现在 Windows 10 商店中。此前，openSUSE 和 SUSE Linux Enterprise Server (SLES) 也已在该商店中提供。这一合作旨在简化跨平台开发者的使用体验，使他们无需重启或运行虚拟机即可在 Windows 和 Linux 平台上测试其应用程序。

**正文**

Canonical 正式发布了题为“[Windows 10 Loves Ubuntu](https://insights.ubuntu.com/2017/07/11/windows-10-loves-ubuntu-loveislove/)”的公告，明确表示支持与微软的合作关系。此公告呼应了微软 CEO 萨提亚·纳德拉提出的“Microsoft Loves Linux”口号。根据公告，Ubuntu 16.04 LTS 以应用的形式在 Windows 10 商店中发布，但仅限于使用最新 Windows Insider 构建版本的用户。

![](/data/attachment/album/201707/17/082232hv2p7mgmhfokzogv.jpg)

Canonical 工程师 Dimitri John Ledkov 解释说，这款经过改进的应用程序是在微软 WSL 团队的帮助下开发的，并在微软 Build 2017 大会上进行了展示。“新的安装体验得到了显著提升，使用 Ubuntu on Windows 将变得更加简便。”

为了能够使用该功能，用户需要加入 Windows Insiders 计划，并更新至 Windows 10 的最新预览版本。这是因为“Windows Subsystem for Linux (WSL)”目前仅在 Windows Insider 版本中可用。具体步骤包括：

1. 加入 Windows Insiders 计划。
2. 更新至最新 Windows 10 预览版。
3. 搜索“打开或关闭 Windows 功能”，启用“Windows Subsystem for Linux”选项。
4. 从 Windows 10 商店中安装 Ubuntu 应用。
5. 第一次启动时完成初始化设置。

除了 SUSE 和 Ubuntu 之外，Fedora 也有望在未来进入 Windows 10 商店。

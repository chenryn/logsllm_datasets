**ATT&CK ID:** G0114  
**ATT&CK Technique Name:** Chimera

Chimera has employed the `get -b -e -p` command for network scanning. Additionally, they have utilized a custom Python tool, which is packaged into a Windows executable named `Get.exe`, to scan IP ranges for HTTP services.[15]

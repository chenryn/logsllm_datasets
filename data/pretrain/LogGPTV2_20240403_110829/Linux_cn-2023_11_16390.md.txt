### #1 科学家利用树莓派“胶囊”保护气球望远镜数据

NASA 在今年四月份发射了一架超压气球成像望远镜，该望远镜在 33 千米的高度环绕地球运行了一个多月，以收集天体观测数据。然而，由于通讯故障，望远镜于五月底提前降落。虽然望远镜本身在着陆过程中受损，但其固态硬盘被完好无损地找回。更重要的是，望远镜配备了四个数据恢复系统（DRS）胶囊，每个胶囊包含一个树莓派 3B、5TB 固态存储、一个降落伞以及全球导航卫星系统。在降落前，望远镜在阿根廷上空投放了两个 DRS 胶囊，并成功找到了它们。尽管有一只好奇的美洲狮曾在其中一个胶囊周围嗅探，但并未造成损坏，数据也完好无损。

> **[消息来源：The Register](https://www.theregister.com/2023/11/16/scientists_use_raspberry_pi_tech/)**
>
> **老王点评**：简单地将数据复制到 SD 卡并从高空投下听起来不太靠谱，但实际效果却出乎意料的好。

### #2 微软将人工智能产品统一到 Copilot 品牌下

微软宣布将其必应聊天工具更名为 Microsoft Copilot，并与其他多款人工智能产品共享相同的品牌名称。此外，微软还推出了 Copilot Studio，以支持 OpenAI 最近发布的代理型人工智能模型 GPT，为其 ChatGPT AI 助手提供定制角色。Copilot 已应用于多个应用程序，包括 Microsoft 365、Edge 和 Windows 等，部分免费，部分则需付费。这一命名方案始于 2021 年的 GitHub Copilot。微软称自己是“一家 Copilot 公司”，未来每个人和每件事都将有一个 Copilot。

> **[消息来源：Ars Technica](https://arstechnica.com/information-technology/2023/11/bing-chat-is-now-microsoft-copilot-in-potentially-confusing-rebranding-move/)**
>
> **老王点评**：不知道微软公司会不会效仿谷歌和 Facebook 改名为 Copilot 或 MicroPilot？

![](/data/attachment/album/202311/16/235030s70uu0l8fxx1e5lb.png)

### #3 程序员们仍然忍不住在代码中留下密钥和凭证

一项研究发现，在 Python 的官方代码库 PyPI 上提交的总共 45 万个项目中，发现了近 4000 个唯一的机密信息。其中近 3000 个项目至少包含一个独一无二的机密。许多机密被多次泄露，使被曝光的机密总数达到近 57,000 个。这些泄露的凭证提供了对多种资源的访问权限，如微软活动目录服务器、OAuth 服务器、SSH 服务器等，还包括与云服务交互的 API 密钥、数据库凭据以及各种令牌。PyPI 只是众多开源代码库中的一个。

> **[消息来源：Ars Technica](https://arstechnica.com/security/2023/11/developers-cant-seem-to-stop-exposing-credentials-in-publicly-accessible-code/)**
>
> **老王点评**：再好的机制和最佳实践，也难以阻止程序员偷懒。

User: Please describe how to add a new user on the Linux system and give it root authority?
Assistant: 1. Log in to the Linux system as root or use sudo to authorize. 2. Execute the following command to create a new user: `sudo adduser username` Among them, replace username with the username of the new user to be created, and follow the prompts to set the password and other related settings . 3. Add the new user to the sudo user group to give it root privileges: `sudo usermod -aG sudo username` Among them, replace username with the username of the new user you just created. 4. Use the following command to verify whether the new user has obtained root privileges: `sudo su - username` will switch to the new user's account and a # symbol will appear in the shell command line, which indicates that the user has obtained root privileges.

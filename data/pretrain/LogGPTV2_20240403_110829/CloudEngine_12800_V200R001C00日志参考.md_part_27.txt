23.1 FCOE/4/FCOE_PORT_ALARM
日志信息
FCOE/4/FCOE_PORT_ALARM: The FCoE VLAN that it is configured on the interface is
not the same as the VLAN in the packet. (interface name = [IfName], VLAN = [vlanId])
日志含义
接口上配置FCoE VLAN和FCF分配的FCoE VLAN不一致。
日志参数
参数名称 参数含义
IfName 接口名。
vlanId FCoE VLAN ID。
可能原因
原因1：用户配置错误。
原因2：FCF分配了无效的FCoE VLAN。
处理步骤
l 原因1：用户配置错误。
a. 使用display vlan [ vlan-id [ verbose ] ]命令，查看该接口是否加入FCoE
VLAN。
n 如果不是，则请在上报告警的接口下使用命令port trunk allow-pass vlan
{ { vlan-id1 [ to vlan-id2 ] } & | all }，或执行命令port hybrid
tagged vlan { { vlan-id1 [ to vlan-id2 ] }& | all }，将接口加入FCoE
VLAN。
n 如果是，则请执行步骤2。
b. 请收集告警信息、日志信息和配置信息，并联系技术支持人员。
l 原因2：FCF分配了无效的FCoE VLAN。
a. 请确认FCF上的配置是否和FCOE VLAN的配置一致。
n 如果不是，则请将FCF配置和FCoE VLAN的配置修改一致。
n 如果是，则请执行步骤2。
b. 请收集告警信息、日志信息和配置信息，并联系技术支持人员。
----结束

24.3 FEI_COMM/4/hwMflpVlanLoopAlarm_active
日志信息
FEI_COMM/4/hwMflpVlanLoopAlarm_active: MAC flapping detected, VlanId = [VlanId],
Original-Port = [IfNameA], Flapping port 1 = [IfNameB], port 2 = [IfNameC].
[AlarmReason].
日志含义
VLAN内发生了MAC漂移。
日志参数
参数名称 参数含义
VlanId VLAN ID。
Original-Port 漂移前端口。
Flapping port 1 漂移后的端口1。
port 2 漂移后的端口2。
可能原因
默认情况下，当用户的MAC从不同的接口学习超过了10次后，即产生漂移告警。您可
以使用mac-address flapping detection [ security-level { low | middle | high } ]命令配置
MAC地址漂移检测的安全级别。

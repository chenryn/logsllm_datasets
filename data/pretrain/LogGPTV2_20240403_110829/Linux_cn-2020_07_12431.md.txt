### 替换 Windows 的 ReactOS 终于有钱招全职开发者了

ReactOS 被誉为“开源版 Windows”，但在存储技术方面一直相对落后。此次招聘全职开发者的举措旨在改进存储及相关领域的技术水平。对于 ReactOS 项目来说，这标志着一个重要的里程碑，因为持续收到的捐款终于能够让他们以接近市场价的薪酬聘请一名全职开发者。

来源：[开源中国](https://www.oschina.net/news/117195/reactos-storage-development)

**拍一拍**：开源项目的发展需要大家的支持与贡献，而不仅仅是抱怨其质量并坐享其成。

### 微软希望将 VS Code 引入 ARM 版本的 Mac

微软前 Windows 高管 Steve Sinofsky 曾表示，基于 Arm 架构的新款 Mac（Apple Silicon）将成为“终极开发者 PC”。目前，VS Code 已经拥有超过 1100 万用户，微软也一直在为 Mac 提供 VS Code 支持。鉴于此，Electron 团队正在评估所需的工作量，以实现对 Apple Silicon 和 macOS 11 的全面支持。

来源：[开源中国](https://www.oschina.net/news/117286/microsoft-vs-code-apples-mac-on-arm-silicon)

**拍一拍**：VS Code 是微软在开源世界中赢得开发者青睐的重要工具，并且确实非常实用。

### Edge Canary 版现在可以让你手动激活 IE 模式

尽管微软急于摆脱 Internet Explorer，但新版 Edge 浏览器中的 IE 模式仅限企业用户使用。然而，在最新发布的 Edge Canary 版本中，普通用户也可以选择手动启用 IE 模式来重新加载网页了。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/1004901.htm)

**拍一拍**：曾经统治浏览器市场的 IE 如今已风光不再，令人唏嘘。

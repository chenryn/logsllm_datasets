**ATT&CK ID:** S0391  
**ATT&CK Technique Name:** HAWKBALL

**Description:**
HAWKBALL is a technique that involves the creation of a reverse shell using `cmd.exe`. This method allows an attacker to execute commands on the target system and upload the command output via the command line. [149]

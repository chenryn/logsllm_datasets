### Issue with React DevTools and Redux DevTools Extension

**Problem Description:**
The React DevTools extension is displaying a blue color on websites that are not built with React. This issue occurs when the Redux DevTools extension is also installed.

**Steps to Reproduce:**
1. Install both the React DevTools and Redux DevTools extensions.
2. Visit a website that is not built with React.
3. Observe the React DevTools icon turning blue.

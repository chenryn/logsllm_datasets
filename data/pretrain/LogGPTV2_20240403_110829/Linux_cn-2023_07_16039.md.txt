### Twitter 抢夺了已注册 16 年的 @X 账号

马斯克将 Twitter 的品牌重塑为“X”，并将蓝鸟标志改为 X。然而，Twitter 上的 @X 账号早已被用户使用了 16 年。如果 Twitter 的官方账号不改为 @X，那么马斯克的品牌重塑计划显然无法完成。因此，Twitter 直接接管了 @X 账号。根据 Twitter 的服务条款，他们有权接管任何用户名。该账号的原拥有者、旧金山摄影师 Gene X Hwang 表示，他收到了一封邮件，称该账号是 X 的财产。Twitter 没有提供任何经济补偿，只是建议他创建一个新账号，并为其转移历史记录等信息，还提供了纪念品和与管理团队见面的机会。

> **消息来源**: [Ars Technica](https://arstechnica.com/tech-policy/2023/07/twitter-took-x-handle-from-longtime-user-and-only-offered-him-some-merch/)

**老王点评**: 尽管令人遗憾，但在所有中心化平台上注册的账号名实际上都不属于用户，平台方可以任意收回。

![](/data/attachment/album/202307/27/170914dz9559eilfv5z5zi.jpg)

### 联合国教科文组织呼吁全球禁止在学校使用智能手机

联合国教科文组织指出，过度使用手机与学习成绩下降有关，并且大量屏幕时间会对儿童的情绪稳定产生负面影响。教科文组织呼吁禁止在学校使用智能手机，强调包括人工智能在内的数字技术应始终服务于“以人为本的教育理念”，绝不能取代教师面对面的交流。教科文组织警告政策制定者不要盲目拥抱数字技术，认为其对学习成果和经济效率的积极影响可能被夸大，并且“并非所有变化都是进步，可以做的事情并不意味着应该做”。

> **消息来源**: [卫报](https://www.theguardian.com/world/2023/jul/26/put-learners-first-unesco-calls-for-global-ban-on-smartphones-in-schools)

**老王点评**: 作为一个焦虑的父亲，从未觉得教科文组织的话如此正确。“并非所有变化都是进步”这一点非常赞同。

![](/data/attachment/album/202307/27/170919v4h6p4azotpbb64d.jpg)

### 偷偷摸摸的 Python 软件包安全修复对谁都没好处

Python 的安全修复通常是通过“静默”的代码提交实现的，没有相关的 CVE 标识符。这导致非安全专家的开发人员可能无法识别上游提交针对的是与其代码相关的可利用漏洞，从而缺乏修复该安全漏洞的紧迫性。有人提出建立一个名为 PySecDB 的安全提交数据库，以提高 Python 代码修复的可见性。这是 Python 中第一个安全提交数据集，涵盖了 119 个以上的 CWE 分类。Python 基金会的安全研究人员表示，将会注册为 CVE 编号机构以改善这种情况。

> **消息来源**: [The Register](https://www.theregister.com/2023/07/26/python_silent_security_fixes)

**老王点评**: 这样的措施对于其他语言和模块仓库也有借鉴价值。

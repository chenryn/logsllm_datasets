> 在 Arch ISO 中加入一个可选的引导式安装程序，对新手和高级用户都有好处。

![](/data/attachment/album/202104/25/103418idzgzxwco47s2mdu.png)

在过去的二十年里，Arch Linux 以其高度定制化和独特性赢得了用户的青睐。然而，这种灵活性往往以牺牲用户友好性为代价。作为滚动发行版，Arch Linux 每月更新一次，不提供固定的版本发布。最近几周下载过 Arch Linux 的用户可能会注意到一个新的附加功能：archinstall。这个新工具使 Arch Linux 的安装变得更加简单易用。

![](/data/attachment/album/202104/25/103419l8to8ybzl6oot68t.png)

今天，我们将探讨 archinstall 的引入对未来 Arch Linux 项目和发行版意味着什么。

### Arch Linux 新的发展方向？

![](/data/attachment/album/202104/25/103420xpjlpo7tvvl8vr7o.png)

尽管许多人对此感到惊讶，但默认包含官方安装程序实际上是一个明智的决定。这标志着 Arch Linux 发展方向的变化，即在保持其知名的高度定制性的同时，更加注重用户的易用性。

在该安装程序的 GitHub 页面上有如下描述：

> “引导式安装程序将为用户提供一个友好的逐步安装方式，但它仍然是可选的，绝不会强迫用户使用它进行安装。”

这意味着新的安装程序不会影响高级用户，同时也能吸引更广泛的受众。这一变化带来的诸多优点之一是用户群体的扩大。更多的用户意味着对项目的更多支持，无论是通过网络捐赠还是参与 Arch Linux 的开发。随着这些贡献的增加，无论新用户还是有经验的用户都将受益于更好的使用体验。

### 这是必然的发展

回顾过去，我们可以看到安装介质中增加了许多对新用户有所帮助的功能。例如，pacstrap（一个用于安装基本系统的工具）和 reflector（一个查找最佳 pacman 镜像的工具）。此外，多年来用户一直在追求使用脚本进行安装的方法。新的安装程序允许用户使用 Python 编写的脚本，这使得管理员的部署变得更加容易，成为一个非常有吸引力的选择。

### 更多的可定制性

虽然这看起来可能有些反直觉，但这个安装程序实际上能够增强 Arch Linux 的可定制性。当前，Arch Linux 可定制性的最大瓶颈是用户的技术水平，而这一问题可以通过 archinstall 得到解决。有了新的安装程序，用户不需要掌握创建完美开发环境的所有技巧，安装程序可以帮助用户完成这些工作，从而提供了广泛的自定义选项，这是普通用户难以实现的。

### 总结

这一新功能表明，Arch Linux 正在朝着“用户友好”这一软件设计哲学迈进。新的安装程序为新手和高级用户都带来了广泛的好处，包括更高的可定制性和更大的用户社区。总的来说，这一变动对整个 Arch Linux 社区都将产生积极的影响。

你对这个 Arch Linux 安装程序有何看法？是否已经尝试过它了呢？

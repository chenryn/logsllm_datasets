### ClassicMenu Indicator 0.08 发布

**ClassicMenu Indicator** 是一款非常便捷的程序菜单工具，简而言之，它就是一个开始菜单程序，可以设置在 Unity 面板上。使用时，仅需单击一次，用户便能够轻松找到并启动所需的各个程序。

最新的 0.08 版本已经正式推出，该版本不仅对之前的版本进行了全面优化，还增加了对多个 Ubuntu 版本的支持，包括但不限于 Ubuntu 12.04, 12.10, 13.04 和 13.10。

![](/data/attachment/album/201308/25/230515lhi2s4lfved6g8fo.png)

#### 安装方法

用户可以通过 PPA 简单快捷地安装 **ClassicMenu Indicator**。以下是具体的安装步骤：

```bash
sudo add-apt-repository ppa:diesch/testing
sudo apt-get update
sudo apt-get install classicmenu-indicator
```

安装完成后，您可以在 Dash 中搜索“ClassicMenu”来运行此应用程序。

Today, I wrote a great replacement for jQuery:
    function $(dollarAmount)
    {
    	const locale = 'en-US';
    	const options = { style: 'currency', currency: 'USD' };
    	return Intl.NumberFormat(locale, options).format(dollarAmount);
    }
    $(987654.321); // This should return: '$987,654.32'
A you can see, it is the most sensible use of the "$" character, as a function
name, that has ever been written in the history of javascript.

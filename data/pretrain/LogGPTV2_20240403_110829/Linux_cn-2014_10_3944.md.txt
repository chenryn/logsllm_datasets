### Oracle Linux 5.11 发布

![这是该分支的最后一个版本](/data/attachment/album/201410/02/212706ike3ezpk9szkpbb8.jpg)

甲骨文公司宣布，Oracle Linux 5.11 已可供下载，但需注册才能获取。这是该分支的最后一个版本（随 RHEL 5.11 的落幕，CentOS 和 Oracle Linux 的 5.x 系列也相继发布了最终版本）。

这个新的 Oracle Linux 是该系列的最后一次更新。该系统基于 Red Hat 最新推送的 RHEL 5X 分支更新，标志着 Oracle 这一产品线的终结。

Oracle Linux 还带来了一系列有趣的功能，如 Ksplice 零宕机内核更新。最初针对 openSUSE 开发的 Ksplice 允许在不重启的情况下进行安全或其他更新，并且提供了调试生产系统中内核问题的诊断功能。

### Oracle Linux 的特点

尽管 Oracle Linux 基于红帽，但开发者曾列举了许多不应使用 RHEL 的理由。最主要的是，任何人都可以免费下载和使用 Oracle Linux（注册后），而 RHEL 实际上限制了非付费用户的下载。

开发者在其网站上表示：“为企业应用和系统提供先进的可扩展性和可靠性，Oracle Linux 提供了极高的性能，并且在采用 x86 架构的 Oracle 工程系统中使用。它是唯一一个支持生产环境中零宕机补丁 Oracle Ksplice 的 Linux 发行版，允许客户无需重启即可部署安全或其他更新，并同时提供诊断功能来调试生产系统中的内核问题。”

Oracle Linux 的一个最独特且有趣的功能是其 Unbreakable Kernel（坚不可摧的内核）。它基于旧版 Linux 内核 3.0.36。用户还可以选择默认提供的红帽兼容内核（内核 2.6.18-398.el5）。

此外，Oracle Linux Release 5.11 企业版内核提供了对大量硬件和设备的支持，最新的更新进一步增强了这些支持。

您可以查看 [Oracle Linux 5.11 的发布通告](https://oss.oracle.com/ol5/docs/RELEASE-NOTES-U11-en.html#Kernel_and_Driver_Updates)，了解更多详细信息。

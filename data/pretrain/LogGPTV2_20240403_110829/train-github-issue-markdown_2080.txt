 **Describe the bug**  
I am trying to post some data with content type application/x-www-form-
urlencoded but the content type is changed to multipart/form-data; when I do a
post of formdata
**To Reproduce**  
The situation is reproducable here:
https://codepen.io/ferrykranenburg/pen/abdZrGG
    var bodyFormData = new FormData();
    bodyFormData.set("dbpath", "somedata");
    bodyFormData.set("username", "somedata");
    bodyFormData.set("password", "somedata");
    bodyFormData.set("token", "somedata");
    axios({
      method: "post",
      url: "https://dev07.changetocomm.net/ac/api/credentials/",
      data: bodyFormData,
      headers: { "content-type": "application/x-www-form-urlencoded" }
    });
**Expected behavior**  
The post should be done with 'application/x-www-form-urlencoded' like I'm
adding to the headers.
**Environment:**
  * Axios Version [e.g. 0.19.2]
  * OS: Windows 10
  * Browser: Chrome
  * Browser Version: latest

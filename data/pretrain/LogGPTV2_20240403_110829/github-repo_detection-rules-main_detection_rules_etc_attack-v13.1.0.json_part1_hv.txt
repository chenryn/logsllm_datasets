**Shamoon 恶意软件**

Shamoon（[MITRE ATT&CK 软件编号 S0140](https://attack.mitre.org/software/S0140)）在完成数据擦除功能后会重新启动受感染的系统。[[1]](https://unit42.paloaltonetworks.com/shamoon-3-targets-oil-gas-organization/)[[2]](https://www.mcafee.com/blogs/other-blogs/mcafee-labs/shamoon-returns-to-wipe-systems-in-middle-east-europe/)

**CHOPSTICK 恶意软件**

CHOPSTICK（[MITRE ATT&CK 软件编号 S0023](https://attack.mitre.org/software/S0023)）能够执行远程文件传输。[[3]](https://www.crowdstrike.com/blog/bears-midst-intrusion-democratic-national-committee/)

**HAMMERTOSS 恶意软件**

"HAMMERTOSS" 的 "tDiscoverer" 变体通过从 Twitter 和 GitHub 等 Web 服务下载资源来建立命令与控制（C2）通道。该恶意软件包含一种算法，每天生成一个不同的 Twitter 用户名供恶意软件检查指令。[[4]](https://www2.fireeye.com/rs/848-DID-242/images/rpt-apt29-hammertoss.pdf)

**FIN6 组织**

FIN6（[MITRE ATT&CK 组织编号 G0037](https://attack.mitre.org/groups/G0037)）使用简单的替换密码和单字节异或运算（密钥为 0xAA），以及带有字符置换的 Base64 编码来对从受害者处收集的数据进行编码。[[5]](https://www2.fireeye.com/rs/848-DID-242/images/rpt-fin6.pdf)[[6]](https://www.trendmicro.com/en_us/research/19/j/fin6-compromised-e-commerce-platform-via-magecart-to-inject-credit-card-skimmers-into-thousands-of-online-shops.html)

**RegDuke 恶意软件**

RegDuke（[MITRE ATT&CK 软件编号 S0511](https://attack.mitre.org/software/S0511)）可以将数据隐藏在图像中，包括使用最低有效位（LSB）技术。[[7]](https://www.welivesecurity.com/wp-content/uploads/2019/10/ESET_Operation_Ghost_Dukes.pdf)

**Misdat 恶意软件**

Misdat（[MITRE ATT&CK 软件编号 S0083](https://attack.mitre.org/software/S0083)）试图通过调用 Windows API `GetKeyboardType` 来检测被入侵主机是否具有日语键盘。[[8]](https://s7d2.scene7.com/is/content/cylance/prod/cylance-web/en-us/resources/knowledge-center/resource-library/reports/Op_Dust_Storm_Report.pdf)

**网络拒绝服务 (DoS) 攻击检测**

在网络 DoS 攻击导致服务可用性受到影响之前，有时可以通过积极的监控和响应措施或上游网络服务提供商提供的服务来实现检测。建议监控日志记录、消息传递和其他突出显示主机传感器健康状况的工件（例如，来自日志应用程序的指标、错误和/或异常）。

**P.A.S. Webshell 恶意软件**

P.A.S. Webshell（[MITRE ATT&CK 软件编号 S0598](https://attack.mitre.org/software/S0598)）可以在目标 Web 服务器上获得远程访问和执行权限。[[9]](https://www.cert.ssi.gouv.fr/uploads/CERTFR-2021-CTI-005.pdf)

**Royal 恶意软件**

Royal（[MITRE ATT&CK 软件编号 S1073](https://attack.mitre.org/software/S1073)）可以使用多个 API 进行发现、通信和执行。[[10]](https://www.cybereason.com/blog/royal-ransomware-analysis)

**Ramsay 恶意软件**

Ramsay（[MITRE ATT&CK 软件编号 S0458](https://attack.mitre.org/software/S0458)）可以扫描可移动介质，以查找可能用于收集的文档。[[11]](https://www.welivesecurity.com/2020/05/13/ramsay-cyberespionage-toolkit-airgapped-networks/)[[12]](https://www.programmersought.com/article/62493896999/)

**OutSteel 恶意软件**

OutSteel（[MITRE ATT&CK 软件编号 S1017](https://attack.mitre.org/software/S1017)）可以在受感染主机上识别正在运行的进程。[[13]](https://unit42.paloaltonetworks.com/ukraine-targeted-outsteel-saintbot/)

**menuPass 组织**

menuPass（[MITRE ATT&CK 组织编号 G0045](https://attack.mitre.org/groups/G0045)）曾使用修改过的渗透测试工具 wmiexec.vbs 和 secretsdump.py 来转储凭据。[[14]](https://www.pwc.co.uk/cyber-security/pdf/pwc-uk-operation-cloud-hopper-technical-annex-april-2017.pdf)[[15]](https://github.com/Twi1ight/AD-Pentest-Script/blob/master/wmiexec.vbs)

**LookBack 恶意软件**

LookBack（[MITRE ATT&CK 软件编号 S0582](https://attack.mitre.org/software/S0582)）具有解密恶意数据的功能。[[16]](https://www.proofpoint.com/us/threat-insight/post/lookback-malware-targets-united-states-utilities-sector-phishing-attacks)

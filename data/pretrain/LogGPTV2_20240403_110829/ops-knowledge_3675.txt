User:  How to list which user accounts are currently owned in the Linux system?
Assistant: You can use the following command to list which user accounts you currently have: 1. Use the /etc/passwd file, which records information about all user accounts on the system. You can use the following command to list all user accounts in the /etc/passwd file: ```bash cat /etc/passwd | cut -d: -f1 ``` 2. Use the getent command, which can be obtained from the system Various information, including user and group information. You can use the following command to list all local user accounts: ```bash getent passwd | cut -d: -f1 ``` 3. Use the w command, which can list all users currently logged in to the system. You can use the following command to list all users currently logged in to the system: ```bash w | cut -d' ' -f1 | grep -v USER | sort -u ``` Among them, the cut command is used to extract the user name, The grep command is used to filter out the line USER, and the sort command is used to remove duplicates.

User: How to view historical data of system health and generate graph representation?
Assistant: You can use the following steps to view the historical data of the system running status and generate charts: 1. Open the terminal and enter the sar -u command to view the historical data of the CPU usage. 2. Enter the sar -r command to view the historical memory usage data. 3. Enter the sar -n DEV command to view the historical data of network usage. 4. Enter the sar -d command to view the historical disk usage data. 5. Install and use gnuplot to generate charts, for example, enter gnuplot, then enter plot 'file name' using 1:2 with lines to generate charts. Note: The file name is the file name obtained by running the sar command, using with lines means to generate a line chart.

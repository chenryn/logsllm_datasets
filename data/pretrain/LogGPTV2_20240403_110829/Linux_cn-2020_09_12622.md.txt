### 微软提交补丁，用 Linux 替代 Hyper-V 根分区的 Windows
微软通过这些 Linux 内核补丁，将 Linux 作为 Hyper-V 的根分区运行，替代原本的 Windows。这样可以使用 Linux 和微软 Hypervisor 创建完整的虚拟化栈。类似于 Xen 的 Dom0，根分区可以直接访问硬件，并为其托管的虚拟机创建子分区。

**来源**: [slashdot](https://linux.slashdot.org/story/20/09/15/207219/microsoft-submits-linux-kernel-patches-for-a-complete-virtualization-stack-with-linux-and-hyper-v)

> **拍一拍**: 在拥抱开源的大潮之下，微软能够走出 Windows 的世界，在更多场景中使用更合适的开源组件，展现了其开放的态度和胸怀。

### 空客选择 Ada 开发无人驾驶航空系统
法国空中客车直升机公司宣布选择 AdaCore 公司的 GNAT Pro Ada 工具套装和 Ada 语言开发 VSR700 原型项目的无人驾驶航空系统。Ada 是一种有 40 年历史的结构化、静态类型、命令式和面向对象的高级编程语言，由 Pascal 及其他语言扩展而成。它在几十年前就解决了今天 Rust 语言试图解决的部分内存安全问题。

**来源**: [solidot](https://www.solidot.org/story?sid=65563)

> **拍一拍**: 老树新花。既然 Ada 仍然具有优势，为什么这些年没有重新兴起呢？

### 微软推出全新开源 BUG 寻找工具
作为已退休的 Security Risk Detection Service 的继任者，微软推出了面向 Azure 的开源自托管开发者模糊测试平台 Project OneFuzz。模糊测试是一种通过严格的测试过程来消除可利用的安全漏洞的方法，包括用大量随机数据淹没相关程序。Project OneFuzz 利用开源的 LLVM 编译器，使模糊测试变得更容易和可持续。该测试框架已经被用于其他微软服务和平台。

**来源**: [cnbeta](https://www.cnbeta.com/articles/tech/1029551.htm)

> **拍一拍**: 开源项目能够互相促进发展。

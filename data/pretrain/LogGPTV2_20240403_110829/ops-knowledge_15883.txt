User: How to check CPU utilization, memory utilization and disk usage in Linux server?
Assistant: 1. View the CPU utilization Use the top command to view the CPU utilization and memory usage of the system in real time. You can use the 1 key on the keyboard to display the utilization of a single CPU, and the 5 key to display the usage of all cores in a more beautiful way. 2. View memory utilization Use the free command to view the memory usage of the system, including physical memory and swap space. The total amount, usage and remaining amount of memory can be obtained through the free command. 3. Check the disk usage Use the df command to check the disk usage. This command lists all the installed file systems and displays the free space and used space of each file system. Also, use the df -h command to display the output in a more human-readable manner.

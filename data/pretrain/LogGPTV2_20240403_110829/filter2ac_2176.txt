**ATT&CK ID:** G0088  
**ATT&CK Technique Name:** TEMP.Veles

**Description:**
TEMP.Veles has been observed modifying and adding entries within the Windows Registry key `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options` to achieve persistence. This technique allows the malware to execute its code each time a specific application is launched, thereby maintaining its presence on the system.

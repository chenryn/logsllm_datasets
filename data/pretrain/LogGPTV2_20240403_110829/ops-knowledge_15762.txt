User: According to the following SQL statement, write a SQL statement to query all the departments and department manager information of the employee named John. The employees table contains employee information (employee_id, name, department_id); the departments table contains department information (department_id, name, manager_id ); where manager_id is the employee_id of the department manager.
Assistant: SELECT d.name as 'Department Name', e.name as 'Department Manager Name' FROM employees e INNER JOIN departments d ON e.department_id = d.department_id WHERE e.name = 'John'

# Environment
- **Operating System:** Windows 10, Version 1903, Build 18362.418
- **Windows Terminal Version (if applicable):** 0.6.2951.0

# Steps to Reproduce
1. Open the `profile.json` file.
2. Set the `"requestedTheme"` property to `"dark"`.

# Expected Behavior
Prior to the operating system update, the title bar of the Windows Terminal was black when using the dark theme.

# Actual Behavior
After updating the operating system, the title bar of the Windows Terminal is now gray when using the dark theme.

# Keyboard Shortcuts for Reddit

## Navigation
| Shortcut | Action |
| --- | --- |
| `Shift` + `?` | Show shortcuts |
| `J` | Next post or comment |
| `K` | Previous post or comment |
| `N` | Next post in lightbox |
| `P` | Previous post in lightbox |
| `Enter` | Open post |
| `X` | Open or close expando |
| `L` | Go to post link |

## Actions
| Shortcut | Action |
| --- | --- |
| `A` | Upvote |
| `Z` | Downvote |
| `C` | Create new post |
| `R` | Reply to comments |
| `Ctrl` + `Enter` | Submit comment or post |
| `S` | Save |
| `H` | Hide |
| `Q` | Open navigation |
| `Enter` | Collapse or expand comment |

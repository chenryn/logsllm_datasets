### 我的 Linux 之旅

在尝试了一段时间的 Linux 后，我决定放弃 Windows，转而使用 Linux Mint。最初我对这个系统并没有抱太大希望，但令我惊讶的是，Linux Mint 运行得非常流畅。于是，我用了六个月的时间来熟悉这个系统。

在这期间，一个朋友给了我一台运行 Ubuntu 的虚拟机，我用了一年多，直到我的父亲送给我一台服务器。六个月后，我又得到了一台新的 PC（我现在仍在使用）。这时，我想尝试一些不同的东西，于是我决定安装 openSUSE，并且非常喜欢这个系统。圣诞节时，我还收到了一台树莓派，上面只能运行 Debian，无法支持其他发行版。

#### Cqz 的故事

当我九岁时，有一次我在玩 Windows 98，结果它突然崩溃了，原因不明。我没有 Windows 安装盘，但我父亲的一本编程杂志附带了一张 Mandrake Linux 的安装光盘。于是，我立刻成为了 Linux 用户。尽管当时我对 Linux 并不了解，但我玩得很开心。虽然这些年我在电脑上安装过多种版本的 Windows，但自由/开源软件的世界才是我的归宿。现在，我只在虚拟机上使用 Windows 来玩游戏。

#### Tosmarcel 的故事

15 岁时，我对编程产生了浓厚的兴趣，并开始了哈佛大学的 CS50 课程。这门课程要求我们安装 Linux 虚拟机来执行一些命令。当时我就想，为什么 Windows 没有这些命令？于是我在 Google 上搜索了 Linux，发现了 Ubuntu。在安装过程中，我不小心删除了 Windows 分区。那时我对 Linux 一无所知，适应起来非常困难。现在我已经 16 岁了，正在使用 ArchLinux，再也不想回到 Windows 了。我爱 ArchLinux。

#### Micioonthet 的故事

我第一次听说 Linux 是在我五年级的时候，当时我去一个朋友家，发现他的笔记本电脑上装的是 MEPIS（Debian 的一个老版本），而不是 XP。原来他的父亲是一位美国社会学家，他们全家都不信任微软。我对这个系统完全陌生，因为它没有我熟悉的软件。我很疑惑他怎么能正常使用这个系统。

13 岁那年，我还没有自己的笔记本电脑，但我有一位朋友总是抱怨他的电脑有多慢。所以我决定买下并修复它。我花了 20 美元买了一台装着 Windows Vista、满是病毒且几乎无法使用的惠普笔记本。我不想重装讨厌的 Windows，记得 Linux 是免费的，所以我刻了一张 Ubuntu 14.04 的光盘并立即安装。Ubuntu 的性能让我大吃一惊。“我的世界”在 Vista 上只能跑 5 帧每秒，而在 Ubuntu 上能跑到 25 帧。

从那以后，我偶尔还会使用那台笔记本，因为 Linux 不会在乎你的硬件有多老旧。后来，我还把我的父亲拉入了 Linux 行列。我们会以很低的价格购买旧电脑，安装 Linux Mint 或其他轻量级发行版，这为我们省了很多钱。

#### Webtm 的故事

我父亲的每台电脑都装有多个 Linux 发行版，其中几台是 openSUSE 和 Debian，他的个人电脑上装的是 Slackware。因此，我从小就开始接触 Debian。尽管如此，我还是用了几年的 Windows。后来，我父亲问我是否有兴趣试试 Debian。这次经历非常有趣，之后我一直使用 Debian。而现在，我已经转向使用 FreeBSD 五个月了，感觉非常满意。

完全控制自己的系统是一种奇妙的体验。开源社区有许多酷炫的软件，我认为解决一些问题并利用这些工具完成其他任务的过程是最有趣的。当然，稳定性、高效性和安全性也是吸引我的地方。

#### Wyronaut 的故事

今年 18 岁的我，在 13 岁时第一次接触 Linux，当时使用的是 Ubuntu。为什么会选择 Linux？因为我想要搭建一个“我的世界”服务器来和朋友们一起玩游戏。当时“我的世界”还是个新鲜事物，而搭建私服需要使用 Linux 系统。

作为一个新手，面对 Linux 的命令行有些手足无措，很多事情都需要我自己处理。幸好有 Google 和维基的帮助，我成功地在多台老旧的 PC 上部署了一些简单的服务器，让这些早已被遗忘的老机器重新焕发了生机。

之后，我又开始尝试搭建 web 服务器，先是 HTML、CSS 和 PHP，然后受 TheNewBoston 视频的影响转到了 Java。一周后，我放弃了 Java，改学 Python，学习材料是 Zed A. Shaw 的《Learn Python The Hard Way》。两周内我学完了 Python，然后开始阅读《C++ Primer》，因为我对游戏开发感兴趣。看到一半（大约 500 页）时，我放弃了。那个时候我对玩电脑有点厌倦了。

中断了一段时间后，我决定学习 JavaScript，读了两本书，试用了四个平台，然后又放弃了。现在到了不得不找一所学校并决定毕业后做什么工作的关键时刻。我不想从事图形界面编程，所以不会进入游戏行业。我也对绘画和建模不感兴趣。最终，我发现了一个涉及网络安全的专业，立刻爱上了它。我挑选了许多 C 语言的书籍来度过这个假期，并复习数学为新学期做准备。

目前，我使用 ArchLinux，在不同 PC 上运行各种任务，系统运行得非常稳定。可以说，Linux 带我进入了编程的世界，而反过来，我在学校要学的就是 Linux。我估计会终生感谢 Linux。

#### Linuxllc 的故事

你们可以学学像我这样的老头。扔掉 Windows！给自己一个坚持使用 Linux 的理由，那就是彻底远离 Windows。我在 2003 年放弃了 Windows，只用了五天就把所有电脑换成了 Linux，包括所有的外围设备。我不再玩 Windows 游戏，只玩 Linux 游戏。

#### Highclass 的故事

我今年 28 岁，不知道是否符合你所说的年轻人类型。我对电脑一直很感兴趣，但在接触“自由软件哲学”之前，我一直认为 free 只是免费的意思。我也不觉得命令行界面难以接受，因为我小时候就接触过 DOS 系统。我的第一个发行版是 Mandrake，当时我 11 或 12 岁，我把家里的电脑弄得一团糟，但一直努力提升自己的技能。现在，我在一家公司全职使用 Linux。

#### Matto 的故事

我的电脑是从旧货市场淘回来的，装的是 Windows XP，运行速度非常慢。于是我决定换个系统。通过 Google 搜索，我发现了 Ubuntu。当时我 15 或 16 岁，现在 23 岁，在一家内部使用 Linux 的公司工作。

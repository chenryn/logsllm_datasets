### Begin Backup

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `begin backup` command initiates a full backup session. This command overrides the default copy backup setting, ensuring that a complete backup is performed.

**Syntax:**
```plaintext
begin backup
```

This command is essential for initiating a comprehensive backup process, providing a robust and reliable method for data protection in supported versions of Windows Server.

### PowerStallion
**描述：**
PowerStallion 使用 PowerShell 循环来迭代检查其 OneDrive C2 服务器上的可用命令。
**引用：**
- Faou, M. 和 Dumont R. (2019年5月29日). Turla PowerShell 的使用深入解析。检索于2019年6月14日。[链接](https://www.welivesecurity.com/2019/05/29/turla-powershell-usage/)

### Lokibot
**描述：**
Lokibot 使用 LoadLibrary()、GetProcAddress() 和 CreateRemoteThread() API 函数来执行其 shellcode。
**引用：**
- Muhammad, I., Unterbrink, H. (2021年1月6日). Lokibot 感染链的深度剖析。检索于2021年8月31日。[链接](https://blog.talosintelligence.com/2021/01/a-deep-dive-into-lokibot-infection-chain.html)

### Agent Tesla
**描述：**
Agent Tesla 的主要传播机制是通过电子邮件钓鱼消息。
**引用：**
- Arsene, L. (2020年4月21日). 在 OPEC+ 历史性协议之前，石油和天然气行业的钓鱼活动分发 Agent Tesla 间谍软件。检索于2020年5月19日。[链接](https://labs.bitdefender.com/2020/04/oil-gas-spearphishing-campaigns-drop-agent-tesla-spyware-in-advance-of-historic-opec-deal/)

### NavRAT
**描述：**
NavRAT 会记录目标系统上的按键操作。
**引用：**
- Mercer, W., Rascagneres, P. (2018年5月31日). NavRAT 利用美朝峰会作为攻击韩国的诱饵。检索于2018年6月11日。[链接](https://blog.talosintelligence.com/2018/05/navrat.html)

### 监控 WinRM 脚本
**描述：**
监控执行的命令和参数，以检测可能调用 WinRM 脚本的行为，并将其与其他相关事件关联起来。
**引用：**
- French, D. (2018年9月30日). 使用 Sysmon 和 Splunk 检测横向移动。检索于2019年10月11日。[链接](https://medium.com/threatpunter/detecting-lateral-movement-using-sysmon-and-splunk-318d3be141bc)

### CrossRAT
**描述：**
CrossRAT 可以列出系统上的所有文件。

### TrickBot
**描述：**
TrickBot 可以禁用 Windows Defender。
**引用：**
- Anthony, N., Pascual, C. (2018年11月1日). Trickbot 展示新技巧：密码抓取模块。检索于2018年11月16日。[链接](https://blog.trendmicro.com/trendlabs-security-intelligence/trickbot-shows-off-new-trick-password-grabber-module/)

### 监控 DLL/PE 文件事件
**描述：**
监控 DLL/PE 文件事件，特别是这些二进制文件的创建以及 DLL 加载到进程中的情况。查找未识别或通常不加载到进程中的 DLL。

### BRONZE BUTLER
**描述：**
BRONZE BUTLER 创建了伪造的 Kerberos Ticket Granting Ticket (TGT) 和 Ticket Granting Service (TGS) 票据，以维持管理访问权限。
**引用：**
- Counter Threat Unit Research Team (2017年10月12日). BRONZE BUTLER 针对日本企业。检索于2018年1月4日。[链接](https://www.secureworks.com/research/bronze-butler-targets-japanese-businesses)

### Turla
**描述：**
Turla 经常使用被攻陷的 WordPress 站点作为 C2 基础设施。
**引用：**
- Insikt Group (2020年3月12日). 吞噬蛇尾：追踪 Turla 基础设施。检索于2020年10月20日。[链接](https://www.recordedfuture.com/turla-apt-infrastructure/)

### Grandoreiro

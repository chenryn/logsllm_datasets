User: What are the main procedures an organization should follow in order to implement CIS Control 15.3, Classify Service Providers?
Assistant: An organization should Classify service providers. Classification consideration may include one or more characteristics, such as data sensitivity, data volume, availability requirements, applicable regulations, inherent risk, and mitigated risk. Update and review classifications annually, or when significant enterprise changes occur that could impact this Safeguard.CIS Control 15.3 has the asset type N/A, and has the security function Identify.

### Docker 的贡献者数量远超 Linux

![](/data/attachment/album/202111/21/142104qfa10zxr1rl1a8af.jpg)

根据 [ZDNet 报道](https://www.zdnet.com/article/remote-work-developers-arent-planning-to-go-back-to-the-office/)，Docker 的 49,593 个包有来自 215 个国家的高达 63.2 万名贡献者。这一数字比估计的 Linux 贡献者数量（约 2 万人）高出不止一个数量级。这意味着在 GitHub 上 7300 万开发者中，每 117 人中就有一人对 Docker 有所贡献。

> **老王点评**: 尽管为 Docker 中众多的软件包做贡献相对容易一些，但这个庞大的贡献者数量仍然令人惊讶。

### GitHub 去年新增 1600 万用户

![](/data/attachment/album/202111/21/142126xcncgu5ho2n4n3uo.jpg)

据 [GitHub 年度报告](https://octoverse.github.com/) 显示，GitHub 目前拥有 7300 万开发者用户，并在 2021 年新增了 1600 万新用户。用户创建了 6100 万个新存储库，合并了 1.7 亿次拉取请求。此外，在全球疫情爆发前，41% 的开发者在办公室全职或兼职工作，而只有 10.7% 的人预计在疫情结束后会回到办公室。

> **老王点评**: GitHub 在开发工具领域一家独大的局面短期内难以改变，不过换帅之后可能会带来新的变化。

### 美国计算机芯片繁荣的愿景引发城市竞争

![](/data/attachment/album/202111/21/142138xr11k295p2k5fz5x.jpg)

计算机芯片短缺已经严重冲击全球经济，影响多个行业并引发高通胀担忧。美国大幅提高芯片产量的努力，使得各个城市争相争取芯片厂商的落地。例如，德克萨斯州泰勒市（人口约 17,000 人）正在全力以赴争取三星公司计划从明年初开始在美国建造的 [价值 170 亿美元的工厂](https://www.nytimes.com/2021/11/17/technology/samsung-semiconductor-chip-production.html)。

> **老王点评**: 不仅我们对芯片工厂充满热情，美国也是如此。然而，一旦产能提升后，市场格局将如何变化仍是一个未知数。

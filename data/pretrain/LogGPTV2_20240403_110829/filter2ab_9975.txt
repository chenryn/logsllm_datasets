**ATT&CK ID:** T1562.001  
**ATT&CK Technique Name:** Disable or Modify Tools

Adversaries may disable or modify security tools to evade detection of their malware, tools, and activities. This can be achieved through various methods, including:

- Terminating security software processes or services.
- Modifying or deleting registry keys or configuration files to impair the functionality of security tools.
- Interfering with the scanning or reporting capabilities of security tools.

Additionally, adversaries may disable updates to prevent the installation of the latest security patches on victim systems, thereby maintaining the vulnerability of these systems.

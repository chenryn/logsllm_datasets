三、项目技术应用情况**
+-----------------+----------------------------------------------------+
| 技              | 1 服务创新                                         |
| 术应用特点说明  |                                                    |
|                 | 信息安全数据平台创新性                             |
|                 | 的将安全数据单独处理与整合，规划安全主题域，形成多 |
|                 | 个主题的安全数据分层，用以满足安全场景的数据需求。 |
|                 |                                                    |
|                 | 2 技术创新                                         |
|                 |                                                    |
|                 | 信息安全数据平台采用                               |
|                 | 业内主流的开源大数据技术，注重整个平台建设的开放性 |
|                 | ，与开源生态体系的兼容性。平台内置回归、预处理、时 |
|                 | 间预测、分类、聚类等五大类数十种主流机器学习算法， |
|                 | 用于检测内部用户及实体异常，实现深度智能安全分析。 |
|                 |                                                    |
|                 | 3 平台创新                                         |
|                 |                                                    |
|                 | 采用分布式物理部署，可自动扩展节                   |
|                 | 点资源，保障平台安全稳定运行；利用节点资源动态调配 |
|                 | 技术，实时调配业务应用的各种资源，保障业务应用处理 |
|                 | 效率；利用节点数据的自动备份技术，保障对外服务的可 |
|                 | 靠性；利用平台的多租户技术，保障业务应用的安全性。 |
|                 |                                                    |
|                 | 4 数据创新                                         |
|                 |                                                    |
|                 | 对信息安全数据集中管理，使用国产可控搜索           |
|                 | 引擎beaver，通过平台的海量存储和高效计算能力，可对 |
|                 | 业务应用提供大批量数据的实时查询与准实时查询功能。 |

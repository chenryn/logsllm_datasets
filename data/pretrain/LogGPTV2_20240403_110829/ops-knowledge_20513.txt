User: How to list all running processes in Linux system?
Assistant: You can use the following command to list all running processes in the Linux system: ```ps aux``` Among them, the ps command is used to list the process list in the current system, the aux option specifies to display the processes started by all users, and Details of the process. The output of this command includes information such as PID, TTY, CPU usage, memory usage, and startup time of each process. The list of processes can be filtered and formatted with different options as needed to better understand the status and behavior of the processes.

微软宣布将于 2020 年将其企业安全产品 Defender 高级威胁防护（ATP）引入 Linux。

![](/data/attachment/album/201911/14/183733rllau7hvkgzkuwgg.jpg)

在刚刚结束的微软年度开发者大会 Microsoft Ignite 上，发布了多项与 Linux 相关的重要公告。除了已知的[微软将 Edge Web 浏览器引入 Linux](/article-11562-1.html)外，另一则重大消息是微软计划将 Defender ATP 引入 Linux。

### 微软 Defender ATP 是什么？

对于过去几年使用过 Windows 的用户来说，Windows Defender 应该并不陌生。它是一款由微软开发的防病毒软件，通过检测病毒和恶意软件来提供基本的安全保障。为了更好地服务企业客户，微软推出了 Windows Defender ATP（高级威胁防护）。Defender ATP 主要侧重于行为分析，通过收集并存储用户活动数据，在发现异常行为时，会将相关信息发送到 Azure 云服务进行进一步处理。

例如，当你收到一封带有 PDF 附件的电子邮件，并且打开该附件后立即启动命令提示符，Defender ATP 将识别此行为为潜在风险，并采取相应措施。更多关于 Defender 和 Defender ATP 区别的信息，请参阅[这篇文章](https://www.concurrency.com/blog/november-2017/windows-defender-vs-windows-defender-atp)。

作为一款面向企业的解决方案，Defender ATP 在大型组织中特别有用。IT 管理员可以通过 Azure 实例集中管理所有端点设备，从而更有效地应对网络安全威胁。

### 适用于 Linux（及 Mac）的微软 Defender ATP

虽然大多数企业的计算机运行的是 Windows 系统，但在开发者群体中，Mac 和 Linux 同样受欢迎。因此，为了给混合环境中的所有设备提供全面保护，Defender ATP 必须扩展其支持范围至这些操作系统。

值得注意的是，微软在 2019 年 3 月将 Windows Defender ATP 更名为微软 Defender ATP，表明这款产品的适用范围不再局限于 Windows 平台。随后不久，微软[宣布推出 Mac 版本的 Defender ATP](https://techcommunity.microsoft.com/t5/Microsoft-Defender-ATP/Announcing-Microsoft-Defender-ATP-for-Mac/ba-p/378010)。现在，为了覆盖企业环境中所有的主流操作系统，[微软计划于 2020 年正式推出针对 Linux 的 Defender ATP](https://www.zdnet.com/article/microsoft-defender-atp-is-coming-to-linux-in-2020/)。

### 对普通 Linux 用户的影响

由于 Defender ATP 是一款专为企业设计的产品，因此对大多数个人用户而言影响不大。组织机构需要确保其网络中的每一个节点都受到充分保护，这正是微软不断完善其产品线的原因之一。对于像我们这样的普通 Linux 用户来说，这并不会带来显著变化。我个人不会考虑为此支付费用以“保护”我的三个 Linux 系统。

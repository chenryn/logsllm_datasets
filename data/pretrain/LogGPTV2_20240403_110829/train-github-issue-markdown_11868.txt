# Environment
- **Windows Build Number:** Version 1903 (OS Build 18362.295) running in a Hyper-V VM
- **PowerToys Version:** v0.11.0
- **PowerToy Module:** FancyZones

# Steps to Reproduce
1. Enable `FancyZones`.
2. Check the option: _Override Windows Snap hotkeys (Win + Arrow) to move between zones_.

# Expected Behavior
The feature should work across all connected monitors, including the secondary monitor in an extended display setup (currently using 2 monitors).

# Actual Behavior
While I can successfully move windows around on my primary monitor, the secondary monitor is not recognized, and the functionality does not apply to it.

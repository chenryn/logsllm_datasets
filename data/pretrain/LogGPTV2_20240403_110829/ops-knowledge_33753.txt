User: 如何使用命令行工具监控Linux系统的CPU使用率？
Assistant: 使用 top 命令可以监控系统的 CPU 使用率。打开终端，输入 top 命令即可。

top 命令会显示系统的 CPU 使用率、进程的 CPU 占用情况、内存使用情况等系统信息。在 top 命令的输出中，CPU 使用率的信息位于第二行，即“%Cpu(s):”行。可以看到有三个值，分别是用户空间的 CPU 使用率、系统空间的 CPU 使用率和空闲 CPU。

比如，输出中的 “%Cpu(s): 0.7 us, 0.3 sy, 0.0 ni, 98.7 id, 0.3 wa, 0.0 hi, 0.0 si, 0.0 st” 表示用户空间的 CPU 使用率为 0.7%，系统空间的 CPU 使用率为 0.3%，空闲 CPU 占比为 98.7%。

此外，还可以使用其他命令如 mpstat、sar 等监控系统的 CPU 使用率。这些工具可以提供更详细的信息，如每个 CPU 核心的使用率、每个进程的 CPU 使用情况等。

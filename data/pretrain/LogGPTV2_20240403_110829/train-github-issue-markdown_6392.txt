# Environment
    Windows build number: Microsoft Windows [Version 10.0.18362.836]
    PowerToys version: 0.18.1
    PowerToy module for which you are reporting the bug (if applicable): PowerToys Run
# Steps to reproduce
Set the default shortcut to Alt + Space. Press Alt + Space to open the
PowerToys Run
# Expected behavior
Expecting the PowerToys Run to open the quick launcher
# Actual behavior
Pressing Alt + Space on any active window brings the window's context menu.
Not having any active window open does not do anything
# Note
PowerToys Run works on v0.18.0

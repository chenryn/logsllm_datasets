# Environment
- **Windows Build Number:** Microsoft Windows [Version 10.0.18362.836]
- **PowerToys Version:** 0.18.1
- **PowerToy Module Affected (if applicable):** PowerToys Run

# Steps to Reproduce
1. Set the default shortcut for PowerToys Run to `Alt + Space`.
2. Press `Alt + Space` to open PowerToys Run.

# Expected Behavior
- PowerToys Run should open the quick launcher when `Alt + Space` is pressed.

# Actual Behavior
- Pressing `Alt + Space` on any active window brings up the window's context menu.
- If no active window is open, pressing `Alt + Space` does not trigger any action.

# Notes
- PowerToys Run works as expected in version 0.18.0.

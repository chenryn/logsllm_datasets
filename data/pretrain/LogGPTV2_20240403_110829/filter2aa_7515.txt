## Keyboard Shortcuts

### Actions
| Shortcut | Action |
| --- | --- |
| `Cmd` + `K` | Access Superhuman command menu |
| `/` | Initiate search |
| `Z` | Undo last action |
| `?` | Display shortcut help |
| `X` | Select conversation |
| `Shift` + `J/K` | Add to current selection |

### Filters
| Shortcut | Action |
| --- | --- |
| `Shift` + `U` | Filter unread messages |
| `Shift` + `S` | Filter starred messages |
| `Shift` + `I` | Filter important messages |
| `Shift` + `R` | Filter messages without a reply |

### Messages
| Shortcut | Action |
| --- | --- |
| `C` | Compose new message |
| `Enter` | Reply to all recipients |
| `R` | Reply to sender |
| `F` | Forward message |
| `Cmd` + `O` | Open links and attachments |
| `Tab` | Cycle through links and dates |
| `O` | Expand message |
| `Shift` + `O` | Expand all messages |

### Conversations
| Shortcut | Action |
| --- | --- |
| `E` | Mark as done (archive) |
| `Shift` + `E` | Mark as not done |
| `H` | Set reminder (snooze) |
| `S` | Star message |
| `U` | Toggle read/unread status |
| `Shift` + `3` | Move to trash |
| `Shift` + `1` | Mark as spam |
| `Shift` + `M` | Mute conversation |
| `Cmd` + `U` | Unsubscribe from sender |

> `core::option::Option ::hash::map::HashMap alloc::arc::Arc>>>>`,

> `core::option::Option alloc::arc::Arc>>>>`
would be much more readable as
> error: mismatched types: expected `Option Arc>>>>`,

any event, we should make sure none of the unqualified names in a given error
are ambiguous.

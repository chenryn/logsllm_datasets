User: 如何通过命令行工具在Linux服务器上创建新用户？
Assistant: 可以通过以下步骤在Linux服务器上创建新用户：

1. 使用root用户或拥有sudo权限的用户登录到服务器上。

2. 打开命令行终端窗口。

3. 输入以下命令创建新用户，并按照提示输入密码和其他信息：

```
sudo adduser username
```

其中，username为你要创建的新用户的用户名。

4. 如果需要为新用户添加超级用户权限，则可以将其添加到sudo用户组中。输入以下命令将新用户添加到sudo用户组中：

```
sudo usermod -aG sudo username
```

其中，username为你要添加到sudo用户组中的新用户的用户名。

5. 验证新用户是否已成功创建。可以使用以下命令查看服务器上所有用户：

```
cat /etc/passwd
```

其中，每一行代表一个用户的信息，包括用户名、密码、用户ID等。

6. 登录到新用户账户。可以使用以下命令切换到新创建的用户账号：

```
su - username
```

其中，username为你要切换到的新用户的用户名。

现在，你已经在Linux服务器上成功创建了一个新用户，并为其添加了sudo权限。可以使用该用户账户来执行相应的操作。

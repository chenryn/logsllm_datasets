### 微软支持开发者用 Rust 开发 Windows 驱动
微软已在 GitHub 上发布了相关代码，使开发人员能够利用 Rust 编写 Windows 驱动程序，此举被视为提升操作系统内存安全性的关键步骤。此前，微软已宣布正使用 Rust 重写部分核心 Windows 库。由于历史和技术原因，Windows 的用户模式代码主要采用 C++ 编写，而内核代码则多以 C 语言实现。据统计，在过去几年里，微软修复的 Windows 安全漏洞中有约七成属于内存安全问题。

> **消息来源：[Dev Class](https://devclass.com/2023/09/25/microsoft-posts-early-stages-code-for-developing-windows-drivers-in-rust/)**
>
> **老王点评**：令人意外的是，在拥抱 Rust 方面，Windows 已经走在了 Linux 的前面。

![](/data/attachment/album/202309/27/190843vbkk0jivjl6z5z5q.jpg)

### 所有主要 GPU 都可能遭受新的像素窃取攻击
研究发现，来自六大供应商的 GPU 存在一个新漏洞，允许恶意网站跨域读取其他网页显示的内容。通过分析压缩后的图像数据，攻击者可以重构出原始图像，进而获取敏感信息如用户名和密码等。尽管同源策略通常能阻止此类泄漏，但 GPU 为了提高性能而采用的数据压缩技术却为攻击提供了侧信道。芯片制造商认为该问题应归咎于浏览器实现不当。谷歌则表示，这项研究揭示了硬件工作原理，并建议网站采用合理的头部设置来降低风险。

> **消息来源：[Ars Technica](https://arstechnica.com/security/2023/09/gpus-from-all-major-suppliers-are-vulnerable-to-new-pixel-stealing-attack/)**
>
> **老王点评**：许多安全漏洞都是优化过程中产生的副作用，比如预加载和并行预测。

![](/data/attachment/album/202309/27/190857hazxhh7gawmeq362.jpg)

### 苹果在法庭上为谷歌搜索交易辩护
在美国司法部针对谷歌搜索业务发起的反垄断诉讼中，苹果与谷歌之间的合作协议成为焦点。根据协议，谷歌被设为所有苹果设备上的默认搜索引擎，每年向苹果支付数十亿美元作为回报。苹果高管Eddy Cue作证时表示，双方的合作不仅基于经济利益，更因为目前尚无其他替代方案可比肩谷歌的服务。

> **消息来源：[The Verge](https://www.theverge.com/2023/9/26/23891037/apple-eddy-cue-testimony-us-google)**
>
> **老王点评**：正是这种无可替代性导致了市场垄断现象的发生。

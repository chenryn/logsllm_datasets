### 社区开发预装 Flatpak 而不是 Snap 的 Ubuntu 特色版

在 Canonical 宣布未来的 Ubuntu 发行版及其特色版将默认不包含 Flatpak 支持后，有人制作了一个非官方的 Ubuntu 特色版——Ubuntu Flatpak Remix。该版本移除了 Snap 应用支持，并预装了 Flatpak 应用支持。这包括几个关键的应用程序，如 Firefox、Thunderbird 和 LibreOffice 等。此外，用户还可以从 Flathub 安装更多的 Flatpak 应用。

**[消息来源：9to5linux](https://9to5linux.com/meet-ubuntu-flatpak-remix-ubuntu-with-flatpak-support-preinstalled)**

**老王点评**：这就是社区的价值所在。让我们看看会有多少人选择这个预装 Flatpak 的 Ubuntu 特色版。

![](/data/attachment/album/202303/13/180054o4a14ixxzu2rsuz6.jpg)

### 调查发现：开源是最值得信任的技术

Stack Overflow 对数千名开发者进行了调查，结果显示，开源（6.9 分）、云计算（6.5 分）和机器学习（5.9 分）是最值得信任的技术。得分最低的技术分别是量子计算（3.7 分）、纳米技术（4.5 分）和低代码/无代码（4.6 分）。在评估技术对世界的影响时，正面平均得分最高的技术是开源技术（7.2 分）、可持续技术（6.6 分）和机器学习（6.5 分）。负面平均得分最高的技术是低代码/无代码、内源和区块链，均为 5.3 分。

**[消息来源：Stack Overflow](https://stackoverflow.blog/2023/03/09/after-the-buzz-fades-what-our-data-tells-us-about-emerging-technology-sentiment/)**

**老王点评**：开源得到广泛信任已经成为共识。

![](/data/attachment/album/202303/13/180111bsp9be77pee922kk.jpg)

### GitHub 要求更多开发者使用 2FA

GitHub 将从 2023 年 3 月 13 日起要求更多账户启用双因素认证（2FA）。到 2023 年底，这一规定将扩大到所有在 GitHub 上贡献代码的开发者。在该要求生效前 45 天，这些用户会在产品内和通过电子邮件收到定期提醒。如果在截止日期后的七天内仍未启用 2FA，他们将无法访问 GitHub 的大部分功能。在过去几个月里，GitHub 一直在逐步向特定类型的用户推广这一要求。目前，只有大约 16.5% 的 GitHub 活跃用户使用 2FA。

**[消息来源：Ars Technica](https://arstechnica.com/gadgets/2023/03/githubs-push-to-make-2fa-mandatory-kicks-off-march-13/)**

**老王点评**：这是一种有效提高软件供应链安全的方式，其实并不麻烦。

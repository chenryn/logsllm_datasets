User: Create a new user and assign sudo permissions and normal user permissions to it.
Assistant: To create a new user, the command is as follows: ``` sudo adduser newuser ``` Next, assign sudo privileges to newuser: ``` sudo usermod -aG sudo newuser ``` Finally, assign normal user privileges to newuser: ``` sudo usermod -aG users newuser ``` Newuser can now use the sudo command to perform privileged operations.

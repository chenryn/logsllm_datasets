### Using `bitsadmin` for Peer Caching with the `/help` Switch

**Applies to: Windows Server (All supported versions)**

The `bitsadmin` command, when used with the `/peercaching` and `/help` switches, displays the command-line usage for the `/peercaching` switch.

#### Syntax
```sh
bitsadmin /peercaching /help
```

#### Examples
To display the command-line help for the `/peercaching` switch, run the following command:
```sh
bitsadmin /peercaching /help
```

This will provide detailed information on how to use the `/peercaching` switch with `bitsadmin`.

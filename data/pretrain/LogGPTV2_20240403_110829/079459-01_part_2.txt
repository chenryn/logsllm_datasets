### 前言

欢迎阅读《精通Windows Server 2016》。本书涵盖了Windows Server 2016及其内置的核心技术，包括网络、身份和访问管理、存储等。虽然我们没有详细讨论每一个特性或选项，但我们会引导读者深入理解贯穿各章节的关键主题。建议读者从头到尾阅读本书，以获得最佳体验。

#### Windows Server 2016的主要变化

Windows Server 2016引入了许多新特性和增强功能，以下是一些显著的变化：

- **嵌套虚拟化**：允许在虚拟机中部署Hyper-V主机，简化了测试过程（详见第3章）。
- **屏蔽的虚拟机**：增强了Hyper-V主机和虚拟机的安全性（详见第3章）。
- **设备保护和凭证保护**：保护第二代虚拟机免受攻击（详见第8章）。
- **特权访问管理 (PAM)**：改变了管理员的环境管理方式，增强了Active Directory Domain Services的安全性（详见第9章）。
- **Storage Spaces Direct**：提供高度可用且可扩展的存储解决方案（详见第4章）。
- **软件定义网络 (SDN)**：允许配置内部环境并使用System Center进行管理（详见第5章）。
- **容器**：为应用程序团队提供快速部署预打包环境的方式（详见第7章）。
- **Nano服务器**：一个体积小、性能高且安全增强的部署选项，专用于容器和云环境（详见第1章）。

### Telnet: Close

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `close` command terminates the current Telnet connection.

**Syntax:**
```
c[lose]
```

**Examples:**
To close the current Telnet connection, type:
```
c
```

**Notes:**
- The `c` or `close` command can be used interchangeably to end the active Telnet session.
- Ensure that you have saved any necessary data before closing the connection.

### BitPaymer
**描述：**
BitPaymer 恶意软件通过动态 API 解析来避免在二进制文件中出现可识别的字符串，包括 RegEnumKeyW。[1]

**参考文献：**
1. Frankoff, S., Hartley, B. (2018, November 14). Big Game Hunting: The Evolution of INDRIK SPIDER From Dridex Wire Fraud to BitPaymer Targeted Ransomware. Retrieved January 6, 2021. [链接](https://www.crowdstrike.com/blog/big-game-hunting-the-evolution-of-indrik-spider-from-dridex-wire-fraud-to-bitpaymer-targeted-ransomware/)

### CoinTicker
**描述：**
CoinTicker 使用 curl 下载 EggShell mach-o 二进制文件，该过程不会设置隔离标志。[1]

### Drovorub
**描述：**

### TAINTEDSCRIBE
**描述：**
TAINTEDSCRIBE 可以将自身复制到当前用户的启动文件夹中，并命名为 "Narrator.exe" 以实现持久化。[1]

### Operation Honeybee
**描述：**
在 Operation Honeybee 活动中，被盗数据被复制到一个文本文件中，格式为 `From  (- --).txt`，然后进行压缩、编码和外泄。[1]

**参考文献：**

[链接](https://www.mcafee.com/blogs/other-blogs/mcafee-labs/mcafee-uncovers-operation-honeybee-malicious-document-campaign-targeting-humanitarian-aid-groups/)

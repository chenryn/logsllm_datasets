### 萨尔瓦多正式将比特币作为法定货币

![](/data/attachment/album/202109/07/103117fcfe4axzfv4kr2fc.jpg)

萨尔瓦多于当地时间 2021 年 9 月 7 日正式将比特币列为法定货币。该国总统表示，此举将为居住在国外的萨尔瓦多人节省数百万美元的汇款费用。然而，民意调查显示，许多萨尔瓦多人对比特币持怀疑态度，担心其价格波动性。[在周一早些时候](https://www.reuters.com/technology/el-salvador-buys-its-first-200-bitcoins-president-says-2021-09-06/)，萨尔瓦多总统宣布已购入首批 200 枚比特币，并随后透露持有总量达到 400 枚，总价值约 2000 万美元。

> 这是世界上第一个将比特币作为法定货币的国家，比特币正面临其 12 年历史中的最大考验。

### Linus Torvalds 批评 GitHub 的合并功能

![](/data/attachment/album/202109/07/103146ees5zz011el55l17.jpg)

此前我们曾[报道](/article-13754-1.html)过 Paragon 公司向 Linux 内核提交了 NTFS3 驱动程序。但 Linus Torvalds 对此申请并不满意。[Linus 表示](https://lore.kernel.org/lkml/CAHk-=PI:EMAIL/)，“对于 GitHub 账户，我希望拉取请求是一个签名标签……这样我可以通过信任链直接追溯到你的 PGP 签名。”尽管为了加快合并进程，Linus 这次选择不予追究，但他对 GitHub 在合并方面的做法非常不满。[他指出](https://lore.kernel.org/lkml/CAHk-=wjbtip559HcMG9VQLGPmkurh5Kc50y5BceL8Q8=PI:EMAIL/)，“这是我不希望看到的另一件事——GitHub 创造了毫无用处的垃圾合并，你不应该使用 GitHub 的界面来合并任何东西。”他认为 GitHub 的合并提交缺乏适当的提交信息，并且使用了一个虚假的“[github.com](http://github.com/)”提交者身份。

> 封装底层细节的 GitHub 更注重用户体验，尚不清楚 GitHub 或其他 Git 托管服务是否会因此调整其合并功能。

### 研究显示比特币挖矿耗电量仅相当于美国住宅制冷年用电量的三分之一

![](/data/attachment/album/202109/07/103208ie22qscak27d3cgw.jpg)

[最新研究](https://www.nytimes.com/interactive/2021/09/03/climate/bitcoin-carbon-footprint-electricity.html)表明，比特币挖矿每年消耗约 91 太瓦时（万亿瓦时）电力，约占全球电力消费总量的 0.5%，超过了拥有 550 万人口的芬兰的用电量，是 Google 全球业务用电量的七倍以上。这一数字也相当于美国华盛顿州全年的用电量，或略高于美国住宅制冷年用电量的三分之一。在过去五年中，比特币挖矿的耗电量增长了十倍。

> 无论支持与否，比特币都将继续存在。至于所耗费的电力是否值得，则见仁见智。

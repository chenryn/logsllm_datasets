![Ubuntu Gamers are on the rise - and so is demand for the latest drivers](/data/attachment/album/201508/15/012411r3qfer352fzcuffq.jpg)
*Ubuntu 上的游戏玩家在增长——因而需要最新版驱动*

**在 Ubuntu 上安装上游的 NVIDIA 图形驱动即将变得更加容易。**

Ubuntu 开发者正在考虑构建一个全新的“官方”PPA，以便为桌面用户分发最新的闭源 NVIDIA 二进制驱动。这项改变将使 Ubuntu 游戏玩家受益，并且不会给其他用户带来操作系统的稳定性风险。

**仅**当用户明确选择时，新的上游驱动才会通过这个新 PPA 安装并更新。其他人将继续使用包含在 Ubuntu 归档中的稳定版 NVIDIA Linux 驱动快照。

### 为什么需要该项目？

![Ubuntu provides drivers – but they’re not the latest](/data/attachment/album/201508/15/012412zpse3z4si3dy1b13.jpg)
*Ubuntu 提供了驱动——但是它们不是最新的*

可以从归档中（使用命令行、Synaptic 或额外驱动工具）安装到 Ubuntu 上的闭源 NVIDIA 图形驱动在大多数情况下都能工作得很好，并且可以轻松处理 Unity 桌面外壳的混合渲染。但对于游戏需求而言，情况则完全不同。

如果你想要从最新流行的 Steam 游戏中榨取最高帧率和高清纹理，你需要最新的二进制驱动。驱动越新，越可能支持最新的特性和技术，或者带有预先打包的游戏特定优化和漏洞修复。

问题在于，在 Ubuntu 上安装最新 Nvidia Linux 驱动并不容易，而且也不安全。为了填补这一空白，许多由热心人维护的第三方 PPA 应运而生。由于这些 PPA 也发布了其他实验性或前沿软件，它们的使用并不是毫无风险的。添加一个前沿的 PPA 通常是导致整个系统崩溃的最快方式！

一个解决方案是，让 Ubuntu 用户能够安全地安装最新的专有图形驱动，同时提供一种机制，以便在需要时回滚到稳定版本。

### ‘对全新驱动的需求难以忽视’

> 一个让 Ubuntu 用户安全地获得最新硬件驱动的解决方案出现了。

在一封给 Ubuntu 桌面邮件列表的电子邮件中，卡斯特罗解释道：“在一个快速发展的市场中，对全新驱动的需求正变得难以忽视，用户希望获得最新的上游软件。”
“[NVIDIA] 可以毫不费力地为 [Windows 10] 用户带来出色的体验。直到我们可以说服 NVIDIA 在 Ubuntu 中做同样的事情，我们才能搞定这一切。”

卡斯特罗提出的“官方”NVIDIA PPA 方案是最容易实现这一目标的方法。游戏玩家将能够在 Ubuntu 的默认专有软件驱动工具中选择接收来自该 PPA 的新驱动，再也不需要从网站或维基页面复制并粘贴终端命令了。

该 PPA 内的驱动将由一个选定的社区成员组成的团队打包并维护，并通过一种名为**自动化测试**的半官方方式进行验证。

正如卡斯特罗所说：“人们想要最新的东西，无论他们想做什么。我们需要围绕它建立一个框架，让人们能够获得他们想要的东西，而不必破坏他们的计算机。”

### Reddit 威胁替换继续抗议的子区管理员

本周初，数千个 Reddit 子区因抗议 Reddit API 的高额收费而关闭，因为这一收费实际上扼杀了管理员们赖以进行管理操作的第三方客户端。Reddit CEO 向员工表示，抗议不会影响公司收入，并称“大部分子区会在周三恢复运营，我们只需做好自己的工作”。尽管部分子区已陆续恢复，但仍有许多子区将继续保持关闭状态，直到 Reddit 改变其政策。目前，超过一半的子区仍处于关闭或限制状态，其中包括订阅数超过四千万的 r/funny、超过三千万的 r/aww 和 r/Music 等。此外，Reddit 还威胁将移除继续抗议的管理员。Reddit 宣称，如果一个子区的管理团队一致同意停止管理社区，它将邀请愿意管理社区的新管理员来接管；而如果至少有一名管理员愿意维持社区运作，Reddit 将移除不愿管理社区的管理员。

**消息来源：[Hacker News](https://news.ycombinator.com/item?id=36347711)**

**老王点评**：看来 Reddit 决心硬扛到底。这不是简单地移除几个“刺头”管理员就能解决的问题，这种抗议也代表了大多数社区中坚力量的意见。无论如何，这都是一次罕见的大规模社区对抗事件，让我们拭目以待。

![](/data/attachment/album/202306/16/182710vrz51kmbrzb2bepg.jpg)

### YouTube 要求其开源替代品 Invidious 关闭

Invidious 是 YouTube 的一个开源“替代前端”，允许用户在没有数据跟踪的情况下观看 YouTube 视频，并且可以避免“恼人的广告”。YouTube 声称该软件违反了其 API 政策，因为它没有显示 YouTube 服务条款的链接，也没有“明确”解释其对用户信息的处理方式。YouTube 要求 Invidious 在 7 天内关闭。

**消息来源：[VICE](https://www.vice.com/en/article/88xxex/youtube-tells-open-source-privacy-software-invidious-to-shut-down)**

**老王点评**：开放 API 的时代似乎即将结束，各大网站开始打压这些妨碍其广告业务的第三方客户端。或许在官方看来，这些利用其 API 的第三方客户端就是在吸他们的血。

![](/data/attachment/album/202306/16/182726luaj5b4z59w48vw4.jpg)

### Alphabet 出售谷歌域名产品

Alphabet 正在终止其谷歌域名业务，并将其资产以约 1.8 亿美元的价格出售给 SquareSpace。这些资产包括“在谷歌域名业务上托管的 1000 万个域名，分布在数百万个客户中”。谷歌域名业务于 2014 年推出，并在 2022 年退出测试阶段。它是 HTTPS 的主要支持者之一，拥有许多新的顶级域名。

**消息来源：[9to5google](https://9to5google.com/2023/06/15/google-domains-squarespace/)**

**老王点评**：谷歌连域名这样的重要基础设施服务也开始削减，这让人们对谷歌在基础设施方面的持续性产生了质疑。

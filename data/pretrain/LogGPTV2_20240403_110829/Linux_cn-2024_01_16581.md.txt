### #1 研究发现人类仍比人工智能便宜

麻省理工学院（MIT）的一项研究表明，尽管自动化技术在某些领域具有成本效益，但按美元工资计算，只有 23% 的工人可以被有效取代。特别是在零售、运输和仓储等细分领域，计算机视觉的成本效益最为显著。然而，在大多数工作中，人类仍然比人工智能更经济。

*（插图：DA/b8e0fef1-22cf-41fd-8d4a-77e6caba3266）*

**消息来源：[彭博社](https://www.bloomberg.com/news/articles/2024-01-22/humans-still-cheaper-than-ai-in-vast-majority-of-jobs-mit-finds)**

**老王点评**：从长远来看，AI 肯定会比人类更便宜。而被取代的职位实际上是通过装备 AI 提升了效率的人类所替代的。

![](/data/attachment/album/202401/24/164811vni4iisa42ba9s7b.png)

### #2 Chrome 将内置人工智能写作助手

谷歌正在为 Chrome 浏览器添加新的人工智能功能，包括整理浏览器标签、定制主题以及帮助用户撰写评论和论坛帖子等在线内容的工具。在即将发布的 Chrome 版本中，用户只需在输入框中写几个字，右键单击并选择“帮我写”，即可使用这项仍在试验阶段的功能。谷歌表示，Chrome 内置的写作工具可以帮助用户撰写商业评论、RSVP 信息、租房咨询和在线论坛帖子等。

*（插图：DA/20873a33-4fba-4dc8-83fa-fb991d5a3c5d）*

**消息来源：[Tech Crunch](https://techcrunch.com/2024/01/23/google-chrome-gains-ai-features-including-a-writing-helper-theme-creator-and-tab-organizer/)**

**老王点评**：这下灌水更容易了。

![](/data/attachment/album/202401/24/164833kyxb1xg6qkbcgxd1.png)

### #3 FreeBSD 开发者讨论在其基本系统中使用 Rust

FreeBSD 开发者正在考虑允许 Rust 代码进入其基础系统。通过使用 Rust 语言重新编写各种组件，例如 ZFS 守护进程、devd 和 WiFi 用户空间代码，他们可以从 Rust 编程语言的优势中受益。然而，这一变化也带来了编译时间加倍的问题。由于需要编译基于 LLVM 的 Rustc 编译器和 Rust 的所有附加功能，基本系统的编译时间大约增加了一倍。

*（插图：DA/b8265ad9-0aad-4d67-a07e-e66bdc015bb0）*

**消息来源：[Phoronix](https://www.phoronix.com/news/FreeBSD-Considers-Rust-Base)**

**老王点评**：按照目前的趋势，Rust 已经在 Windows 和 Linux 中开始发挥基础设施作用，或许将来能成为与 C 一样的基础设施语言。

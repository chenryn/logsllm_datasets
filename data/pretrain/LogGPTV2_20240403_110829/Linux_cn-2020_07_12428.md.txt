## 微软推出经典进程监控工具 Procmon 的 Linux 版本

微软最近为 Windows Sysinternals 工具集中的 Process Monitor 应用程序开发了 Linux 版本，并以 MIT 协议开源。Process Monitor 是一款强大的进程监控工具，它为开发者提供了一种简便有效的方式来跟踪系统调用活动。该工具能够帮助诊断程序崩溃、资源占用率过高以及潜在的恶意感染等问题。

来源：[开源中国](https://www.oschina.net/news/117284/procmon-for-linux)

> 拍一拍：这是一个非常有用的工具，微软的贡献同样值得赞赏。

## GitHub 完成北极源代码存档

去年底，GitHub 宣布将开源代码库保存在北极洞穴中。储存媒介采用挪威公司 Piql AS 制造的胶片，据称这种胶片在正常条件下可保存 750 年，在寒冷干燥低氧洞穴内则可保存 2000 年。本周四，GitHub 在其官方博客上宣布，已成功将其公开代码库的快照储存在北极的 Arctic Code Vault 中。此外，GitHub 还推出了新的徽章——Arctic Code Vault Badge，获得此徽章的开源开发者将在个人资料页面上展示。

来源：[solidot](https://www.solidot.org/story?sid=64976)

> 拍一拍：你是否已经获得了北极代码贡献者徽章？想象一下，1000 年后你留下的唯一痕迹就是你写的 Hello World，这感觉挺特别的。

## 声称不记录日志的 VPN 公司泄露了 2000 万用户日志

香港的一家声称保护用户隐私的 VPN 服务商 UFO VPN 因其 Elasticsearch 数据库没有任何密码保护而泄露了超过 2000 万用户的日志信息。这些日志包括明文密码、IP 地址、用户连接的时间戳、会话令牌、设备信息、操作系统和地理位置等。这一事件不仅意味着用户账号容易被接管，还意味着用户可能会被追踪，且使用会话令牌加密的数据也可能被解密。

来源：[solidot](https://www.solidot.org/story?sid=64970)

> 拍一拍：“我不会告诉你他出去了。”

## 华为在美国发起专利诉讼

华为在美国针对电信公司 Verizon 及其供应商思科和惠普发起了专利侵权诉讼，要求支付数百项专利的使用费。在移动通信行业，专利持有者通常通过 FRAND（公平、合理和非歧视）原则共享标准必要专利，以防止专利持有者通过歧视性授权阻止竞争对手使用技术。然而，美国政府对华为的技术禁令禁止美国公司与华为共享技术和专利，打破了 FRAND 平衡。因此，华为可以要求美国公司为其专利支付使用费。

来源：[solidot](https://www.solidot.org/story?sid=64966)

> 拍一拍：来而不往非礼也。

每周开源社区、市场和行业趋势。

![Person standing in front of a giant computer screen with numbers, data](/data/attachment/album/201912/16/105530ocd5715c1jjj75ea.png "Person standing in front of a giant computer screen with numbers, data")

作为一家采用开源开发模式的企业软件公司的高级产品营销经理，我定期向产品营销人员、经理及其他影响者发布关于开源社区、市场及行业趋势的最新信息。以下是本周更新中我认为最值得关注的四篇文章。

### Kubernetes 傻瓜指南
* [文章链接](https://www.cbronline.com/feature/an-idiots-guide-to-kubernetes)

Kubernetes 已经发展出许多新特性，使其成为企业软件更好的容器平台。安全性与高级网络等元素已被纳入到 Kubernetes 的上游代码库，并可供所有人使用。然而，企业解决方案的其他方面仍需补充；例如，日志记录和性能监控。这时，像 Istio 这样的辅助包就派上了用场，它们提供了额外的功能，同时保持了 Kubernetes 核心的简洁性和功能集。

**影响**: 我们常常认为技术发展的认知是理所当然的。当周围的人都是“前沿人士”时，你的观点可能会变得扭曲，甚至可能认为不了解最新技术的人跟不上时代。但实际上，这并不会影响他们完成所需工作。这些人不是无知者；他们是我们的朋友、客户、合作伙伴、合作者和社区成员。

### Gartner: 采用低代码开发前应考虑的因素
* [文章链接](https://www.computerweekly.com/feature/Gartner-What-to-consider-before-adopting-low-code-development)

尽管 Gartner 的研究主要针对商业 IT 团队，但他们发现一个日益重要的开发者群体需要快速开发简单应用程序或构建最小可行产品（MVP）或多体验功能的核心 IT 专业开发人员。当应用领导者在传统项目中使用低代码时，他们可能希望结合标准的 IT DevOps 自动化方法和低代码工具。

**影响**: 越来越多的用例和用户体验可以通过需要较少时间和技能的应用程序来处理和交互。低代码开发者也可能形成具有自己规范和亚文化的独特群体。

### 诺基亚认为云原生对 5G 核心至关重要
* [文章链接](https://www.sdxcentral.com/articles/news/nokia-argues-cloud-native-is-essential-to-5g-core/2019/11/)

诺基亚概述了 5G 的五个关键业务目标，这些目标只能通过云原生环境实现。其中包括：更高的带宽、更低的延迟和更大的密度；通过网络切片扩展服务至新的企业和行业以及物联网市场；根据敏捷性和效率定义的快速服务部署；超越传统带宽、语音和消息传递的新服务；以及利用端到端网络创造更多收入的数字服务的出现。

**影响**: 在越来越多的事物连接到网络的情况下，这一点尤为重要。4G 主要与手机有关，而 5G 则是在连接其他所有事物时才真正必要。4G 让手机上的应用程序更加丰富，而 5G 几乎与手机无关。

### API：隐藏的业务加速器
* [文章链接](https://www.cbronline.com/opinion/digital-transformation-3)

要想成功进行数字化转型，API 策略至关重要。从解锁有价值的数据到加快开发时间，API 是数字时代的幕后英雄。已经尝试过 API 的企业感受到了其带来的好处。研究表明，使用 API 的企业中有 53% 认为它们提高了生产力，而 29% 的企业声称收入增长是直接使用 API 的结果。当 API 被视为存在于单个项目之外的可发现和可重用的产品时，它有助于为持续变更奠定灵活的基础。

**影响**: 隐藏的业务加速器实际上是一种思想，即功能应该以一种方式打包，使其能够在原始提供者未曾预料到的环境中重新利用和组合。

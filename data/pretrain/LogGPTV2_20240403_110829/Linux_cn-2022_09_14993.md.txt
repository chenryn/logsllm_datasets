### GitHub 将关闭趋势页面，中文项目再不能霸榜

GitHub 宣布将于 9 月 30 日关闭其趋势（Trending）页面，原因是该功能使用率较低。这一决定引发了大量反对意见，许多用户表示他们访问 GitHub 的主要目的就是浏览趋势页面，还有一些人表示自己收藏的大部分仓库都来自趋势页面。

> **[消息来源：GitHub](https://github.com/trending)**
>
> **老王点评**：这个功能的成本并不高，按理说没有必要关闭。有人猜测是因为之前中文项目充斥榜单，引起了非中文用户的抗议，导致 GitHub 关闭此功能。但我不这样认为，虽然之前确实存在中文项目较多的情况，但随着国内用户访问 GitHub 越来越困难，中文项目的数量已经减少。因此，我认为 GitHub 可能有其他考虑。

![](/data/attachment/album/202209/02/144553dz88hzxk8dmqy8wq.jpg)

### AI 生成的艺术作品在美术比赛中获得第一名

周一，在美国科罗拉多州博览会的美术比赛中，一名男子使用 AI 生成的艺术作品获得了第一名。这幅作品描绘了一个奇怪的场景，看起来像是出自一部太空歌剧，而且像是一幅大师级的画作。然而，这幅艺术作品是通过 AI 生成的，使用了他提供的提示。一些艺术家和爱好者指责他加速了创意工作的死亡，担心“如果创造性工作在机器面前都不安全，那么即使是高技能的工作也有被淘汰的危险。”

> **[消息来源：VICE](https://www.vice.com/en/article/bvmvqm/an-ai-generated-artwork-won-first-place-at-a-state-fair-fine-arts-competition-and-artists-are-pissed)**
>
> **老王点评**：不要怀疑 AI 的能力，只是时间早晚的问题。

![](/data/attachment/album/202209/02/144556z1l2ck7l5743uyhc.jpg)

### 多达 140 万安装量的 Chrome 扩展会注入推广代码

五个提供群发 Netflix 视频、屏幕截图等各种服务的 Chrome 扩展程序被发现偷偷跟踪用户的浏览历史，并在用户访问的特定电子商务网站中插入跟踪代码。该代码修改了网站的 cookie，使扩展程序的作者能够收到购买物品的推广付款。谷歌已经移除了这些扩展程序，但已经安装了这些扩展程序的 140 万用户不会自动卸载，仍将受到影响。

> **[消息来源：ARS Technica](https://arstechnica.com/information-technology/2022/08/google-removes-extensions-that-covertly-tracked-users-and-injected-javascript/)**
>
> **老王点评**：安装浏览器扩展时需要小心，因为这些扩展的行为往往没有经过严格检查。

单值百分比图是饼图的一种，与基本饼图相比，单值百分比图适用于展示单个类别与总体的数据占比情况，不支持多数据系列的配置。本文介绍单值百分比图各配置项的含义。
**配置**
**图表尺寸**、**图表位置**、**旋转角度**和**透明度**可以参考组件指南的配置项说明。
* **饼图属性**
+
image::images/galaxee/single-value/column_chart.png[single-value_column_chart, 200, 200]
** **内半径**：饼图的内半径
** **外半径**：饼图的外半径
** **饼图填充底色**：没有数据的区域的填充色
* **扇形配置**
+
image::images/galaxee/single-value/column_chart_1.png[single-value_column_chart_1, 200, 200]
** **色值**：有数据的区域的填充色
** **动画持续时间**：饼图从起点逐渐增长直至末端的持续时间。
* **中间百分比系列**
+
image::images/galaxee/single-value/value_label.png[single-value_value_label, 200, 200]
** **文本样式**
+
[cols="1, 4"]
|===
|参数 |说明
|字体
|文本的字体样式
|文字粗细
|文本的文字粗细
|文字大小
|文本的文字大小
|文字颜色
|文本的文字颜色
|===
** **百分比符号**：中间百分比的显示符号
** **百分比位置**：中间百分比的位置，前置或后置
** **小数点位数**：中间百分比保留的小数点位数
**数据**
image:images/galaxee/single-value/data.png[single-value_data, 250]
上图中的示例JSON代码如下。
[source,javascript]
----
[
    {
        "value": 0.738
    }
]
----
***表1.字段说明***
[cols="1, 4"]
|===
|字段 |说明
|value
|扇形所占饼图的百分比，值为小数，1表示100%。
|===
***表2.配置项说明***
[cols="1, 4"]
|===
|配置项 |说明
|数据源类型
| 分为静态数据和SPL数据。可在配置数据中实时更改静态数据或动态SPL配置。
|数据响应结果
| 展示数据请求的响应结果。当数据源发生改变时，可实时查看数据响应结果。
|===

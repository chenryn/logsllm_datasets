User: How to check the system load and CPU usage on the Linux command line?
Assistant: The following commands can be used: 1. uptime: Displays the current load, running time and number of users of the system. 2. top: Displays the usage of processes in the system in real time, including CPU usage, memory usage, etc. 3. sar: Display the historical records of system resource usage, including CPU, memory, disk and other usage rates. 4. vmstat: Display the virtual memory usage of the system, including memory usage, CPU usage, etc. 5. ps: List the running processes and their resource usage, including CPU usage, memory usage, etc.

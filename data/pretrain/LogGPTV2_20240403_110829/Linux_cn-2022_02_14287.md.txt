### 微软新专利威胁到下一代 JPEG 格式

新的 JPEG XL 格式正准备取代已有30年历史的 JPEG 图像格式。JPEG XL 提供了三倍于 JPEG 的压缩率、相似的计算成本，并与 JPEG 兼容。此外，它还支持渐进式解码以及 HDR、alpha 通道、无损压缩和动画等特性。然而，微软的一项新专利可能对正在推广的 JPEG XL 构成威胁。该专利涉及一种名为 rANS 的数据编码技术的改进版本，这是非对称数字系统（ANS）系列的一个变种。尽管 ANS 的创造者已说服谷歌放弃与其相关的专利申请，但微软的新专利仍可能影响 JPEG XL 和 CRAM 项目，以及其他由 Meta、NVIDIA 等公司运营的开源项目。

> 老王点评：希望微软能够及时收手，不要让过去三十年的努力付之东流。

![](/data/attachment/album/202202/19/200555jnqtq7fxr77n9gfs.jpg)

### Steam Deck 正在测试对原生 Linux 游戏的支持

Steam Deck 基于 Linux 操作系统，其 Proton 兼容层允许原本仅支持 Windows 的游戏在 Linux 上运行，而无需单独移植。直到最近，Valve 一直倾向于使用 Proton，即使某些游戏已有原生 Linux 版本。[Valve 表示](https://www.pcgamer.com/valve-is-testing-native-linux-ports-for-the-steam-deck-in-addition-to-proton/) 它将优先测试游戏的原生 Linux 版本，只有当“兼容性测试失败或遇到重大问题”时才会通过 Proton 运行游戏。但也有人对此持不同意见，认为一些游戏的 Linux 版本不如 Windows 版本完善。

> 老王点评：我认为 Valve 对 Linux 推广所做的贡献堪比安卓。

![](/data/attachment/album/202202/19/200619egkl4dffdjwcoild.jpg)

### Project Zero 称 Linux 平台较竞争对手更安全

Project Zero 团队的一项新研究显示，Linux 平台比 Windows 和 macOS 更加安全。当然，这并不意味着 Linux 在所有方面都具有压倒性优势，因为人们在 Linux 中也发现了许多安全漏洞。不过，得益于维护平台的开发者们的积极修复态度，这些缺陷通常能得到迅速解决。他们发现修复 Linux 问题平均只需 25 天，现在通常都能在两周内完成。相比之下，苹果平均需要约 69 天、谷歌需 44 天、Mozilla 需 46 天左右。

> 老王点评：这表明，在内核社区的管理下，Linux 已经超越了专有企业。

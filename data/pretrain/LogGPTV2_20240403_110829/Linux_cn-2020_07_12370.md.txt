### Redis 作者辞去项目领导者和维护者职务

Redis 的创始人 antirez 决定将项目的领导权交给社区，并邀请了两位同事 Yossi Gottlieb 和 Oran Agra 继续负责维护。antirez 将成为 Redis Labs 顾问委员会的一员，为 Redis 的未来发展贡献自己的想法。他表示，从本质上讲，他更愿意作为一个糟糕的艺术家而不是一名优秀的程序员被人们记住。

来源：[开源中国](https://www.oschina.net/news/116838/antirez-stepped-down)

**拍一拍：**
此前，Python 之父 Guido van Rossum 已经卸下了 Python “仁慈独裁者”的身份。开源软件领袖也会感到疲惫，不愿意将一生都投入到一件看似永无止境的工作中。最近，Linus Torvalds 也表达了内核维护工作让他感到“无聊”，并指出内核社区的核心成员中五十岁以下的年轻人较少，而年轻一些的人尚不足以承担整个社区的管理责任。

### KDE 社区迁移至 GitLab

2019 年 11 月，KDE 社区宣布计划迁移到 GitLab。目前，该社区已顺利完成迁移的第一阶段，并加入了 GitLab 的开源计划。KDE 社区共同创建和维护了超过 200 个应用程序、无数插件、1000 多个存储库、80 多个 Qt 框架以及 2600 多个项目。在迁移过程中，KDE 面临的最大挑战是处理庞大的数据量及其与众多工具的集成问题。

来源：[开源中国](https://www.oschina.net/news/116835/kde-moving-to-gitlab)

**拍一拍：**
这是一个艰巨的任务，而且看起来 GitLab 在某些方面比 GitHub 更加灵活自由。

### GnuCash 4.0 发布

GNU 财务软件项目 GnuCash 通过邮件列表宣布发布了最新版本 GnuCash 4.0。新版本引入了一个独立的可执行文件 `gnucash-cli`，用于命令行操作，如更新账簿中的价格。

来源：[Solidot](https://www.solidot.org/story?sid=64808)

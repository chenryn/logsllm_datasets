**ATT&CK ID:** S0093  
**ATT&CK Technique Name:** Backdoor.Oldrea

The Backdoor.Oldrea payload collects detailed server information, including the following:
- CLSID (Class Identifier)
- Server name
- Program ID
- OPC (OLE for Process Control) version
- Vendor information
- Running state
- Group count
- Server bandwidth

This comprehensive data helps to identify the server's role within the control process.

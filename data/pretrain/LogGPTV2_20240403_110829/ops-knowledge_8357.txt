User: How to create a new user on a Linux server via command line tools?
Assistant: You can create a new user on the Linux server through the following steps: 1. Log in to the server as the root user or a user with sudo privileges. 2. Open a command line terminal window. 3. Enter the following command to create a new user, and enter the password and other information as prompted: ``` sudo adduser username ``` where username is the username of the new user you want to create. 4. If you need to add superuser privileges to a new user, you can add them to the sudo user group. Enter the following command to add a new user to the sudo user group: ``` sudo usermod -aG sudo username ``` where username is the username of the new user you want to add to the sudo user group. 5. Verify that the new user was created successfully. You can use the following command to view all users on the server: ``` cat /etc/passwd ``` Among them, each line represents a user's information, including user name, password, user ID, etc. 6. Log in to the new user account. You can use the following command to switch to the newly created user account: ``` su - username ``` where username is the username of the new user you want to switch to. Now, you have successfully created a new user on your Linux server and added sudo privileges to it. You can use this user account to perform corresponding operations.

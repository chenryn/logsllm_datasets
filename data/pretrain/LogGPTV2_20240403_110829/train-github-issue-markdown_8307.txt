Simply add anywhere in template the following:
`{{123456 | number}}`  
or  
`{{123456 | number:'4.2-2'}}`
In desktop browsers (Firefox, Chrome, IE10) the value is correctly
interpolated as:
    123,456
or
    123,456.00
but on mobile browsers on iPad iOS 9.2.1 (Chrome 47, Safari, Firefox 2.0) it
fails resulting in empty value.

User: How to use the Linux command line tool to query the port status of the remote server?
Assistant: You can use the Linux command line tool nc (netcat) or telnet to query the port status of the remote server. Use the nc command: Open the terminal and enter the following command: nc -zv server IP port number For example, to check whether port 80 of 192.168.1.100 is open, you can enter the following command: nc -zv 192.168.1.100 80 If port 80 is open, it will The following information is returned: Connection to 192.168.1.100 80 port [tcp/http] succeeded! If port 80 is not open, the following information is returned: nc: connect to 192.168.1.100 port 80 (tcp) failed: Connection refused Use the telnet command: Open the terminal and enter the following command: telnet server IP port number For example, to check whether port 80 of 192.168.1.100 is open, you can enter the following command: telnet 192.168.1.100 80 If port 80 is open, the following will be displayed: Trying 192.168.1.100 ... Connected to 192.168.1.100. Escape character is '^]'. If port 80 is not open, the following will be displayed: Trying 192.168.1.100... telnet: Unable to connect to remote host: Connection refused Otherwise In addition, there are some other command line tools that can be used to query the port status, such as nmap or tcpdump, etc.

User: What are the main procedures an organization should follow in order to implement CIS Control 13.9, Deploy Port-Level Access Control?
Assistant: An organization should Deploy port-level access control. Port-level access control utilizes 802.1x, or similar network access control protocols, such as certificates, and may incorporate user and/or device authentication.CIS Control 13.9 has the asset type Devices, and has the security function Protect.

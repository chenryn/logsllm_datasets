User: 请选择VersionControl领域相关问题的正确答案及解释简述Git Flow的分支类型和作用 ？

A: Git Flow 主要有 Master、Develop、Feature、Release、Hotfix 五种类型的分支。Master 分支存放稳定发布版本的代码；Develop 分支存放开发过程中的代码，是开发的主分支；Feature 分支是新功能开发的分支，开发完成后合并到Develop分支；Release 分支是准备要发布版本的分支，用于灰度测试和Bug修复；Hotfix分支是紧急修复线上Bug的分支，修复完成后需要合并到 Master 分支和 Develop分支。
B: Git Flow 只有 Master 和 Develop 两种分支，其他并没有特定的分支类型。
C: Git Flow 主要有 Master、Develop、Feature、Release、Hotfix 五种类型的分支。Master 分支存放开发过程中的代码；Develop 分支存放稳定发布版本的代码。
D: Git Flow 只有 Master 分支，所有的开发工作都在这一分支进行。

User: Please provide some common Linux commands and their functions, such as ls, ps, grep, awk, etc.
Assistant: 1. ls: List all files and subdirectories in the current directory. 2. cd: Change directory. 3. cat: display the contents of the file. 4. pwd: Display the current working directory. 5. rm: Delete files. 6. mkdir: Create a new directory. 7. rmdir: Delete a directory. 8. cp: Copy files. 9. mv: Move or rename files. 10. ps: Displays the currently running processes. 11. grep: Find matches in text files. 12. awk: A text processing tool for performing various operations on text files. 13. sed: A text processing tool for performing various substitutions and other operations in text files. 14. top: View system operating status and resource usage. 15. ifconfig: Display current network configuration and interface information. 16. ping: Test whether the network connection is available. 17. netstat: View network connection status and statistics. 18. ssh: Remote login and execute commands. 19. scp: Copy files between local and remote machines. 20. tar: pack and compress files and directories.

User: There is a server running Windows Server 2016 operating system. Now it is necessary to add a new storage device for data backup. Please provide a guide for adding a storage device in the Windows Server 2016 system.
Assistant: 1. Plug the storage device into an available USB or SATA port on the server 2. Open Server Manager 3. Click the Storage option on the left menu 4. Select Logical Units and click Add on the right Logical Unit Wizard" 5. Follow the instructions of the wizard to select the storage device to be used. You can choose to create a new logical unit or add the device to an existing logical unit 6. After completing the wizard, you can format and partition the newly added storage device, as well as set permissions and sharing options.

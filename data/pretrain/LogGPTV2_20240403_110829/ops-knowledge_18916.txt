User: There is a CentOS server, and nginx needs to be installed and configured to provide external services as a web server. Please provide a complete installation and configuration scheme.
Assistant: The following is a complete solution to install and configure nginx: 1. Update the system Use the command to update the system package: ``` sudo yum update ``` 2. Install Nginx Install nginx using the following command: ``` sudo yum install epel-release sudo yum install nginx ``` 3. Configure the firewall In order to allow HTTP and HTTPS traffic, the following firewall rules need to be enabled: ``` sudo firewall-cmd --permanent --add-service=http sudo firewall-cmd --permanent - -add-service=https sudo firewall-cmd --reload ``` 4. Start the Nginx service Run the following command to start nginx: ``` sudo systemctl start nginx ``` You can use the following command to add the nginx service to the startup item : ``` sudo systemctl enable nginx ``` 5. Configure the site Before configuring the site, you need to create a site directory. Create a directory with the following command: ``` sudo mkdir -p /var/www/example.com/html ``` Copy the site files to this directory: ``` sudo cp /path/to/site/index.html / var/www/example.com/html/ ``` Create nginx configuration file: ``` sudo nano /etc/nginx/conf.d/example.com.conf ``` Add the following: ``` server { listen 80; listen [::]:80; server_name example.com www.example.com; root /var/www/example.com/html; index index.html; location / { try_files $uri $uri/ =404; } } ``` Save and close the file. Then reload the Nginx configuration: ``` sudo systemctl reload nginx ``` Your site can now be accessed using your IP address or domain name, for example: ``` http://your_ip_address/ http://example.com/ ` `` Summary The above is a complete solution for installing and configuring Nginx on CentOS 7. After configuring and starting the Nginx server, you can host websites on your web server.

### 百度在 GitHub 上发布量子机器学习工具包

百度近期在 GitHub 上发布了其量子机器学习平台 Paddle Quantum 的工具包，旨在为开发者提供构建和训练量子神经网络模型的能力。该工具包基于百度的深度学习平台 PaddlePaddle 构建，涵盖了量子化学库、优化工具及三个量子应用：量子机器学习、量子化学模拟和量子组合优化。此外，百度还推出了七款新工具，并为 PaddlePaddle 引入了 27 项增强功能，其中包括一个名为 Paddle.js 的 JavaScript 库，它允许开发者在浏览器内或微信小程序中使用 AI。

来源：[zdnet](https://www.zdnet.com/article/baidu-releases-quantum-machine-learning-toolkit-on-github/)

> **硬核老王点评**：这是一个非常有意义的技术进步。专注于开发有用的技术是值得赞扬的。

### 德州仪器取消部分计算器编程支持以防止考试作弊

为了阻止学生规避考试模式下的限制，德州仪器公司已经移除了某些型号计算器的部分编程功能。这些设备包括广受欢迎的 TI-84 Plus CE、TI-83 Plus CE-T 和 TI-83 Premium CE。尽管用户仍然可以使用如 TI-BASIC 或 Python 等语言进行编程，但汇编或 C 语言程序已不再被支持。值得注意的是，在最新的固件更新后，目前没有方法能够恢复之前的功能。

来源：[cnBeta.COM](https://www.cnbeta.com/articles/tech/984039.htm)

> **硬核老王点评**：虽然理解这种做法背后的意图，但对于那些出于兴趣而非作弊目的探索编程的学生来说，这无疑是一种打击。不过现在入手一台可编程计算器应该还不晚吧？:D

### 谷歌 TAG 报告揭露印度“黑客雇佣”公司活动

谷歌威胁分析小组（TAG）在其首份季度报告中指出，多家总部位于印度的“黑客雇佣”公司正在创建虚假 Gmail 账户并冒充世界卫生组织 (WHO) 人员，针对多个国家内的金融服务、咨询和医疗保健行业领导者实施攻击。“这些行为反映了他们毫无底线地利用全球公共卫生危机谋取私利。”谷歌 TAG 负责人 Shane Huntley 如此评价道。

来源：[zdnet](https://www.zdnet.com/article/google-highlights-indian-hack-for-hire-companies-in-new-tag-report/)

> **硬核老王点评**：此类非法活动不仅损害了个人隐私权，也严重破坏了网络安全环境。必须提高警惕，防范来自印度及其他地区的国家级别黑客及其相关产业。

### Qt 5.15 LTS 正式推出

Qt 项目团队宣布正式推出 Qt 5.15 版本，这是 Qt 5 系列最后一个功能性更新版本，之后将重点转向即将于年底发布的 Qt 6。作为长期支持版本，Qt 5.15 将为商业用户提供为期三年的技术支持，并与此前所有 Qt 5.x 版本保持完全兼容性。此次发布主要集中在错误修复上，使其成为迄今为止最稳定的 Qt 5 发行版之一。

来源：[solidot](https://www.solidot.org/story?sid=64481)

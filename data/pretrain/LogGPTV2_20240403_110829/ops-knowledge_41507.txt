User: How to make an API request through cURL and save the request result to a file
Assistant: You can use the following command to send an API request using cURL and save the request result to a file: ``` curl -o <filename> -X <request method> -H <header> <API endpoint> ``` - use` The -o` parameter specifies the filename to save the response to. - Use the `-X` parameter to specify the request method. For example, use `-X GET` to send a GET request. - Use the `-H` parameter to specify the request header. For example, use `-H "Content-Type: application/json"` to send a request in JSON format. - Specify the URL of the API endpoint in the last parameter. Here is an example: ``` curl -o response.json -X GET -H "Authorization: Bearer YOUR_ACCESS_TOKEN" https://api.example.com/data ``` This command sends a GET request, setting the request header to Authorize header with access token and save the response to a file called `response.json`.

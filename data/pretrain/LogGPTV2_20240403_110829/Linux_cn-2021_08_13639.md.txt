### 得州仪器的新计算器将可以运行 Python 程序

![](/data/attachment/album/202108/01/090500b8zt8ux3pr8t9fxb.jpg)

尽管得州仪器每年从半导体业务中获得约 140 亿美元的收入，但其图形计算器依然是其最为人熟知的消费产品之一。最新款的 [TI-84 型号](https://www.prnewswire.com/news-releases/texas-instruments-new-ti-84-plus-ce-python-graphing-calculator-introduces-students-to-programming-301341889.html)允许用户通过按下“alpha”键来输入字母，并支持编写和运行 Python 程序。此外，该设备还配备了一个文件管理器，方便用户快速访问存储在计算器中的 Python 脚本。计算器还可以与计算机连接，实现数据传输、系统更新以及屏幕截图功能。

得州仪器认为，这款新计算器甚至有可能替代教室中的传统电脑，他们表示：“通过加入 Python 支持，我们希望让所有学生都能更容易地接触并学习编程，从而减少教师为教授这些重要技能而单独设置计算机实验室的需求。”

> 得州仪器的计算器简直强大得像一台微型计算机。

### FSF 认为 Copilot 是不可接受和不公正的

![](/data/attachment/album/202108/01/090526kzm2j46o437tib93.jpg)

由微软和 OpenAI 共同开发的 GitHub Copilot 工具能够基于大量代码库训练的 AI 模型提供自动完成建议。然而，这种做法是否侵犯了原始编码者的版权？开发者们对此表示担忧，尤其是在考虑到使用他们的软件训练神经网络是否构成合理使用的问题时。专有软件公司在他人劳动成果的基础上建立服务，是否存在根本性的不公平？

目前，自由软件基金会（FSF）呼吁对这些问题进行更深入的研究，并宣布将资助有关微软“GitHub Copilot”问题的论文撰写工作。

> 尽管 Copilot 可能在未来会进一步取代普通程序员的工作，但它仍然面临着一系列值得探讨的非技术性争议。

### YouTube-dl 事件后，GitHub 的 DMCA 程序现在包括免费法律帮助

![](/data/attachment/album/202108/01/090546kj5n1hjxrbgjrssn.jpg)

开源项目开发者经常发现自己处于《数字千年版权法案》（DMCA）的风口浪尖上，即便他们并没有违反任何规定，也往往不得不遵从删除请求。由于缺乏足够的资源，自由开发者或小型团队很难对抗 DMCA 请求。为此，GitHub 宣布与斯坦福大学法学院[合作](https://venturebeat.com/2021/07/27/github-offers-open-source-developers-legal-counsel-to-combat-dmca-abuse/)，为面临此类挑战的开发者提供法律援助。今后，每当 GitHub 向开发者发出有效的盗版通知时，都会同时告知他们可以选择申请免费的独立法律顾问服务。

> 过去，面对删除要求时，最简单且安全的做法通常是直接关闭代码仓库。现在，GitHub 提供了更多支持选项。

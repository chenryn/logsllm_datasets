|             |                                                        |
|             | 1)政策风险及对策                                       |
|             |                                                        |
|             | 软件与信息技术服务行业处于快速发展阶段，是国           |
|             | 家鼓励发展的新兴产业，公司未来能否持续成长，受到宏观经 |
|             | 济、产业政策、政府扶持等方面的影响，如相关政策因素对公 |
|             | 司持续成长发生不利变化，且公司未能及时采取措施积极应对 |
|             | ，将导致公司存在成长性下降或者不能达到预期的潜在风险。 |
|             |                                                        |
|             | 公司将实时关注并遵守国家相关政                         |
|             | 策、最新动态，及时调整公司发展目标和经营战略，并专注于 |
|             | 为用户提供实时搜索、关联分析、监控告警、多维统计和数据 |
|             | 可视化等服务，帮助企业进行运维监控、安全审计及业务数据 |
|             | 挖掘，缩短技术成果转化周期，减少政策变化所带来的风险。 |
|             |                                                        |
|             | 2)技术风险及对策                                       |
|             |                                                        |
|             | 经过近6年的积累，公司形成了较                          |
|             | 强的自主创新能力，建立了完备的技术体系和高质量的技术人 |
|             | 才队伍。一方面，公司积极围绕核心技术申请知识产权；另一 |
|             | 方面，公司通过建立完善的研发过程规范化管理、健全内部保 |
|             | 密制度、技术档案管理制度等相关措施，未来仍不能排除核心 |
|             | 技术流失的可能。在市场竞争中，一旦出现掌握核心技术的人 |
|             | 员流失、核心技术信息失密、专利管理疏漏，导致核心技术泄 |
|             | 密，公司技术创新、新产品开发、生产经营将受到不利影响。 |
|             |                                                        |
|             | 公司将紧密跟踪行业技术发展趋                           |
|             | 势，不断加强在现有技术基础上的前瞻性技术研究与储备，及 |
|             | 时跟踪前沿信息化技术，保持公司在重点应用领域的技术优势 |
|             | 。同时将继续通过完善薪酬设计、股权激励办法、塑造企业文 |
|             | 化等方式，提升员工队伍的凝聚力，防止核心技术人员流失。 |
|             |                                                        |
|             | 3）市场风险及对策                                      |
|             |                                                        |
|             | 公司的产                                               |
|             | 品的主要用户以IT化程度比较高的金融、电信运营商、能源企 |
|             | 业为主，其他行业为辅，目前金融、电信运营商、能源企业的 |
|             | IT需求市场还在扩大，但细分市场会越来越精细，如果细分领 |
|             | 域的市场竞争加剧、产品更新换代、新市场需求的培育等因素 |
|             | 均可能导致下游市场需求发生波动。如果未来公司现有主要产 |
|             | 品市场需求出现持续下滑或市场竞争加剧，同时公司未能及时 |
|             | 培育和拓展新的应用市场，将导致公司主营业务收入、净利润 |
|             | 面临下降的风险，盈利能力的稳定性和持续性也将受到考验。 |
|             |                                                        |
|             | 公司将加快新产品的开发进                               |
|             | 度和对行业前沿技术的研究与探索，增强市场应变能力，丰富 |
|             | 产品结构，同时进一步提高产品性能，降低产品成本，利用技 |
|             | 术、成本、品牌等优势提高产品综合竞争力，进一步完善市场 |
|             | 销售渠道，制定合理的销售价格策略，增强公司的盈利能力。 |
|             |                                                        |
|             | 4）管理风险及对策                                      |
|             |                                                        |
|             | 软件与信息技术服务是人才密集型行业，作                 |
|             | 为高新技术企业，公司核心技术人员的稳定性是公司赖于存在 |
|             | 和发展的基础。优秀的项目管理人员需要具有综合业务知识、 |
|             | 专业知识和协调管理能力，而技术人员则需要能够将专业知识 |
|             | 和信息化项目应用有效结合起来，并能在日常工作中发挥出信 |
|             | 息化的优势。项目管理人员和技术人员的素质、能力能力等对 |
|             | 项目顺利实施有着重要的影响，对公司的发展有一定的风险。 |
|             |                                                        |
|             | 公司多年来一直坚持                                     |
|             | 弘扬"以人为本，团结合作"的精神，增强企业的凝聚力，营造 |
|             | 和谐的工作环境，发挥每一位员工的创造力，把每一位员工的 |
|             | 成就汇聚为企业的成功，强调员工和公司共同发展，公司将完 |
|             | 善现有的薪酬制度，实施具有可操作性的人才引进、人才培养 |
|             | 、人才激励计划，在管理上坚持以人为本，使人才充分享受公 |
|             | 司发展带来的成果，与公司紧密结合在一起，实现共同发展。 |

> Firefox 正在不断取得进展，让我们一起看看最新的变化吧！

Mozilla Firefox 是一款优秀的开源多平台浏览器，以其出色的隐私保护功能而闻名。此外，它还是大多数 Linux 发行版的默认浏览器，因此成为 Linux 用户的首选。

### Firefox 当前状况
尽管 Firefox 一直以隐私优先著称，并且是谷歌 Chrome 的最佳开源替代品之一，但它仍然未能获得大众的广泛认可。根据 StatCounter 的数据，谷歌 Chrome 占据了 63.56% 的市场份额，而 Firefox 仅占 2.94%。

![](/data/attachment/album/202310/03/001920uofzb8ewwmy7qsaa.png)

为什么会出现这种情况呢？详细来看，Firefox 在 2021 年几乎失去了 5000 万用户。简而言之，Mozilla Firefox 已经尽其所能，但仍不足以吸引公众的注意。

### Firefox 用户体验的提升
幸运的是，在过去的两年中，Mozilla Firefox 专注于逐步提升用户体验，并更多地向公众展示其改进。例如，他们为 Firefox 117 提供了自动翻译功能支持。最近的更新简化了用户界面，并提供了更多使用 Firefox 的理由。

当然，Firefox 已经提供了诸如 Firefox Relay 等隐私特性，但并不是每个用户都关心这些底层的安全优化。许多用户希望浏览器能够无缝支持他们喜爱的网络服务，这是用户体验的核心部分。

### 谷歌服务在 Firefox 上的表现更好
好消息是，Mozilla 正在全力以赴做出改变。在最近的一篇博客文章中，他们宣布，谷歌 Meet 以及其他谷歌服务如 Workspace、Voice、YouTube、Keep、地图、翻译和相册在 Firefox 上运行得比以往任何时候都要好。

![](/data/attachment/album/202310/03/001921tmm6gvh8h71vvv7v.png)

他们与谷歌的 Workspace 团队紧密合作，确保兼容性问题得到解决。现在，只需使用 Firefox 115 或更高版本，你就可以在加入视频通话前后应用视效或背景。

![](/data/attachment/album/202310/03/001921mir899tigg8r0m9t.jpg)

许多网络用户都在使用谷歌的服务，如果他们在 Firefox 上的体验也很棒，那么 Firefox 就可能成为一个替代 Chrome 的选择。

**所以，谷歌的网络服务在 Firefox 上的体验得到了改进？这又是一个使用 Firefox 的理由。**

我认为，Mozilla 为了与谷歌 Chrome 及其他竞争对手保持竞争力，并增强用户对流行网络服务的体验，这么做是明智之举。

58.1 NVO3/4/NVO3_TUNNEL_DOWN
日志信息
NVO3/4/NVO3_TUNNEL_DOWN: The status of the vxlan tunnel changed to down.
(SourceIpAddress=[SrcIpAddr], DestinationIpAddress=[DstIpAddr],
TunnelStatus=[TnlStatus])
日志含义
VXLAN隧道状态变为Down。
日志参数
参数名称 参数含义
SourceIpAddress VXLAN隧道源端VTEP地址。
DestinationIpAddress VXLAN隧道远端VTEP地址。
TunnelStatus VXLAN隧道状态。
可能原因
原因1：终端用户下线。
原因2：网络拓扑变化，导致网络三层无法互通。
处理步骤
l 原因1：终端用户下线。
a. 请确认终端用户是否是正常下线。
n Y => 正常运行信息，无需处理。
n N => 请执行步骤2。
b. 请确认终端用户异常下线原因，故障解决后，查看告警是否仍然存在。如果
告警仍存在，则请执行步骤3。
c. 请收集日志信息和配置信息，并联系技术支持人员。
l 原因2：网络拓扑变化，导致网络三层无法互通。
a. 按实际组网拓扑检查设备间路由是否可达。
n Y => 请执行步骤3。
n N => 请执行步骤2。
b. 检查网络物理链路并正确修复。如果告警仍存在，则请执行步骤3。
c. 请收集日志信息和配置信息，并联系技术支持人员。
----结束

59.1 OPENFLOW/2/hwOFPSessionDown_active
日志信息
OPENFLOW/2/hwOFPSessionDown_active: Openflow connection is down. IP
Type=[OpenflowIpType], Local IP=[OpenflowLocalIp], Remote IP=[OpenflowRemoteIp],
VrfName=[VrfName], DatapathId=[DatapathId], AuxiliaryId=[AuxiliaryId],
SubReason=[SubReason], Reason=[Reason]
日志含义
OpenFlow连接中断。
日志参数
参数名称 参数含义
IP Type IP地址类型。
Local IP OpenFlow转发器的IP地址。
Remote IP OpenFlow控制器的IP地址。
VrfName OpenFlow所在的Vrf索引值。
DatapathId OpenFlow转发器的唯一标识。
AuxiliaryId 辅助连接ID。
SubReason OpenFlow连接中断的详细原因。
Reason OpenFlow会话Down的原因。
可能原因
原因1：TCP连接异常。
原因2：OpenFlow心跳检测报文超时。
原因3：OpenFlow会话控制器发送的报文非法或处理错误。
处理步骤
l 原因1：TCP连接异常。
a. 检查OpenFlow的控制器配置是否有变化，是否删除或更改了OpenFlow的配
置。
n 如果有变化，则请解决OpenFlow的配置问题。

n 如果没有变化，则请执行步骤2。
b. 在OpenFlow会话两端分别使用命令ping检查网络是否连通。
n 如果连通，则请执行步骤3。
n 如果不连通，则请解决网络连通问题。连通性问题解决后，请执行步骤
3。
c. 使用命令display sdn openflow session查看OpenFlow会话状态。
n 如果OpenFlow会话状态不是REGISTERED，则请执行步骤4。
n 如果OpenFlow会话状态是REGISTERED，则请执行步骤5。
d. 请收集告警信息、日志信息和配置信息，并联系技术支持人员。
e. 结束。
l 原因2：OpenFlow心跳检测报文超时。
a. 在OpenFlow会话两端分别使用命令ping检查网络是否连通。
n 如果连通，则请执行步骤2。
n 如果不连通，则请解决网络连通问题。连通性问题解决后，请执行步骤
2。
b. 使用命令display sdn openflow session查看OpenFlow会话状态。
n 如果OpenFlow会话状态不是REGISTERED，则请执行步骤3。
n 如果OpenFlow会话状态是REGISTERED，则请执行步骤4。
c. 请收集告警信息、日志信息和配置信息，并联系技术支持人员。
d. 结束。
l 原因3：OpenFlow会话控制器发送的报文非法或处理错误。
a. 请收集告警信息、日志信息和配置信息，并联系技术支持人员。
----结束

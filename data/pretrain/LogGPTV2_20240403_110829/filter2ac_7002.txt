**ATT&CK ID:** C0025  
**ATT&CK Technique Name:** 2016 Ukraine Electric Power Attack

In the 2016 Ukraine Electric Power Attack, the Sandworm Team added a login to a SQL Server using the `sp_addlinkedsrvlogin` stored procedure. This technique allowed them to create a linked server and configure the security context for remote connections, thereby facilitating further malicious activities.

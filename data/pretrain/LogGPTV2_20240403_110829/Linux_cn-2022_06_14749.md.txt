### 外媒报道中国芯片行业高速发展

根据最新数据显示，在过去四个季度中，全球增长最快的20家芯片公司中有19家位于中国。据中国半导体行业协会的数据，2021年中国芯片制造商和设计公司的总销售额增长了18%，达到了创纪录的超过1万亿元人民币。中芯国际和华虹半导体是中国最大的合同芯片制造商，其中中芯国际最近报告其季度销售额增长了67%，超过了GlobalFoundries和台积电。视频芯片设计公司上海富瀚微电子的收入平均增长了37%，并计划进入电动汽车和人工智能领域。集成电路软件工具开发商上海概伦电子在过去四个季度的平均销售额翻了一番，并声称其软件可用于制造3纳米芯片。

> **消息来源**: [彭博社](https://www.bloomberg.com/news/articles/2022-06-20/us-sanctions-helped-china-supercharge-its-chipmaking-industry)

**老王点评**: 尽管美国的制裁一度压制了中国企业的增长，但也促进了中国芯片产业的发展。

### Rust 有可能在 Linux 5.20 中合并

在近日举行的开源峰会上，Linus Torvalds 讨论了在 Linux 内核中使用 Rust 的可能性，并表示它可能很快就会被引入，甚至可能是在下一个内核周期。尽管目前还没有任何关于 Rust for Linux 的拉取请求被发送和合并，但许多初始的 Rust 启用代码已经准备就绪。上个月发布了最新的 Rust for Linux 内核补丁，使更多功能得以实现，并完成了更多的审查。

> **消息来源**: [Phoronix](https://www.phoronix.com/scan.php?page=news_item&px=Rust-For-Linux-5.20-Possible)

**老王点评**: Rust 支持是值得期待的特性之一，但距离 Rust 在内核中的广泛应用可能还有很长的路要走。

![](/data/attachment/album/202206/23/202206aco68vx8r7lkcfgw.jpg)

### 内核开发者仍在实验 -O3 级优化

Linux 内核的编译通常采用 `-O2` 级别的优化。虽然采用更高级别的 `-O3` 编译可以带来更高的性能，但由于内核的复杂性和庞大的代码库，这可能导致低级别的内核代码出现错误或不正确的行为。因此，采用 `-O3` 编译的内核并没有得到广泛认可。一些内核开发者甚至不希望看到它成为 Kconfig 选项。然而，仍有一些爱好者对 `-O3` 优化内核以获得更高性能感兴趣，或者试图发现 Linux 内核代码和编译器本身的优化错误。最近有开发者发出一组补丁，使其可以在所有架构上使用该优化级别。

> **消息来源**: [Phoronix](https://www.phoronix.com/scan.php?page=news_item&px=O3-Optimize-Kernel-2022-Patches)

**老王点评**: 上游的 Linux 内核开发者对新的编译器功能通常是保守的，只是为了提高性能而引入新功能可能会导致微妙且难以检测的问题。但我认为作为实验性功能进行探索是有价值的。

**ATT&CK ID:** M1054  
**ATT&CK Technique Name:** Software Configuration

**Recommendations:**
- **Allowlist MIB Objects and Implement SNMP Views:** Configure your network devices to allow only specific Management Information Base (MIB) objects. Additionally, implement Simple Network Management Protocol (SNMP) views to restrict access to sensitive information.
- **Disable Smart Install (SMI):** If the Smart Install feature is not in use, it is recommended to disable it to reduce potential attack vectors.

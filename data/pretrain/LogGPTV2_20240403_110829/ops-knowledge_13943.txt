User: How to install an Apache server on a Linux system to access static web pages in a web browser?
Assistant: The following are the steps to install the Apache server on the Linux system and access static web pages in a web browser: 1. Open the terminal and enter the following command to install the Apache server: ``` sudo apt-get update sudo apt-get install apache2 ``` 2. After the installation is complete, run the following command to start Apache: ``` sudo systemctl start apache2 ``` 3. Confirm that the Apache server has been started successfully, and you can enter the IP address of the server in a web browser to access it. 4. Create a static web page under the default root directory `/var/www/html/` of the Apache server, such as `index.html`, and enter the following content: ``` <!DOCTYPE html> <html> <head> <title>Welcome to my website</title> </head> <body> <h1>Hello World!</h1> </body> </html> ``` 5. Access the IP of the server in a web browser Address, you can see that the static web page just created can be accessed normally.

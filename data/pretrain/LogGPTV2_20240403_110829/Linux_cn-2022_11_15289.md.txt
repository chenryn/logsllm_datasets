### 谷歌的秘密项目教 AI 编写和修改代码

一个名为 Pitchfork 的秘密项目现已转至谷歌实验室。该项目旨在通过机器学习技术训练 AI 自动编写、修复 Bug 以及更新代码。Pitchfork 的初步目标是构建一种工具，能够将谷歌的 Python 代码库升级到最新版本。随着时间的发展，项目目标扩展为开发一个通用系统，以减少人类开发者编写及维护代码的需求，同时保持高质量标准。

> **消息来源**：[Business Insider](https://www.businessinsider.com/google-ai-write-fix-code-developer-assistance-pitchfork-generative-2022-11)

**老王点评**：我相信未来大部分编程工作都将由计算机自行完成。

### 谷歌安全团队称安卓手机厂商未能及时修补漏洞

据谷歌 Project Zero 团队透露，在今年 6 月和 7 月间发现了五个新的安全问题，并已通知 ARM 公司。ARM 在随后的 7 月和 8 月发布了相关修复补丁。然而，直到三个月后，包括三星、小米、Oppo 和谷歌在内的主要智能手机制造商仍未部署这些补丁来解决上述漏洞。利用这些漏洞，恶意黑客可以绕过安卓系统的权限模型，从而获得对设备的完全控制权。

> **消息来源**：[Engadget](https://www.engadget.com/best-black-friday-tv-deals-2022-144506723.html)

**老王点评**：仅靠上游供应商的努力不足以彻底解决安全问题。

### GitHub 正在测试在 Markdown 文件中嵌入 JavaScript

GitHub 目前正在内测一项新功能——允许在其仓库内的 Markdown 文件中嵌入 JavaScript 代码，这一特性被称为 “GitHub Blocks”。通常情况下，Markdown 主要用于撰写代码仓库的文档说明。通过集成 JavaScript 并调用 GitHub 提供的 RESTful API，可以使原本静态的 Markdown 文档具备动态交互能力。例如，用户可以使用该功能渲染字体样式、显示 JSON 数据表、呈现仓库分析报告（如贡献者列表、问题统计等）、绘制图表或演示应用程序的功能。

> **消息来源**：[DevClass](https://devclass.com/2022/11/24/github-blocks/)

**老王点评**：这是否意味着 Markdown 正逐渐偏离其最初作为简单标记语言的设计理念呢？

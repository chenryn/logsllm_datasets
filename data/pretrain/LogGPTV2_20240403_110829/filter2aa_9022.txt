# Jira Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action |
| --- | --- |
| `G` `D` | Go to Dashboard |
| `G` `P` | Browse to a Project |
| `G` `A` | Go to Agile |
| `G` `I` | Go to Issue Navigator |
| `G` `G` | Administration Search Dialog Box |
| `/` | Quick Search |
| `C` | Create an Issue |
| `?` | Open Shortcut Help |
| `Ctrl` `Alt` `D` | Dashboards Drop-down Menu |
| `Ctrl` `Alt` `P` | Projects Drop-down Menu |
| `Ctrl` `Alt` `I` | Issues Drop-down Menu |
| `Ctrl` `Alt` `G` | Agile Drop-down Menu |
| `Ctrl` `Alt` `S` | Form Submit |
| `Ctrl` `Alt` `\` | Cancel Form |

## Agile Shortcuts

| Shortcut | Action |
| --- | --- |
| `J` | Next Issue |
| `K` | Previous Issue |
| `N` | Next Column |
| `P` | Previous Column |
| `-` | Expand/Collapse All Swimlanes |
| `T` | Hide/Show Detail View |
| `O` | View Issue |
| `Enter` | Add a New Line or Submit Changes |
| `Esc` | Cancel Changes |
| `I` | Assign to Me |
| `S` `T` | Send to Top |
| `S` `B` | Send to Bottom |
| `E` | Edit Issue |
| `Z` | Projector Mode |

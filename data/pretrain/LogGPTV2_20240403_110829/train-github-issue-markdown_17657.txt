  * OpenCV => 4.1.0

##### Detailed description
The build of `opencv_highgui` fails with several errors of the form `X already
defined in window.obj`. The other projects compile successfully.
##### Steps to reproduce
  1. Clone the `opencv` and `opencv_contrib` repos, checkout `4.1.0` in both.
  2. Run CMake, enable `WITH_CUDA`, `OPENCV_ENABLE_NONFREE` and set `OPENCV_EXTRA_MODULES_PATH` to the `opencv_contrib\modules` directory.
  3. Generate the solution.
  4. Open the solution in Visual Studio 2017 and launch the build.

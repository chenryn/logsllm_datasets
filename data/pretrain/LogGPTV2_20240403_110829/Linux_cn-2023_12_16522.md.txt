### 1. GitHub 正式发布 Copilot 编程助手
GitHub 的 Copilot Chat 聊天机器人功能于 7 月份推出了有限的测试版，并在 9 月份面向个人开发者推出了测试版。从今天起，Copilot Chat 可在 VS Code 和 Visual Studio 的侧边栏中使用。作为 GitHub Copilot 付费计划的一部分，它还免费提供给经过验证的教师、学生和某些开源项目的维护者。该聊天机器人由 GPT-4 支持，并专门针对开发场景进行了微调。自测试版发布以来，Copilot Chat 的其他方面几乎没有变化。

> **[消息来源：Tech Crunch](https://techcrunch.com/2023/12/29/github-makes-copilot-chat-generally-available-letting-devs-ask-questions-about-code/)**
>
> **老王点评**：试用了一下，感觉还不错，帮我优化了两段代码。AI 在降低编程门槛方面的作用越来越重要。

### 2. 迪士尼、华纳、康卡斯特和派拉蒙等考虑合并
全球最大的传统娱乐公司——迪士尼、华纳兄弟、康卡斯特和派拉蒙，在去年流媒体业务亏损超过 50 亿美元后，正在考虑削减开支并可能进行合并，以更好地与 Netflix 竞争。尽管这些公司在传统媒体领域苦苦挣扎，但十多年前开创流媒体模式的 Netflix 已经成为这场视频分销变革中的赢家。Netflix 最近一个季度的盈利飙升，新增了 900 万订户，超过了华尔街的预期。

> **[消息来源：Ars Technica](https://arstechnica.com/culture/2023/12/its-shakeout-time-as-losses-of-netflix-rivals-top-5-billion/)**
>
> **老王点评**：每个时代都有一些被淘汰的巨头。

### 3. 人们越来越少在社交媒体上发帖
根据一项研究，虽然每月有数十亿人使用社交媒体，但真正发帖的人却越来越少，他们更倾向于被动地浏览内容。调查显示，61% 拥有社交媒体账户的美国成年受访者表示，他们对自己发布的内容变得更加挑剔。这种潜水者心态普遍存在于 Instagram、Facebook 以及 X 和 TikTok 等平台。然而，Instagram 负责人亚当·莫塞里（Adam Mosseri）持有不同观点：“人们在 Feed 上的分享越来越少，但在 Stories 上的分享却越来越多。与其说人们减少了分享，不如说他们在以不同的方式分享。”

*（插图：DA/a80ac39e-aca6-472e-b515-e0c6c5dcb27b）*

> **[消息来源：MSN](https://www.msn.com/en-us/news/technology/we-aren-t-posting-on-social-media-as-much-anymore-will-we-ever/ar-AA1lXyIS)**
>
> **老王点评**：主动创作难，但对别人的内容进行互动则相对容易。

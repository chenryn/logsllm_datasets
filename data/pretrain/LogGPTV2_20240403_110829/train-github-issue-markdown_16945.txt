### Environment
- **Windows Build Number:** 10.0.18362.10019
- **Windows Terminal Version (if applicable):** Windows Terminal (Preview) Version: 0.5.2681.0
- **Other Software:** None

### Steps to Reproduce
1. Open the Windows Terminal.
2. Drag and select some text within the terminal.
3. Right-click the selected text.

### Expected Behavior
- The terminal should not exit.
- The selected text should be pasted at the cursor position.

### Actual Behavior
- The terminal window closes without any error message or warning.

### Additional Information
- This issue occurs consistently every time the steps are followed.
- No other software is running that might interfere with the terminal's operation.

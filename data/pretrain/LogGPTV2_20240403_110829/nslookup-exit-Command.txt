### nslookup /exit

**Description:**
Exits the `nslookup` command-line tool.

**Syntax:**
```
nslookup /exit
```

**Parameters:**

- **/exit**: Exits the `nslookup` command-line tool.
- **/? or /help**: Displays help information at the command prompt.

**Usage:**
To exit the `nslookup` tool, simply type `nslookup /exit` and press Enter. For help on using `nslookup`, you can use either `/?` or `/help` as parameters.

excerpt: |
  Linux 在我们的生活中无处不在：从家庭到公司，从大学到实验室，甚至在太空空间站中都能看到它的身影。然而，在过去，它并不像现在这样成为计算机发展的核心力量。这一切都始于一位芬兰大学生的个人爱好。如今，Linux 已经从一个个人项目发展成了一场计算机革命。本文将向您展示最完整的 Linux 23 年发展历程。

  **1991**
  - 8月25日：21岁的芬兰学生 Linus Benedict Torvalds 在 comp.os.minix 新闻组上宣布他正在编写一个免费的操作系统。
  - 9月1日：Linux 0.01 版本在网上发布。

  **1992**
  - 1月5日：Linux v0.12 发布版本的内核以 GNU GPL 协议重新发布。原来的许可证禁止商业用途。通过这次协议变更，发布和出售修改或未修改版的 Linux 成为了可能，只要你将这些复制版本以相同的 GPL 许可证发布，并且有相对应完整的源代码。Linus 后来在一次采访中提到：“让 Linux 遵守 GPL 绝对是我干过的最正确的事。”
  - 1月29日：Andrew S. Tanenbaum 向 comp.os.minix 邮件列表发送了一封名为 “LINUX is obsolete” 的邮件。这场争论主要围绕 Linux 和内核架构展开，Tanenbaum 认为微内核比宏内核更先进，因此认为 Linux 是过时的。
  - 4月5日：第一个 Linux 新闻组 comp.os.linux 由 Ari Lemmke 提议并开通。
  - 5月21日：Peter MacDonald 发布了第一个独立的 Linux 安装包 SLS。该安装包可以通过软盘安装，包括先进的 TCP-IP 网络支持和 X Window 系统。建议至少预留 10MB 的磁盘空间来安装。

  **1993**
  - 6月17日：Patrick Volkerding 发布了 Slackware Linux。Slackware 被认为是第一个取得广泛成功的 Linux 发行版，至今仍在使用。
  - 8月16日：Ian Murdock（Debian 中的 'ian'）发布了第一个 Debian Linux 发行版。Debian 是最有影响力的 Linux 发行版之一，是 MEPIS、Mint、Ubuntu 和许多其他发行版的基础。
  - 8月19日：Matt Welsh 撰写的《Linux Installation and Getting Started》第一版出版，这是第一本关于 Linux 的书籍。

  **1994**
  - 3月14日：Linux 内核 V1.0 发布。它支持基于 i386 单处理器的计算机系统。在这三年里，内核代码库已经增长到了 176,250 行。
  - 3月26日：第一期《Linux Journal》杂志发行。这一期杂志的特点是发表了一篇对 Linus Torvalds 的采访以及 Phil Hughes、Robert “Bob” Young、Michael K. Johnson、Arnold Robbins、Matt Welsh、Ian A 等人的文章。
  - 8月15日：Willian R. Della Croce, Jr. 申请了“Linux”商标，并于9月进行了注册。Della Croce 在不知道 Linux 社区财政状况的情况下，向著名的“Linux 公司”写信要求支付“Linux”商标使用费。直到 1997 年，这次风波以将商标转让给代表所有请愿者和 Linux 用户的 Linus Torvalds 而告终。

[我们周围到处都有 Linux 的身影](http://linux.cn/article-2480-1.html)，无论是在家中、公司、大学、实验室，还是在太空空间站。但在过去，它并没有像现在这样成为计算机发展的核心力量。这一切都始于一位芬兰大学生的个人爱好。如今，Linux 已经从一个个人项目发展成了一场计算机革命。在这篇文章中，我们将向您展示最完整的 Linux 23 年发展历程。

### 1991
**8月25日**：21岁的芬兰学生 Linus Benedict Torvalds 在 comp.os.minix 新闻组上宣布他正在编写一个免费的操作系统。
**9月1日**：Linux 0.01 版本在网上发布。

### 1992
**1月5日**：Linux [v0.12 release](https://www.kernel.org/pub/linux/kernel/Historic/old-versions/RELNOTES-0.12) 版本的内核重新以 GNU GPL 协议发布。原来的许可证禁止任何商业用途。通过这次协议变更，发布和出售修改或未修改版的 Linux 成为了可能，只要你将这些复制版本以相同的 GPL 许可证发布，并且有相对应完整的源代码。在后来的一次[采访](http://j.mp/fs-pragmatist)中，Linus 对这次许可证的更改说了这样一句话：
> "让 Linux 遵守 GPL 绝对是我干过的最正确的事。"

**1月29日**：Andrew S. Tanenbaum 向 comp.os.minix 邮件列表发送了一封名为 [LINUX is obsolete](https://en.wikipedia.org/wiki/Tanenbaum%E2%80%93Torvalds_debate) 的邮件。总的来说，这次被一些人升级为“战火”的争论是关于 Linux 和内核架构的。Tanenbaum 争辩说微内核比宏内核更加高级，所以 Linux 是过时的。
**4月5日**：第一个 Linux 新闻组 comp.os.linux 由 Ari Lemmke 提议和开通。
**5月21日**：Peter MacDonald 发布了第一个独立的 Linux 安装包 SLS。可以通过软盘安装，包括先进的 TCP-IP 网络支持和 X Window 系统。建议至少预留 10MB 的磁盘空间来安装。

### 1993
**6月17日**：Patrick Volkerding 发布了 Slackware Linux。[Slackware](http://www.slackware.com/) 被认为是第一个取得广泛成功的 Linux 发行版，而且它现在还在使用。
**8月16日**：Ian Murdock（Debian 中的 'ian'）发布了第一个 Debian Linux 的发行版。Debian 是最有影响力的 Linux 发行版之一，是 MEPIS、Mint、Ubuntu 和很多其他发行版的鼻祖。
**8月19日**：Matt Welsh 撰写的《Linux Installation and Getting Started》第一版出版，这是第一本关于 Linux 的书籍。

### 1994
**3月14日**：[Linux 内核 V1.0](https://www.kernel.org/pub/linux/kernel/v1.0/) 发布。它支持基于 i386 单处理器的计算机系统。这三年来，内核代码库已经增长到了 176,250 行。
**3月26日**：第一期《Linux Journal》杂志发行。这一期杂志的特点是发表了一篇对 Linus Torvalds 的采访和一些 Phil Hughes、Robert “Bob” Young、Michael K. Johnson、Arnold Robbins、Matt Welsh、Ian A 等人的文章。
**8月15日**：Willian R. Della Croce, Jr. 申请了“Linux”商标，并于9月进行了注册。Della Croce 在不知道 Linux 社区财政窘迫的前提下，向煊赫的“Linux 公司”写了一封信，要求他们为“Linux”商标使用支付费用。直到 1997 年，这次风波以将商标转让给代表所有请愿者和 Linux 使用者的 Linus Torvalds 而告终。

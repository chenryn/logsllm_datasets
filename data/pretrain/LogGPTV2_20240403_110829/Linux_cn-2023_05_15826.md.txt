### 引言
如果你是 Fedora 的粉丝，那么你可能会喜欢 risiOS 提供的调整和开箱即用的工具。

![risiOS](/data/attachment/album/202305/18/223854kz2zfrtptm1k8itr.jpg)

我们已经介绍了大量基于 Arch 和 Ubuntu 的发行版。那么基于 Fedora 的发行版呢？为什么你需要它？

risiOS 是 [最佳基于 Fedora 的发行版之一](https://itsfoss.com/best-fedora-linux-distributions/?ref=news.itsfoss.com)，旨在让用户体验更加轻松。虽然 Fedora 本身并不复杂，但在首次设置时，一些细节可能会让人感到困惑。

### risiOS 概述

![risiOS 主屏幕](/data/attachment/album/202305/18/223922qahqzqcggqvgsgha.jpg)

与原生 Fedora 相比，你不应期望有巨大的变化。相反，risiOS 主要专注于通过某些 GUI 工具和功能来简化用户体验。例如，你会看到一个欢迎屏幕，提醒你在安装 Fedora 后需要执行的基本步骤。这不仅提供了信息，还帮助自动完成一些重要的初始设置。

当然，你还会注意到与原生 Fedora 不同的品牌、强调色和新的默认壁纸。

**risiOS 还有哪些不同之处？** 让我强调一下其余的优点：

- **欢迎应用**：提供快速操作（使用 risiScripts）以便轻松设置。
- **risiTweaks 应用**：用于自定义 GNOME 体验。
- **Chromium 网络浏览器**：而不是 Firefox。
- **ONLYOFFICE 桌面编辑器**。
- **源自 Linux Mint 的 Web 应用功能**。

> risiOS 是一个相对较新的发行版。无论你是想试用还是日常使用，都取决于你自己。

### 初步印象

请记住，你正在安装的是带有开箱即用实用程序的 Fedora，这些实用程序可帮助你快速进行设置。因此，在安装完 risiOS 后，你会看到相同的 Fedora 初始步骤，如选择用户名和调整隐私设置。

通过后，你会注意到“risiWelcome”应用弹出窗口。

![risiOS 欢迎应用](/data/attachment/album/202305/18/223922higsc51c5cgpigs0.jpg)

它列出了所有可以通过单击执行的重要操作。为了实现这一目标，该发行版依赖于其特色之一——“risiScripts”。

![risiOS Flathub 设置](/data/attachment/album/202305/18/223923wyzt76ds38778738.jpg)

每当你单击一个操作来设置 Flathub 或 RPMFusion 时，它都会提示你运行脚本。随着你的继续，这个过程变得自动化，你可以毫不费力地实现你的目标。

同样，你可以向下滚动以探索其他选项，例如使用“risiTweaks”调整用户体验：

![risiTweaks](/data/attachment/album/202305/18/223923gtcvnnrz83nzi8y5.jpg)

你可以使用此应用控制要自定义的内容，包括动画设置、强调色、主题调整、字体、背景徽标定制等。

正如你所注意到的，risiOS 内置了一些 [GNOME 扩展](https://itsfoss.com/best-gnome-extensions/?ref=news.itsfoss.com)。你可以在此处使用相同的应用禁用或配置它们。

![risiOS](/data/attachment/album/202305/18/223924m8tp1ut1kwlout8o.jpg)

如果你探索 risiScript 菜单，你会找到安装程序来帮助你安装 Brave、Edge、Chrome、Opera 或 Vivaldi 浏览器。默认情况下，你会通过 risiOS 获得 Chromium。

此外，如果你是一个动手达人，你可以用 [Fedora Nobara](https://nobaraproject.org/?ref=news.itsfoss.com)（游戏发行版）附带的 Fsync 内核替换原装内核。

因此，risiOS 对所有类型的用户来说似乎都是一个有趣的选择。

最重要的是，risiOS 包含一些可能会增强你的用户体验的额外功能：

- 预装 ONLYOFFICE 桌面编辑器
- 来自 Linux Mint 的 Web 应用

对于那些喜欢 ONLYOFFICE 作为文档编辑器的用户，risiOS 已经为你准备好了。而来自 Linux Mint 的“Web 应用”对于用户来说是一个不错的选择，可以将网站作为应用运行，并可以在其发行版中默认使用相同应用。

![risiOS Web 应用](/data/attachment/album/202305/18/223924ztifydd4vu4zfidd.jpg)

对于坚持使用网络应用，并且主要使用云来完成所有任务的用户来说，这是一个很好的补充。

### 尝试 risiOS

你可以前往其官方网站下载最新可用的 ISO 文件。我们建议你先在虚拟机或备用系统上尝试新发行版，然后再将其安装到主系统上。

目前，它基于 [Fedora 37](https://news.itsfoss.com/fedora-37-features/)。你可以期待他们接下来发布基于 Fedora 38 的 ISO。

# 常见环境英文缩写对照表

## 环境缩写与定义
| 缩写    | 英文全称                  | 中文释义               | 使用场景                     |
|---------|---------------------------|------------------------|------------------------------|
| DEV     | Development               | 开发环境               | 代码开发调试                 |
| SIT     | System Integration Test   | 系统集成测试环境       | 联调联测                     |
| UAT     | User Acceptance Test      | 用户验收测试环境       | 客户验证                     |
| PET     | Performance Evaluation Test | 性能评估测试环境       | 压力测试                     |
| SIM     | Simulation                | 仿真环境               | 模拟生产数据/流程            |
| PRD/PROD | Production               | 生产环境               | 对外提供服务                 |
| PRE/PPE | Pre-Production           | 预生产环境             | 上线前最终验证               |

## 环境层级体系详解

### 1. 开发环境（DEV）
- **核心特征**：  
  - 开发人员专属  
  - 版本快速迭代  
  - 本地/容器化部署  
  - 无外部依赖约束  

### 2. 测试环境（SIT/UAT/PET）
- **功能划分**：
  - **SIT**：系统间接口联调
  - **UAT**：业务流程验证
  - **PET**：高并发压力测试
- **典型配置**：  
  - 隔离网络环境  
  - 模拟生产数据集  
  - 自动化测试框架支持  

### 3. 预生产环境（PRE）
- **关键特性**：  
  - 1:1复现生产架构  
  - 外部服务沙箱化对接  
  - 灰度发布支持  
  - 监控告警全量覆盖  
- **特殊价值**：  
  - 验证环境差异风险  
  - 应急预案实战演练  

### 4. 生产环境（PROD）
- **运行标准**：  
  - 多可用区容灾  
  - 实时监控告警  
  - 访问控制严格  
  - 自动伸缩策略  
- **变更规范**：  
  - 变更窗口限制  
  - 回滚机制完备  
  - 影响范围评估  

## 环境演进流程图
```
DEV → SIT → UAT → PET → PRE → PROD
```

### 文件保存与导出功能

#### 保存文件
- **保存 (Ctrl+S)**
  - 如果您已经保存过文件，此选项将变为灰色不可选。
- **另存为 (Shift+Ctrl+S)**
  - 您可以选择将当前文件保存为另一个文件。这将弹出一个“另存为”对话框（参见第 5.3.1 节 “保存捕获文件为”对话框）。

**注意：**
- 动态捕捉的文件在结束捕捉前不能保存。必须先停止捕捉才能进行保存操作。

#### 文件集合
- **文件集合 > 列表文件**
  - 显示已打开文件的列表。将会弹出一个对话框显示这些文件（参见第 5.5 节 “文件集合”）。
- **文件集合 > 下一个文件**
  - 如果当前加载的文件是文件集合的一部分，该选项将跳转到下一个文件。如果不是，则会跳转到最后一个文件，并且该选项会变为灰色。
- **文件集合 > 上一个文件**
  - 如果当前文件是文件集合的一部分，该选项将跳转到前一个文件。如果不是，则会跳转到文件集合的第一个文件，并且该选项会变为灰色。

#### 导出文件
- **导出 > 作为纯文本文件**
  - 将捕获文件中的所有或部分数据包导出为 ASCII 文本格式。将弹出 Wireshark 导出对话框（参见第 5.6.1 节 “导出为纯文本文件”对话框）。
- **导出 > 作为 PostScript 文件**
  - 将捕获文件的全部或部分导出为 PostScript 格式。将弹出导出对话框（参见第 5.6.2 节 “导出为 PostScript 文件”对话框）。
- **导出 > 作为 CSV 文件**
  - 将捕获文件的全部或部分摘要导出为 CSV 格式（适用于电子表格）。将弹出导出对话框（参见第 5.6.3 节 “导出为 CSV 文件”对话框）。
- **导出 > 作为 PSML 文件**
  - 将捕获文件的全部或部分导出为 PSML 格式的 XML 文件。将弹出导出对话框（参见第 5.6.4 节 “导出为 PSML 文件”对话框）。
- **导出 > 作为 PDML 文件**
  - 将捕获文件的全部或部分导出为 PDML 格式的 XML 文件。将弹出导出对话框（参见第 5.6.5 节 “导出为 PDML 文件”对话框）。
- **导出 > 选定的数据包字节**
  - 将当前在数据包字节面板中选择的字节导出为二进制文件。将弹出导出对话框（参见第 5.6.6 节 “导出选定的数据包字节”对话框）。

#### 打印
- **打印 (Ctrl+P)**
  - 打印捕获文件的全部或部分内容。将弹出打印对话框（参见第 5.7 节 “打印数据包”）。

#### 退出
- **退出 (Ctrl+Q)**
  - 退出 Wireshark。如果未保存文件，Wireshark 会提示是否保存。

### 跳转菜单
Wireshark 的“跳转”菜单包含以下项目：

| 菜单项 | 快捷键 | 描述 |
| --- | --- | --- |
| 返回 | Alt+Left | 跳转到最近浏览的数据包，类似于浏览器中的页面历史记录。 |
| 前进 | Alt+Right | 跳转到下一个最近浏览的数据包，类似于浏览器。 |
| 跳转到指定数据包 (Ctrl+G) | 无 | 打开一个对话框，输入指定的数据包序号，然后跳转到对应的数据包（参见第 6.8 节 “跳转到指定的数据包”）。 |
| 跳转到对应的应答数据包 | 无 | 跳转到当前数据包的应答数据包。如果不存在，该选项将变为灰色。 |
| 上一个数据包 (Ctrl+Up) | 无 | 移动到数据包列表中的前一个数据包，即使数据包列表面板不是当前焦点也是可用的。 |
| 下一个数据包 (Ctrl+Down) | 无 | 移动到数据包列表中的后一个数据包，同上。 |
| 第一个数据包 | 无 | 移动到列表中的第一个数据包。 |
| 最后一个数据包 | 无 | 移动到列表中的最后一个数据包。 |

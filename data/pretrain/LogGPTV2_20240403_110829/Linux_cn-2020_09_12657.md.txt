### 一个无聊的开发者创造了“DOS 的 Linux 子系统”

一位来自墨尔本的软件工程师在城市封锁期间，开发了一个类似于 Windows 10 中集成 Linux 环境的新工具。该工具允许用户从 DOS 命令行启动 Linux，并且在 Linux 运行后仍可继续执行 DOS 操作。Linux 在启动过程中会保留第一兆字节的内存（即 DOS 所在的位置），因此只需跳回正确的位置即可继续运行 DOS。

来源：[itnews](https://www.itnews.com.au/news/dos-subsystem-for-linux-breaks-cover-553747)

> 拍一拍：虽然这个项目看起来有些“无聊”，但这种探索精神值得鼓励。

### 福布斯：微软更新 Edge 新功能挑战 Chrome

微软最近为 Edge 浏览器添加了一系列新功能，以吸引更多企业用户。这些功能包括直接从地址栏搜索工作文件、让 IT 管理员管理特定与工作相关的应用程序及浏览环境，以及支持 Microsoft Endpoint DLP 原生策略。此外，即将推出的 Sleeping Tabs 功能可以将内存使用率提高 26%，CPU 使用率降低 29%。

来源：[forbes](https://www.forbes.com/sites/kateoflahertyuk/2020/09/26/microsoft-updates-edge-with-a-set-of-stellar-new-features-to-beat-chrome/)

> 拍一拍：通过这些更新，微软希望吸引更多的企业用户选择 Edge 浏览器。

### JetBrains：世界上最大的 Java 程序员集中地在亚洲和德国

根据 JetBrains 的数据，Java 开发者最集中的地区是亚洲，其中中国和印度等国家有超过 250 万开发者将其作为主要编程语言。这可能是因为离岸开发人员在构建 Android 应用时常常被雇佣。使用 Java 作为主要语言的开发者比例最高的六个国家分别是：中国、韩国、印度、德国、西班牙和巴西。

来源：[jetbrains](https://blog.jetbrains.com/idea/2020/09/a-picture-of-java-in-2020/)

> 拍一拍：无论你是否喜欢，Java 仍然非常流行。

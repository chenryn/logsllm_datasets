## 三分之二的开发者在使用 JavaScript

![](/data/attachment/album/202107/17/165712kf31hmw1fh3znwz2.jpg)

JetBrains 对近 3.2 万名开发者的[调查](https://www.jetbrains.com/lp/devecosystem-2021/)显示，JavaScript 的使用率为 69%，其中 39% 的开发者将其作为主要编程语言。相比之下，Java 的使用率为 32%，PHP 为 22%。

该调查还揭示了一些显著的地区差异：
- Java 在中国和韩国最受欢迎。
- 英国的 JavaScript、PHP 和 Python 使用率高于平均水平。
- 美国的 JavaScript 和 Python 使用率较高。
- 印度的 JavaScript、Python 和 C++ 使用率名列前茅。

> 没有人会想到，一种最初设计用来快速解决问题的脚本语言，如今已经成为了如此庞大的生态系统。

## DeepMind 开源 AlphaFold

![](/data/attachment/album/202107/17/165733v8hbgd996979o8fr.jpg)

去年底，Google 子公司 DeepMind 宣布了一项重大突破：AlphaFold 系统能够根据蛋白质的氨基酸序列预测其三维结构。这一突破有望大大加快新药的开发过程。AlphaFold 能在数天内提供准确结果，精度达到原子级别。本周，DeepMind 在《自然》期刊上发表[论文](https://www.nature.com/articles/s41586-021-03819-2)，宣布 AlphaFold 的精度在大多数情况下能与实验室结果相媲美，远超其他方法。此外，DeepMind 还在 GitHub 上以 Apache 许可证[开源](https://github.com/deepmind/alphafold)了 AlphaFold。

> 尽管 OpenAI 解散了机器人团队，但 AI 的发展仍然充满希望。

## 针对 IoT 设备的攻击在两年间增长了 700%

![](/data/attachment/album/202107/17/165753x8aj866ppa676ztt.jpg)

新冠大流行导致许多企业办公室空无一人，但仍有大量工作在进行中。各种联网设备如打印机、数字标牌、智能手表和其他物联网设备被留在办公室。这些设备成为了网络攻击的目标。研究发现，大部分攻击源自印度和美国，而中国、美国和爱尔兰是受攻击最严重的国家。约 98% 的 IoT 攻击受害者来自医疗保健、零售、批发、制造和技术部门。研究显示，在 18,000 台主机上发现了 900 种不同的恶意载荷，涉及 70 多个不同制造商的设备。

> 这些无人管理且未更新的 IoT 设备可能会成为重大的安全隐患。

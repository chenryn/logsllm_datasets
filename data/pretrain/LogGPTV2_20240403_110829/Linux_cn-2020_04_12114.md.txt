深度桌面环境（DDE）以其直观且美观的用户界面著称，最初由深度 Linux 开发团队创建。尽管深度 Linux 最初基于 Ubuntu，但后来切换到了 Debian。

### 深度桌面环境简介

深度桌面环境（DDE）是由深度 Linux 开发团队打造的一个美观且用户友好的桌面环境。然而，深度 Linux 的一个主要缺点是其下载服务器位于中国境内，导致国际用户的下载速度较慢。

对于那些希望在 Ubuntu 上使用深度桌面环境的用户来说，[UbuntuDDE](https://ubuntudde.com/) 提供了一个便捷的选择。UbuntuDDE 项目通过在 Ubuntu 基础上提供开箱即用的深度桌面体验，简化了安装和配置过程，从而节省了用户的时间和精力。

![UbuntuDDE 截图](/data/attachment/album/202004/15/193232byo44gv5i4s4vag8.jpg)

### UbuntuDDE：结合 Ubuntu 的强大与深度桌面的美丽

需要注意的是，UbuntuDDE 并非官方的 Ubuntu 变体。UbuntuDDE 的开发人员与 Ubuntu 团队无关，目前它是一个混合发行版，目标是在未来的版本中成为官方变体之一。

UbuntuDDE 的开发得到了包括 Ubuntu Snapcraft 团队成员 Alan Pope、Ubuntu Budgie 团队以及 [Ubuntu Cinnamon](https://itsfoss.com/ubuntu-cinnamon/) 团队的支持。此外，还有其他开发者如 Hualet Wang 和 Felix Yan 也提供了帮助。

在与 FOSS 的对话中，主要开发人员 Arun 强调该项目的重点在于定期维护 Ubuntu 的 DDE 软件包，并帮助用户充分享受 DDE 的全部功能。首先，他们从上游（即深度仓库）获取最新版本进行维护和打包，然后将其与 Ubuntu 20.04 focal 结合，生成一个可供用户直接安装的镜像文件，而无需先安装常规的 Ubuntu 再手动安装深度桌面。

![Ubuntu Deepin Edition 登录界面](/data/attachment/album/202004/15/193233d9hq0gigilg0e828.jpg)

与 Deepin Linux 不同，UbuntuDDE 使用的是 Ubuntu 软件中心而非深度应用商店。这消除了关于深度 Linux 间谍软件谣言的担忧。

### 下载 UbuntuDDE 20.04 Beta

UbuntuDDE 计划与 Ubuntu 20.04 同步发布首个正式稳定版本。类似于其他 Ubuntu 变体，UbuntuDDE 20.04 beta 版本现已开放下载并可供试用。

> **警告！**
>
> UbuntuDDE 是一个正在开发中的新手项目。请勿在主系统上使用。建议在虚拟机或备用系统中进行测试。

* [下载 Ubuntu 20.04 DDE Beta](https://ubuntudde.com/download/)

![安装 UbuntuDDE](/data/attachment/album/202004/15/193235dxxa0f7sca6j6atw.jpg)

由于本质上是基于 Ubuntu，因此安装 UbuntuDDE 与安装 Ubuntu 的过程相同。你可以参考这篇教程，了解如何在 VirtualBox 中安装 Ubuntu。

虽然有人可能认为这只是“另一个 Ubuntu” 或者“只是在 Ubuntu 上安装深度桌面”，但对于一部分用户而言，这样的项目确实带来了便利。你怎么看？

User: Please describe how to use iptables to configure firewall rules, including adding, deleting and modifying rules.
Assistant: iptables is used for the firewall function in Linux system. You can use the following commands to add rules: 1. Add a single rule: iptables -A INPUT -s 192.168.1.0/24 -p tcp --dport 22 -j ACCEPT 2. Add multiple rules: iptables -A INPUT -s 192.168.1.0 /24 -p tcp --dport 22 -j ACCEPT iptables -A FORWARD -s 10.0.0.0/8 -j DROP Note: After adding a rule, to confirm whether the rule is in effect, you can use the iptables -L command to view and delete the rule. You can use the following command : 1. Delete a single rule: iptables -D INPUT -s 192.168.1.0/24 -p tcp --dport 22 -j ACCEPT 2. Delete all rules: iptables -F To modify a rule, you need to delete the old rule first and then add a new rule, you can Use the following commands: 1. Delete the old rule: iptables -D INPUT -s 192.168.1.0/24 -p tcp --dport 22 -j ACCEPT 2. Add the new rule: iptables -A INPUT -s 192.168.1.0/24 -p tcp --dport 80 -j ACCEPT Note: After modifying the rules, you also need to confirm whether the rules are in effect, you can use the iptables -L command to check

`anaconda`{.command} 和 Red Hat Enterprise Linux
系统都使用相同的软件组件集合。关键技术的详细信息，请访问下面列表的网站：
:::
::: variablelist
[引导装载程序]{.term}
:   ::: para
    Red Hat Enterprise Linux 使用[]{#ap-techref.html#idm139817209421568
    .indexterm}`GRUB`{.command} 引导装载程序。详情请参考
    。
    :::
[磁盘分区]{.term}
:   ::: para
    Red Hat Enterprise Linux 使用 `parted`{.command}
    来完成磁盘分区作业。更多信息请参考
    。
    :::
[存储管理]{.term}
:   ::: para
    []{#ap-techref.html#idm139817212619152
    .indexterm}逻辑卷管理（LVM）为管理员提供一系列管理存储的工具。默认情况下，
    Red Hat Enterprise Linux 安装进程将驱动器格式化为 LVM 卷。详情请参考
    。
    :::
[音频支持]{.term}
:   ::: para
    Red Hat Enterprise Linux 使用的 Linux 内核整合
    []{#ap-techref.html#idm139817182478944 .indexterm} 音频服务器。有关
    PulseAudio
    的详情请参考项目文档：。
    :::
[图形系统]{.term}
:   ::: para
    安装系统和 Red Hat Enterprise Linux
    都使用[]{#ap-techref.html#idm139817181335968
    .indexterm}`Xorg`{.command} 套件来提供图形功能。 `Xorg`{.command}
    的套件为用户使用的桌面环境管理显示、键盘鼠标。详情请参考
    。
    :::
[远程显示]{.term}
:   ::: para
    Red Hat Enterprise Linux 和 `anaconda`{.command}
    包含[]{#ap-techref.html#idm139817175966096 .indexterm}
    VNC（虚拟网络计算）软件启用对图形显示的远程访问。有关 VNC
    的详情请参考 RealVNC
    网站中的文档：。
    :::
[命令行界面]{.term}
:   ::: para
    默认情况下， Red Hat Enterprise Linux 用 GNU `bash`{.command} shell
    来提供一个命令行界面。GNU 核心程序完成命令行环境。有关
    `bash`{.command} 详情请参考
    。要了解更多关于 GNU
    核心程序的知识请参考 。
    :::
[远程系统访问]{.term}
:   ::: para
    Red Hat Enterprise Linux 整合[]{#ap-techref.html#idm139817197346784
    .indexterm} OpenSSH 组件，该组件提供对系统的远程访问。SSH
    []{#ap-techref.html#idm139817205123040
    .indexterm}服务启用了很多功能，其中包括从其他系统访问命令行，远程命令执行以及网络文件传输。安装过程中，`anaconda`{.command}
    可使用 OpenSSH 的 `scp`{.command}
    []{#ap-techref.html#idm139817200488960
    .indexterm}功能将崩溃报告传送到远程系统中。有关 OpenSSH
    网站详情请参考：
    :::
[访问控制]{.term}
:   ::: para
    []{#ap-techref.html#idm139817116384896 .indexterm}SELinux
    提供强制访问控制（MAC）功能，它可补充标准的 Linux
    安全性特性。有关详情请参考 SELinux
    项目页面：。
    :::
[防火墙]{.term}
:   ::: para

    框架来提供[]{#ap-techref.html#idm139817127521856 .indexterm}
    防火墙功能。Netfilter 项目网站提供了 `netfilter`{.command} 文件和
    `iptables`{.command}
    管理工具：。
    :::
[软件安装]{.term}
:   ::: para
    Red Hat Enterprise Linux 使用[]{#ap-techref.html#idm139817114204704
    .indexterm}`yum`{.command} 管理组成系统的 RPM 软件包。详情请参考
    。
    :::

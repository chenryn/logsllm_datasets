User: How to install Nginx on CentOS 7?
Assistant: The following are the steps to install Nginx on CentOS 7: 1. Open a terminal on CentOS 7 2. Add the official repository of Nginx `sudo vi /etc/yum.repos.d/nginx.repo` 3. Add the following content to the file `[nginx]` `name=nginx repo` `baseurl=http://nginx.org/packages/mainline/centos/7/$basearch/` `gpgcheck=0` `enabled=1` 4. Install Nginx `sudo yum install nginx` 5. Start Nginx `sudo systemctl start nginx` 6. Set Nginx to boot automatically `sudo systemctl enable nginx` 7. Verify that Nginx is working normally. Enter the IP address of the server in the browser. If you see the welcome page of Nginx, it means that it has been successfully installed. `http://<server_IP_address>/` Above are the steps to install Nginx on CentOS 7.

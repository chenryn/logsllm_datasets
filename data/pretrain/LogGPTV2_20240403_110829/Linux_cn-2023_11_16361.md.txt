### 苹果公司暂停开发明年的操作系统，以修复软件错误
上周，苹果公司在内部宣布了这一决定。公司的工程师们目前的主要任务是修复错误和改进性能，而不是添加新功能。上个月，苹果完成了明年发布的 iPhone、iPad 和 Mac 操作系统的第一个里程碑版本，但在内部测试中发现了大量问题。因此，第二个里程碑版本的发布工作被推迟，新功能开发暂停一周，以便集中精力修复错误。不过，这次暂停不太可能影响新版本的最终发布。

> **[消息来源：彭博社](https://www.bloomberg.com/news/articles/2023-11-07/apple-delays-work-on-ios-18-macos-15-watchos-11-due-to-problems-with-bugs)**
>
> **老王点评**：这是集体修 bug 的一周。我在写这条消息时，刚刚收到了前两天升级的 macOS 14.1 的一个更新。

### Servo 浏览器引擎获得资金支持 HTML 表格
Servo 浏览器引擎最初由 Mozilla 开发，目前由欧洲 Linux 基金会等组织共同维护。今年，Servo 项目取得了显著进展，目标是成为一个易于嵌入的网络引擎，并不断增加新功能。最近，Servo 获得了 NLnet 基金会的资助，用于完善浮动支持、支持更多语言的内联布局，并初步实现对“表格”标签的支持。值得注意的是，HTML 表格尚未在 Servo 中实现，此次开发的重点是满足维基百科上 HTML 表格的需求。

> **[消息来源：Phoronix](https://www.phoronix.com/news/Servo-Grant-Funding-Table)**
>
> **老王点评**：我很惊讶居然还有连表格都不支持的浏览器引擎。

### 必应聊天的 GPU 不够用了，微软将向甲骨文租用 GPU
由于人工智能服务需求的激增，微软需要更多的计算资源。甲骨文恰好拥有数以万计的英伟达 GPU 可供租用。这两家云计算提供商利用了微软 Azure 的甲骨文互联服务，该服务允许 Azure 中运行的服务与甲骨文云基础设施中的资源进行交互。此前，他们曾利用这项服务让客户将运行在 Azure 中的工作负载连接到甲骨文云上的数据库。

> **[消息来源：The Register](https://www.theregister.com/2023/11/07/bing_gpu_oracle/)**
>
> **老王点评**：连地主家都没有余粮了。不过，最近感觉 Azure 的 ChatGPT 有些笨了。

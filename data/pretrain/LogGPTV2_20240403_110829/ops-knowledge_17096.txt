User: 如何在Linux系统上设置定时任务？请提供常用的命令行工具和操作步骤。
Assistant: Linux系统上设置定时任务一般使用crontab命令来完成，具体步骤如下：

1. 打开终端（Terminal）。

2. 输入以下命令来编辑定时任务：

   crontab -e

   如果是第一次编辑，则系统会提示选择编辑器，可以选择vi或nano等常用的编辑器。

3. 在打开的文件末尾添加想要执行的命令，每个命令占一行。

   crontab格式如下：

   ```
   *     *     *     *     *  command to be executed
   -     -     -     -     -
   |     |     |     |     |
   |     |     |     |     +----- day of the week (0 - 6) (Sunday=0)
   |     |     |     +------- month (1 - 12)
   |     |     +--------- day of the month (1 - 31)
   |     +----------- hour (0 - 23)
   +------------- min (0 - 59)
   ```

4. 按照上面的格式来填写时间和命令，例如：

   ```
   0 1 * * * /home/user/script.sh
   ```

   表示在每天的凌晨1点执行/home/user/script.sh。

5. 完成后，保存文件并关闭编辑器。

6. 输入以下命令来查看当前用户的定时任务：

   crontab -l

7. 如果需要删除某个定时任务，可以输入以下命令：

   crontab -r

   或者使用以下命令来编辑已有的定时任务：

   crontab -e

注意事项：

- 使用crontab命令需要具有管理员权限（sudo）。
- 定时任务执行的命令需要具有可执行权限（chmod +x）。
- 时区可能会影响定时任务的执行时间。

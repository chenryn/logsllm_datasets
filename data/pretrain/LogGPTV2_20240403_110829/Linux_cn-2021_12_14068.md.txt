> Zorin OS 16 Lite 带来了 Zorin OS 16 的所有优点和 Xfce 桌面环境。让我们来了解一下。

![](/data/attachment/album/202112/10/100703m0r10v1e2515zzra.jpg)

[Zorin OS 16](https://news.itsfoss.com/zorin-os-16-release/) 是一个令人印象深刻的版本。然而，如果你正在寻找一个轻量级的发行版或适合旧电脑的选择，它可能并不理想。Zorin OS 为每个版本都提供了一个基于 Xfce 桌面环境的 Lite 版本，以供资源有限的计算机使用。直到最近，Zorin OS 16 才推出了这个 Lite 版。

是的，Zorin OS 16 Lite 终于发布了，并且可以与 Pro 版一起下载。如果你已经购买了 Zorin OS 16 Pro，那么 Lite 版也包含在内。（LCTT 译注：Zorin OS 的 Pro 版是收费的，而 Core 版和 Lite 版是免费的。）

### Zorin OS 16 Lite: 新增内容

![](/data/attachment/album/202112/10/100704i7xfq7u8fufl0x5l.jpg)

在这个版本中，Zorin OS 16 Lite 旨在通过保持最低的资源使用率，带来所有 [Zorin OS 16 中引入的改进](https://news.itsfoss.com/zorin-os-16-features/)。我在制作视频点评时简要测试了该发行版，对其印象深刻。

有趣的是，Zorin OS 16 Lite 提供了类似于 Xfce 的用户体验和精美的桌面环境。通常我不喜欢 Xfce 桌面环境（仅出于审美原因），但 Zorin OS 16 Lite 却与众不同。我制作了一个关于 Zorin OS 16 Lite 专业版的视频，你可以看到它的运行情况。

#### 视觉更新

![](/data/attachment/album/202112/10/100705ktx53zfffuiemm6e.png)

带有 Xfce 4.16 的 Zorin OS 16 Lite 提供了令人耳目一新的用户体验，其精美的主题与 Zorin OS 16（GNOME 版）非常接近。你可以调整主题、切换暗色模式，并从外观菜单中选择一种重点颜色。外观选项也得到了刷新，使你在 Zorin OS 16 Lite 上轻松定制体验。

不幸的是，Lite 版不支持像 Zorin OS 16 一样的果冻模式。因此，如果你需要华丽的动画效果，Lite 版可能不是最佳选择。

除了新的视觉变化和功能外，你还会发现一些新的背景壁纸。有些壁纸可能只限于 Pro 版。

![](/data/attachment/album/202112/10/100706y1yeqeigdyb22c2g.png)

#### Pro 版的桌面布局

虽然 Core 版和 Lite 版中有一些桌面布局的选择，但 Pro 版又为你提供了两个额外的桌面布局。

![](/data/attachment/album/202112/10/100707lkvuqpauzjgsjfov.jpg)

其中一个布局类似于经典的 Windows 设计，另一个布局看起来接近 ChromeOS。

![](/data/attachment/album/202112/10/100708go8yfgdj0yqr0lhr.jpg)

你还可以在类似 macOS、GNOME 2 和类似 Windows 列表的布局之间进行选择。作为 [Windows 用户的最佳发行版之一](https://itsfoss.com/windows-like-linux-distributions/)，其默认的布局类似 Windows，这应该是一个很好的体验。

#### 带有窗口预览的任务栏

![](/data/attachment/album/202112/10/100709ir8y0ypxyttyiryu.png)

桌面功能有一个重要的改进，即现在当你从任务栏悬停在一个活动窗口上时，可以很容易地查看该窗口的预览。在用预先发布的 ISO 进行测试时，这个体验并不流畅。但是，考虑到他们已经修复了一些错误，这是 Zorin OS 16 Lite 中的一个很好的功能。

#### 默认启用的 Flathub

应用中心（软件中心）也提供了 Flathub 集成，开箱即用。你应该可以从软件中心很快地找到 Flatpak 应用程序，如果你不喜欢 Flatpak 软件包，可以更改软件包源。

#### 其他改进

![](/data/attachment/album/202112/10/100710awt66zd55zx5bmt4.png)

除了关键的变化外，你还会发现一些改进，如禁用遥测的 Firefox 浏览器、新的声音记录器应用、高分辨率显示器的小数点级缩放以及 Rythmbox 应用。Thunar 文件管理器也有一些常规的小改进，并与其他设计融为一体。你应该能够使用 Parole 视频播放器来播放视频。

[Linux 内核 5.11](https://news.itsfoss.com/linux-kernel-5-11-release/) 也改进了一些技术兼容性。

总的来说，Zorin OS 团队在将 Zorin OS 16 的漂亮设计和易用性延续到其 Lite 版方面做得很好。大部分时间，你甚至可能没有意识到你在使用 Xfce 桌面环境。

### 下载 Zorin OS 16 Lite

你可以前往其下载页面获取 Zorin OS 16 Lite 的 ISO。如果你已经购买了 Zorin OS 16 Pro 版，你可以使用电子邮件中的链接来下载 Lite 版。

* [Zorin OS 16 Lite](https://zorin.com/os/download/)

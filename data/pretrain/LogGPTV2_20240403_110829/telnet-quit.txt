### Telnet: Quit Command

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `quit` command is used to exit a Telnet session.

**Syntax:**
```
q[uit]
```

**Examples:**
To exit a Telnet session, type:
```
q
```

**Explanation:**
- The `q` or `quit` command will terminate the current Telnet session and return you to the command prompt.
- This command is case-insensitive, meaning both `q` and `QUIT` are valid.

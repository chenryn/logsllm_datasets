User: How to query the size of all files in a directory by using Shell commands, and display the results in increasing order?
Assistant: You can use the following Shell command to query the size of all files in a directory, and display the results in increasing order: ``` du -hs * | sort -n ``` Explanation: - `du -hs *`: use `du The `command queries the size of all files and directories in the current directory. The `-h` option means to display in a readable format. The `-s` option means to only display the total size instead of the size of each file. `*` means to display All files and directories are queried. - `sort -n`: Use the `sort` command to sort the results by size, the `-n` option means sorting by numerical size (instead of sorting by lexicographical order). Results will be presented in increasing order.

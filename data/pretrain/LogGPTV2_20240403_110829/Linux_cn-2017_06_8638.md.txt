> Linus Torvalds 和 VMware 开源负责人 Dirk Hohndel 上周在中国 LinuxCon 上进行了一次“炉边聊天”。

上周一，Linus Torvalds 首次来到中国参加在北京举行的 LinuxCon + ContainerCon + CloudOpen 大会。在近 2000 名观众面前，他与 VMware 开源负责人 Dirk Hohndel 进行了一次“炉边聊天”，讨论了哪些事情仍然让他感到惊讶和激励，以及有志于开源开发的新手该如何入门。以下是他们谈话的一些亮点。

### Linux 开发中令人惊讶的事情

> “我觉得有趣的是，我认为已经稳定的代码仍然在不断得到改进。有些东西我们已经很多年没有碰过，但有人来改进了它们，或者在我以为根本不会有人用的东西上提交了 Bug 报告。我们有了新的硬件，开发了新的功能，但 25 年后，我们仍然有老的、非常基础的东西，并且人们依然在关心和改善着它们。”

### 什么在激励着他

> “我真的很喜欢我正在做的事情。我喜欢醒来时有一个技术上有趣而富有挑战性的工作，而且不要太紧张，这样我可以长时间地投入其中；或者做一些我觉得真正有意义的事情，不仅仅是对我自己，而是对整个社区。”

> “我偶尔会在工作中休息一下，例如我在 Git 上工作两到三周就会开始休息。但如果休息时间太长，我会感到无聊。当我出去潜水一周后，就想着要回来工作，我从来没有觉得需要一个更长的假期。”

（LCTT 译注：此处“在 Git 上工作”是指 Linus 在 Git 版本仓库中开发 Linux 内核，而不是开发 Git 软件——事实上，Linus 在早期开发完 Git 的原型之后，主要的 Git 开发已经有其他人接手了。而“潜水”是真的指潜水运动，Linus 喜欢玩潜水。）

### Linux 的未来领导力

> “我们的工作进程不会只持续 25 年，我们仍然有一个非常强大的维护团队。我们常常抱怨没有足够的维护者——这是真的，我们只有几十名顶级维护者做日常合并的工作，这对于一个开源项目来说是一个非常强大的团队。随着这些顶级维护者逐渐变老，我们不断有新人加入。一个新人成长为顶级维护者需要几年的时间，因此我不担心 Linux 的下一个 20 年。”

### Linux 会被替代吗？

> “或许将来会有新的项目表明他们比我们做得更好，但我并不担心这一点。有很多非常成功的 Linux 分支，人们不会把它们看作是分支，因为它们很和谐。如果有人想改变一切并让内核变得更好，我的感觉是：去做吧，证明你自己。我可能认为那是个坏主意，但你可以证明我是错的。”

（LCTT 译注：此处所说的分支，应该是指类似 Android、AGL 等 Linux 分支并没有分裂 Linux 生态，而是彼此补充。）

### 对 Git 的看法

> “我对 Git 的广泛传播感到非常惊讶。显然我非常高兴，这验证了我对分布式开发的看法。然而，那时已经有如此多的源码版本控制工具，很难再去推出一个新的版本控制系统。我预计它主要限于内核开发——因为它是为了我们的需求而设计的。”

> “在最初的三到四年里，关于 Git 的抱怨是它与众不同且难以使用。大约五年前，情况发生了变化。有足够的项目和开发者开始使用 Git，它变得不再与众不同；人们习惯了使用 Git。他们开始利用这种开发模式，使用 Git 的安全感意味着任何东西都不会损坏或丢失。”

> “在某些方面，Git 比 Linux 更为人所知。Linux 通常被隐藏起来，例如安卓手机运行在 Linux 之上，但你并不知道。但使用 Git 时，你确切地知道你在使用 Git。”

### 分支 Linux

> “当我坐下来开始写 Git 时，一个首要原则就是你应该能够 fork 并在此基础上做自己的事情。如果你有友好的 fork（能够证明我错了并且能够改进内核），在这种情况下，人们可以回来说我们实际上改进了内核，这没有什么不好的感觉。我会采纳你的改进并将其合并进来。这就是为什么你应该鼓励 fork。你也希望让良好的回馈变得简单。”

### 开源开发者应该如何开始

> “对我来说，我总是自我激励，知道自己想要做什么，我从来没有被告知要去做什么。我不确定我的例子是否适合所有人效仿。如果你是一个新手程序员，你可以从成千上万的开源项目中找到你感兴趣的，长期关注这个项目，了解它的代码，以至于你可以在某个部分的代码上成为专家，不需要是整个项目。没有人是整个内核的专家，但你可以很好地了解其中一个领域。”

> “如果你能成为社区的一部分，提交补丁，那将不仅仅是编程，还有开源社会方面的意义。你作为一个程序员提升自己并与外界联系起来。你可以向外展示——我做了这些改进，我有能力在我的社区或工作中走得更远。你不得不花费一定的时间来学习一个项目，但你将有很大的上升空间——不仅仅是在职业方面，而且在生活中也有一个惊人的项目。”

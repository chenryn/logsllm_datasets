**iLinux Icons 是 Linux 平台上最大的图标收藏库之一，其设计灵感来源于 Linux、Windows 和 Mac OS 系统。**

![](/data/attachment/album/201310/14/201157ieottti45ikwligt.png)

尽管有数百种不同的软件包可用，但在 Linux 平台上高质量的图标仍然稀缺。iLinux Icons 的开发者从 Ubuntu、Windows 和 Mac OS 这三个平台收集并整理了这些图标，有些甚至比原版更出色。

安装过程非常简单，可以通过添加 PPA 来完成。只需在终端中输入以下命令：

```bash
sudo add-apt-repository ppa:noobslab/icons2
sudo apt-get update
sudo apt-get install ilinux-icons
```

为了应用这些图标，你还需要使用 Ubuntu Tweak 或 GNOME Tweak Tool。更多详细信息，请访问开发者的 [官方网站](http://haniahmed.deviantart.com/art/iLinux-icons-v-8-5-354470161)。

Assistant: 1) Install unzip yum install -y unzip zip 2) Enter the path where the jar package is located, use the command unzip jar package name to decompress to get the directory 3) Use the vim command to edit the files in the decompressed directory 4) Save and exit the vim editor , use the command zip -r jar package name directory name to repackage the edited file into a jar package

User: You need to install and configure Nginx on a CentOS server so that it can act as a web server to provide website services. Please provide a detailed procedure.
Assistant: The following are the detailed steps to install and configure Nginx on the CentOS server: 1. Update the server package: use the following command: sudo yum update 2. Install the Nginx package: use the following command: sudo yum install nginx 3. Start the Nginx service: use The following command: sudo systemctl start nginx 4. Verify that Nginx is running: Use the following command: sudo systemctl status nginx If Nginx is running, you should see an "active" status. 5. Configure the firewall to allow HTTP and HTTPS traffic: Use the following commands: sudo firewall-cmd --permanent --zone=public --add-service=http sudo firewall-cmd --permanent --zone=public --add- service=https sudo firewall-cmd --reload 6. Visit the default Nginx page: Enter your server's IP address or domain name in your browser. You should see a "Welcome to nginx!" message. 7. Configure Nginx to serve websites: You can edit Nginx configuration files to create website configurations. The default configuration file is located in the "/etc/nginx/nginx.conf" file. You can create a new configuration file and place it in the "/etc/nginx/conf.d/" directory. For example, the following is an example of a configuration file that creates a virtual host named "example.com": server { listen 80; server_name example.com; root /var/www/example.com; index index.html; } 8. Change the file system permissions to allow the Nginx server to access the website directory: Use the following command: sudo chown -R nginx:nginx /var/www/example.com 9. Reload the Nginx configuration for the changes to take effect: Use the following command: sudo systemctl reload nginx 10. Test your website service: enter your domain name or IP address in a browser and you should see your website.

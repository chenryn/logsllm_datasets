### elementaryOS 5.1.4 引入屏幕时间和限制功能

此次更新中，家长控制被重新命名为“屏幕时间和限制”，并且引入了新功能，允许为任何账户设置特殊规则，包括管理员账户。此前，这一功能仅限于非管理员账户。此外，系统设置中的搜索功能也进行了改进，使用户更容易找到特定配置选项。本次更新还包含了许多修复，涉及视频、Gala 窗口管理器、照片应用和媒体键等方面。

**来源**: [softpedia](https://news.softpedia.com/news/elementaryos-5-1-4-is-a-massive-update-new-features-announced-529880.shtml)

**硬核老王点评**: 现在连自己都可以限制使用时间了，这是怕自己上网成瘾吗？

### 全球虚拟专用网络使用量在 3 月激增

根据英国虚拟专用网络研究和测试公司 Top10VPN.com 的报告，在 3 月 13 日至 3 月 23 日期间，美国商用虚拟专用网络的需求增加了 41%。需求增幅最大的国家是埃及（224%）、斯洛文尼亚（169%）和智利（149%）。持续增长最显著的是埃及（154%）、秘鲁（119%）和南非（105%）。在美国和英国，有 51% 的人在公共 Wi-Fi 网络上使用虚拟专用网络来保护隐私。另有 44% 的受访者表示，匿名浏览是使用虚拟专用网络的主要原因，其次是安全通讯。

**来源**: [networkworld](https://www.networkworld.com/article/3542032/global-vpn-use-exploded-in-march.html)

**硬核老王点评**: “佛跳墙”虽然常被认为是一个敏感词，但它确实有严肃的企业和个人用途。

### Valve 决定放弃 macOS 版 SteamVR

Valve 在 2017 年年中发布了 macOS 版本的 SteamVR。尽管该公司没有解释具体原因，但宣布开发者将继续为苹果操作系统构建应用，只是未来不会发布新的功能或 bug 修复。从现在起，Valve 将专注于 Windows 和 Linux 平台的开发。

**来源**: [softpedia](https://news.softpedia.com/news/valve-drops-steamvr-for-macos-as-linux-and-windows-now-the-key-focus-529884.shtml)

### LLVM 10 导致 Rust 编译时间变长

据 Phoronix 报道，与 LLVM/Clang 9.0 相比，Clang 10.0 的基准测试结果总体良好，未出现明显倒退。然而，LLVM 10.0 导致 Rust 编译时间增加了 7% 至 8%。

**来源**: [开源中国](https://www.oschina.net/news/115541/rust-hurt-on-llvm-10)

### 微软保留 Win7/8.1 免费升级至 Win10 的福利

尽管 Windows 10 的免费升级优惠已在多年前正式结束，但通过实际测试发现，该免费升级漏洞仍然有效。用户可以通过下载微软官方提供的 Media Creation Tool 并选择“Upgrade this PC now”进行升级。不清楚微软为何保留这个漏洞，可能是因为对 Windows 10 达到 10 亿用户的速度不满意。

**来源**: [快科技](https://www.cnbeta.com/articles/tech/977369.htm)

**硬核老王点评**: 只要有利可图，那就睁一只眼闭一只眼吧。

### Edge 浏览器将隐藏垃圾通知

跟随谷歌和 Mozilla 的脚步，微软正在测试一项新功能，旨在减少浏览器内的垃圾通知和权限请求。这项“更安静的通知”功能预计将在今年晚些时候推出。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/977297.htm)

**硬核老王点评**: 即便是再好的机制，也会因滥用而受到重重束缚。

### 任天堂法务部面临挑战：马里奥 64 移植版失控扩散

任天堂法务部门正在全力打击种子站、YouTube 视频、谷歌搜索结果和 Reddit 论坛帖子中关于马里奥 64 移植版的讨论。尽管如此，许多玩家已经下载了可执行程序并通过社交媒体继续传播。后续上传的移植版视频也非常多，难以完全封禁。

**来源**: [3DMGame](https://hot.cnbeta.com/articles/game/977249.htm)

**硬核老王点评**: 一旦失控，法律手段也难以完全遏制。

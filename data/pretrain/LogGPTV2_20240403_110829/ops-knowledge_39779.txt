User: What are the main procedures an organization should follow in order to implement CIS Control 8.5, Collect Detailed Audit Logs?
Assistant: An organization should Configure detailed audit logging for enterprise assets containing sensitive data. Include event source, date, username, timestamp, source addresses, destination addresses, and other useful elements that could assist in a forensic investigation.CIS Control 8.5 has the asset type Network, and has the security function Detect.

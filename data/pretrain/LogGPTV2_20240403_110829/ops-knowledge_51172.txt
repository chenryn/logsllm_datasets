User: How to check the CPU and memory usage of a process in Linux?
Assistant: Use the top command to view the CPU usage, memory usage, and system load of processes running in the system. Use the top command: 1. Open a terminal window 2. Enter the top command 3. View the CPU and memory usage of the process in the top window. The first line displayed in the top window is the load of the system, and the second line starts to display the load of each process Information, including process ID, process name, CPU usage, memory usage and other parameters. If you want to refresh the process information in real time, you can press the "Shift" + "R" keys, and the top window will continuously refresh to display the latest process information. If you want to exit the top command, you can enter the "q" key, or press the "Ctrl" + "C" key.

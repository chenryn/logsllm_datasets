User: What are the main procedures an organization should follow in order to implement CIS Control 16.13, Conduct Application Penetration Testing?
Assistant: An organization should Conduct application penetration testing. For critical applications, authenticated penetration testing is better suited to finding business logic vulnerabilities than code scanning and automated security testing. Penetration testing relies on the skill of the tester to manually manipulate an application as an authenticated and unauthenticated user. CIS Control 16.13 has the asset type Applications, and has the security function Protect.

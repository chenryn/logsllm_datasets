# Environment
    Windows build number:  [Version 10.0.18362.836]
    PowerToys version: v0.18.1
    PowerToys: PowerToys Run
# Steps to reproduce
With PowerToys installed and running, press `Alt` \+ `Space` from any screen.
# Expected behavior
PowerToys Run popup shows up to start typing a command or a search term
# Actual behavior
window menu (Restore / Move / Size / Minimize / Maximize / Close) shows up in
the top left corner of the focused window if the focus is on a window or
nothing at all if the focus is on the desktop.

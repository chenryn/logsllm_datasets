# Environment
- **Operating System:** Windows 10, Build Number: 10.0.18362.836
- **PowerToys Version:** v0.18.1
- **Feature in Use:** PowerToys Run

# Steps to Reproduce
1. Ensure that PowerToys is installed and running on your system.
2. Press the `Alt` + `Space` key combination from any screen.

# Expected Behavior
- The PowerToys Run popup should appear, allowing you to type a command or search term.

# Actual Behavior
- If the focus is on a window, the window menu (containing options such as Restore, Move, Size, Minimize, Maximize, and Close) appears in the top-left corner of the focused window.
- If the focus is on the desktop, no action occurs, and nothing is displayed.

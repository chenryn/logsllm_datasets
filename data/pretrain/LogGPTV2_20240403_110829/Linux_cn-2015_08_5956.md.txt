不久前，开源模式还被成熟的工业级厂商以怀疑的态度视为叛逆小孩的玩物。如今，开源的促进会和基金会在一长列供应商的支持下正蓬勃发展，并将开源模式视为创新的关键。

![](/data/attachment/album/201508/05/000602vcozsyhfgskshxlk.jpg)

### 技术的开放发展驱动着创新

在过去的二十年间，开源技术已成为驱动创新的重要因素。即使是那些曾经视开源为威胁的公司也开始接受这一观点，例如微软，现在在多个开源项目中表现活跃。尽管大多数开源推进都集中在软件领域，但社区已经开始向开源硬件倡议扩展。本文将介绍七个成功地在硬件和软件方面同时促进和发展开源技术的组织。

### OpenPOWER 基金会

![](/data/attachment/album/201508/05/000604jnj76yxys27dj4iy.jpg)

[OpenPOWER 基金会](http://www.linuxfoundation.org/) 由 IBM、Google、Mellanox、Tyan 和 NVIDIA 于 2013 年共同创建，旨在推动开放协作硬件的发展。IBM 通过开放其基于 Power 架构的硬件和软件技术，以及提供使用 Power IP 的独立硬件产品的许可证，为基金会的建立奠定了基础。如今，超过 70 个成员共同协作，为基于 Linux 的数据中心提供定制的开放服务器、组件和硬件。

去年四月，OpenPOWER 推出了一个技术路线图，基于新的 POWER8 处理器的服务器比最新的基于 x86 系统的数据分析能力快 50 倍。同年七月，IBM 和 Google 发布了一个固件堆栈。去年十月，NVIDIA GPU 为加速 POWER8 系统提供了支持，Tyan 也推出了第一个 OpenPOWER 参考服务器。

### Linux 基金会

![](/data/attachment/album/201508/05/000605sslwahyew114hsha.jpg)

[Linux 基金会](http://www.linuxfoundation.org/) 成立于 2000 年，如今已成为历史上最大的开源协同开发成果之一，拥有超过 180 个合作成员，以及许多独立成员和学生成员。它赞助 Linux 核心开发者的工作，促进、保护和推进 Linux 操作系统，并协调软件的协作开发。

其最成功的协作项目包括 Code Aurora Forum（为移动无线产业服务的企业财团）、MeeGo（为移动设备和 IVI [车载信息娱乐系统] 构建基于 Linux 内核的操作系统的项目）和 Open Virtualization Alliance（开放虚拟化联盟，促进自由和开源软件虚拟化解决方案的采用）。

### 开放虚拟化联盟

![](/data/attachment/album/201508/05/000605w6eeu9pzpb77uwe6.jpg)

[开放虚拟化联盟 (OVA)](https://openvirtualizationalliance.org/) 的目标是通过提供使用案例和支持具有互操作性的通用接口和 API 的发展，来促进自由和开源软件虚拟化解决方案（如 KVM）的采用。KVM 将 Linux 内核转变为一个虚拟机管理程序。如今，KVM 已成为与 OpenStack 共同使用的最常见的虚拟机管理程序之一。

### OpenStack 基金会

![](/data/attachment/album/201508/05/000606oqtnhdghh9h6gfnn.jpg)

最初作为 IaaS（基础设施即服务）产品由 NASA 和 Rackspace 于 2010 年启动，[OpenStack 基金会](http://www.openstack.org/foundation/) 已成为最大的开源项目之一，拥有超过 200 家公司成员，其中包括 AT&T、AMD、Avaya、Canonical、Cisco、Dell 和 HP。

大约每六个月发布一次新版本，基金会的 OpenStack 项目开发用于通过基于 Web 的仪表盘、命令行工具或 RESTful 风格的 API 来控制或调配数据中心中的处理存储池和网络资源。迄今为止，基金会支持的协同开发已经孕育出了一系列 OpenStack 组件，包括 OpenStack Compute（云计算网络控制器）、OpenStack Networking（管理网络和 IP 地址的系统）和 OpenStack Object Storage（可扩展的冗余存储系统）。

### OpenDaylight

![](/data/attachment/album/201508/05/000608cba9cd9l6o4y9o4p.jpg)

作为来自 Linux 基金会的另一个协作项目，[OpenDaylight](http://www.opendaylight.org/) 是一个由 Dell、HP、Oracle 和 Avaya 等行业厂商于 2013 年 4 月联合发起的倡议。其任务是建立一个由社区主导、开源且有工业支持的针对软件定义网络（SDN）的包含代码和蓝图的框架。其目标是提供一个可直接部署的全功能 SDN 平台，而不需要其他组件，供应商可以提供附加组件和增强组件。

### Apache 软件基金会

![](/data/attachment/album/201508/05/000609xzvhe9qhx07z50d0.jpg)

[Apache 软件基金会 (ASF)](http://www.opencompute.org/) 是近 150 个顶级项目的聚居地，这些项目涵盖了从开源企业级自动化软件到与 Apache Hadoop 相关的分布式计算的整个生态系统。这些项目分发企业级、可免费获取的软件产品，而 Apache 协议则使商业用户和个人用户能够更方便地部署 Apache 的产品。

ASF 成立于 1999 年，是一个会员制非营利公司，以精英为核心 —— 要成为其成员，必须首先在基金会的一个或多个协作项目中做出积极贡献。

### 开放计算项目

![](/data/attachment/album/201508/05/000610feuzte3bswdbuuwz.jpg)

作为 Facebook 重新设计其 Oregon 数据中心的副产物，[开放计算项目](http://www.opencompute.org/) 旨在开发针对数据中心的开源硬件解决方案。OCP 包括廉价无浪费的服务器、针对 Open Rack（为数据中心设计的机架标准，以便将其集成到数据中心的基础设施中）的模块化 I/O 存储和一个相对“绿色”的数据中心设计方案。

OCP 董事会成员包括来自 Facebook、Intel、Goldman Sachs、Rackspace 和 Microsoft 的代表。

OCP 最近宣布了两种可选的许可证：一种类似于 Apache 2.0 的允许衍生工作的许可证，另一种更为严格的鼓励将更改回馈到原有软件的许可证。

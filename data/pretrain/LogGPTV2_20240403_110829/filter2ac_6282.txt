**ATT&CK ID:** S0085  
**ATT&CK Technique Name:** S-Type

**Description:**
S-Type may create a file named `Realtek {Unique Identifier}.lnk` in the `%HOMEPATH%\Start Menu\Programs\Startup` directory. This shortcut file points to a malicious `msdtc.exe` executable, which is already placed in the `%CommonFiles%` directory. [31]

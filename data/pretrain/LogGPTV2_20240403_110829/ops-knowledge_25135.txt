User: How to create a new branch when using Git for code version management? How to merge new branch with master branch?
Assistant: The command to create a new branch is: git branch <branch name> For example, the command to create a new branch named develop is: git branch develop The command to switch to a new branch is: git checkout <branch name> For example, switch to the develop branch The command is: git checkout develop If you want to complete the above two operations in one step, you can use the following command: git checkout -b <branch name> For example, the command to create and switch to a new branch named feature1 is: git checkout -b feature1 The command to merge the new branch with the main branch is: git merge <branch name> For example, the command to merge the develop branch into the main branch is: first switch to the main branch, and then execute git merge develop

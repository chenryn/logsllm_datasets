# Environment
- **Windows Build Number:** 10.0.18363.836
- **PowerToys Version:** 0.18.2
- **PowerToy Module:** Run

# Steps to Reproduce
1. Open PowerToys Run.
2. Search for "Docker" or "Docker Desktop".
3. Observe that only the `.lnk` (shortcut) is listed.

# Expected Behavior
- The search should return the Docker Desktop application, not just the shortcut.

# Actual Behavior
- Only the `.lnk` (shortcut) for Docker Desktop is found and displayed in the search results.

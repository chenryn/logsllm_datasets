## 北斗“收官之星”成功发射

2020 年 6 月 23 日上午 9 时 43 分，西昌卫星发射中心成功发射了北斗三号全球卫星导航系统的最后一颗组网卫星。这标志着北斗三号全球卫星导航系统星座部署全面完成。

来源：[雷锋网](https://www.cnbeta.com/articles/tech/994787.htm)

**拍一拍**：这是一个值得庆祝的里程碑。依赖美国 GPS 可能带来潜在风险，而目前大部分中国的安卓手机已经使用了北斗系统。相比之下，iPhone 等产品仍然主要依赖 GPS。

## Electron 成为 OpenJS 基金会的 Impact 项目

2019 年 12 月，OpenJS 基金会宣布 Electron 加入其孵化计划。在最近的 OpenJS World 上，Electron 正式从 OpenJS Foundation 的孵化计划毕业，成为该基金会的 Impact 项目。OpenJS 基金会是由 Node.js 基金会与 JS 基金会合并而成，托管了许多开源 JavaScript 项目，包括 jQuery、Node.js 和 webpack。Electron 是一个基于 Node.js 和 Chromium 的跨平台桌面应用开发工具，支持使用 JavaScript、HTML 和 CSS 编写跨平台的桌面应用程序。它被广泛应用于许多知名软件，如 Discord、Microsoft Teams、Skype、Slack 和 Visual Studio Code。

来源：[开源中国](https://www.oschina.net/news/116694/electron-openjs-impact-project)

**拍一拍**：尽管 Electron 是一套 JavaScript 框架，但它已经被广泛用于开发许多我们日常使用的应用程序。

## Microsoft Edge 擅自导入 Firefox 数据

用户反馈称，在通过 Windows Update 更新到新版 Edge 后，即使没有授权，Edge 也会静默地从 Firefox 导入数据。除非通过任务管理器关闭它，否则无论如何它都会复制数据。此外，微软还会取消系统的默认浏览器设置，使得用户在点击 URL 时需要重新选择默认浏览器。对此，微软尚未作出回应。

来源：[开源中国](https://www.oschina.net/news/116691/import-firefox-data-to-edge)

**拍一拍**：这种情况确实令人担忧，希望微软能够尽快澄清此事。

## CNCF 宣布首个中国原创项目 Harbor 毕业

Harbor 是一个开源制品仓库，提供策略和基于角色的访问控制来保护制品（如容器镜像、Helm Chart 等），扫描镜像并避免安全漏洞，并对镜像签名以确保内容可信。该项目起源于 2014 年 VMware 中国研发中心云原生实验室的一个内部项目，旨在解决容器开发人员的镜像管理问题。许多企业和初创公司在生产系统中使用 Harbor，包括中国移动、京东、才云科技、网易云、三星 SDS、Hyland 软件、Mulesoft、腾讯云、趋势科技和 VMware 等。

来源：[开源中国](https://www.oschina.net/news/116677/harbor-graduate-from-cncf)

**拍一拍**：中国在开源领域的贡献越来越多，这是一个非常积极的趋势。

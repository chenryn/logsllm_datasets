如何修改 manager server 和 agent 的配置文件？

Assistant: 要修改 manager server 和 agent 的配置文件，需要登录到相应的服务器上，找到相应的配置文件并进行编辑。对于 manager server，需要编辑 `/opt/rizhiyi_manager_server/server/conf/app.conf` 文件中的相关配置项。对于 manager agent，需要编辑 `/opt/rizhiyi_manager_agent/manager/conf/config.ini` 文件中的 `enable_https` 配置项。

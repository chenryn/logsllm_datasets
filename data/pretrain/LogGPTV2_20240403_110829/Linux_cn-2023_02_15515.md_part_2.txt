# NeoChat

![neochat](/data/attachment/album/202302/06/163903skig3dnnzic8ytak.png)

NeoChat 是一个简洁的 Matrix 客户端，目前由 KDE 社区积极开发。它专为 Linux 和 Windows 设计，特别适用于 KDE Plasma 桌面环境。当然，你也可以在其他桌面环境中使用它。

NeoChat 可通过 KDE 的“发现”(Discover) 软件中心、Flathub 以及 Snap 商店进行安装。尽管它主要面向桌面用户，但最新版本也支持 Android（可在 F-Droid 中添加 KDE 仓库后下载）和 macOS。因此，对于寻求简单易用的 Matrix 客户端的桌面用户来说，NeoChat 是一个不错的选择。

了解更多详情，请访问其 [源代码仓库](https://invent.kde.org/network/neochat) 或 [官网](https://apps.kde.org/neochat/)。

---

### FluffyChat

![fluffychat](/data/attachment/album/202302/06/163904cw22w4ks7kp5mnht.png)

FluffyChat 是一款美观且直观的 Matrix 客户端，旨在提供愉悦的用户体验。如果你需要一个既支持桌面又支持移动设备（包括安卓和 iOS）的客户端，那么 FluffyChat 将是一个理想选择。

Linux 用户可以从 Snap 商店或 Flathub 下载该应用。虽然它不提供 Windows 和 macOS 的原生应用程序，但用户可以在网页浏览器中使用 FluffyChat。更多相关信息，请访问 [GitLab 页面](https://gitlab.com/famedly/fluffychat) 或 [官方网站](https://fluffychat.im/)。

---

### Fractal

![fractal](/data/attachment/album/202302/06/163904h0x1zbg1b0e50k5g.png)

Fractal 是 GNOME 桌面环境下的一款 Matrix 聊天客户端，采用 Rust 语言编写。这款应用针对大型团队协作进行了优化设计。由于 Fractal 以 Flatpak 形式发布，任何 Linux 发行版均可轻松安装，不受限于特定桌面环境。

若你喜欢响应迅速的应用程序，不妨考虑尝试 Fractal。更多详情，请参考 [GitLab 页面](https://gitlab.gnome.org/GNOME/fractal) 或 [项目主页](https://wiki.gnome.org/Apps/Fractal)。

---

### Hydrogen Web（实验性）

![hydrogen](/data/attachment/album/202302/06/163904d2vja537oo5u9eoj.png)

寻找性能优越的轻量级 Matrix 客户端？Hydrogen 聊天客户端具备离线功能，并兼容多种浏览器，提供了快速流畅的体验。尽管该项目仍在开发中，但由 Element 团队倾力打造，未来值得期待。感兴趣的朋友可以关注其 [GitHub 页面](https://github.com/vector-im/hydrogen-web/) 获取最新进展。

---

### Matrix Commander（基于命令行）

Matrix Commander 是一款专为终端用户设计的 Matrix 网络聊天工具。尽管并非所有功能都可通过命令行实现，但通过创建 cron 任务等方式，仍可满足消息通知、机器人等需求。该工具可在 [PyPi](https://pypi.org/project/matrix-commander/) 或 Docker Hub 上获取。更多信息请访问 [GitHub 仓库](https://github.com/8go/matrix-commander)。

---

### Gomuks（基于命令行）

![gomuks](/data/attachment/album/202302/06/163905bz7yk4k41y0pg1vz.png)

Gomuks 是一款用 Go 语言编写的终端 Matrix 客户端，适合喜欢使用命令行界面的用户。你可以从 [GitHub Releases 页面](https://github.com/tulir/gomuks/releases) 下载适用于 Linux、Windows 和 macOS 的二进制文件。进一步了解请访问 [项目主页](https://maunium.net/go/gomuks/)。

---

### Syphon（Alpha 版）

![syphon](/data/attachment/album/202302/06/163905zho1pinnh17dbhbq.jpg)

Syphon 是一款专为手机设计的 Matrix 客户端，正处于 Alpha 测试阶段。如果你希望在安卓或 iOS 设备上安装类似 Signal 的开源替代品，那么 Syphon 会是个不错的选择。尽管其用户界面与现有应用相似，但绝非完全复制。想要尝鲜的朋友可以前往 [官方网站](https://syphon.org/) 下载试用。

---

## 总结

尽管 Matrix 协议尚未普及至所有组织和个人之中，但它作为开源项目，在构建隐私保护及去中心化网络方面表现卓越。最重要的是，用户可以根据自身喜好自由选择合适的客户端，无需受限于特定应用程序才能跨设备沟通。

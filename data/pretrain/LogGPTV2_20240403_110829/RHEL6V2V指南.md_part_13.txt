::: {.procedure xmlns:d="http://docbook.org/ns/docbook"}
1.  ::: para
    以 root 身份挂载 [**rhel-6.x-p2v.iso**]{.application}
    :::
    ``` screen
    mkdir /mnt/p2vmount
    ```
    ``` screen
    mount -o loop rhel-6.x-p2v.iso /mnt/p2vmount
    ```
2.  ::: para
    在一个终端中以 root 身份运行
    [**livecd-iso-to-pxeboot**]{.application} 脚本：
    :::
    ``` screen
    bash /mnt/p2vboot/LiveOS/livecd-iso-to-pxeboot /PATH/TO/rhel-6.x-p2v.iso
    ```
    ::: para
    当命令完成后，运行命令的目录中会出现一个 tftpboot 目录。
    :::
3.  ::: para
    对新创建的 tftpboot 目录重新命名：
    :::
    ``` screen
    mv tftpboot/ p2vboot/
    ```
4.  ::: para
    把 p2vboot/ 子目录复制到 /tftpboot 目录：
    ``` screen
    cp -R p2vboot/ /tftpboot/
    ```
    设置您的 DHCP、TFTP 和 PXE 服务器来使用
    `/tftpboot/p2vboot/pxeboot.0`{.filename}。
    :::
    ::: note
    ::: admonition_header
    **注意**
    :::
    ::: admonition
    ::: para
    initrd 映像会包括整个 CD ISO。在 PXE 引导时，您会注意到 initrd
    可能需要很长时间来下载，这是一个正常的现象。
    :::
    :::
    :::

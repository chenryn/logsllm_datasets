User: 请列出常见的Linux服务（如SSH、HTTP、FTP），并对它们的功能和端口号进行说明。
Assistant: 1. SSH (Secure Shell)：允许远程连接到服务器并执行命令或管理系统。默认端口号为22。
2. HTTP (Hypertext Transfer Protocol)：用于在客户端和服务器之间传输Web页面和资源。默认端口号为80。
3. HTTPS (HTTP over SSL/TLS)：与HTTP相似，但通过SSL或TLS协议进行加密。默认端口号为443。
4. FTP (File Transfer Protocol)：用于传输文件，允许用户在两个计算机之间进行文件传输。默认端口号为21。
5. SMTP (Simple Mail Transfer Protocol)：用于在邮件客户端和邮件服务器之间传输邮件。默认端口号为25。
6. IMAP (Internet Message Access Protocol)：用于从邮件服务器中检索邮件。默认端口号为143。
7. POP3 (Post Office Protocol version 3)：用于从邮件服务器中检索邮件。默认端口号为110。
8. DNS (Domain Name System)：将域名转换为IP地址以便计算机进行通信。默认端口号为53。
9. DHCP (Dynamic Host Configuration Protocol)：用于自动分配IP地址和其他IP网络参数。默认端口号为67和68。
10. NFS (Network File System)：允许将文件在网络上共享并允许远程客户端访问文件。默认端口号为2049。

### 功能描述
SET 使用内置目录来识别含有敏感信息的表单字段和参数。它以红色高亮显示潜在的用户名和密码字段，提醒攻击者注意这些敏感信息。一旦获取到目标的敏感信息，通过按 `CTRL-C` 可生成 XML 和 HTML 格式的报告。SET 的 Web 服务器是多线程的，能够处理大量请求，并自动将表单字段信息分开存储，以便生成可读报告。此外，还可以将获取的信息导出为 XML 兼容格式，便于后续工具的导入。

## 社会工程学工具包（Social-Engineer Toolkit, SET）指南

### 10.3.4 标签页劫持攻击
在标签页劫持攻击中，当目标用户打开多个浏览器标签页并访问恶意网页时，点击恶意链接后会看到“页面正在加载，请等待”的提示。切换标签页时，恶意网页检测到焦点转移并重写当前页面，提示用户“请等待”正在跳转至目标网站。实际上，用户被引导至一个伪装的钓鱼页面，输入敏感信息后会被重定向至合法网站，从而不知不觉地泄露了敏感信息。此攻击技术可通过 SET 的 Web 攻击向量接口实现。

### 10.3.5 中间人攻击
中间人攻击利用 HTTP Referer 机制或跨站脚本漏洞 (XSS) 将目标用户的敏感信息传递给攻击者的 HTTP 服务器。若发现 XSS 漏洞，发送特制的恶意 URL 给目标用户，点击链接后，虽然网页正常运行，但在登录系统时，敏感信息会被传递给攻击者。中间人攻击向量同样可在 SET 的 Web 攻击向量接口中找到。

### 10.3.6 网页劫持
网页劫持攻击允许创建一个克隆网站并通过声称网站已迁移的链接展示给目标用户。当鼠标悬停在该链接上时，显示的是正常的 URL，但点击后则被替换为恶意克隆网站。例如，克隆 https://gmail.com 时，用户看到的仍然是 https://gmail.com，但实际上已被恶意服务器替代。这种攻击使用基于时间的框架替换技术，确保用户在点击前看到的是合法 URL。攻击者可通过修改配置文件调整启动时间。

### 10.3.7 综合多重攻击方法
综合攻击向量允许攻击者组合多种单一攻击方法，实施多重攻击。配置时，选择 Java Applet 攻击、Metasploit 客户端渗透攻击及网页劫持攻击等，生成超级恶意网页。当目标用户点击链接后，依次执行设定的攻击向量。例如，选择 IE7 客户端渗透攻击，即使目标使用 IE6 浏览器，其他攻击方法也可能成功。最后，选择默认的 Meterpreter 反弹式攻击载荷及编码和监听端口，配置完成即可发起攻击。

### 10.4 传染性媒体生成器
传染性媒体生成器创建一个包含 Autorun.inf 文件的文件夹，可用于 CD/DVD 或 USB 驱动器。插入目标主机后，Autorun.inf 自动加载并执行指定攻击。目前支持加载可执行文件（如 Meterpreter）及文件格式漏洞攻击（如 Adobe 漏洞）。

### 慕尼黑市市长透露重返 Windows 的费用

**作者**: Avishek Kumar  
**分类**: 新闻  
**译者**: wxy  
**审校者**: 无  
**摘要**: 慕尼黑市市长透露了在该市摆脱微软十年之后再次放弃 Linux 重返 Windows 的费用，大约需要数以百万计的欧元。

---

慕尼黑市市长近日披露，若该市决定重新采用 Windows 操作系统，将需要花费上百万欧元购买新的硬件。今年早些时候，新当选的市长提出慕尼黑可能重返 Windows，尽管市当局已经用了若干年才迁移到基于 Linux 的操作系统和开源软件。

作为最著名的从微软迁移到 Linux 桌面系统的案例，慕尼黑投向开源软件的做法一直引发各种争议和讨论。慕尼黑的迁移始于2004年，还有一些德国的地方当局也追随其脚步转向开源。

![](/data/attachment/album/201410/18/110513dwnljnjwnnp3opwj.jpg)

目前还没有制定好返回 Windows 桌面的具体计划，但当局正在调研哪种操作系统和软件包（包括专有软件和开源软件）更适合他们的需求。调研报告还将统计迁移到开源软件所花费的费用。

市长 Dieter Reiter 在回应慕尼黑绿党的问询时透露了重返 Windows 的费用。他表示，迁移到 Windows 7 需要替换超过14,000名职员的所有个人电脑，此举将花费 315万欧元。这还不包括软件许可证费用和基础设施的投入。如果迁移到 Windows 8，费用将更高。

Reiter 还提到，返回微软将导致之前迁移到 Limux、OpenOffice 及其他开源软件所花费的1400万欧元付诸东流。部署 Limux 并从微软 Office 迁移的项目实施、支持、培训、系统修改以及相关软件授权等工作都将被搁置。

他还透露，迁移到 Limux 为市政府节约了约1100万欧元的许可证和硬件费用，因为基于 Ubuntu 的 Limux 操作系统对硬件的要求较低，不需要频繁升级较新版本的 Windows。

在接受 Stadtbild 杂志采访时，Reiter 表示自己是微软粉丝，但这不会影响 IT 审计报告的结果。他强调，用户满意度是关键因素之一，与个人偏好无关。

此外，他还提到了 Windows 和基于 Linux 的操作系统的相对安全性问题。根据德国国家安全局 BSI 的信息，Linux 虽然漏洞更多，但由于使用量较少，实际风险并不一定比 Windows 高。不过他也指出，这种比较可能存在不同的解释。

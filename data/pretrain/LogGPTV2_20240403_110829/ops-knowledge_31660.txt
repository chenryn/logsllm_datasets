User: What are the main procedures an organization should follow in order to implement CIS Control 9.1, Ensure Use of Only Fully Supported Browsers and Email Clients?
Assistant: An organization should Ensure only fully supported browsers and email clients are allowed to execute in the enterprise, only using the latest version of browsers and email clients provided through the vendor.CIS Control 9.1 has the asset type Applications, and has the security function Protect.

在2016年3月30日于美国旧金山莫斯康展览中心召开的 Build 2016 开发者大会上，微软除了介绍自家生态系统的产品和技术外，还为开发者带来了 Ubuntu on Windows 功能。

Bash 是 Linux 和 Unix 上非常流行的命令行 Shell，也是 Ubuntu、RHEL 等 Linux 发行版以及苹果 macOS 操作系统的默认命令行 Shell。

通过 Windows Subsystem for Linux (WSL) 这一新特性，Bash on Ubuntu on Windows 在 Windows 中原生运行大多数 Linux 命令行程序成为可能。

> 其实，在 Windows 7 之前的操作系统中，都包含一个 POSIX 子系统，以便将 UNIX 的程序源代码编译为 Windows 程序。微软为 POSIX 子系统提供了众多的 UNIX 工具，这些工具基于 POSIX 子系统直接使用 GNU 的原生代码编译实现。你也可以在这个 POSIX 子系统中运行 C Shell、Korn Shell 等命令行 Shell。不过，从 Windows 7 以后的操作系统中，默认移除了 POSIX 子系统。如果感兴趣的话，可以在 Windows 7 操作系统中的“启用或关闭 Windows 功能”中启用“基于 UNIX 的应用程序子系统”，即可体验 POSIX 子系统。

启用 WSL 功能后，如果使用 Bash，则 Windows 10 操作系统会下载一个由 Canonical 创建的 Ubuntu 用户模式镜像，然后 Bash 程序以及其他 Linux 二进制程序就可以运行在这个 Ubuntu 镜像上。此外，微软承诺未来会提供更多 Linux 发行版供选择。

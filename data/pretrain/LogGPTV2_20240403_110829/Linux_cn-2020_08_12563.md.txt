### 去中心化的消息应用 Riot 改名为 Element

Riot 是一款基于开源 Matrix 协议的去中心化即时通讯应用。2020年6月底，Riot 宣布将更名为 [Element](https://element.io/)。本文将探讨 Riot 更名的原因以及这一变化带来的其他影响。

#### 为什么从 Riot 改名为 Element？

![Riot 更名公告](/data/attachment/album/202008/29/224635yovu6ss557hsisoz.png)

在深入探讨更名原因之前，我们先回顾一下 Riot 在博客中提到的三个主要原因：

1. **商标问题**：据称，一家大型游戏公司多次阻止 Riot 注册其产品名称和域名（如 [Riot.im](http://Riot.im)）。虽然没有明确指出，但推测这家“游戏公司”可能是 [Riot Games](https://en.wikipedia.org/wiki/Riot_Games)。

2. **品牌形象**：最初选择 Riot 这个名字是为了传达一种破坏性和活力的感觉，但现在担心这个名字会被误解为与暴力有关。因此，他们决定更换一个更加积极正面的名字。

3. **品牌简化**：Riot 由 New Vector 公司创建，并托管在 Modular 平台上，这导致了一系列的品牌混淆。通过改名为 Element，他们希望简化命名系统，使用户更容易识别和记住他们的产品。

#### 元素即一切

![Element 应用界面](/data/attachment/album/202008/29/224641ecvuvqs83q0wqhhc.jpg)

自2020年7月15日起，Riot 的应用名称和公司名称正式改为 Element。他们的 Matrix 托管服务现称为 Element Matrix Services。以下是官方对 Element 的描述：

> “对于那些首次接触我们的用户来说，Element 是 Matrix 通信网络中的旗舰级安全协作应用。它不仅让你拥有自己的端到端加密聊天服务器，还能与更广泛的 Matrix 网络中的其他人连接。”

选择 Element 这个名字是因为它反映了设计 RiotX 时对简洁和清晰的重视。此外，这个名字还强调了他们致力于打造最优雅、最实用的主流通讯应用的决心，并且能够唤起数据所有权和自我主权的概念。

#### 除了改名之外

![Element 客户端界面](/data/attachment/album/202008/29/224651ovu6b8dpxd833lop.jpg)

此次更名不仅仅是名称上的改变。Element 还发布了新一代的安卓版 Matrix 客户端，该客户端前身是 RiotX，现已更名为 Element。新版客户端进行了彻底的重写，支持 VoIP 通话和小部件功能。同时，Element 也在 iOS 平台上推出，支持 iOS 13，并提供全新的推送通知支持。

Element Web 客户端也得到了更新，包括 UI 优化、新的易读字体以及改进的房间列表控件。这些改进包括增加房间预览、按字母顺序排列、可调整列表大小及改进的通知界面等。此外，他们还在努力提升端到端加密性能。

#### 最后的思考

尽管短期内可能会因为改名而失去部分用户，但从长远来看，品牌简化有助于 Element 在竞争激烈的市场中脱颖而出。值得一提的是，这是该应用历史上第三次更名：2016年发布时命名为 Vector，同年晚些时候改为 Riot。希望这次更名后，Element 能够长久地使用下去。

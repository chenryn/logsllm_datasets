[Ubuntu Studio](https://ubuntustudio.org/) 是一个流行的 [Ubuntu 官方变种](https://itsfoss.com/which-ubuntu-install/)，专为从事音频制作、视频、图形、摄影和书籍出版的创意内容创作者设计。它提供了多种多媒体内容创作应用程序，开箱即用，体验极佳。

在最近发布的 20.04 LTS 版本之后，Ubuntu Studio 团队在其[官方公告](https://ubuntustudio.org/2020/04/ubuntu-studio-20-04-lts-released/)中强调了一些重要的信息。其中关键的一点是关于 Ubuntu Studio 的未来方向。

Ubuntu Studio 20.04 将是最后一个使用 [Xfce 桌面环境](https://xfce.org)的版本。未来的所有发行版都将改用 [KDE Plasma](https://kde.org/plasma-desktop)。

### 为什么 Ubuntu Studio 放弃 Xfce？

![](/data/attachment/album/202005/16/113713pl190wbu8788rq9q.jpg)

根据团队的解释，Ubuntu Studio 并不专注于任何特定的外观或感觉，而是致力于提供最佳用户体验。经过评估，KDE 被认为是一个更好的选择。

> 实践证明，Plasma 为图形艺术家和摄影师提供了更强大的工具，例如在 Gwenview、Krita 和文件管理器 Dolphin 中都能体现出来。此外，它对 Wacom 平板的支持也优于其他桌面环境。
>
> 目前，大部分 Ubuntu Studio 团队成员都在日常工作中使用通过 Ubuntu Studio 安装程序添加了 Ubuntu Studio 的 Kubuntu。鉴于我们中的许多人都已经在使用 Plasma，因此在下一个版本中过渡到 Plasma 是一个合理的选择。

当然，每个桌面环境都有其独特的特点，适用于不同的用户需求。在此背景下，他们认为 KDE Plasma 是取代 Xfce 的最佳选择，能够为所有用户提供更好的体验。

尽管用户的反应可能因个人偏好而异，但如果现有用户对 KDE 没有异议，这一转变应该不会带来太大问题。

值得注意的是，Ubuntu Studio 还提到了为何 KDE 可能是更好的选择：

> 在没有 Akonadi 的情况下，Plasma 桌面环境的资源使用与 Xfce 一样轻量级，甚至更轻。像 Fedora Jam 和 KXStudio 这样以音频为重点的 Linux 发行版历史上一直使用 KDE Plasma，并且在音频方面表现良好。

此外，他们还引用了 [Jason Evangelho 在福布斯杂志上的一篇文章](https://www.forbes.com/sites/jasonevangelho/2019/10/23/bold-prediction-kde-will-steal-the-lightweight-linux-desktop-crown-in-2020)，其中一些基准测试表明 KDE 几乎与 Xfce 一样轻量级。尽管这是一个积极的信号，但最终还需要等待用户试用 KDE 驱动的 Ubuntu Studio，才能确定这一决定是否正确。

### 更改后，Ubuntu Studio 用户将面临什么变化？

在带有 KDE 的 Ubuntu Studio 20.10 及更高版本中，整体工作流程可能会受到影响（或改善）。然而，从 20.04 升级到 20.10 可能会导致系统损坏。因此，全新安装 Ubuntu Studio 20.10 或更高版本将是唯一可行的方法。

团队还提到，他们将持续评估预装应用是否存在重复。相信在未来会有更多细节公布。

Ubuntu Studio 是最近一段时间内第二个改变主桌面环境的发行版。此前，[Lubuntu](https://itsfoss.com/lubuntu-20-04-review/) 从 LXDE 切换到了 LXQt。

User: How to check file system usage in Linux? Please list common file system management commands and parameters.
Assistant: 1. View the current disk space and usage command: df -h Parameter description: -h: display usage in a human-readable manner 2. View the space usage command of the specified directory: du -sh /path/to/directory/ Parameter description: -s: only display the total usage -h: display the usage in a human-readable way 3. View the size of all files and folders in the current directory: du -sh * parameter description: -s: only display Total usage -h: display usage in a human-readable way *: wildcard, representing all files and folders 4. Delete a file or folder Command: rm -rf /path/to/file_or_folder Parameter description: -r: Recursive deletion -f: Forced deletion without prompt 5. Backup file or folder command: tar -cvzf /path/to/back/up.tar.gz /path/to/source Parameter description: -c: Create a new backup file -v: display the backup process -z: compress the backup file -f: specify the path and file name of the backup file 6. Restore the backup file command: tar -xvzf /path/to/back/up.tar.gz -C /path /to/restore parameter description: -x: restore data from backup file -v: display restore process -z: decompress backup file -f: specify path and file name of backup file -C: specify directory to restore data 7. View disk partition and usage command: fdisk -l parameter description: -l: list all partitions 8. Format disk command: mkfs.ext4 /dev/sda1 parameter description: ext4: formatted file system type /dev/sda1 : The disk or partition to be formatted 9. Command to mount the partition: mount /dev/sda1 /mnt Parameter description: /dev/sda1: The partition to be mounted /mnt: Mount point 10. Command to unmount the partition: umount /mnt parameter description: /mnt: mount point

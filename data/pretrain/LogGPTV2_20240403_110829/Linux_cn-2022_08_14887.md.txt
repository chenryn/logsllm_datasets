### Linus 在 Macbook 上发布了 Linux 内核 5.19

今天，Linus Torvalds 发布了 Linux 5.19。新版本增加了对龙芯 CPU 和 Zen 4 CPU 的支持，并引入了 Zstd 压缩固件的支持，进一步提升了性能。有趣的是，此次发布是在他的苹果 MacBook 笔记本上完成的，这台笔记本运行了 Asahi Linux 为苹果 M1 Macbook 打造的 Linux 系统。Linus 表示：“这是我期待已久的事情，现在终于实现了。”他曾多次使用苹果硬件进行 Linux 开发，但目前他仅将 MacBook Air 用于版本测试、启动和发布工作。

**消息来源：[Phoronix](https://www.phoronix.com/news/Linux-5.19-Released)**

**老王点评：看来 Linus 对 Asahi Linux 的表现非常满意。我也在考虑是否要将我的 M1 Macbook 换成 Asahi Linux。**

![](/data/attachment/album/202208/01/200544oo9h326e6w6is6u9.jpg)

### Linux 内核可能跳过版本 5.20 直接进入 6.0

正在开发中的 Linux 5.20 内核很可能被称为 Linux 6.0。根据 Linus Torvalds 对现代版本号的划分，Linux 又到了采用下一个主要版本号的时候。通常当小版本更新到 19 或 20 时，他会提升主要版本号。例如，Linux 3.19 之后是 Linux 4.0，而 Linux 4.20 之后是 Linux 5.0。在周日发布的 Linux 5.19 公告中，Linus 表示：“我可能会将其称为 6.0，因为我开始担心再次被大数字迷惑。”

**消息来源：[Phoronix](https://www.phoronix.com/news/Linux-5.20-Is-Linux-6.0)**

**老王点评：其实 Linux 内核的大版本号并没有特别的意义，只是进位而已。你可以把内核版本号看作是 20 进制的。有时候甚至会忘记进位。**

![](/data/attachment/album/202208/01/200654tax48nximqizvbn4.jpg)

### Python 开发者讨论从邮件列表转移到论坛

Python 项目自 2018 年起开始尝试使用论坛作为交流平台。作为实验的一部分，Python 核心开发者使用的邮件列表 python-committers 切换到了 Discourse 论坛。尽管该邮件列表仍然存在，但主要用于发布公告，大部分讨论已转移到论坛上。一位开发者表示，由于不常访问 Discourse，他错过了感兴趣的新 PEP 讨论，希望新 PEP 能继续发布在 python-dev 邮件列表上。这一提议引发了讨论。最终，Python 语言指导委员会决定将讨论媒介切换到 Discourse 论坛，Python 的邮件列表可能将全面转向论坛。

**消息来源：[Solidot](https://www.solidot.org/story?sid=72316)**

**老王点评：虽然许多老程序员习惯了邮件列表，但与时俱进地采用更丰富、更现代的交流方式是必要的。同时，可以保留旧的交流方式，直到它不再被广泛使用或因缺乏重要功能而被废弃。**

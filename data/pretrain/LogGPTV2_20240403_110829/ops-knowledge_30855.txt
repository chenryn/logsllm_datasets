User: How to use the iptables command of Linux to configure the network firewall?
Assistant: iptables is a command used in the Linux operating system to configure and manage network firewalls. The following are the steps to configure the network firewall: 1. View iptables rules: Execute the following command to view all rules in the system: ``` iptables -L -v ``` 2. Clear all rules: Execute the following command to delete all rules: ` `` iptables -F ``` 3. Set the default policy: Execute the following command to set the default policy: ``` iptables -P INPUT DROP //reject all input iptables -P OUTPUT ACCEPT //allow all output iptables -P FORWARD DROP //Reject all forwarding``` 4. Allow local access: Execute the following command to allow local access: ``` iptables -A INPUT -i lo -j ACCEPT ``` 5. Allow specific IP address access: Execute the following Command to allow specific IP addresses to access: ``` iptables -A INPUT -s 192.168.1.100 -j ACCEPT ``` 6. Allow specific protocols and ports: Execute the following commands to allow specific protocols and ports: ``` iptables -A INPUT -p tcp --dport 22 -j ACCEPT //Allow SSH to access iptables -A INPUT -p udp --dport 1194 -j ACCEPT //Allow OpenVPN access``` 7. Defense DDoS attack: Execute the following iptables command to Limit the maximum number of client connections: ``` iptables -A INPUT -p tcp --syn --dport 80 -m connlimit --connlimit-above 20 --connlimit-mask 32 -j DROP ``` The above are only iptables commands For the basic use of iptables, see the iptables command manual for more details.

### 优化后的文本

许多人误以为应用商店和手机应用程序是苹果公司的首创，但实际上，初代 PalmPilot 已经拥有了一个非常早期的应用商店。这是一个第三方平台，开发者们在这里创作了大量创新性的应用程序，并通过该商店将这些应用同步到设备上。

**Saron Yitbarek**：
他们的目标是在第一年销售10万台设备。在最初的六个月里，月销量稳定在大约1万台左右。但随后，销量开始迅速增长。

**Donna Dubinsky**：
实际上，在短短18个月内，我们售出了100万台设备，这成为了当时美国历史上增长最快的产品之一。能够在18个月内生产出100万台设备，这确实是一个惊人的成就。

**Saron Yitbarek**：
Palm 公司开创了一个全新的硬件类别，本应成为这一领域的领导者。然而，Palm 的母公司 US Robotics 却被另一家公司 3Com 收购。受微软商业模式的影响，3Com 决定将 Palm OS 授权给其他想要开发自己的个人数字助理（PDA）的公司。

**Donna Dubinsky**：
显然，这是微软在个人电脑领域取得成功的策略。但我们并不认为这种模式适用于掌上设备。我们认为掌上设备需要高度集成的设计。然而，他们始终认为这是一个错误的决定，坚持认为我们应该将操作系统授权给所有商业伙伴。我们无法认同他们的观点。

**Saron Yitbarek**：
他们坚信自己的愿景。因此，在帮助母公司革新市场后，Jeff Hawkins、Donna Dubinsky、Ed Colligan 和 Rob Haitani 等人离开了 Palm，成立了一家新公司——Handspring。他们获得了自己创建的 Palm OS 的许可，并将其安装到了新的 PDA 上。他们曾创造过巨人，现在打算以弱胜强，用“大卫与歌利亚”的方式打败它。1999年，Handspring 成立并发布了名为 Visor 的全新系列 PDA，内置 Palm OS。Ed Colligan 还记得这款新产品大受欢迎。

**Ed Colligan**：
果然，一夜之间我们就占据了25%的市场份额。

**Saron Yitbarek**：
Palm 公司的硬件销量受到了这一举动的冲击。讽刺的是，正是 3Com 决定将 Handspring 拆分为独立公司的决定，使得这一局面得以实现。尽管如此，当时超过90%的掌上设备运行的是 Palm 公司的操作系统。事实上，有一段时间，Palm 的市值甚至超过了福特和通用汽车的总和。人们认为它会成为下一个微软。当然，Handspring 也有自己的计划。

**Donna Dubinsky**：
成立 Handspring 后，我们意识到掌上设备的终极形态将是通信设备。因此，我们在产品上增加了一个硬件插槽，用于实验和整合多种类型的通信模块，如呼机卡或语音卡。这样我们可以从中学习，深入了解通信领域。

**Ed Colligan**：
未来属于智能手机，所有这些技术都在朝着无线化发展。我们的下一步是将 PDA 和手机功能结合起来。

**Saron Yitbarek**：
于是，Handspring 开始研发智能手机。他们用键盘取代了手写笔，并将新产品命名为 Treo。Jeff、Donna 和 Ed 一起会见了另一位科技企业家——史蒂夫·乔布斯。

**Ed Colligan**：
史蒂夫在白板上画了一台 Macintosh 计算机，并在其周围像卫星一样画出了照片、视频等附加功能。他说：“我们的战略是让 Macintosh 成为一切的核心。iTunes、iPhoto 等所有功能都围绕它设计。” Jeff 反驳道：“不，人们将会拥有一台掌上设备，所有这些东西都将以它为中心。”

**Saron Yitbarek**：
我们知道最终结果如何。Jeff Hawkins 的观点更接近事实，但在21世纪初，乔布斯和其他行业人士对此持怀疑态度。

**Ed Colligan**：
我曾去游说 Sprint 和 Verizon 等公司，试图说服他们未来属于智能手机。虽然今天看来难以置信，但当时他们的反应就像是在说：“嗯，这是一种新奇的设备” 或者 “谁会在口袋里发邮件呢？” 我则坚称：“我相信这将成为现实。”

**Saron Yitbarek**：
在等待智能手机时代到来的同时，他们面临了一个更紧迫的问题。2001年，科技泡沫破裂，股市暴跌，资金短缺，投资枯竭。这威胁到了 Handspring、Palm 以及几乎所有产品的未来。为了制造 Treo，Handspring 再次合并回 Palm 公司。尽管经历了多次变动，Treo 最终成为市场上最受欢迎的智能手机。然而，那时 Palm OS 已经显示出老化迹象。新的竞争对手如 RIM 公司的黑莓手机也进入了市场。

**Saron Yitbarek**：
当无线技术成为潮流时，专家们开始质疑 Palm 操作系统是否适合下一代设备。因此，在2005年，Palm 公司发布了第一款不使用 Palm 操作系统的 Treo，转而采用了 Windows Mobile 操作系统。到2007年，Palm 已经成为一个没有自研操作系统的纯硬件公司。Palm 团队所设想的未来仍在继续发展，但似乎不再需要与 Palm 公司同行。

**Saron Yitbarek**：
正当 Palm 公司需要帮助时，John Rubinstein 挺身而出。他在苹果公司开发了 iPod，并在苹果公司发布 iPhone 时加入了 Palm 公司，担任其产品开发的新主管。两年后，Palm 团队推出了他们的新设备：Palm Pre 和一款名为 Web OS 的新操作系统。它们于2009年在消费电子展（CES）上亮相。有人认为这是有史以来最好的科技主题报告之一。Ed Colligan 在活动中表示：

**Ed Colligan**：
它叫做 Palm Web OS，我们非常高兴今天能把它带给大家。它是为开发人员而构建的，基于行业标准的 Web 工具。只要您了解 HTML、CSS 和 JavaScript，就可以为这个平台开发应用程序。

**Saron Yitbarek**：
没有人见过像 Web OS 这样的东西，它奠定了现代智能手机体验的基础。实际上，iOS 和 Android 都从中学到了很多。Web OS 的功能包括多个同步日历、统一的社交媒体和联系人管理、曲面显示器、无线充电、集成文本和 Web 消息、非侵入式通知等。只需将其置于开发模式即可轻松升级，并且还支持无线更新。尽管 Web OS 是一项伟大的成就，但仍然不够。

**Ed Colligan**：
我认为我们在某些方面做得很好，但还是太少太迟了，因为那时苹果已经发布了 iPhone。他们执行得又快又好，所以市场被他们占领了。但我认为我们的设计对他们影响深远。直到今天，iPhone 和 macOS 才真正赶上了 Web OS 的所有功能。

**Saron Yitbarek**：
然而，Ed 认为真正的杀手锏是另一部手机。

**Ed Colligan**：
导致他们失败的杀手锏其实是谷歌和 Android。凭借搜索业务的支持，谷歌不需要通过卖手机赚钱。

**Saron Yitbarek**：
谷歌基本上免费提供了 Android，这对微软的 Windows Phone 和 Palm Pre 的 Web OS 来说是个大问题。

**Ed Colligan**：
我们没有那样的商业模式，这对我们造成了巨大的影响，导致我们再也无法恢复过来。

**Saron Yitbarek**：
从 PalmPilot 创造了一个全新的技术类别，到 Palm OS 主导移动软件，再到打造第一款智能手机 Treo，以及用 Web OS 重塑移动操作系统，经过所有这些创新和迭代之后，Palm 公司在2010年被惠普收购，后来又被 LG 收购。2012年，惠普发布了基于 Linux 的开源 Web OS。

**Saron Yitbarek**：
一旦开源，Web OS 就成为其他智能设备、电视、手表和物联网的底层操作系统。关于硬件和软件融合的争论，让我们听听 Donna Dubinsky 的看法。

**Donna Dubinsky**：
硬件和软件实际上是不可分割的。你不能在好硬件上运行坏软件，也不能用坏硬件来支持好软件。这个问题几乎毫无意义，两者必须保持一致。这些随身携带的软硬件高度集成在一起，人们甚至不知道硬件在哪里结束，软件在哪里开始。

**Saron Yitbarek**：
在 Jeff Hawkins 的故事中，无论是硬件还是软件，都是从他衬衫口袋里的那块小木块开始的。25年后，这块小木块演变成了数百万乃至数十亿部智能手机。

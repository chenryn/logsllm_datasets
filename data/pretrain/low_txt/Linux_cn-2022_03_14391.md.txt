### OpenJDK 18 正式发布

Oracle 公司今日 [正式发布了](https://jdk.java.net/18/) Java 18 的参考实现 JDK/OpenJDK 18。OpenJDK 18 现已可用于生产环境，并引入了多项新功能及数百个较小的改进和一千多个错误修复。其中，UTF-8 已成为 Java SE API 的默认字符集。此外，还包含了一个名为 Jwebserver 的简单 Web 服务器，这是一个新的命令行工具，用于启动基于 Java 的最小静态 Web 服务器。

> 老王点评：时间飞逝，Java 已经发展到第 18 版本了。

### 一名 16 岁英国少年被疑为 Lapsus$ 黑客组织的主谋

据[彭博社报道](https://www.bloomberg.com/news/articles/2022-03-23/teen-suspected-by-cyber-researchers-of-being-lapsus-mastermind?sref=ylv224K8)，网络安全研究人员在调查 Lapsus$ 黑客组织针对微软和英伟达等科技公司的攻击时，发现这些攻击与一位居住在英国牛津附近的 16 岁少年有关，怀疑他是该组织的主谋。他的个人信息，包括住址及其父母的信息，已被对手公布在网上。Lapsus$ 的另一名成员则疑似是一名居住在巴西的青少年，可能还有其他人员参与了该组织的活动。

> 老王点评：真是“无所畏惧”的年轻人，但令人惊讶的是，一些世界级的大公司竟因此被攻破。

![](/data/attachment/album/202203/24/172155n9lb2ob96fnly9lh.jpg)

### 微软采用新的机器学习技术改善其 AI 翻译

[微软宣布](https://techcrunch.com/2022/03/22/microsoft-improves-its-ai-translations-with-z-code/) 将基于 Z-Code 项目对其翻译服务进行更新，该项目采用了“混合专家”方法。这些新模型在盲评中得分提高了 3% 至 15%。“混合专家”并不是一项全新的技术，但在翻译领域特别有用。系统将任务分解成多个子任务，并将其分配给更小、更专业的模型（即“专家”）。随后，系统根据预测结果决定将哪些任务委托给哪个“专家”。

> 老王点评：坦白说，无论是谷歌还是微软的必应，它们的翻译质量仍然有待提高，有时甚至显得荒谬。

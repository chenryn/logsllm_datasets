# Environment
- **Windows Build Number:** [Please run "ver" at a command prompt to obtain this information]
- **PowerToys Version:** 0.18.1
- **PowerToy Module (if applicable):** PowerToys Run

# Steps to Reproduce
1. Search for a program shortcut located in the Documents folder using PowerToys Run.

# Expected Behavior
- The program shortcut should appear in the search results. In the previous version, a program shortcut stored in `C:\Users\x\Documents\` was successfully indexed and could be launched via PowerToys Run.

# Actual Behavior
- The program shortcut is not included in the PowerToys Run search results.

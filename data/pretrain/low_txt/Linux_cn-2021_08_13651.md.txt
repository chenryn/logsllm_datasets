### 美国国税局本财年已查获价值 12 亿美元的加密货币

![](/data/attachment/album/202108/05/170931a5u77cdv6xce7djd.jpg)

美国政府定期拍卖其扣押的比特币、以太坊、莱特币等加密货币。一旦案件结案，这些扣押的加密货币会被兑换成法定货币。根据美国国税局的数据，在 2019 财年，他们扣押了价值约 70 万美元的加密货币；到 2020 年，这一数字上升至 1.37 亿美元；而截至 2021 年目前，已经达到了 [12 亿美元](https://www.cnbc.com/2021/08/04/irs-has-seized-1point2-billion-worth-of-cryptocurrency-this-year-.html)。随着网络犯罪活动的增加，预计美国政府的加密货币库存将继续增长。

> 这确实是一条“生财之路”，当然，那些用于非法用途的加密货币也理应受到这样的惩罚。

### Rust 连续第六年成为 Stack Overflow 用户最喜欢的语言

![](/data/attachment/album/202108/05/171015k1wvz1pi5wj15f10.jpg)

编程问答网站 Stack Overflow 发布的[调查](https://insights.stackoverflow.com/survey/2021#technology-most-loved-dreaded-and-wanted)显示，Rust 已经连续第六年被评为最受用户喜爱的编程语言，紧随其后的是 Clojure、TypeScript、Elixir、Julia、Python 和 Dart。其他调查结果包括：

- Redis 连续第五年成为最受欢迎的数据库。
- AWS 是最受欢迎和最常用的云计算平台。
- Svelte 是最受欢迎的 Web 框架。
- Tensorflow 是程序员最希望掌握的库。
- Pytorch 是最受喜爱的库。
- Git 是绝大多数程序员都会使用的工具。
- JavaScript 是最常用的编程语言。
- MySQL 是最常用的数据库。
- React.js 是最常用的 Web 框架。
- Visual Studio Code 是最常用的集成开发环境（IDE）。

> Rust 的受欢迎程度正逐渐接近巅峰。

### 美国国务院和其他三个美国机构的网络安全等级为 D 级

![](/data/attachment/album/202108/05/171033iibgrqi24r1aagqz.jpg)

美国参议院委员会周二发布的一份[报告](https://www.hsgac.senate.gov/imo/media/doc/Federal%20Cybersecurity%20-%20America's%20Data%20Still%20at%20Risk%20(FINAL).pdf)指出，八个联邦机构的网络安全状况非常糟糕。其中四个机构获得了 D 级评分，三个机构获得 C 级，仅有一个机构获得 B 级。这八个部门分别是：美国社会安全局、国土安全部、国务院、交通部、住房和城市发展部、农业部、卫生与公共服务部以及教育部。两年前，另一份[报告](https://www.hsgac.senate.gov/imo/media/doc/2019-06-25%20PSI%20Staff%20Report%20-%20Federal%20Cybersecurity%20Updated.pdf)同样发现这些联邦机构在遵守联邦网络安全标准方面存在系统性问题。检查中发现的问题长期困扰着联邦机构。

> 我们必须以此为鉴，加强自身的网络安全防护措施。

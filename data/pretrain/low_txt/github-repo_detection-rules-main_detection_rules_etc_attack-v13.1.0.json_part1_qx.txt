**Bumblebee 数据传输**
- **描述**: Bumblebee 恶意软件能够以 JSON 格式将收集到的数据发送至命令与控制（C2）服务器。
- **引用来源**: Stolyarov, V. (2022年3月17日). Exposing initial access broker with ties to Conti. 从2022年8月18日检索。[链接](https://blog.google/threat-analysis-group/exposing-initial-access-broker-ties-conti/)

**APT28 使用工具进行信息窃取**
- **描述**: APT28 使用一种工具，该工具通过感染的 USB 设备从隔离网络中的计算机捕获信息，并在插入联网计算机时转移这些信息。
- **引用来源**: Anthe, C. 等人 (2015年10月19日). Microsoft Security Intelligence Report Volume 19. 从2015年12月23日检索。[链接](http://download.microsoft.com/download/4/4/C/44CDEF0E-7924-4787-A56A-16261691ACE3/Microsoft_Security_Intelligence_Report_Volume_19_English.pdf)

---

**监控获取 root 权限的过程**
- **描述**: 监控可能获取 root 权限的过程，允许它们读取安全守护程序的安全内存，然后扫描内存以找到正确的密钥序列，在相对较少的尝试中解密用户的登录密钥链。

**Lazarus Group 在 Operation Dream Job 中使用恶意宏**
- **描述**: 在 Operation Dream Job 行动中，Lazarus Group 执行了一个用 VBA 编写的恶意宏，受害者下载了恶意 DOTM 文件；此外，他们还使用 Visual Basic 宏代码提取了双重 Base64 编码的 DLL 植入物。
- **引用来源**: 
  - Cashman, M. (2020年7月29日). Operation North Star Campaign. 从2021年12月20日检索。[链接](https://www.mcafee.com/blogs/other-blogs/mcafee-labs/operation-north-star-a-job-offer-thats-too-good-to-be-true/?hilite=%27Operation%27%2C%27North%27%2C%27Star%27)
  - ClearSky Research Team. (2020年8月13日). Operation 'Dream Job' Widespread North Korean Espionage Campaign. 从2021年12月20日检索。[链接](https://www.clearskysec.com/wp-content/uploads/2020/08/Dream-Job-Campaign.pdf)

### WSL2 可以运行在 Windows Server 2022 上了

微软已正式宣布，WSL 2 发行版现在支持 Windows Server 2022。这对有特殊需求的开发者来说是个好消息，因为他们可以在 Windows Server 2022 上启动 Linux 而无需依赖完整的 Hyper-V 会话。WSL2 更加轻量级。此前，微软曾表示，尽管该技术对桌面用户非常有用，但在服务器上使用它可能会导致其被用于非预期用途。此外，微软没有计划将 WSL2 移植回 Windows Server 2019。

> **[消息来源：The Register](https://www.theregister.com/2022/06/22/wsl2_windows_server/)**
>
> **老王点评**: 微软再次向 Linux 展开攻势，不仅面向开发者，这次还瞄准了运维人员。

![](/data/attachment/album/202206/24/145301frrzhjecgbfxpj6t.jpg)

### 美国参议员将加密货币监管法案草案发布到 GitHub 上

本周三，美国参议员 Cynthia Lummis 对即将出台的加密货币监管法案采取了新措施，将其托管在 GitHub 上，邀请公众对草案提出意见。GitHub 用户已经提出了几十条评论和拉取请求，其中一些提出了有意义的补充。例如，建议通过征税来增加工作量证明加密货币的价值。另一条评论则表达了对稳定币算法支持的担忧。

> **[消息来源：GitHub](https://github.com/responsible-financial-innovation-act22/RFIA-bill)**
>
> **老王点评**: 这才是真正的征求意见，以开源的方式透明地推动进步。

![](/data/attachment/album/202206/24/145322l2f4y4myej5fy4fm.jpg)

### 首个全链路国产操作系统 OpenCloudOS 发布

根据官方公众号的消息，OpenCloudOS 已正式发布了“首个源社区（L1）项目及首个全量软件包（L3）版本”，并公布了技术研发路线图。OpenCloudOS 是去年12月由腾讯牵头成立的开源操作系统社区，成员包括北京红旗、飞腾、浪潮、龙芯中科、兆芯等多家操作系统生态厂商及用户。OpenCloudOS 宣称是“我国首个具备全链路国产化能力的服务器操作系统”，并提出了 L1 源社区、L2 商业版、L3 社区稳定版和 L4 衍生发行版的层级划分。据称，OpenCloudOS 的社区及衍生版已经装机超过千万台。

> **[消息来源：OpenCloudOS](https://mp.weixin.qq.com/s/YD-r1dT8JOshfsIJDvOBlQ)**

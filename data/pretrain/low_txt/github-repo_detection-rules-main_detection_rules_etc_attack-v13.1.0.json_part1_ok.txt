### Web Service 检测
**描述：**
一旦对手利用 Web 服务作为基础设施（例如用于命令和控制），可能可以通过查找已知的对手软件的独特特征来进行检测。（引用来源：[ThreatConnect Infrastructure Dec 2020](https://threatconnect.com/blog/infrastructure-research-hunting/)）许多此类活动将发生在目标组织视线之外，使得检测这种行为变得困难。检测工作可以集中在对手生命周期的相关阶段，例如在命令和控制（[Web 服务](https://attack.mitre.org/techniques/T1102)）或[通过 Web 服务进行数据外泄](https://attack.mitre.org/techniques/T1567)。

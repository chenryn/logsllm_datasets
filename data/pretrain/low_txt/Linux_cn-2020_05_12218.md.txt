### 华为否认提交给 Linux 内核的不安全补丁 HKSP 来自官方

上周，一名华为工程师向 Linux 内核提交了一个名为 HKSP（华为内核自我防护）的内核强化补丁。然而，在代码审查过程中，发现该补丁存在安全漏洞。根据 ZDNet 的报道，华为在周一发布声明称，尽管该项目名称中使用了“华为”，并且由其一名顶级安全工程师开发，但华为并未正式参与 HKSP 项目。华为还表示，该项目代码从未在任何正式的华为产品中实际使用过：“这只是个人用于与开源社区 Openwall 进行技术讨论的演示代码。”

**来源**: [开源中国](https://www.oschina.net/news/115632/huawei-denies-involvement-in-buggy-linux-kernel-patch-proposal?tdsourcetag=s_pctim_aiomsg)

> **硬核老王点评**: 本来是一个普普通通的代码质量问题，但在某些人的解读下却变得复杂起来。不过，这样的个人项目冠以公司名义的做法在华为是被允许的吗？

### 阿里云自研数据仓库 AnalyticDB 再夺 TPC 全球冠军

5月14日，TPC 官方网站宣布，阿里云自主研发的 AnalyticDB 通过了 TPC-DS 全流程测试，将前世界纪录的性能提升了29%，并将单位成本降低了三分之二，成功夺得全球数据仓库的桂冠。

**来源**: [CSDN](https://mp.weixin.qq.com/s/rpZ8gbEw-Ak9_7Sc2eQShg)

> **硬核老王点评**: 似乎对阿里云热衷于赢得各种比赛的冠军已经有点麻木了。

### 腾讯因疫情收入大幅增长

新冠病毒疫情推动了“宅经济”的发展。中国最大的游戏和互联网巨头腾讯控股周三公布，第一季度净利润同比增长6%，而非国际财务报告准则（Non-GAAP）净利润增幅更是达到了29%。内地防疫居家隔离措施带动了网络游戏及网络广告收入的增长。然而，随着各地复工复产，腾讯预计游戏用户使用时长及消费活动将恢复到正常水平。尽管如此，公司相信游戏行业已从结构上扩大了其长期受众及吸引力。

**来源**: [solidot](https://www.solidot.org/story?sid=64354)

> **硬核老王点评**: 尽管不能说游戏本身是邪恶的，但一家公司因这种情况带来的收入大幅增长，总让人感觉有些不对劲。当然，曾经有公司信誓旦旦不做游戏，最终还是投资了游戏行业。其实，公司都是为了盈利而生，这也不足为奇。只希望这些公司在追求利润的同时不要忘记社会责任感。

### Deno 1.0 发布

Deno 是 Ryan Dahl 在 Node.js 之后的又一力作，它是一个新的运行时环境，用于在 Web 浏览器之外执行 JavaScript 和 TypeScript，采用 Rust 编写而成。Deno 旨在提供一个独立的工具来快速编写复杂的脚本，并且始终是一个单一的可执行文件。就像 Web 浏览器一样，它知道如何获取外部代码。在 Deno 中，单个文件可以定义任意复杂的行为，而无需其他工具。Ryan Dahl 认为他在设计 Node.js 时犯了一些错误，包括安全性、构建系统、package.json、node_modules、index.js 等等，并表示 Node.js 存在的问题导致了许多严重问题且无法避免。鉴于当前 JavaScript 及其周边软件基础设施的巨大变化，值得进行简化，于是他重新设计了 Deno。

**来源**: [开源中国](https://www.oschina.net/news/115625/deno-1-0-released)

> **硬核老王点评**: 确实，Node.js 在取得成功的同时也暴露出一些先天不足。这个新项目值得期待。

### Windows 10 应用商店冷清

无论是 iOS 还是 Android，系统内置的应用商店都十分受欢迎。然而，在 Windows 10 上情况却截然不同——自带的应用商店门可罗雀。至今，绝大多数用户在 Windows 10 中安装软件的主要方式仍然是自行下载安装，Windows 10 应用商店似乎已成为一个鸡肋般的存在。

**来源**: [太平洋电脑网](https://www.cnbeta.com/articles/tech/978733.htm)

> **硬核老王点评**: 相比之下，Linux 桌面的应用商店（软件中心）或许更受用户欢迎。

### CreepRank 算法帮助谷歌 Play 商店剔除 813 款 Android 蠕虫应用

纽约大学、康奈尔科技学院以及 NortonLifeLock（原赛门铁克）的研究人员对这些蠕虫软件进行了深入分析。这里提到的蠕虫特指不具备完整间谍或追踪功能的移动应用。即便如此，恶意开发者仍可以通过这些应用对用户进行间接追踪、骚扰、欺诈或威胁。研究团队开发的 CreepRank 算法能够识别移动应用中的疑似蠕虫行为，并对其进行评分。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/978495.htm)

### UCloud 发布物联网边缘网关 IoT Edge，实现边云协同

用户可以将 UIoT Edge 运行系统直接安装到符合要求的 X86 或 ARM 硬件网关上。该系统实现了子设备数据的采集、解析、清洗、加工、缓存，本地场景的实时控制与联动，广泛应用于工业制造、能源电气、智慧社区、物业楼宇、智能农业、新零售等领域。UIoT Edge 边缘网关提供了子设备接入、函数计算、消息路由、一键部署/远程运维和本地 Web Portal 五大功能。目前，UIoT Edge 已进入公测阶段。

**来源**: [UCloud](https://mp.weixin.qq.com/s/B6JNQvkyOca25o1V6hIMHQ)

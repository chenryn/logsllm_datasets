### Issue Description

When using PowerToys Run to search for files and folders, the functionality works as expected when launched without administrative privileges. However, if PowerToys Run is launched with administrative privileges, all file and folder search queries return no results. Other search functionalities, such as searching for the calculator or applications, continue to work correctly.

### Environment Details
- **Windows Build Number:** 10.0.19624.1000
- **PowerToys Version:** v0.18.1
- **PowerToy Module:** Run

This issue specifically affects the "Run" module of PowerToys.

**ATT&CK ID:** G0034  
**ATT&CK Technique Name:** Sandworm Team

The Sandworm Team has exploited several vulnerabilities, including:

- **Microsoft PowerPoint via OLE Objects (CVE-2014-4114):** This vulnerability allows the execution of arbitrary code through specially crafted OLE objects embedded in PowerPoint files.
- **Microsoft Word via Crafted TIFF Images (CVE-2013-3906):** This vulnerability enables the execution of arbitrary code by embedding maliciously crafted TIFF images in Word documents.

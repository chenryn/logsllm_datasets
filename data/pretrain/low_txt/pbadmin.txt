### pbadmin

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2003
- Windows Server 2003 R2
- Windows Server 2003 with SP1
- Windows Server 2003 with SP2

**Important Note:**
This command has been deprecated.

**Description:**
The `pbadmin` command is used to manage phone books. When run without any parameters, it launches the Phone Book Administrator interface.

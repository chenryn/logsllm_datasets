### 云环境中实例或虚拟机的创建
**创建时间**: 2022-03-30T14:26:51.840Z  
**描述**: 创建新的实例或虚拟机是许多云环境中的常见操作。事件不应孤立地看待，而应作为可能导致其他活动的行为链的一部分。例如，由新用户帐户创建实例或意外创建一个或多个快照后创建实例可能表明可疑活动。  
在AWS中，CloudTrail日志会捕获RunInstances事件中的实例创建，在Azure中，VM的创建可能被捕获在Azure活动日志中。(引用: AWS CloudTrail Search; Azure Activity Logs) Google的Admin Activity审计日志可以在Cloud Audit日志中用于检测gcloud compute instances create的使用。(引用: Cloud Audit Logs)  
**外部引用**:  
- Amazon. (无日期). 搜索EC2实例的CloudTrail日志. [链接](https://aws.amazon.com/premiumsupport/knowledge-center/cloudtrail-search-api-calls/)  
- Google. (无日期). 审计日志. [链接](https://cloud.google.com/logging/docs/audit#admin-activity)

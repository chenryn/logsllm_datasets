### Web 中间件

Web 中间件是指在客户端和后端应用之间提供各种服务的软件组件。它主要包括 Web 容器和 Web 服务器。

- **Web 容器**：Web 容器为应用程序组件（如 ASP 和 JSP）提供了一个运行环境。它是中间件的一个重要组成部分，负责解析和执行动态语言代码。通过 Web 容器，开发者可以更方便地编写和部署动态网页应用。

- **Web 服务器**：Web 服务器主要用于处理 HTTP 协议请求。它可以响应针对静态页面或图片的请求，并进行页面跳转。对于动态内容的请求，Web 服务器通常会将这些请求委托给其他程序（例如应用服务器或脚本解释器）来处理。

综上所述，Web 容器和 Web 服务器共同构成了 Web 中间件，它们各自承担不同的职责，协同工作以提供完整的 Web 服务。

# Sub-domain Enumeration Checklist

## Search Engines
- **Google** ([google.com](https://google.com)) - Use the `site:` operator to find subdomains.
- **Bing** ([bing.com](https://bing.com)) - Use the `site:` operator to find subdomains.

## DNS Information Aggregators
- [VirusTotal](https://www.virustotal.com/)
- [ViewDNS](https://viewdns.info)
- [DNSdumpster](https://dnsdumpster.com/)
- [ThreatCrowd](https://www.threatcrowd.org/)

## Certificate Transparency Logs
- [crt.sh](https://crt.sh/)
- [Censys](https://censys.io/)
- [Facebook CT](https://developers.facebook.com/tools/ct/)
- [Google Transparency Report](https://transparencyreport.google.com/https/ct/)

## Tools
- [Sublist3r](https://github.com/aboul3la/Sublist3r) - A fast subdomain enumeration tool.
- [Altdns](https://github.com/infosec-au/altdns) - A tool to generate and test permutations of subdomains.
- [massdns](https://github.com/blechschmidt/massdns) - A high-performance DNS stub resolver.
- [enumall](https://github.com/jhaddix/domain) - A comprehensive subdomain enumeration tool.
- [DNSRecon](https://github.com/darkoperator/dnsrecon) - A DNS reconnaissance tool.
- [Domain Analyzer](https://github.com/eldraco/domain_analyzer) - A tool for analyzing domain names.
- [XRay](https://github.com/evilsocket/xray) - A tool for discovering and visualizing the network infrastructure of a target.
- [Aquatone](https://github.com/michenriksen/aquatone) - A tool for taking screenshots of websites and organizing them.
- [ldns-walk](https://www.nlnetlabs.nl/projects/ldns/) - A tool for walking through DNS zones.
- [NSEC3 Walker](https://dnscurve.org/nsec3walker.html) - A tool for walking NSEC3 chains in DNS.

## Data Sources
- [Project Sonar](https://sonar.labs.rapid7.com/) - A data collection platform for security research.
- [Certificate Transparency Logs](https://www.certificate-transparency.org/known-logs) - A list of known certificate transparency logs.
- [domains-index.com](http://domains-index.com) - An aggregator for registered domain names and DNS records.
- [domainsdb.info](http://domainsdb.info) - A search API for registered domain names.

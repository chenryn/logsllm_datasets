# Summary of the New Feature/Enhancement
The proposed enhancement introduces the capability to perform web searches directly from PowerToys Run.

# Proposed Technical Implementation Details (Optional)
It is suggested that an option be added to PowerToys Run, enabling users to search the web. To provide flexibility and personalization, the following features should be included:

- **Selectable Browser:** Users should be able to choose their preferred web browser for executing the search.
- **Selectable Search Engine:** Users should also have the ability to select their preferred search engine for the query.

This enhancement will improve the functionality and user experience of PowerToys Run by integrating a commonly used feature in a customizable manner.

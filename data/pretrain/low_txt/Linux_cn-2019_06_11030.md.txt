在企业纷纷实施 IT 转型以适应当前和未来的业务需求之际，SUSE® 近日宣布推出 SUSE Linux Enterprise 15 Service Pack 1 操作系统。该操作系统旨在为企业传统的和不断发展的容器化工作负载提供坚实的基础。

![](/data/attachment/album/201906/28/173812rusha2txl0oxtcyl.jpg)

IT 转型已成为近年来的一个重要趋势。例如，人工智能（AI）的部署是数字化转型中的一个典型项目。据预测，到 2022 年，数据中心中 AI 的部署比例将从 2018 年的 4% 增长至 50%。这类新部署推动了 IT 工具和软件的变革。此外，在制造业，经过数字化转型的企业能够实现年均 2.3% 的利润增长，而未进行数字化转型的企业则面临年均 2.1% 的利润下降。由此可见，拥抱数字化转型的企业将在竞争中占据优势。

SUSE 提供了一套开放性平台工具，帮助企业完成数字化转型。这种开放性确保了不会出现供应商锁定的情况。无论是公有云还是自建云平台，SUSE SLES 多模态操作系统都是其基础组件。这里提到的“多模”包括稳态（传统数据中心等）和敏态（如容器及其他现代化组件）。许多公司都具备这种多模系统的现状。

在 DevOps 时代，企业需要具备设计、部署和运行云原生及微服务应用程序的能力。为此，企业必须利用 Kubernetes 等编排工具来交付现代容器化应用程序，并确保这些应用程序在边缘、本地以及混合云和多云环境下的安全性和灵活性。同时，传统系统仍广泛用于承载关键任务工作负载，如数据库和 ERP 系统。

![](/data/attachment/album/201906/28/173847a4bs1wltjzwtl9ca.jpg)

目前，许多企业拥有能够运行多模工作负载的多模 IT 基础设施。SUSE Linux Enterprise 作为全球首款多模操作系统，为传统基础设施与云基础设施之间架起桥梁，助力企业持续创新、竞争和成长。

SUSE 工程、产品和创新总裁 Thomas Di Giacomo 表示：“SUSE Linux Enterprise 是一款现代模块化操作系统，它简化了多模 IT，提高了传统 IT 基础设施的运营效率，并为开发人员提供了一个具有吸引力的平台。因此，企业可以跨本地和公有云环境轻松部署和迁移业务关键型工作负载。SUSE 秉承开放的理念，与客户青睐的合作伙伴及供应商合作，尽可能减少服务中断，帮助客户根据业务需求对系统进行创新和升级。”

SUSE Linux Enterprise 15 SP1 对多模操作系统模型进行了改进，包括加强通用代码库的核心原则、模块度和社区开发，同时强化关键业务特性，如优化工作负载、增强数据安全以及缩短停机时间。主要亮点如下：

* **从社区级 Linux 到企业级 Linux 的快速转换**——开发者和运维人员只需几次点击即可将 openSUSE Leap 系统转移到 SUSE Linux Enterprise Server。完善的 SUSE Package Hub 也为企业在生产系统和基于社区级 openSUSE Leap 开发系统上构建一流应用程序提供了巨大支持。
* **强化对边缘到 HPC 工作负载的支持**——Service Pack 1 中，SUSE Linux Enterprise Server for Arm 15 对 SoC 处理器的支持数量翻倍，提升了存储支持，并增强了 64 位 Arm 服务器和物联网设备上的工业自动化应用程序的支持。例如，64 位树莓派设备现在支持全 HDMI 音频和视频，并提供 ISO 镜像以便快速安装。
* **优化工作负载，降低数据延迟**——SUSE Linux Enterprise Server 搭载英特尔® 傲腾™ DC 持久内存和第二代英特尔® 至强® 可扩展处理器，使解决方案能够应对海量数据挑战，在提高速度和灵活性的同时降低基础设施成本和管理成本。
* **提升基于硬件的数据安全性**——Service Pack 1 全面支持 AMD 安全加密虚拟化（SEV）技术，该技术已在 SUSE Linux Enterprise Server 15 的开发者预览版中体现。这使得客户虚拟机能够在加密内存中运行，免受来自管理程序的内存刮取攻击。SP1 同样支持 AMD 安全内存加密（SME），该技术使用单码加密系统内存。
* **缩短升级停机时间**——事务处理升级（Service Pack 1 技术预览版）可缩短维护窗口时间并增加运行时间。
* **通过完善的 Modular+ 简化安装过程**——SUSE Linux Enterprise 15 中推出的 Unified Installer 为客户简化了 SUSE 产品的使用方法，还可用于安装 SUSE 的其他系列产品，如 SUSE Manager、SUSE Linux Enterprise Real Time 和 SUSE Linux Enterprise Point of Service。

![](/data/attachment/album/201906/28/174043rmdzne9oe3cd9d39.jpg)

此外，SUSE 近日还推出了 SUSE Manager 4 和 SUSE Manager for Retail 4 开源基础设施管理解决方案。这两款新产品能够帮助全球企业的 DevOps 和 IT 运营团队降低 IT 资产复杂性，重新获取对 IT 资产的掌控力，在遵守安全策略的前提下提高效率，通过自动化的优化操作降低成本。

![](/data/attachment/album/201906/28/174111dvdjsnjxvxxaoggw.jpg)

作为软件定义基础设施的重要组成部分，SUSE Manager 4 可以实现以下功能：

* 降低成本，简化管理，加强内容生命周期管理，在整个 DevOps 周期内简化软件包的移动和管理。基于 Salt 的虚拟机管理解决方案能够实现实时管理数百台服务器。
* 在下一代基于 Prometheus 的监控堆栈中进一步增强监控和预警能力，提高可用性和可见性，帮助客户轻松定位并解决问题。
* 加强操作系统支持并提供新的 SAP HANA 部署能力，简化 SAP HANA 的安装和配置，从而降低不同 IT 环境的复杂性，重获对其的掌控力。

SUSE Manager for Retail 是专为零售业优化和量身打造的开源基础设施管理解决方案。无论是主机、HPC 集群、裸机服务器、虚拟机、容器，还是服务点/销售点终端、自助服务站、自助服务系统和回收系统，SUSE Manager for Retail 都能帮助客户降低成本，优化运营，确保整个零售 IT 基础设施的合规性。

SUSE Manager 4 现已上线，镜像也可在 Amazon Web Services、Google 云平台和 Microsoft Azure 上找到。

目前，通过 SUSE Linux Enterprise Server 12 和 SUSE Linux Enterprise Server 15 认证的软件应用程序已达到近 3300 个。认证应用的多样性表明，多模工作负载已得到企业的广泛采纳。去年，在新认证的应用程序中，软件存储类占 16%，管理软件占 15%，企业资源管理应用占 11%，酒店和旅游软件占 10%，安全软件占 9%。其余 40% 包括数据管理软件、客户关系管理（CRM）应用等。

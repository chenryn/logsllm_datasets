# GitHub Dorking: Search Terms for Sensitive Information

## Introduction
GitHub Dorking is a technique used to search for sensitive information, such as access keys, passwords, and other credentials, that have been inadvertently committed to public repositories. Below is a comprehensive list of search terms and patterns that can be used to identify such information.

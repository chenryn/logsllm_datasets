**PowerSploit**
- **创建时间**: 2018年4月18日
- **描述**: PowerSploit模块是用PowerShell编写的，并通过PowerShell执行。(引用: GitHub PowerSploit May 2012, PowerSploit Documentation)
- **外部引用**:
  - PowerShellMafia. (2012, 5月26日). PowerSploit - A PowerShell Post-Exploitation Framework. 检索于2018年2月6日.
  - PowerSploit. (n.d.). PowerSploit. 检索于2018年2月6日.

**Indrik Spider**
- **创建时间**: 2021年1月29日
- **描述**: Indrik Spider使用组策略对象来部署批处理脚本。(引用: Crowdstrike Indrik November 2018)
- **外部引用**:
  - Frankoff, S., Hartley, B. (2018, 11月14日). Big Game Hunting: The Evolution of INDRIK SPIDER From Dridex Wire Fraud to BitPaymer Targeted Ransomware. 检索于2021年1月6日.

**APT37**
- **创建时间**: 2018年4月18日
- **描述**: APT37通过注册表键HKCU\Software\Microsoft\CurrentVersion\Run\添加持久性。(引用: FireEye APT37 Feb 2018, Talos Group123)
- **外部引用**:
  - FireEye. (2018, 2月20日). APT37 (Reaper): The Overlooked North Korean Actor. 检索于2018年3月1日.
  - Mercer, W., Rascagneres, P. (2018, 1月16日). Korea In The Crosshairs. 检索于2018年5月21日.

**Agent Tesla**
- **创建时间**: 2019年1月29日
- **描述**: Agent Tesla可以通过将自身添加到注册表启动项中来建立持久性。(引用: Fortinet Agent Tesla April 2018, SentinelLabs Agent Tesla Aug 2020)
- **外部引用**:
  - Zhang, X. (2018, 4月5日). Analysis of New Agent Tesla Spyware Variant. 检索于2018年11月5日.
  - Walter, J. (2020, 8月10日). Agent Tesla | Old RAT Uses New Tricks to Stay on Top. 检索于2020年12月11日.

**Winnti for Windows**
- **创建时间**: 2022年1月11日
- **描述**: Winnti for Windows的dropper组件可以验证单个命令行参数的存在，并在未找到时终止或将其用作解密密钥。(引用: Novetta Winnti April 2015)
- **外部引用**:
  - Novetta Threat Research Group. (2015, 4月7日). Winnti Analysis. 检索于2017年2月8日.

**HDoor**
- **创建时间**: 2017年12月14日
- **描述**: HDoor会杀死受害者系统上的反病毒软件。(引用: Baumgartner Naikon 2015)
- **外部引用**:
  - Baumgartner, K., Golovkin, M. (2015, 5月). The MsnMM Campaigns: The Earliest Naikon APT Campaigns. 检索于2019年4月10日.

**监控命令执行**
- **创建时间**: 2022年3月30日
- **描述**: 监控可能包含登录脚本的已执行命令和参数，以检测异常用户或异常时间的访问。

**Hi-Zor**
- **创建时间**: 2017年12月14日
- **描述**: Hi-Zor在其DLL有效载荷文件执行时删除其RAT安装程序文件。(引用: Fidelis INOCNATION)
- **外部引用**:
  - Fidelis Cybersecurity. (2015, 12月16日). Fidelis Threat Advisory #1020: Dissecting the Malware Involved in the INOCNATION Campaign. 检索于2016年3月24日.

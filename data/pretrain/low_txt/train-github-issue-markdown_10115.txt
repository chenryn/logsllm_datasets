### Issue Description
When clicking on "HEAD" in the sidebar of the documentation page, a 404 error is encountered.

### Expected Behavior
Clicking on "HEAD" should direct the user to the latest (HEAD) version of the GitHub documentation, which should correspond to version 0.20.0.

### Current Behavior
A 404 error is displayed when "HEAD" is selected from the sidebar.

### Steps to Reproduce
1. Navigate to the documentation website.
2. Open the sidebar menu.
3. Attempt to change the version to "HEAD."
   - Note: All other versions are accessible and functional, except for "HEAD."

None

No specific environment details are required for this issue.

#### Common Signs of Mental Health Conditions
- Excessive worrying or fear
- Feeling excessively sad or low
- Confused thinking or problems concentrating and learning
- Extreme mood changes, including uncontrollable "highs" or feelings of euphoria
- Prolonged or strong feelings of irritability or anger
- Avoiding friends and social activities
- Difficulties understanding or relating to other people
- Changes in sleeping habits or feeling tired and low energy
- Changes in eating habits such as increased hunger or lack of appetite
- Changes in sex drive
- Difficulty perceiving reality (delusions/hallucinations)
- Inability to perceive changes in oneâ€™s own feelings, behavior, or personality
- Abuse of substances like alcohol or drugs
- Multiple physical ailments without obvious causes
- Thoughts of suicide, or suicidal planning
- Inability to carry out daily activities or handle daily problems and stress

#### Steps to Support Mental Health
- Inform yourself about the illness.
- Start dialogues with family and friends.
- Seek professional help in cases of acute psychiatric distress.
- Communicate openly and ask for help when needed.
- Join support groups.
- Reassure loved ones that you care.
- Offer practical help with everyday tasks.
- Include them in your plans and continue to invite them.
- Take care of your own well-being.
- Avoid falling into the role of "fixer" and "savior."
- Offer objectivity, compassion, and acceptance.
- Have realistic expectations about the recovery process.

## Information Security Resources
### Android Debug Bridge (ADB)
- **Purpose:** A versatile command-line tool for communicating with an Android device.
- **Common Commands:**
  - `adb devices`: Lists connected devices.
  - `adb root`: Restarts adbd with root permissions.
  - `adb start-server`: Starts the adb server.
  - `adb kill-server`: Kills the adb server.
  - `adb reboot`: Reboots the device.
  - `adb devices -l`: Lists devices by product/model.
  - `adb shell`: Starts a background terminal.
  - `exit`: Exits the background terminal.
  - `adb help`: Lists all commands.
  - `adb -s <device>`: Redirects command to a specific device.
  - `adb -d`: Directs command to only attached USB device.
  - `adb -e`: Directs command to only attached emulator.

#### Package Installation
- `adb shell install <path_to_apk>`: Installs an app.
- `adb shell install -r <path_to_apk>`: Reinstalls an app.
- `adb shell uninstall <package_name>`: Uninstalls an app.

#### Paths
- `/data/data/<package_name>/databases`: App databases.
- `/data/data/<package_name>/shared_prefs/`: Shared preferences.
- `/data/app/`: APKs installed by the user.
- `/system/app/`: Pre-installed APK files.
- `/mnt/asec/`: Encrypted apps.
- `/mnt/emmc/`: Internal SD Card.
- `/mnt/sdcard/`: External/Internal SD Card.
- `/mnt/sdcard/external_sd/`: External SD Card.

#### File Operations
- `adb push <local> <remote>`: Copies a file/directory to the device.
- `adb pull <remote> <local>`: Copies a file/directory from the device.
- `run-as <package_name> cat <file_path>`: Accesses private package files.

#### Phone Info
- `adb get-state`: Prints device state.
- `adb get-serialno`: Gets the serial number.
- `adb shell dumpsys iphonesybinfo`: Gets the IMEI.
- `adb shell netstat`: Lists TCP connectivity.
- `adb shell pwd`: Prints current working directory.
- `adb shell dumpsys battery`: Shows battery status.
- `adb shell pm list features`: Lists phone features.
- `adb shell service list`: Lists all services.
- `adb shell dumpsys activity <activity>`: Shows activity info.
- `adb shell ps`: Prints process status.
- `adb shell wm size`: Displays the current screen resolution.
- `dumpsys window windows | grep -E 'mCurrentFocus|mFocusedApp'`: Prints the current app's opened activity.

#### Package Info
- `adb shell pm list packages`: Lists package names.
- `adb shell pm list packages -r`: Lists package name + path to APKs.
- `adb shell pm list packages -3`: Lists third-party package names.
- `adb shell pm list packages -s`: Lists only system packages.
- `adb shell pm list packages -u`: Lists package names + uninstalled.
- `adb shell dumpsys package <package_name>`: Lists info on one package.
- `adb shell pm path <package_name>`: Path to the APK file.

#### Configure Settings
- `adb shell dumpsys battery set level <level>`: Changes the battery level (0-100).
- `adb shell dumpsys battery set status unknown`: Changes the battery status to unknown.
- `adb shell dumpsys battery reset`: Resets the battery.
- `adb shell dumpsys battery set usb <ON|OFF>`: Changes the USB connection status.
- `adb shell wm size <W>x<H>`: Sets the resolution to WxH.

#### Device-Related Commands
- `adb reboot-recovery`: Reboots into recovery mode.
- `adb reboot fastboot`: Reboots into fastboot mode.
- `adb shell screencap -p "/path/to/screenshot.png"`: Captures a screenshot.
- `adb shell screenrecord "/path/to/record.mp4"`: Records the device screen.
- `adb backup -apk -all -f backup.ab`: Backs up settings and apps.
- `adb backup -apk -shared -all -f backup.ab`: Backs up settings.
- `adb backup -apk -nosystem -all -f backup.ab`: Backs up non-system apps.
- `adb restore backup.ab`: Restores a previous backup.
- `adb shell am start|startservice|broadcast <action> -a <intent_action> -c <category>`: Starts an activity intent.

This document provides a comprehensive list of resources and tools for both information security and mental health, along with detailed instructions for using the Android Debug Bridge (ADB).

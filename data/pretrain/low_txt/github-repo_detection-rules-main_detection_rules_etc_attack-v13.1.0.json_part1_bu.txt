**监测运行进程的上下文数据**

- **描述**: 监测运行进程的上下文数据，包括环境变量、镜像名称、用户/所有者等信息，以识别可能过时、易受攻击或不支持更新安全控制（如日志记录）的系统功能版本。例如，通过监测Windows事件ID 400中的EngineVersion字段，可以揭示PowerShell版本，从而检测恶意降级攻击。
- **引用**: (Hastings, 2014)

**Azorult 恶意软件**

- **描述**: Azorult 恶意软件能够检查系统上安装的软件，通过访问注册表键 `Software\Microsoft\Windows\CurrentVersion\Uninstall` 来获取相关信息。
- **引用**: (Yan et al., 2018)

**JPIN 恶意软件**

- **描述**: JPIN 恶意软件可以列出正在运行的服务。
- **引用**: (Microsoft, 2016)

**BISCUIT 恶意软件**

- **描述**: BISCUIT 恶意软件具有从系统中收集用户名的功能。
- **引用**: (Mandiant, n.d.)

**metaMain 恶意软件**

- **描述**: metaMain 恶意软件可以通过调用 `WinExec`、`WriteFile` 和 `ReadFile` 等函数来执行操作员提供的 Windows 命令。
- **引用**: (Ehrlich et al., 2022; SentinelLabs, 2022)

**ZxShell 恶意软件**

- **描述**: ZxShell 恶意软件的相关信息和使用情况。
- **引用**: (Allievi et al., 2014; Esler et al., 2014)

**3PARA RAT 恶意软件**

- **描述**: 3PARA RAT 恶意软件使用 HTTP 进行命令与控制通信。
- **引用**: (Crowdstrike, 2014)

**AAA 系统在网络安全中的应用**

- **描述**: 使用认证、授权和计费 (AAA) 系统可以限制管理员的操作，并提供用户操作的历史记录以检测未经授权的使用和滥用。TACACS+ 可以通过配置身份验证和命令授权来控制管理员可使用的命令。
- **引用**: (Cisco, n.d.a; Cisco, n.d.b)

**MegaCortex 恶意软件**

- **描述**: MegaCortex 恶意软件会在注册表中添加用于勒索联系信息的条目。
- **引用**: (Del Fierro & Kessem, 2020)

**Lazarus Group 的钓鱼攻击**

- **描述**: Lazarus Group 创建新的电子邮件账户来进行鱼叉式网络钓鱼攻击。
- **引用**: (Kopeytsev & Park, 2021)

**Avenger 恶意软件**

- **描述**: Avenger 恶意软件能够浏览文件目录，如 Program Files 和桌面。
- **引用**: (Chen et al., 2019)

**HEXANE 组织的内部钓鱼攻击**

- **描述**: HEXANE 组织对高管、人力资源和 IT 人员进行了内部鱼叉式网络钓鱼攻击，以获取信息和访问权限。
- **引用**: (SecureWorks, 2019)

# Environment
- Microsoft Windows Version: 10.0.18995.1
- Windows Terminal Version (if applicable): 0.5.2681.0

# Steps to Reproduce
1. Open the PowerShell profile settings.
2. Set the `colorScheme` property to one of the schemes listed in the settings file.

# Expected Behavior
The custom color theme specified in the `colorScheme` setting should be applied.

# Actual Behavior
The standard blue PowerShell theme is still displayed, despite the custom `colorScheme` setting.

# Notes
Other profiles appear to correctly apply and use the configured color themes.

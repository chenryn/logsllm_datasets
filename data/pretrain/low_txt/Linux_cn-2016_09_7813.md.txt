在 Distrowatch.com 上列出的近 300 个 Linux 发行版中，几乎任何一个都可以用作服务器系统。然而，有些发行版在这方面尤为突出。以下是一些专为服务器设计的发行版。

![](/data/attachment/album/201609/27/195515uun7unfdbacfdecf.jpg)

### Red Hat Enterprise Linux (RHEL)
RHEL 是最著名的 Linux 服务器发行版之一，以其在高需求任务中的稳定性而闻名，例如纽约证券交易系统的运行。红帽公司也提供了业内最佳的支持服务。然而，RHEL 的支持订阅费用较高，尽管物有所值。也有便宜的第三方 RHEL 服务，但选择前应进行充分研究。

### CentOS
CentOS 是 RHEL 的一个分支，适用于那些喜欢 RHEL 但不想支付红帽支持费用的人。该项目始于 2004 年，并在 2014 年获得了红帽的官方支持。部署 CentOS 需要具备 Linux 技能的员工，因为缺乏技术支持。不过，CentOS 社区提供了丰富的资源，如邮件列表、Web 论坛和聊天室，帮助用户解决问题。

### Ubuntu Server
Canonical 推出 Ubuntu Server 时曾受到质疑，但它迅速站稳脚跟。Ubuntu Server 基于 Debian，继承了其优秀的基因，并通过提供可负担的技术支持、出色的硬件支持和开发工具填补了市场空白。目前，Ubuntu Server 在 OpenStack 和 Amazon Elastic Compute Cloud 上部署最多，超过了 Amazon Linux 和 Windows。它也是使用最多的 Linux Web 服务器。

### SUSE Linux Enterprise Server (SLES)
SLES 在欧洲拥有庞大的用户群，在 Novell 引发公关问题之前，它在美国也曾是领先的服务器发行版。SLES 稳定且易于维护，付费的 7x24 小时快速响应技术支持使其适合关键任务部署。惠普企业公司将 SUSE 作为首选 Linux 合作伙伴。

### ClearOS
ClearOS 基于 RHEL，简单易用，即使是非专业人士也能配置。它面向中小型企业，也可以用于家庭娱乐服务器。最新的 7.2 版本包括了许多功能，如对微软 Hyper-V 技术的支持、XFS 和 BTRFS 文件系统支持以及 LVM 和 IPv6 支持。这些功能在免费版或价格合理的专业版中都可用。

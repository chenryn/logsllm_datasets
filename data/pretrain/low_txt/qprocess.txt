### qprocess

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `qprocess` command displays information about processes that are currently running on a Remote Desktop Session Host (RDSH) server. This utility is particularly useful for administrators who need to monitor and manage processes on remote desktop servers.

**Usage:**
To find out what's new in the latest version of Remote Desktop Services, refer to the documentation on [What's New in Remote Desktop Services in Windows Server](#).

**Note:**
The `qprocess` command is functionally equivalent to the `query process` command.

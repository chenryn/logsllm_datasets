### 微软将在 Windows 中放弃对 PHP 的支持
PHP 8.0 计划于 11 月发布，但届时 Windows 将不再提供对该版本的支持，具体原因尚未公布。微软表示，在 PHP 7.2、7.3 和 7.4 版本的生命周期内，将继续为其提供错误修复和安全补丁。然而，对于即将发布的 PHP 8.0 及以后的版本，微软将不会在任何方面继续支持 PHP for Windows。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/1002019.htm)

> **拍一拍**: 希望这一决定是基于用户数量不足的考量。

### Zig 语言成立基金会以替代 C 语言
Zig 语言官方宣布成立 Zig 软件基金会（Zig Software Foundation, ZSF），致力于促进、保护并推动 Zig 编程语言的发展，同时支持并鼓励多元化和国际化的 Zig 开发者社区的成长。Zig 是一种通用编程语言，特别注重稳定性、可维护性和性能，目标是成为系统级编程中 C 语言的最佳替代品。

**来源**: [开源中国](https://www.oschina.net/news/117118/announcing-zig-software-foundation)

> **拍一拍**: C 语言：“想取代我的多着呢……”

### Mozilla 暂停 Firefox Send 文件共享服务以防黑客滥用
Firefox Send 服务自 2019 年 3 月上线以来，已被恶意软件社区广泛利用。攻击者通过该平台上传加密的恶意软件文件，并通过电子邮件与受害者分享下载链接。鉴于此情况，Mozilla 决定暂时关闭这项服务。

**来源**: [cnBeta.COM](https://www.cnbeta.com/articles/tech/1002131.htm)

> **拍一拍**: 自由总是伴随着被滥用的风险。

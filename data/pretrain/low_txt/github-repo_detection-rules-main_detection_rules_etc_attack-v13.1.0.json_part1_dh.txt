**API 调用监控**

**描述：**
监控 API 调用可能会生成大量数据，在特定情况下收集可能对防御无用，因为良性使用 API 函数很常见且难以与恶意行为区分开来。将其他事件与 API 函数调用相关的行为进行关联，可以为事件提供额外的上下文，有助于确定其是否由恶意行为引起。通过进程 ID 对进程谱系的活动进行关联可能就足够了。

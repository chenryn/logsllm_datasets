### 3D 和动画库

- **[three.js](https://github.com/mrdoob/three.js)** - 一个强大的JavaScript 3D库，用于在Web上创建和显示3D图形。[演示](https://threejs.org/)
- **[loaders.css](https://github.com/ConnorAtherton/loaders.css)** - 提供多种CSS加载动画效果。[演示](https://connoratherton.com/loaders)
- **[Hover.css](https://github.com/IanLunn/Hover)** - 一款基于 CSS3 的悬停特效合集。[演示](http://ianlunn.github.io/Hover/)
- **[Effeckt.css](https://github.com/h5bp/Effeckt.css)** - 包含众多精妙的 CSS3 切换和动画效果库。[演示](http://h5bp.github.io/Effeckt.css/)
- **[Magic Animations](https://github.com/miniMAC/magic)** - 一个独特的 CSS3 动画特效包。[演示](https://minimamente.com/example/magic_animations/)
- **[Transformicons](https://github.com/grayghostvisuals/transformicons)** - 结合 SVG、CSS 和 HTML 技术，为图标、按钮和符号添加变种动画效果。[演示](http://www.transformicons.com/)
- **[SpinKit](https://github.com/tobiasahlin/SpinKit)** - 一款高度自定义的 CSS 加载动画合集。[演示](http://tobiasahlin.com/spinkit/)
- **[d3-ease](https://github.com/d3/d3-ease)** - 一个使动画更为平滑的 Easing 库。
- **[ScrollMagic](https://github.com/janpaepke/ScrollMagic)** - 一个 JavaScript 库，用于创建魔幻滚动交互。[演示](http://scrollmagic.io/)
- **[ScrollReveal](https://github.com/jlmakes/scrollreveal)** - 一种页面滚动显示动画，可以播放一次或无限次。[演示](https://scrollrevealjs.org/)
- **[RELLAX.js](https://github.com/dixonandmoe/rellax)** - 一款轻量级的纯 JavaScript 视差效果库。[演示](https://dixonandmoe.com/rellax/)
- **[CountUp.js](https://github.com/inorganik/CountUp.js)** - 用于以有趣的方式显示数值数据的动画库。[演示](https://inorganik.github.io/countUp.js/)
- **[Dynamics.js](http://dynamicsjs.com/)** - 可以创建物理运动动画效果的 JavaScript 库。[演示](https://github.com/michaelvillar/dynamics.js)
- **[Mojs](https://github.com/legomushroom/mojs)** - 一个拥有极简声明式 API 的工具库，用于轻松掌控运动轨迹。[演示](http://mojs.io/)
- **[React FLIP Move](https://github.com/joshwcomeau/react-flip-move)** - 解决列表顺序变化时项目列表动画化的问题。[演示](http://joshwcomeau.github.io/react-flip-move/examples/#/shuffle?_k=og3e8m)
- **[tween.js](https://github.com/tweenjs/tween.js)** - 一个JS 平滑动画库。[演示](http://tweenjs.github.io/tween.js/examples/12_graphs_custom_functions.html)
- **[Typed.js](https://github.com/mattboldt/typed.js)** - 一个JS 打字动画库。[演示](https://mattboldt.com/typed.js/)
- **[vivus](https://github.com/maxwellito/vivus)** - 使SVG绘制动画的JavaScript库。[演示](http://maxwellito.github.io/vivus/)
- **[Choreographer-js](https://github.com/christinecha/choreographer-js)** - 一个用于处理复杂动画的简单库。[演示](https://christinecha.github.io/choreographer-js/)
- **[minirefresh](https://github.com/minirefresh/minirefresh)** - 优雅的H5下拉刷新，零依赖。[演示](https://minirefresh.github.io/minirefresh-doc/minirefresh-doc/guid/guid_effect.html)

### Summary: Auto-Fill Feature for FancyZones

**Category:** Feature Request

**Description:**
When the auto-fill feature is enabled, new windows (excluding those that are ignored) will automatically snap to the next available FancyZone. This functionality would enable users to quickly and efficiently populate layouts with a high number of zones.

**Benefits:**
- **Enhanced Usability:** Users can easily manage and organize multiple windows without manual intervention.
- **Efficiency:** Ideal for workflows that require frequent window management, especially in complex layouts with many zones.

This feature would significantly improve the user experience by streamlining the process of arranging and managing windows within FancyZones.

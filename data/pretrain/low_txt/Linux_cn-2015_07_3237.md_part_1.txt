### 简介

Fedora Linux（第七版以前为 Fedora Core）是较具知名度的 Linux 发行版之一，由 Fedora 项目社区开发、红帽公司赞助，旨在创建一套新颖、多功能且开源的操作系统。Fedora 基于 Red Hat Linux，在 Red Hat Linux 终止发布后，红帽公司以 Fedora 取代其在个人领域的应用，而 Red Hat Enterprise Linux 则取代其在商业领域的应用。对于用户而言，Fedora 是一个功能完备、更新迅速的免费操作系统；而对于红帽公司而言，它是一个新技术的测试平台。目前，Fedora 最新的版本是 Fedora 27。

官方网站：[Fedora Project](https://getfedora.org/)

### 下载

#### Fedora 27 Workstation Live

Fedora Workstation Live 镜像允许您为自己电脑制作完整的立即可用的 Fedora Workstation 系统介质。您可以使用 Live 镜像测试和体验 Fedora，而无需改动硬盘内容。当您满意之后，可以从 Live 镜像安装 Fedora 到您的硬盘。要使用该镜像，您需要可创建或刻录 DVD 的驱动器或者至少与镜像大小相同的 USB 闪存盘。从版本 27 开始，不再提供 32 位的 Live 安装镜像，如需安装 32 位版本的 Fedora 27，只能通过网络安装。

- **Intel & AMD/兼容 PC 64 位**，适用于 64 位 PC  
  [点此下载](https://download.fedoraproject.org/pub/fedora/linux/releases/27/Workstation/x86_64/iso/Fedora-Workstation-Live-x86_64-27-1.6.iso)

#### Fedora 27 Server

Fedora Server 安装镜像允许您为计算机制作介质以便引导安装程序将 Fedora Server 直接安装到您的硬盘。

- **Intel & AMD/兼容 PC 64 位**，适用于 64 位 PC  
  [点此下载](https://download.fedoraproject.org/pub/fedora/linux/releases/27/Server/x86_64/iso/Fedora-Server-dvd-x86_64-27-1.6.iso)

### 发行方式

Fedora 项目通过以下几种方式发行 Fedora：

- **Fedora DVD/CD** – 包含所有主要软件包的 DVD 或 CD 套装；
- **Live 镜像** – CD 或 DVD 大小的光盘镜像，可用于创建 Live CD 或从 USB 设备启动，并可选安装到硬盘；
- **最小 CD** – 用于通过 HTTP、FTP 或 NFS 安装。

您可以使用 [Fedora Live USB Creator](https://fedorahosted.org/liveusb-creator/) 或 UNetbootin 创建 Live USB 版本的 Fedora。

此外，Fedora 项目还发布自定义的 Fedora 版本，称为 Fedora Spins。这些版本包含特定的软件包集合以及各种桌面环境（如 KDE Plasma、Xfce、LXDE、MATE 和 Cinnamon），以满足特定类型用户的需求。Fedora Spins 由对 Fedora 有特殊兴趣的小组开发。

### Extra Packages for Enterprise Linux (EPEL)

Extra Packages for Enterprise Linux (EPEL) 是由来自 Fedora 项目的志愿者发起的社区力量，旨在创建高质量的附加软件仓库，以补充 RHEL 和其他兼容版本的软件库。

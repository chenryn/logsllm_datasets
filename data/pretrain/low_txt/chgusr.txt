### chgusr

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `chgusr` command is used to change the installation mode for the Remote Desktop Session Host (RDSH) server.

**Important Note:**
This command has been deprecated and replaced by the `change user` command. For detailed information, including syntax and parameters, please refer to the documentation for the `change user` command.

Firefox 111 已正式发布，引入了多项新功能和改进，包括原生通知、新增的本地化语言以及增强的 Web 平台功能。

![](/data/attachment/album/202303/19/144348oyfh999yy292f0f9.jpg)

### Firefox 111 的主要更新

![在 Linux 上运行的 Firefox 111](/data/attachment/album/202303/19/144419jtbffdssydtdtttr.jpg)

Firefox 是一个广受欢迎的网络浏览器，最新发布的版本 111 带来了多项新功能和改进。其中一个重要更新是对 Windows 原生系统通知的支持，这为用户提供了更流畅和一致的体验。

此外，Firefox Relay 用户现在可以直接从 Firefox 凭据管理器中创建 Relay 邮箱。此功能仅适用于登录了 Firefox 账户的用户，为他们的在线活动提供了额外的隐私和安全保护。

Firefox 111 还增加了两种新的语言支持：弗留利语 (fur) 和撒丁语 (sc)，进一步提高了全球用户的可访问性和包容性。

在 Web 平台方面，Firefox 在表单元素上引入了 `rel` 属性，使开发人员可以更轻松地指定当前文档与表单目标之间的关系。这一功能与其他浏览器兼容，并简化了 Web 应用程序的开发过程。

最后，Firefox 启用了原点私有文件系统访问 (Origin Private File System Access, OPFS)，这是一种新的存储 API，允许 Web 应用程序在沙箱环境中存储和检索文件系统中的数据。这一增强功能为开发人员提供了更多控制权，确保用户数据的安全性和隐私性。

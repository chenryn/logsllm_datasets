Mozilla 已将最终版本的 Firefox 46.0 上传至其 FTP 服务器，尽管目前官方尚未正式更新下载链接。此次发布的 Firefox 46.0 被广泛期待已久，它终于实现了对 GNU/Linux 平台上 GTK3 的集成。这一承诺从 Firefox 42.0 开始就一直存在，但在 2015 年 11 月的 42.0 版本中并未兑现。经过多次推迟，现在终于实现了。

![](/data/attachment/album/201604/26/103535a8nzr1brrdrwwcrc.jpg)

除了 GTK3 集成外，新版本还改进了 JavaScript JIT 编译器的安全性，并支持使用内容解密模块 (CDM) 作为 H.264 和 AAC 流的回退解码器。总体来看，Firefox 46.0 是一个较小的更新版本，从之前的 Beta 版变更日志可以看出，它的改进包括 WebRTC API 的稳定性和性能提升、HTML5 `document.elementsFromPoint` 的支持以及 Web Crypto API 中 HKDF 的支持。

对于开发者而言，Firefox 46.0 在内存工具中增加了显示 dominator trees 的功能，在性能面板中添加了分配和垃圾回收的暂停分析，并允许在样式编辑器中直接启动响应模式。

# 重要 KDE 版本及 Linux 发展历程

## 2000
![](/data/attachment/album/201510/26/070801uvqvujivuuvuoj5j.jpg)
**Dell 支持 Linux** - Dell 成为首个支持 Linux 的大型硬件供应商。

## 2001
![](/data/attachment/album/201510/26/070801drfq9bfnf8hp88d7.jpg)
**《操作系统革命》纪录片发布** - 该纪录片记录了 GNU、Linux、开源软件和自由软件的 20 年发展历程，并采访了多位顶尖黑客。

## 2002
![](/data/attachment/album/201510/26/070802biwaiyruhuyyislw.jpg)
**BitKeeper 引入** - Linux 开始使用 BitKeeper，这是一种商业化的分布式版本控制系统。

## 2003
![](/data/attachment/album/201510/26/070803i0wip5h0jkiptpri.png)
**Novell 收购 SUSE** - Novell 以 2.1 亿美元的价格收购了 SUSE Linux AG。同年，SCO 集团针对 IBM 和 Linux 社区发起了关于 Unix 版权的法律诉讼。
![](/data/attachment/album/201510/26/070804taa2n9gxgm1ykgha.png)
**Fedora 发布** - 红帽公司与 Linux 社区合作首次发布了 Fedora Linux。

## 2004
![](/data/attachment/album/201510/26/070806q5lm1gllynln0o9p.png)
**X.Org 基金会成立** - XFree86 解散后，其成员加入了新成立的 [X.Org 基金会](http://www.x.org/)，加速了 X Window 系统的开发。
![](/data/attachment/album/201510/26/070807pzaapxrnjajvz8pj.jpg)
**Ubuntu 首次亮相** - Ubuntu 4.10 正式发布，标志着 Ubuntu 的诞生。

## 2005
![](/data/attachment/album/201510/26/070807mkmbi5mm55bljxbz.png)
**openSUSE 启动** - [openSUSE](https://en.opensuse.org/Main_Page) 项目启动，提供了 Novell 操作系统的免费版本。此外，OpenOffice.org 开始支持 OpenDocument 标准。

## 2006
![](/data/attachment/album/201510/26/070808qb6znwgnh3glar35.png)
**Oracle Linux 发布** - Oracle 推出了基于红帽企业版 Linux 的 Oracle Linux。同时，微软和 Novell 在 IT 和专利保护方面展开合作。

## 2007
![](/data/attachment/album/201510/26/070809bwm5wbh661edwwwz.jpg)
**Dell 预装 Linux 笔记本** - Dell 发布了首款预装 Linux 操作系统的笔记本电脑。

## 2008
![](/data/attachment/album/201510/26/070810tbjbjb8psoyjsqyb.jpg)
**KDE 4.0 发布** - KDE 4.0 正式发布，但由于稳定性问题，许多用户转向 GNOME。

## 2009
![](/data/attachment/album/201510/26/070812nmmu88puziji48b9.jpg)
**红帽市值突破 26 亿美元** - 红帽公司在这一年取得了显著成功，市值达到 26 亿美元。微软也在这一年内向 Linux 内核提交了第一个补丁（基于 GPLv2 协议）。

## 2010
![](/data/attachment/album/201510/26/070813t1260scjdl1uvusq.png)
**Novell 被 Attachmate 收购** - Novell 以 22 亿美元的价格被 Attachmate Group, Inc. 收购。SUSE 和 Novell 成为新公司的独立产品线。同年，[systemd](https://en.wikipedia.org/wiki/Systemd) 首次发布，开启了 Linux 系统的新篇章。

## 2011
![](/data/attachment/album/201510/26/070815hkloo10w6zosrzz7.png)
**Unity 桌面环境发布** - Ubuntu Unity 桌面环境正式推出，但受到了不少用户的批评。
![](/data/attachment/album/201510/26/070818k0yr7my72nl7no7e.png)
**GNOME 3.0 发布** - GNOME 3.0 正式发布，尽管 Linus Torvalds 将其称为“一团糟”，但还是收到了大量负面反馈。Linux 内核 3.0 版本也于同年发布。

## 2012
![](/data/attachment/album/201510/26/070819np1gu4jixbixjvbi.png)
**Linux 内核代码量达 1500 万行** - Linux 内核的代码量达到了 1500 万行，微软成为主要贡献者之一。

## 2013
![](/data/attachment/album/201510/26/070820pqtlpmdiil4ildjq.png)
**Kali Linux 1.0 发布** - Kali Linux 1.0 正式发布，这是一个基于 Debian 的 Linux 发行版，专用于渗透测试和数字取证。同年，CentOS 及其代码开发者加入了红帽公司。

## 2014
![](/data/attachment/album/201510/26/070821qqfbql6g0klk0rlq.jpg)
**systemd 成为主流初始化系统** - systemd 成为了 Ubuntu 和其他主流 Linux 发行版的默认初始化系统。此时，Ubuntu 已拥有 2200 万用户。安卓在移动设备市场的份额达到 75%。

## 2015
![](/data/attachment/album/201510/26/070822vxybqvzxyvyqxpy7.jpg)
**Linux 4.0 发布** - Linux 4.0 正式发布。Mandriva 公司宣告破产，但仍有多个分支继续发展，其中最著名的是 Mageia。

**APT28**
- **描述**: APT28 (https://attack.mitre.org/groups/G0007) 保存了具有隐藏文件属性的文件。(引用: Talos Seduploader Oct 2017)
- **外部引用**:
  - Mercer, W., et al. (2017, October 22). "Cyber Conflict" Decoy Document Used in Real Cyber Conflict. Retrieved November 2, 2018. [链接](https://blog.talosintelligence.com/2017/10/cyber-conflict-decoy-document.html)

**Ajax Security Team**
- **描述**: Ajax Security Team (https://attack.mitre.org/groups/G0130) 使用了个性化的鱼叉式网络钓鱼附件。(引用: Check Point Rocket Kitten)
- **外部引用**:
  - Check Point Software Technologies. (2015). ROCKET KITTEN: A CAMPAIGN WITH 9 LIVES. Retrieved March 16, 2018. [链接](https://blog.checkpoint.com/wp-content/uploads/2015/11/rocket-kitten-report.pdf)

**DustySky**
- **描述**: DustySky (https://attack.mitre.org/software/S0062) 通过在 HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run 中创建注册表项来实现持久性。(引用: DustySky)
- **外部引用**:
  - ClearSky. (2016, January 7). Operation DustySky. Retrieved January 8, 2016. [链接](https://www.clearskysec.com/wp-content/uploads/2016/01/Operation%20DustySky_TLP_WHITE.pdf)

**JHUHUGIT**
- **描述**: JHUHUGIT (https://attack.mitre.org/software/S0044) 使用 .bat 文件执行 .dll 文件。(引用: Talos Seduploader Oct 2017)
- **外部引用**:
  - Mercer, W., et al. (2017, October 22). "Cyber Conflict" Decoy Document Used in Real Cyber Conflict. Retrieved November 2, 2018. [链接](https://blog.talosintelligence.com/2017/10/cyber-conflict-decoy-document.html)

**Pay2Key**
- **描述**: Pay2Key (https://attack.mitre.org/software/S0556) 具有收集受害者主机名的能力。(引用: Check Point Pay2Key November 2020)
- **外部引用**:
  - Check Point. (2020, November 6). Ransomware Alert: Pay2Key. Retrieved January 4, 2021. [链接](https://research.checkpoint.com/2020/ransomware-alert-pay2key/)

**InnaputRAT**
- **描述**: InnaputRAT (https://attack.mitre.org/software/S0259) 的一些变种通过修改注册表键 HKU\Software\Microsoft\Windows\CurrentVersion\Run:%appdata%\NeutralApp\NeutralApp.exe 来建立持久性。(引用: ASERT InnaputRAT April 2018)
- **外部引用**:
  - ASERT Team. (2018, April 04). Innaput Actors Utilize Remote Access Trojan Since 2016, Presumably Targeting Victim Files. Retrieved July 9, 2018. [链接](https://asert.arbornetworks.com/innaput-actors-utilize-remote-access-trojan-since-2016-presumably-targeting-victim-files/)

**Rising Sun**

- **外部引用**:
  - Sherstobitoff, R., Malhotra, A., et. al.. (2018, December 18). Operation Sharpshooter Campaign Targets Global Defense, Critical Infrastructure. Retrieved May 14, 2020. [链接](https://www.mcafee.com/enterprise/en-us/assets/reports/rp-operation-sharpshooter.pdf)

- **外部引用**:

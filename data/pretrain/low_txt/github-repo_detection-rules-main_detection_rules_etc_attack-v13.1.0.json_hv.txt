**Waterbear (S0579)**
- **描述**: Waterbear 恶意软件通过删除特定的注册表项来加载恶意 DLL 文件。(引用: Trend Micro, 2019 年 12 月)
- **引用**: Su, V. 等人 (2019 年 12 月 11 日). "Waterbear Returns, Uses API Hooking to Evade Security". [链接](https://www.trendmicro.com/en_us/research/19/l/waterbear-is-back-uses-api-hooking-to-evade-security-product-detection.html)

**Emissary (S0082)**
- **描述**: Emissary 恶意软件能够将自身配置为服务。(引用: Emissary Trojan Feb 2016)
- **引用**: Falcone, R. 和 Miller-Osborn, J. (2016 年 2 月 3 日). "Emissary Trojan Changelog: Did Operation Lotus Blossom Cause It to Evolve?". [链接](http://researchcenter.paloaltonetworks.com/2016/02/emissary-trojan-changelog-did-operation-lotus-blossom-cause-it-to-evolve/)

**Magic Hound (G0059)**
- **描述**: Magic Hound 组织在 Administrators 和 Remote Desktop Users 组中添加了一个名为 DefaultAccount 的用户。(引用: DFIR Report APT35 ProxyShell March 2022)
- **引用**: DFIR Report (2022 年 3 月 21 日). "APT35 Automates Initial Access Using ProxyShell". [链接](https://thedfirreport.com/2022/03/21/apt35-automates-initial-access-using-proxyshell)

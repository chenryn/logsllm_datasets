### Log Field Definitions

The following log fields are used in the FortiOS 6.2.5 log messages:

- **ip**: IP address (Length: 39)
- **level**: Log level (Length: 11)
- **logdesc**: Log description (Length: 4096)
- **logid**: Log ID (Length: 10)
- **msg**: Log message (Length: 4096)
- **sn**: Serial number (Length: 64)
- **subtype**: Log subtype (Length: 20)
- **time**: Time (Length: 8)
- **type**: Log type (Length: 16)
- **tz**: Time zone (Length: 5)
- **vd**: Virtual domain (Length: 32)
- **date**: Date (Length: 10)
- **devid**: Device ID (Length: 16)
- **eventtime**: Event time (Length: 20, DataType: uint64)

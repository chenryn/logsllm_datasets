### Lotus Notes 的重生

HCL 最近发布了群件服务器 Domino 12，并通过视频宣布“Notes 和 Domino 已经转型”。这款曾由 IBM 拥有、后于 2018 年被 HCL 收购的软件，现已转型为云原生和低代码产品，成为 Exchange 的一个可行替代方案。Domino 应用程序不仅可以在浏览器中运行，还提供了一个低代码开发环境。此外，该软件支持包括 AWS、Azure 和 Google 在内的多种云平台，甚至可以在 Kubernetes 和 OpenShift 上运行。

> 我以为 Lotus Notes 只存在于一些大型企业中，没想到它已经与时俱进，实现了现代化转型。

### RHEL 上部署 SQL Server 的流行度

![](/data/attachment/album/202106/08/235735xm8f8spzqkfmqpik.jpg)

红帽公司在 2020 年底对 SQL Server 专业协会的 300 多名成员进行了调查，其中 54% 的受访者是数据库管理员。结果显示，31% 的受访者在其站点上至少运行了一个 Linux 操作系统，而 12% 的受访者表示他们正在 RHEL 上运行 SQL Server。预计在未来两年内，采用 Linux 作为 SQL Server 平台的人数将增加两倍以上。

> 尽管微软早已发布 Linux 版的 SQL Server，但我还是惊讶于其在 Linux 上的实际使用率。

### iOS 15 中的新定位功能

![](/data/attachment/album/202106/08/235750osonojoejoju8csm.jpg)

在 iOS 15 之前，iPhone 关机后无法通过“查找我的网络”进行定位。然而，在最新的 iOS 15 中，即使设备已关机或被重置，仍可定位其位置。这一新功能使得 iPhone 在关机状态下保持低功耗模式，类似于 AirTag，允许附近的 iOS 设备接收蓝牙信号并报告位置信息。当然，用户可以选择关闭此功能。

另外，当手机被恢复到出厂设置时，如果启用了激活锁，位置追踪将继续有效。若丢失的 iPhone 被转售给他人，欢迎屏幕上会显示“设备已锁定，可被定位”的提示。

> 关机后的 iPhone 或许仍在悄悄工作……

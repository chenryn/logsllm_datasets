### 监控第三方应用日志
**描述**：监控可能利用软件漏洞导致应用程序或系统崩溃并拒绝服务的第三方应用程序日志、消息和其他工件。[11] 针对 Web 应用程序的攻击可能会在 Web 服务器、应用程序服务器和数据库服务器中生成日志，这些日志可以用于识别攻击类型。外部监控可能成为端点拒绝服务目标的服务可用性。

**引用来源**：

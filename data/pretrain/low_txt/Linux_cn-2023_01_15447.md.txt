### Copilot 新增 Photoshop 式代码“笔刷”功能

在 Copilot 实验室的 VS 扩展中，新添了一款笔刷工具箱。该工具允许用户通过选择特定代码行并应用不同类型的笔刷来修改代码。目前提供的笔刷包括类型笔刷、错误修复笔刷、调试笔刷及易读性笔刷等，并计划未来加入更多选项，甚至支持开发人员自定义笔刷。

该项目旨在通过简单直观的界面赋予开发者更大的灵活性与控制力，而非单纯追求自动化处理。当前订阅此笔刷工具箱服务的费用为每月 10 美元。
![](/data/attachment/album/202301/15/192126en0uau64a6t0u6xr.gif)

**消息来源：[GitHub Next](https://githubnext.com/projects/code-brushes)**

> 老王点评：设想一下，将来或许可以选择“卡马克”笔刷，从而轻松写出像大神一样的代码！

### Ubuntu 实时内核即将全面上线

Canonical 在 Ubuntu 22.04 LTS 中推出了实时内核测试版。经过一年的发展，现在官方宣布带实时内核的 Ubuntu 即将全面上市（GA）。值得注意的是，这项功能作为 Ubuntu Pro 或 Ubuntu Advantage 订阅的一部分对个人用户免费提供。同时，上游实时补丁系列也已基本准备就绪，仅剩一些小问题待解决。如果今年能够顺利将实时内核支持集成进 Linux 内核主线，则将进一步简化其他发行版引入实时内核的过程。

**消息来源：[Phoronix](https://www.phoronix.com/news/Ubuntu-RT-Kernel-GA-Coming)**

> 老王点评：得益于社区各家企业和开发者的共同努力，Linux 的实时内核终于接近完成阶段，这对拓展 Linux 应用场景具有重要意义。

![](/data/attachment/album/202301/15/191731zu7atc037zctytfp.jpg)

### 大量第三方 Twitter 应用失效

近期，众多第三方 Twitter 客户端突然停止工作，但具体原因尚不清楚。Twitter 官方尚未对此作出解释。有消息称，由于新管理层解雇了负责维护 API 运行顺畅的技术团队成员，导致这些客户端无法正常访问 API 并收到“401 未授权”的错误提示。推测可能是因为这些第三方应用过滤广告、按时间顺序展示推文或者用户基数超过了一定阈值所致。

**消息来源：[The Register](https://www.theregister.com/2023/01/14/third_party_twitter_clients_stop_working/)**

> 老王点评：记得国内某款模仿 Twitter 的社交平台走向衰落的一个征兆就是其 API 不稳定且逐渐限制第三方接入。

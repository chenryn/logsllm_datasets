### 概述

KDE 是有史以来功能最强大的桌面环境之一，开源且可自由使用。19年前，即1996年10月14日，德国程序员 Matthias Ettrich 开始了这个美观的桌面环境的开发。KDE 提供了用户界面以及其他许多日常使用的程序。如今，KDE 被成千上万的 Unix 和 Windows 用户使用。对于一个软件项目来说，19年是一个相当长的时间。现在是时候回顾一下这一切是如何开始的。

![](/data/attachment/album/201607/09/092813oufgdpuf4w8l4w14.jpg)

K Desktop Environment (KDE) 在多个方面进行了创新：新设计、美观、一致的用户体验、易于使用，并且为普通用户和专业用户提供了强大的应用库。“KDE”这个名字是对“通用桌面环境”（Common Desktop Environment）的一个谐音游戏，“K”代表“Cool”。第一代 KDE 使用了 Trolltech 公司专有的 Qt 框架（现 Qt 的前身），并在双许可证下发布：开源的 QPL（Q Public License）和商业专利许可证。2000年，Trolltech 公司将部分 Qt 库在 GPL 许可证下发布；Qt 4.5 则在 LGPL 2.1 许可证下发布。自2009年起，KDE 桌面环境由三部分组成：Plasma Workspaces（用于交互界面）、KDE Applications 和作为 KDE Software 编译基础的 KDE Platform。

### 各发布版本

#### 预发布版本 – 1996年10月14日

![](/data/attachment/album/201601/29/080552b56119trtfzhh00r.png)

当时名为 Kool Desktop Environment；“Kool”这个词很快就被弃用了。最初，所有 KDE 组件都是单独发布在开发社区中，并没有被整合到一个大型项目中。开发组邮件列表中的首选通信方式是发送至 [PI:EMAIL](mailto:PI:EMAIL) 邮件列表。

#### KDE 1.0 – 1998年7月12日

![](/data/attachment/album/201601/29/080553wpyoopprwlpdwd09.png)

该版本收到了颇具争议的反馈。许多人反对使用 Qt 框架，因为当时的 FreeQt 许可证与自由软件许可证不兼容，他们建议开发组使用 Motif 或 LessTif 替代。尽管如此，KDE 仍然受到了许多用户的青睐，并成功作为第一个 Linux 发行版的桌面环境被集成。

![1999年1月28日](/data/attachment/album/201601/29/080555sqqc111x5d1a1aa7.png)

有一次升级，**K Desktop Environment 1.1**，更快、更稳定的同时加入了许多小的改进。这个版本还加入了新的图标、背景和材质纹理。同时出现的是 Torsten Rahn 绘制的全新 KDE 图标——一个放在齿轮前的字母 K；这个图标的修改版一直沿用至今。

#### KDE 2.0 – 2000年10月23日

![](/data/attachment/album/201601/29/080556hhn9h3h4qs0y0qsh.png)

重大更新：
* DCOP (Desktop COmmunication Protocol)，一个端到端的通信协议
* KIO，一个应用程序 I/O 库
* KParts，组件对象模型
* KHTML，一个符合 HTML 4.0 标准的渲染引擎

![2001年2月26日](/data/attachment/album/201601/29/080557hqmzqcq15qsod6dd.png)

**K Desktop Environment 2.1** 首次发布了媒体播放器 noatun，它采用了模块化和插件设计。为了方便开发者，K Desktop Environment 2.1 还打包了 KDevelop。

![2001年8月15日](/data/attachment/album/201601/29/080600iy3w2lx42f1eby46.png)

**KDE 2.2** 版本在 GNU/Linux 上加快了50%的应用启动速度，同时提高了 HTML 渲染、JavaScript 稳定性和性能，并增加了 KMail 的一些功能。

#### KDE 3.0 – 2002年4月3日

![](/data/attachment/album/201601/29/080602nj32hyd828ydw4fp.png)

K Desktop Environment 3.0 加入了更好的限制使用功能，这一功能在电话亭、网咖和企业公用电脑上广泛需求，它可以禁止用户完全使用软件的某些能力。

![2003年1月28日](/data/attachment/album/201601/29/080603fafu14dhd3uuwfpa.png)

**K Desktop Environment 3.1** 加入了新的默认窗口样式（Keramik）和图标样式（Crystal）以及其他一些改进。

![2004年2月3日](/data/attachment/album/201601/29/080604nq256msd1i5zo22m.png)

**K Desktop Environment 3.2** 加入了诸如在网页表格和书写邮件中进行拼写检查的新功能，增强了邮件和日历功能，完善了 Konqueror 中的标签机制，并支持 Microsoft Windows 桌面共享协议（RDP）。

![2004年8月19日](/data/attachment/album/201601/29/080608dil9hce9j266azm6.png)

**K Desktop Environment 3.3** 侧重于组合不同的桌面组件。Kontact 被整合进群件应用 Kolab 并与 Kpilot 结合。Konqueror 的加入使 KDE 有了更好的即时通讯功能，如支持发送文件和其他 IM 协议（如 IRC）的支持。

![2005年3月16日](/data/attachment/album/201601/29/080609v77j8qj8e63ms936.png)

**K Desktop Environment 3.4** 侧重于提高易用性。这次更新为 Konqueror、Kate 和 KPDF 加入了文字-语音转换功能，并在桌面系统中加入了独立的 KSayIt 文字-语音转换软件。

![2005年11月29日](/data/attachment/album/201601/29/080611fplq9l4qqxzj4bbi.png)

**The K Desktop Environment 3.5** 发布时加入了 SuperKaramba，为桌面环境提供了易于安装的小部件机制。Konqueror 还加入了广告屏蔽功能，并成为历史上第二个通过 Acid2 CSS 测试的浏览器。

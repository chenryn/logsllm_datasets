LinuxCon 将首次在中国举行，定于下周一在北京国家会议中心隆重开幕。届时，Linus Torvalds 大神将首次现身中国。不仅如此，微软公司也将再次亮相 LinuxCon，自 2015 年以来，微软已多次参加该活动，并于去年 11 月成为 Linux 基金会的白金会员。

![](/data/attachment/album/201706/16/182108dn7744s99do417yr.png)

尽管有些人对微软和开源的合作仍持谨慎态度，但近年来，微软在开源领域的举措已经非常显著。为了深入了解此次微软在 LinuxCon 的演讲内容，我特别联系了 Linux 基金会的 Maggie，以获取更多信息。

### 《我变故我在——微软与开源共舞》

根据日程安排，微软将派遣多位重量级代表出席本次 LinuxCon，其中包括微软中国云计算与企业事业部资深合伙人及高级总监梁戈碧女士。她不仅会在技术女性午餐会上发言，还将带来题为《我变故我在——微软与开源共舞》的演讲。

![](/data/attachment/album/201706/16/182155kq6mm7bbbqh298kc.jpg)
*梁戈碧女士*

梁戈碧女士拥有威廉玛丽学院的计算机科学硕士学位。在加盟微软之前，她在英特尔工作了 15 年，担任多个技术和领导职务。她在云计算领域有卓越成就，曾领导设计 Windows Azure 存储等面向全球客户的云计算产品和服务。她还积极推动微软内部的开源技术和解决方案，并积极参与女性平权活动，为 IT 领域的女性发展做出了巨大贡献。

让我们来猜测一下梁女士的演讲内容：

**✿ 回顾微软的开源历史**
> 
> 2014 年 10 月 20 日，微软首席执行官萨提亚·纳德拉首次公开表示微软对 Linux 和开源的支持。从那时起，“把 Linux 放在重中之重”已成为微软内部的准则。
> 
> —— 《[拥抱开源，微软是认真的](https://news.microsoft.com/zh-cn/拥抱开源，微软是认真的/)》
> 

![](/data/attachment/album/201706/16/182505f6cb7jfbnfjjf9nm.jpg)
*微软开源简史*

如今，微软在开源和 Linux 方面投入了大量资源，如 PowerShell、Visual Studio Code 和 Edge JavaScript 引擎等项目均已开源。此外，Windows 10 内置了 Ubuntu，收购 Xamarin 后也开源了其软件开发工具包。微软还与 Red Hat 和 SUSE 合作，在 Azure 云上推出了官方支持的 Linux 镜像，并发布了 Linux 版 SQL Server。

![](/data/attachment/album/201706/16/182546k9ny2ttj1f21ollz.jpg)
*开源在整个微软公司蔚然成风*

**✿ 解释微软如何与 Linux 社区及其他社区合作**
据我们了解，Linux 基金会并不是微软加入的第一个行业领袖级 Linux 团体。微软已参与了多个 Linux 基金会项目，包括 Node.js 基金会、OpenDaylight、开放容器促进联盟、R 社团和开放 API 促进联盟等。

**✿ 分享微软的开源战略**
根据去年的章鱼猫报告，GitHub 上开源贡献者最多的组织是微软，超过了 Facebook、Docker 和谷歌。

![](/data/attachment/album/201706/16/182715cvzy2oddzoo2666f.jpg)
*GitHub 的 2016 章鱼猫报告中，微软以贡献者数量最多而排名第一*

这充分证明了微软的开源策略的有效性。微软的开源策略分为四个方面：支持、整合、发布和贡献。

![](/data/attachment/album/201706/16/182749lifdvptdjqfftgjg.jpg)
*微软的开源策略*

**✿ 通过与社区密切合作，让核心创新为客户带来更多敏捷性和创造性**
随着微软在开源方面的不断努力，其核心产品如 SQL Server、Windows 10 内置的 Linux 子系统以及 .NET Core 和 TypeScript 等开发环境和工具语言纷纷开源或支持 Linux。

![](/data/attachment/album/201706/16/182835s036cn1fdqphvd0y.jpg)
*微软核心产品开源或支持 Linux*

Azure 云业务也在各个层面上全面支持开源，包括基础设施、容器、数据库、中间件、平台框架和应用程序，均采用了大量的开源组件。

![](/data/attachment/album/201706/16/182906lezzezm2oj0mwg3q.jpg)
*Azure 在各个层次全面支持开源*

梁女士的演讲时间虽然只有短短 15 分钟，但她所表达出的微软对开源的信心和坚定态度令人瞩目。开源社区应该正视这个软件巨人在开源世界中的新角色。

### LinuxCon 技术女性午餐会

除了这段演讲外，我还非常关注由微软主持的技术女性午餐会。虽然作为一个中年大叔，但我坚决支持女性在技术和开源领域的发展（我不是为了混进午餐会才这么说的！）。据说，有人引荐的话，像我这样的进步分子是可以参加技术女性午餐会的。我正在考虑是否要推掉下周一中午的其他邀约……

国内技术圈对技术女性的处境和发展尚未形成硅谷那样的潮流，但我们也看到了一些相关活动和观点。事实上，国内技术女性的发展仍处于早期阶段，因此关怀她们的发展是技术界应重视的事情。这也是为什么 LinuxCon 上会有专门的技术女性午餐会的原因。

请大家期待我的第一时间火线报道吧！:-D

那么，LinuxCon 见！

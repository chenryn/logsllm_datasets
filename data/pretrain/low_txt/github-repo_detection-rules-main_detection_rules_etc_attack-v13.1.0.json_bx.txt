### PoetRAT
**Description:**
PoetRAT is a remote access trojan (RAT) first identified in April 2020. It has been used in multiple campaigns targeting both private and public sectors in Azerbaijan, including Industrial Control Systems (ICS) and Supervisory Control and Data Acquisition (SCADA) systems in the energy sector. The STIBNITE activity group has been observed using this malware. PoetRAT derives its name from references to the poet William Shakespeare found in its code.

**References:**
- [Talos PoetRAT April 2020](https://blog.talosintelligence.com/2020/04/poetrat-covid-19-lures.html)
- [Talos PoetRAT October 2020](https://blog.talosintelligence.com/2020/10/poetrat-update.html)

**Description:**
CHOPSTICK is a modular backdoor malware family used by APT28 since at least 2012. It is typically deployed as second-stage malware but has also been used as first-stage malware in some cases. CHOPSTICK has both Windows and Linux variants.

**ATT&CK ID:** T1505  
**ATT&CK Technique Name:** Server Software Component

Adversaries may exploit legitimate extensible development features of server applications to gain persistent access to systems. Many enterprise server applications offer functionality that allows developers to write and install additional software or scripts, thereby extending the core application's capabilities. Adversaries can leverage these features to install malicious components, which in turn enable them to extend and abuse the server application for their own purposes.

> Java 是免费的？你想多了。Oracle 的律师已经在赶来的路上了。

### Oracle 开始追索 Java SE 许可费用

在收购 Sun 微系统公司六年后，Oracle 终于开始大规模审计 Java 客户，声称他们违反了许可协议。不仅客户受到影响，连合作伙伴也未能幸免。

![](/data/attachment/album/201612/18/215414pg1ga6ww7ikwnwji.jpg)

2010 年，随着 Oracle 收购 Sun 微系统公司，它也拥有了 Java。然而直到现在，其许可管理服务部门 (License Management Services, LMS) 才开始积极追讨许可费用。今年，Oracle 在全球雇佣了 20 名员工，专门负责向违反 Java 许可协议的企业追索费用。行业合规专家组也在扩招人马，预计 2017 年将有更多的针对 Java 的行动。此举涉及巨额金额，使用 Java 的客户可能面临数万甚至数十万美元的追缴。

引起争议的 Java 版本是 Java SE，有三个付费分支版本，每用户费用为 $40 到 $300，每处理器费用为 $5000 到 $15000。据报道，某零售业拥有八万台 PC 的客户已接到 Oracle 通知，声称其违反了 Java 协议。另一个 Java 客户则被 Oracle 索要十万美金，后降至三万美金。

Palisade Compliance 的 CEO 兼创始人 Craig Guarente 表示，Oracle 此举不仅针对客户，还针对合作伙伴。“Oracle 同样针对其合作伙伴。这让人们感到愤怒，因为他们一直在帮助 Oracle。”他说道。

### Java 是开源的，但 Java SE 安装包不是免费的

根本原因是大家对 Java 存在误解，认为它是“免费”的。这种观点可以追溯到 Sun 时代；Sun 时代的 Java 是免费可用的，在 Oracle 下也是如此。然而，Sun 曾对 IBM 和蓝光播放器制造商等收取许可证费用，尽管对大多数用户不收费。这是因为 Sun 利用 Java 作为“楔子的尖端”，推动其系统的销售。

Oracle 决定进一步商业化 Java，推出了多个付费版本，如 Java SE Advanced Desktop、Java SE Advanced 和 Java SE Suite。这些版本包括额外的功能，如 Java Mission Control 和 Flight Recorder。此外，还有用于大规模 Java 部署的 Microsoft Windows Installer Enterprise JRE Installer。

Java SE 在通用计算场合（如桌面电脑、笔记本、智能手机和平板电脑）是免费的，但在专业嵌入式计算机中使用时则需付费。由于无法在下载时将付费组件从免费的 Java SE 中分离出来，因此用户必须支付整个套件的费用。

“人们没有意识到，他们以为 Java 是免费的 - 因为它是开源的，所以你可以使用它。这里有一个基本的误解。”Guarente 解释道。

### 为什么 Oracle 现在才收网？

为什么 Oracle 在通过收购 Sun 获得 Java 所有权六年之后才采取行动？普遍认为，LMS 用了很长时间来设计审计方法并详细了解客户的 Java 使用情况。LMS 现已准备好在 2017 年向 Java SE 用户开刀。

“我预计 Oracle 在 2017 年会继续更加关注此事。所有趋势显示，Oracle 的 LMS 审计团队变得更加强势，试图获得比往年更高的收入。我不认为 2017 年 Oracle 会变得温和仁慈。”Guarente 说道。

### 你们该怎么做？

专家建议下载 Java SE 时要格外小心，已经下载的客户应重新审视使用方式，以应对 LMS 的审查。据报道，2016 年下半年，寻求 Java 许可帮助的客户数量显著增加。

“如果你下载了 Java，你就得到了全部软件，你需要确保只安装你被授权的组件，并移除未使用的部分。”专家警告说，“如果你已经有了 Java 应用，先确定实际使用的特定组件及其使用方式，然后在 Oracle 找上门之前验证是否有使用问题。”

目前尚未看到 Oracle 对此的官方回应。

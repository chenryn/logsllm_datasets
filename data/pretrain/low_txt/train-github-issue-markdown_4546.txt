# Environment
- Version: 0.18.1
- Application: PowerToys Run

# Steps to Reproduce
1. Open PowerToys Run.
2. Search for any item on your system (e.g., an application, file, or folder).

# Expected Behavior
- Search results should be displayed instantaneously.

# Actual Behavior
- When searching for an application, there is a noticeable delay before the search results are displayed.

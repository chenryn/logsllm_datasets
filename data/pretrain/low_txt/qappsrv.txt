### qappsrv

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `qappsrv` command displays a list of all Remote Desktop Session Host (RDSH) servers on the network. This is useful for administrators who need to manage or monitor RDSH servers.

**Usage:**
To see what's new in the latest version of Remote Desktop Services, refer to the documentation: [What's New in Remote Desktop Services in Windows Server](#).

**Note:**
The `qappsrv` command is functionally equivalent to the `query termserver` command. Both commands provide the same output and can be used interchangeably.

# Rule Coverage

ATT&CK Navigator layer files are generated when a package is built using the `make release` command or by running `python -m detection-rules`. These files can be downloaded from all successful builds and can be used in a custom ATT&CK Navigator session. For convenience, the links to these files are provided below. You can include multiple layers across different tabs in a single session, but it is not recommended to upload all of them simultaneously, as this may overload your browser's resources.

## Current Rule Coverage

The source files for these links are regenerated with every successful merge to the `main` branch. These files represent the coverage based on the state of the rules in the `main` branch.

### Lazarus Group Malware: IndiaIndia

**Description:**
The Lazarus Group malware, known as IndiaIndia, gathers information about the victim and saves it to a file. This file is then compressed using Zlib, encrypted, and uploaded to a Command and Control (C2) server.

**References:**
- Novetta Threat Research Group. (2016, February 24). Operation Blockbuster: Remote Administration Tools & Content Staging Malware Report. Retrieved March 16, 2016.
- Sherstobitoff, R. (2018, February 12). Lazarus Resurfaces, Targets Global Banks and Bitcoin Users. Retrieved February 19, 2018.

Avaddon ransomware searches for and attempts to stop database processes on the infected system.

### Fysbis Backdoor

**Description:**
Fysbis, a Linux backdoor, establishes persistence by installing itself as an autostart entry under `~/.config/autostart/dbus-inotifier.desktop`.

**Reference:**
- Doctor Web. (2014, November 21). Linux.BackDoor.Fysbis.1. Retrieved December 7, 2017.

### FatDuke Malware

**Description:**
FatDuke malware can call the `ShellExecuteW` function to open the default browser on the URL `localhost`.

**Reference:**

**监测 Python 使用情况**

- **创建时间**：2022年3月30日 14:26:51
- **描述**：监测系统中的异常 Python 使用情况和 python.exe 行为，这可能是恶意活动的指示器。理解标准使用模式对于避免大量误报非常重要。如果脚本对普通用户是受限的，那么任何尝试启用脚本运行的行为都应被视为可疑。如果系统中不常用脚本，但启用了脚本，则不在补丁或管理员功能周期内运行的脚本应被视为可疑。尽可能从文件系统中捕获脚本来确定其行为和意图。脚本可能会执行各种影响系统的操作，具体取决于所使用的监控类型。监控可能滥用 Python 命令和脚本执行的已执行命令和参数。

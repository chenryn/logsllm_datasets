# Collaborative Document Creation Tools

- **Riseup Pad**: A secure, collaborative text editor that allows multiple users to work on a document simultaneously. Accessible at [pad.riseup.net](https://pad.riseup.net).

- **Rustpad**: A self-hostable, real-time collaborative editor. It can be hosted on your own server for enhanced privacy and control. More information and the source code are available on [GitHub](https://github.com/ekzhang/rustpad). You can also try it out at [rustpad.io](https://rustpad.io).

# Collaborative Terminal Tools

- **sshx**: A tool designed for collaborative terminal sessions. While it appears to be a promising solution, it's important to note that it may use their servers to facilitate the connection, which could raise privacy concerns. For more details, visit [sshx.io](https://sshx.io/).

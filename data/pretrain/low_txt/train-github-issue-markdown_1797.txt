### Environment
- **Windows Build Number:** 10.0.18362.175

### Steps to Reproduce
1. Install Ubuntu and Ubuntu 16.04 LTS.
2. Open the new Windows Terminal.
3. Check the tools menu for options to open the installed distributions.

### Expected Behavior
The tools menu in the new Windows Terminal should display options to open the installed Ubuntu distributions, along with PowerShell and the old Windows Command Prompt (CMD).

### Actual Behavior
The tools menu in the new Windows Terminal only shows options for PowerShell and the old Windows CMD. The installed Ubuntu distributions do not appear in the menu.

### SLOTHFULMEDIA

**SLOTHFULMEDIA** is a remote access Trojan (RAT) written in C++ that has been in use by an unidentified, sophisticated cyber actor since at least January 2017. This malware has targeted government organizations, defense contractors, universities, and energy companies in Russia, India, Kazakhstan, Kyrgyzstan, Malaysia, Ukraine, and Eastern Europe.

- **References:**
  - CISA MAR SLOTHFULMEDIA October 2020
  - Costin Raiu IAmTheKing October 2020
  - USCYBERCOM SLOTHFULMEDIA October 2020
  - Kaspersky IAmTheKing October 2020
  - ESET PowerPool Code October 2020

In October 2020, Kaspersky Labs identified **SLOTHFULMEDIA** as part of an activity cluster referred to as "IAmTheKing." Additionally, ESET noted code similarities between **SLOTHFULMEDIA** and droppers used by a group known as "PowerPool."

- **Aliases:**
  - SLOTHFULMEDIA
  - JackOfHearts
  - QueenOfClubs

### FALLCHILL

**FALLCHILL** is a Remote Access Trojan (RAT) used by the Lazarus Group since at least 2016. It targets the aerospace, telecommunications, and finance industries. The malware is typically dropped by other Lazarus Group malware or delivered when a victim unknowingly visits a compromised website.

- **References:**
  - US-CERT FALLCHILL Nov 2017

### Small Sieve

**Small Sieve** is a Python-based backdoor that uses the Telegram Bot API. It has been distributed using a Nullsoft Scriptable Install System (NSIS) Installer and has been used by MuddyWater since at least January 2022. Security researchers have also linked its use to UNC3313, which may be associated with MuddyWater.

- **References:**
  - DHS CISA AA22-055A MuddyWater February 2022
  - NCSC GCHQ Small Sieve Jan 2022
  - Mandiant UNC3313 Feb 2022

- **Aliases:**
  - Small Sieve
  - GRAMDOOR

### Net

**Net** is a command-line utility included in the Windows operating system. It is used for managing users, groups, services, and network connections. **Net** provides a wide range of functionalities, including gathering system and network information, lateral movement through SMB/Windows Admin Shares, and interacting with services. The `net1.exe` utility is sometimes executed for specific functionality when `net.exe` is run.

- **References:**
  - Microsoft Net Utility
  - Savill 1999

- **Aliases:**
  - Net
  - net.exe

### ShimRatReporter

**ShimRatReporter** is a tool used by the suspected Chinese adversary Mofang to automatically conduct initial discovery. The gathered details are used to customize follow-on payloads and set up faux infrastructure mimicking the adversary's targets. It has been used in campaigns targeting multiple countries and sectors, including government, military, critical infrastructure, automobile, and weapons development.

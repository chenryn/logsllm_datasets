## 在《Minecraft》中启动 Windows 95 并运行 Doom

近期，《Minecraft》社区推出了一款新的虚拟机电脑 Mod，允许玩家在游戏世界中构建一台能够启动 Windows 95 及其他操作系统的计算机。该 Mod 采用了 VirtualBox 技术。通过 VM Computers Mod，一些玩家甚至成功地在《Minecraft》内部运行了经典游戏 Doom。

**来源**：[cnBeta.COM](https://hot.cnbeta.com/articles/game/1007723.htm)

> **拍一拍**：也许我们正生活在一个巨大的“我的世界”之中。

## 马斯克谈 Twitter 安全：私信多为表情包

在 2020 年 7 月 15 日的一场比特币骗局中，超过 100 位知名人士的 Twitter 账号遭到黑客入侵，其中包括特斯拉 CEO 埃隆·马斯克。针对此次事件，马斯克表示：“虽然我的私信中可能有一些听起来不太好的内容，但大部分都是与朋友们互换的表情包。”

**来源**：[凤凰网科技](https://www.cnbeta.com/articles/tech/1007787.htm)

> **拍一拍**：用表情包交流也挺有趣的吧？哈哈

## Ubuntu Web：Firefox 支持下的新型 Web 操作系统

Ubuntu Unity Remix 和 Ubuntu Education 的主要开发者 Rudra Saraswat 正计划开发一款名为 Ubuntu Web 的全新操作系统。这款基于 Web 的操作系统将结合 Ubuntu Linux 和 Mozilla Firefox 浏览器的功能，作为 Chrome OS 的替代品。与后者不同的是，Ubuntu Web 将使用 Firefox 作为其核心浏览器，并且有望支持各种类型的计算机设备。

**来源**：[开源中国](https://www.oschina.net/news/117457/ubuntu-web-firefox-chrome-os)

> **拍一拍**：期待这个项目能够打破谷歌在这一领域的垄断地位。

## 未知黑客成功干扰 Emotet 僵尸网络

经过五个月的沉寂后，Emotet 僵尸网络于上周重新活跃起来。然而，一位身份不明的黑客迅速采取行动，通过替换恶意载荷中的 GIF 动画阻止了潜在受害者的感染。Emotet 是一种极具威胁性的恶意软件，它能够窃取银行账户信息并安装多种类型的恶意程序。据追踪数据显示，有四分之一的 Emotet 恶意负载下载受到了干扰。

**来源**：[solidot](https://www.solidot.org/story?sid=65049)

> **拍一拍**：干得漂亮！

### 谷歌 CEO 提醒员工不要将乐趣与金钱等同

在最近的一次内部会议上，谷歌员工对公司在创纪录利润后削减福利和差旅预算表达了不满。对此，谷歌 CEO 桑达尔·皮查伊（Sundar Pichai）回应称：“我记得当谷歌规模很小的时候，我们也很艰难。……我们不应该总是把乐趣等同于金钱。我认为，在一家努力工作的创业公司里，人们可能正在享受乐趣……”。员工还询问了高管薪酬的问题，但据说皮查伊并未直接回答。据估计，他的个人财富超过 10 亿美元，去年获得了 630 万美元的报酬。

> **[消息来源：CNBC](https://www.cnbc.com/2022/09/23/google-ceo-pichai-fields-questions-on-cost-cuts-at-all-hands-meeting-.html)**
>
> **老王点评**：尽管这番话有道理，但亿万富翁这样的回答总让人感觉有些不足。

![](/data/attachment/album/202209/26/165552bj9oms7z9m799rmo.jpg)

### Mozilla 承诺继续支持 WebRequest API

目前大多数基于 Chromium 的扩展程序使用的是 Manifest V2 (MV2)，而谷歌计划从明年开始采用新的扩展规范 Manifest V3 (MV3)，限制 MV2 中 WebRequest API 的功能。从明年 1 月起，使用旧 API 的扩展将不再被列出，并最终被禁用。像 uBlock Origin 这样的广告拦截扩展依赖 WebRequest API 在广告下载前进行拦截，若该 API 被禁用，用户体验将受到影响，例如页面闪烁等问题。上周，Mozilla 重申它将继续支持 MV3，同时也承诺会继续支持 MV2 的 WebRequest API。

> **[消息来源：ghacks](https://www.ghacks.net/2022/09/24/mozilla-reaffirms-that-firefox-will-continue-to-support-current-content-blockers/)**
>
> **老王点评**：这就是浏览器垄断带来的问题，无论你是否愿意接受。这次我觉得 Mozilla 做得不错，至少给了用户选择的权利。

![](/data/attachment/album/202209/26/165609kjqv9iiqa2ibqvnq.jpg)

### 中国拥有全球近半工业机器人

根据国际机器人联合会（IFR）的一份新报告，中国现在几乎拥有全球所有工业机器人装置的一半，而且这一领先优势正在迅速扩大。2021 年，中国的工业机器人安装量达到了 243,300 台的新纪录，同比增长 44%。十年前，中国每万名员工中只有 10 个工业机器人，而到了 2020 年，这一数字已增至 246 个，位居世界第九。

> **[消息来源：IFR](https://ifr.org/ifr-press-releases/news/china-robot-installations-grew-by-44-percent)**
>
> **老王点评**：虽然人均排名仅第九，但如此庞大的总量仍然非常可观。

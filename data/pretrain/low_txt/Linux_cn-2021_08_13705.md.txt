Zorin 团队宣布发布了全新的 Zorin OS 16，带来了许多急需的更新和改进。本文将对这一版本进行详细总结。

![Zorin OS 16 桌面版](/data/attachment/album/202108/21/121824htnmnnx3q2s62qzt.jpg)

Zorin OS 是一款开源且美观的 Linux 发行版，现已发布其最新的稳定版本——Zorin OS 16。该版本将在2025年前提供增强和更新支持。开发团队在确保性能不下降的同时，还引入了一些独特而有用的特性。Zorin OS 使用自己的软件源，同时也兼容 Ubuntu 的软件仓库。

### Zorin OS 16 – 主要新特性

Zorin OS 16 基于 Linux 内核 5.11（HWE 栈），并基于 Ubuntu 20.04 LTS。此版本最显著的变化之一是默认包含了 Flathub 软件仓库。因此，Zorin 应用商店成为 Linux 发行版中最大的应用程序集合之一。它不仅支持 Flathub，还支持 Snap 商店、Ubuntu 软件仓库、Zorin 自有仓库以及 AppImage。

Zorin OS 以其出色的视觉效果而闻名，在这个版本中，有许多改进：
- **新的图标和色彩方案**：默认主题更加精致。
- **预装的设计和壁纸**：提供了新的设计和壁纸。
- **锁屏模糊效果**：锁屏界面现在可以展示自选壁纸的模糊效果，提供更简洁的视觉体验。
- **任务栏改进**：任务栏图标启用了活动指示器和带有计数的通知气泡，用户可以在任务栏图标中获取信息应用的未读消息计数等信息。此外，任务栏还增加了自动隐藏、透明度调整和图标移动等功能。

![新的任务栏通知气泡](/data/attachment/album/202108/21/121824tfmd23d2ztb232pp.png)

新版还进行了许多内部提升，虽然具体细节尚未公布，但据团队表示，所有 Zorin 风格的整体桌面体验相比前一版本 [Zorin 15](https://www.debugpoint.com/2020/09/zorin-os-15-3-release/) 有了显著改进。

此版本引入了两个新应用：
- **Tour 应用**：首次安装后，用户可以通过 Tour 应用概览 Zorin 桌面。
- **录音应用**：新增了一个录音应用。

对于笔记本用户，应用和工作区之间的切换变得更加快捷和简便。Zorin OS 16 支持多点触控手势，开箱即用。用户可以通过上下滑动四个手指以流畅的 1:1 动作在工作区之间切换，或通过三个手指撮合打开活动概览，查看当前运行的所有应用程序。

Zorin OS 16 还支持高分辨率显示器的分数缩放。安装程序现在包含了 NVIDIA 驱动，并支持加密。

详细的更新日志请参阅 [这里](https://blog.zorin.com/2021/08/17/2021-08-17-zorin-os-16-is-released/)。

### Zorin OS 16 最低系统要求

**Zorin OS Core、Education 和 Pro**:
- **CPU**: 1 GHz 双核处理器，Intel/AMD 64 位处理器
- **RAM**: 2 GB
- **存储**: 15 GB（Core & Education）或 30 GB（Pro）
- **显示器**: 800 × 600 分辨率

**Zorin OS LITE**:
- **CPU**: 700 MHz 单核，Intel/AMD 64 或 32 位处理器
- **RAM**: 512 MB
- **存储**: 10 GB
- **显示器**: 640 × 480 分辨率

### 下载 Zorin OS 16

值得注意的是，Zorin 推出了一个售价约 $39 的 PRO 版本，具有类似 Windows 11 风格等额外特性。不过，你仍然可以随时下载免费版本：Zorin OS 16 Core 和 Zorin OS 16 LITE（适用于低配置电脑）。你可以参考 [功能比较](https://zorin.com/os/pro/#compare) 了解更多详情。

你可以从以下链接下载最新的 .iso 文件，然后使用 [Etcher](https://www.debugpoint.com/2021/01/etcher-bootable-usb-linux/) 或其他工具创建可启动 USB 安装盘。

- [下载 Zorin OS 16](https://zorin.com/os/download/)

### 从 Zorin 15.x 升级

目前还没有从 Zorin OS 15 升级到最新版本的路径，但据团队称，未来将会有简单的升级方法。

### 结束语

Zorin OS 的一大优势在于其独特的应用生态系统处理方式。它可能是唯一一个开箱即用的 Linux 桌面发行版，可以通过其软件商店从 Flathub、Snap 商店、AppImage、Ubuntu/自有软件仓库搜索和安装应用。你无需手动为 Snap 或 Flatpak 配置系统。尽管如此，它仍然是一个基于 GNOME 的修改版。有些人可能不喜欢 Zorin OS，因为它预装了许多功能而显得臃肿。但从某种意义上说，它是 Linux 桌面新手的理想选择，特别是那些需要类似 Windows/macOS 系统感觉的现成 Linux 功能的用户。

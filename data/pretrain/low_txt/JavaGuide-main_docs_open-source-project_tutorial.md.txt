## Java

- **[JavaGuide](https://github.com/Snailclimb/JavaGuide)**：这是一份全面的 Java 学习和面试指南，涵盖了大部分 Java 程序员需要掌握的核心知识。
- **[toBeBetterJavaer](https://github.com/itwanger/toBeBetterJavaer)**：一份通俗易懂、风趣幽默的 Java 学习指南，内容包括 Java 基础、集合框架、并发编程、JVM 和企业级开发（如 Git、SSM、Spring Boot）等知识点。
- **[interview-guide](https://github.com/csguide-dabai/interview-guide)**：总结了后端面试中的常见问题，旨在帮助准备互联网开发岗位校招面试的同学。
- **[advanced-java](https://github.com/doocs/advanced-java)**：涵盖高并发、分布式、高可用性、微服务和海量数据处理等领域的进阶知识，适用于希望提升技能的互联网 Java 工程师。
- **[toBeTopJavaer](https://github.com/hollischuang/toBeTopJavaer)**：为有志于成为顶尖 Java 工程师的人提供的一系列教程。
- **[technology-talk](https://github.com/aalansehaiyang/technology-talk)**：汇总了 Java 生态圈中常用的技术框架、开源中间件、系统架构、数据库管理、大型公司架构案例、第三方库使用技巧、项目管理和个人职业发展等内容。
- **[JCSprout](https://github.com/crossoverJie/JCSprout)**：一个正在成长中的 Java 核心知识库。
- **[bestJavaer](https://github.com/crisxuan/bestJavaer)**：一系列旨在帮助你成为更优秀 Java 程序员的学习资料。
- **[java-design-patterns](https://github.com/iluwatar/java-design-patterns)**：展示了多种设计模式在 Java 中的应用示例。

## 计算机基础

- **[cs-self-learning](https://github.com/PKUFlyingPig/cs-self-learning)**：提供了一份详尽的计算机自学路线图，整理了欧美多所名校提供的高质量课程资源。
- **[CS-Notes](https://github.com/CyC2018/CS-Notes)**：包含了技术面试所需的基础知识、LeetCode 解题思路、后端及 Java 靖面指南、春招秋招信息以及操作系统、计算机网络和系统设计等相关内容。
- **[Waking-Up](https://github.com/wolverinn/Waking-Up)**：对计算机基础知识（如计算机网络、操作系统、数据库、Git 等）进行了深入浅出地讲解，并附带了大量的面试题目解析。

## 系统设计

### Spring Boot

- **[springboot-guide](https://github.com/Snailclimb/springboot-guide)**：基于最新版 Spring Boot 的核心知识点总结。
- **[SpringAll](https://github.com/wuyouzhuguli/SpringAll)**：逐步引导读者学习 Spring Boot 及其相关技术栈，如 Shiro、Cloud、Security & OAuth2 等。
- **[Springboot-Notebook](https://github.com/chengxy-nds/Springboot-Notebook)**：通过整合 Redis、RabbitMQ、Elasticsearch、MongoDB、Spring Cloud、Kafka 和 Skywalking 等现代 Web 技术，展示了一系列综合性案例。
- **[springboot-learning-example](https://github.com/JeffLi1993/springboot-learning-example)**：适合初学者及想要巩固已有技能者参考的最佳实践案例集。
- **[spring-boot-demo](https://github.com/xkcoding/spring-boot-demo)**：包含超过 50 个集成示例的深度学习与实战项目。
- **[SpringBoot-Labs](https://github.com/YunaiV/SpringBoot-Labs)**：一系列关于 Spring Boot 的教学文章和代码示例。

### Spring Cloud

- **[SpringCloudLearning](https://github.com/forezp/SpringCloudLearning)**：方志朋编写的《史上最简单的 Spring Cloud 教程》配套源码。
- **[springcloud-learning](https://github.com/macrozheng/springcloud-learning)**：一套完整的 Spring Cloud 教程，覆盖了主要组件的使用方法。
- **[SpringCloud](https://github.com/zhoutaoo/SpringCloud)**：一个基于 Spring Cloud 2.1 的微服务开发脚手架，集成了 spring-security-oauth2、nacos、feign、sentinel 和 springcloud-gateway 等功能模块。

### Nginx

- **[nginx-tutorial](https://github.com/dunwu/nginx-tutorial)**：一系列简明扼要的 Nginx 教程，涉及 HTTP/HTTPS 反向代理、负载均衡器配置、静态网站托管等方面的知识点。

## 大数据

- **[juicy-bigdata](https://github.com/datawhalechina/juicy-bigdata)**：介绍大数据技术入门课程，以轻松有趣的方式带领读者了解这一领域。
- **[flink-learning](https://github.com/zhisheng17/flink-learning)**：提供了 Apache Flink 入门指南、概念解释、实际操作案例分析、性能优化策略以及源码解读等丰富内容。

## 开源书籍

- **[《高并发的哲学原理》](https://github.com/johnlui/PPHC)**：探讨高并发背后隐藏的一个重要原则——识别并分解单一瓶颈点。
- **[《Effective Java（第 3 版）》中英对照版](https://github.com/clxering/Effective-Java-3rd-edition-Chinese-English-bilingual)**：该书各章节的双语版本，方便中文读者学习。
- **[《DDIA（设计数据密集型应用）》中文版](https://github.com/Vonng/ddia)**：Martin Kleppmann 所著《Designing Data-Intensive Applications》一书的中文翻译。
- **[《凤凰架构》](https://github.com/fenixsoft/awesome-fenix)**：专注于构建可信赖的大规模分布式系统的讨论。
- **[《分布式系统模式》中文版](https://github.com/dreamhead/patterns-of-distributed-systems)**：Brendan Burns 撰写《Patterns of Distributed Systems》一书的中文译本。

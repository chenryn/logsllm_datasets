### chgport

**Applies to:**
- Windows Server 2022
- Windows Server 2019
- Windows Server 2016
- Windows Server 2012 R2
- Windows Server 2012

**Description:**
The `chgport` command is used to list or modify COM port mappings to ensure compatibility with MS-DOS applications.

**Note:**
This command has been superseded by the `change port` command. For detailed information, including syntax and parameters, please refer to the documentation for the `change port` command.

- **外部引用**: 
  - Grange, W. (2020年7月13日). Anchor_dns 恶意软件跨平台传播。检索于2020年9月10日。

**UBoatRAT (S0333)**
- **描述**: UBoatRAT 使用 GitHub 和香港的一个公共博客服务进行 C2 通信。(引用: PaloAlto UBoatRAT Nov 2017)
- **外部引用**: 
  - Hayashi, K. (2017年11月28日). UBoatRAT 导航东亚。检索于2018年1月12日。
  - URL: <https://researchcenter.paloaltonetworks.com/2017/11/unit42-uboatrat-navigates-east-asia/>

**Inception (G0100)**
- **描述**: Inception 使用浏览器插件从 Internet Explorer、Chrome、Opera、Firefox、Torch 和 Yandex 中窃取密码和会话。(引用: Symantec Inception Framework March 2018)
- **外部引用**: 
  - Symantec. (2018年3月14日). Inception Framework：仍然活跃且隐藏在代理背后。检索于2020年5月8日。

**Skidmap (S0468)**
- **描述**: Skidmap 能够将其操作者公钥添加到 `authorized_keys` 文件中，以维持在受感染主机上的持久性。(引用: Trend Micro Skidmap)
- **外部引用**: 
  - Remillano, A., Urbanec, J. (2019年9月19日). Skidmap Linux 恶意软件利用 rootkit 功能隐藏加密货币挖矿负载。检索于2020年6月4日。

**Felismus (S0171)**

- **外部引用**: 
  - Somerville, L. 和 Toro, A. (2017年3月30日). 猫鼠游戏：介绍 Felismus 恶意软件。检索于2017年11月16日。

**Ryuk (S0446)**
- **描述**: Ryuk 通过调用 `CreateToolhelp32Snapshot` 来枚举所有正在运行的进程。(引用: CrowdStrike Ryuk January 2019)
- **外部引用**: 
  - Hanel, A. (2019年1月10日). 大规模猎杀 Ryuk：另一种高利润的目标勒索软件。检索于2020年5月12日。

- **描述**: Ryuk 通过注入远程进程来加密文件，使用了 `VirtualAlloc`、`WriteProcessMemory` 和 `CreateRemoteThread` 的组合。(引用: CrowdStrike Ryuk January 2019)

**CozyCar (S0046)**

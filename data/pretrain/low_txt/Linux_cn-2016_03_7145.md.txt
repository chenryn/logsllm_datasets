还记得 Pear OS 吗？这是一个外观酷似 Mac OS X 的 GNU/Linux 发行版。遗憾的是，它已被一家未知的大公司收购，官方已停止了 Pear OS 和 Pear Cloud 的下载服务。

去年我们曾报道过，葡萄牙开发者 Rodrigo Marques 创建了一个名为 PearOS 的克隆版，并将其发布在 SourceForge 上。当时，PearOS 的初始版本让许多习惯于精美且高度模仿 Mac OS X 的老用户感到失望。

然而，经过一段时间的发展，PearOS 最新的更新版本带来了重新设计的 Mac OS X 外观，基于流行的 Ubuntu LTS 版本 14.04 Trusty Tahr。

### PearOS 9.3 发布，基于 Ubuntu 14.04.1 LTS

应读者要求，我们下载了 PearOS 9.3 Live CD 并对其进行了评测。不得不说，这次更新相比初次发布有了显著的进步！

整个 Mac OS X 风格的界面是基于 GNOME 桌面环境构建的，并对 GNOME Shell 用户界面进行了大量调整和优化。建议您自行[下载 PearOS 9.3 ISO 镜像](http://linux.softpedia.com/get/System/Operating-Systems/Linux-Distributions/Pear-Linux-76309.shtml)体验一下。

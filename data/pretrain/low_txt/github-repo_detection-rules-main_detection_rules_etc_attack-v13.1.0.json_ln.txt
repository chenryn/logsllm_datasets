**LSA保护**

- **创建日期**: 2020年1月24日
- **修改日期**: 2020年3月25日
- **描述**: Windows 8.1, Windows Server 2012 R2及更高版本可以通过设置注册表键`HKLM\\SYSTEM\\CurrentControlSet\\Control\\Lsa\\RunAsPPL`使LSA作为受保护进程轻量级（PPL）运行，这要求所有SSP DLL都必须由Microsoft签名。(引用: Graeber 2014, Microsoft Configure LSA)
- **外部参考**:
  - 描述: Graeber, M. (2014年10月). 恶意安全支持提供者DLL的分析。检索于2017年3月1日。

    - 来源: Microsoft Configure LSA

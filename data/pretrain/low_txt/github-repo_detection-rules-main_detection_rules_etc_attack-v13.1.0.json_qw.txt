**SolarWinds Compromise (C0024)**
- **APT29 (G0016)** 在 [SolarWinds 攻击](https://attack.mitre.org/campaigns/C0024) 期间获取了 PKI 密钥、证书文件以及 Active Directory Federation Services (AD FS) 容器中的私有加密密钥，以便解密相应的 SAML 签名证书。(引用: [Microsoft 365 Defender Solorigate](https://www.microsoft.com/security/blog/2020/12/28/using-microsoft-365-defender-to-coordinate-protection-against-solorigate/) 和 [Cybersecurity Advisory SVR TTP May 2021](https://www.ncsc.gov.uk/files/Advisory-further-TTPs-associated-with-SVR-cyber-actors.pdf))

**Naikon (G0019)**
- **Naikon (G0019)** 曾使用 `schtasks.exe` 在受攻击的网络中进行横向移动。(引用: [Bitdefender Naikon April 2021](https://www.bitdefender.com/files/News/CaseStudies/study/396/Bitdefender-PR-Whitepaper-NAIKON-creat5397-en-EN.pdf))

**macOS.OSAMiner (S1048)**
- **macOS.OSAMiner (S1048)** 可以解析本地 `system_profiler` 工具的输出，以确定机器是否运行在四核上。(引用: [SentinelLabs reversing run-only applescripts 2021](https://www.sentinelone.com/labs/fade-dead-adventures-in-reversing-malicious-run-only-applescripts/))

**Green Lambert (S0690)**
- **Green Lambert (S0690)** 可以从受攻击主机收集日期和时间信息。(引用: [Objective See Green Lambert for OSX Oct 2021](https://objective-see.com/blog/blog_0x68.html) 和 [Glitch-Cat Green Lambert ATTCK Oct 2021](https://www.glitch-cat.com/blog/green-lambert-and-attack))

**RTM (S0148)**
- **RTM (S0148)** 通过伪装成 PDF 文档的鱼叉式钓鱼附件进行传播。(引用: [Unit42 Redaman January 2019](https://unit42.paloaltonetworks.com/russian-language-malspam-pushing-redaman-banking-malware/))

**Turla (G0010)**
- **Turla (G0010)** 可能会尝试使用 `net use` 命令和预定义的密码列表连接受害网络内的系统。(引用: [Kaspersky Turla](https://securelist.com/the-epic-turla-operation/65545/))

**SolarWinds Compromise (C0024)**
- **APT29 (G0016)** 在 [SolarWinds 攻击](https://attack.mitre.org/campaigns/C0024) 期间修改了后门的时间戳，使其与合法 Windows 文件相匹配。(引用: [Microsoft Deep Dive Solorigate January 2021](https://www.microsoft.com/security/blog/2021/01/20/deep-dive-into-the-solorigate-second-stage-activation-from-sunburst-to-teardrop-and-raindrop/))

**Gold Dragon (S0249)**
- **Gold Dragon (S0249)** 在建立持久性后删除端点上的一个文件 `2.hwp`。(引用: [McAfee Gold Dragon](https://www.mcafee.com/blogs/other-blogs/mcafee-labs/gold-dragon-widens-olympics-malware-attacks-gains-permanent-presence-on-victims-systems/))

**Fox Kitten (G0117)**
- **Fox Kitten (G0117)** 使用 Volume Shadow Copy 访问 NTDS 中的凭据信息。(引用: [CISA AA20-259A Iran-Based Actor September 2020](https://us-cert.cisa.gov/ncas/alerts/aa20-259a))

**Machete (G0095)**
- **Machete (G0095)** 发送包含指向外部服务器链接的钓鱼邮件，该服务器上有 ZIP 和 RAR 压缩包。(引用: [Cylance Machete Mar 2017](https://threatvector.cylance.com/en_us/home/el-machete-malware-attacks-cut-through-latam.html) 和 [ESET Machete July 2019](https://www.welivesecurity.com/wp-content/uploads/2019/08/ESET_Machete.pdf))

**HOPLIGHT (S0376)**
- **HOPLIGHT (S0376)** 具备连接远程主机上传和下载文件的能力。(引用: [US-CERT HOPLIGHT Apr 2019](https://www.us-cert.gov/ncas/analysis-reports/AR19-100A))

**Grandoreiro (S0531)**
- **Grandoreiro (S0531)** 利用被攻陷的网站和 Google 广告诱骗受害者下载其安装程序。(引用: [Securelist Brazilian Banking Malware July 2020](https://securelist.com/the-tetrade-brazilian-banking-malware/97779/) 和 [IBM Grandoreiro April 2020](https://securityintelligence.com/posts/grandoreiro-malware-now-targeting-banks-in-spain/))

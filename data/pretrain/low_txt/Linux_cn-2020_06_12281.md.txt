如果您拥有一台老旧的 PC 或者是基于单板计算机的小型系统，那么运行一个占用资源较少且易于下载的 Linux 版本可能是您的最佳选择。以下是几个非常轻量级的 Linux 发行版，除了 Damn Small Linux 外，其他都是最近更新过的。每个发行版都可以在内存不足 1GB 的机器上运行，大多数甚至可以在更小的资源下运行。下面的列表按照字母顺序排列。

### Alpine Linux
![Alpine Linux - 130MB download](/data/attachment/album/202006/04/143612dnwezg1qdwglk0g0.jpg)
**下载大小**: 130 MB  
最初为虚拟服务器和嵌入式设备设计，Alpine Linux 直接运行在内存中，并以其安全性著称。虽然面向最终用户的功能有限，但它也支持桌面环境。最后一次公开发布是在 2019 年 10 月。

### antiX Linux
![antiX Linux - about 1GB download](/data/attachment/album/202006/04/143431u4nsx8llwwrnvuxd.png)
**下载大小**: 约 1GB  
antiX 发行版宣称完全不使用 systemd，这是现代 Linux 发行版的核心组件之一。取而代之的是所谓的“antiX Magic”。该安装程序提供了许多选项，可能会让新手感到困惑。这是一个基于 Debian 的构建版本，最新一次更新发生在 2019 年 10 月。

### ArchBang
![ArchBang - 600-700MB download](/data/attachment/album/202006/04/143433yo2b71t8kk7881dk.png)
**下载大小**: 600 - 700MB  
受 CrunchBang 启发，ArchBang 使用了 Arch Linux 的 Openbox 窗口管理器。最近一次发布是在 2019 年 11 月。

### Bodhi Linux
![Bodhi Linux - 700MB download](/data/attachment/album/202006/04/143436ljbv6jhuoub1ybhm.png)
**下载大小**: 700MB  
基于 Ubuntu 的轻量级发行版，采用 Moksha（一种基于 Enlightenment 的桌面环境）。它具有高度模块化和可定制性，允许用户根据需要构建简单或复杂的系统。最近刚刚进行了更新。

### BunsenLabs Helium
![BunsenLabs Linux Helium - 600MB to 1.2GB download](/data/attachment/album/202006/04/143437z2bzbbps3f5ibxxy.jpg)
**下载大小**: 600MB 至 1.2GB  
继承了 CrunchBang 和 CrunchBang++ 的遗产，BunsenLabs Helium 基于 Debian 并采用了 Openbox 窗口管理器。最新的版本发布于 2019 年 7 月。

### Damn Small Linux
![Damn Small Linux - 50MB download](/data/attachment/album/202006/04/143437i4hiqu7qjyyyjcab.jpg)
**下载大小**: 50MB  
尽管自 2008 年以来没有正式发布新版本，但 Damn Small Linux 在 2012 年曾推出过一个候选版本。对于旧硬件来说，这仍然是一个不错的选择，但由于较新的安全漏洞尚未修补，因此不适合频繁上网使用。其最轻量级的形式可以在仅 16MB 内存中运行，完全加载则需要 128MB 内存。

### Elive
![Elive - 2.4GB download](/data/attachment/album/202006/04/143443nt5mf5tmqtd6ttqm.png)
**下载大小**: 2.4GB  
Elive 是一款预装了 Enlightenment 桌面环境的发行版，附带 Elpanel 控制面板。遵循典型的 Linux 风格，它的口号是：“虽然 Elive 不适合所有人，但你是例外吗？” 它只需要 256MB 内存即可运行。最后一次更新是在 2019 年 10 月。

### Linux Lite
![Linux Lite - 1.4GB download](/data/attachment/album/202006/04/143444moorzykujrolhr7x.png)
**下载大小**: 1.4GB  
虽然不像某些微型发行版那样小巧，但 Linux Lite 也不会显得臃肿。它基于 Ubuntu，提供了一个令人愉悦的桌面体验，配备了所有必需的组件。它可以运行在 768MB 内存中，但推荐使用 1GB 内存。最近刚刚进行了更新。

### Lubuntu
![Lubuntu - 1.6GB download](/data/attachment/album/202006/04/143445btt3kvivvi2ozvuq.png)
**下载大小**: 1.6GB  
Lubuntu 的名字意味着“轻量级 Ubuntu”，它使用 LXQt 桌面环境，允许用户自由选择安装哪些典型的 Ubuntu 软件包。最近刚刚发布了新版本。

### LXLE
![LXLE - 1.2GB download](/data/attachment/album/202006/04/143446gnromiflgqtomesq.jpg)
**下载大小**: 1.2GB  
基于 Ubuntu，LXLE 的口号是“重新唤醒那台老旧的 PC”。它旨在通过丰富的功能和应用程序将一台老旧的 PC 转变为一台实用的工作站，满足基本的计算需求。最后一次公开发布是在 2019 年 9 月。

### Puppy Linux
![Puppy Linux - around 300 MB download](/data/attachment/album/202006/04/143447v51bt5zrldraxo8z.jpg)
**下载大小**: 约 300MB  
Puppy Linux 并不是一个单一的发行版，而是有 11 个不同的变体。尽管不再像以前那样紧凑，但 1GB 的内存足以让它良好运行。它是最早的轻量级发行版之一，最后一次更新是在 2019 年 3 月。

### Raspberry Pi OS
![Raspberry Pi OS - 400MB - 1.2GB](/data/attachment/album/202006/04/142434e0pmwt6j6cm6gp9p.jpg)
**下载大小**: 400MB 至 1.2GB  
专为树莓派设计的 Debian 发行版（前身为 Raspbian），Raspberry Pi OS 是我在日常生活中使用的发行版之一。我将其部署在三台正在运行的树莓派服务器上，以及几台暂时闲置的设备上。我不使用图形界面，而是连接到 OctoPi 网络接口来控制我的 3D 打印机。最近刚刚进行了更新。

### SliTaz
![SliTaz - 50MB download](/data/attachment/album/202006/04/143448vpza6fdbpsdgrd6u.png)
**下载大小**: 50MB  
SliTaz 是一款可以直接从 CD-ROM 或 USB 驱动器启动并完全运行在内存中的 Linux 发行版。尽管体积仅有约 50MB，但它具备强大的安全特性，既可以选择只安装基础系统，也可以选择带有精简 X 环境的版本。最新的滚动发布版本出现在 2018 年。

### SparkyLinux 基础版
![SparkyLinux base edition - 540MB download](/data/attachment/album/202006/04/143450clh4wwhhc67hes3a.png)
**下载大小**: 540MB  
另一个基于 Debian 的发行版，Sparky 设计初衷是为了给家庭用户提供一个相对简单的开箱即用的 Enlightenment 体验，同时支持大约 20 种窗口管理器和桌面环境。它在 2019 年 11 月进行了更新。

### Tiny Core Linux
![Tiny Core Linux — 11MB download](/data/attachment/album/202006/04/143450c6ql58me5l5t7t53.png)
**下载大小**: 11MB  
共有三个版本，最小的一个仅有 11MB 的下载量。如果不需要图形界面，Tiny Core Linux 可以在 64MB 内存中运行。最后一次更新是在 2019 年 1 月。

### 结语
以上就是我们整理的一些轻量级 Linux 发行版。您尝试过其中的哪一个呢？或者是否有其他未提及的轻量级发行版值得推荐？欢迎您在下方留言分享您的经验。

### Re-Loader Activator 介绍

Re-Loader Activator 是一款用于激活 Windows 系统和 Office 产品的工具，支持全系列的 Office 和 Windows 产品。该工具具有以下特点：

- **多语言支持**：自带简体中文界面。
- **离线使用**：无需联网即可使用。
- **多种激活方式**：
  - KMS 激活
  - AntiWPA 激活
  - OEM 激活
- **其他功能**：
  - 自定义 KMS
  - 移除 KMS
  - 移除水印
  - 个性化 OEM 系统信息

### 支持的激活方式

- **AntiWPA 激活**：Windows XP
- **OEM 激活**：Windows Vista, Windows 7, Windows Server 2008 (R2), Windows Server 2012 (R2), Windows Server 2016 RTM
- **KMS 激活**：Windows Vista, Windows 7, Windows 8.0, Windows 8.1, Windows 10, Windows Server 2008 (R2), Windows Server 2012 (R2), Windows Server 2016 RTM
- **KMS 激活**（适用于 Office）：Office 2010, Office 2013, Office 2016 所有版本

### 系统要求
- Win7 需要安装 .NET Framework 4.0 或更高版本。
- Win8/Win10 无需额外安装框架。

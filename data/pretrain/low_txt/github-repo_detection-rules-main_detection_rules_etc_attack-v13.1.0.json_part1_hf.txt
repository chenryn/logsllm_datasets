**用户账户监控（2022年3月）**

- **创建时间**：2022-03-30T14:26:51.832Z
- **描述**：监控用户账户的更改，以检测意外的属性修改，如密码或状态（启用/禁用）。Windows 事件日志可能会记录与对手尝试移除账户访问权限相关的活动：
  - 事件 ID 4723：尝试更改账户密码
  - 事件 ID 4724：尝试重置账户密码
  - 事件 ID 4725：用户账户被禁用
- **警告**：这些事件 ID 可能会产生大量误报，因此应将其与基线知识进行比较，并尽可能将修改事件与其他恶意活动指示相关联。
- **修改时间**：2022-03-30T14:26:51.832Z

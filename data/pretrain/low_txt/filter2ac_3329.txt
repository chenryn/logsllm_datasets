**ATT&CK ID:** M1006  
**ATT&CK Technique Name:** Use Recent OS Version

### Description
Recent versions of operating systems have introduced various privacy and security enhancements. For example, Android 11 enhanced package visibility by filtering the results returned from the package manager. Similarly, iOS 12 removed a private API that was previously used to list installed applications on non-App Store applications.

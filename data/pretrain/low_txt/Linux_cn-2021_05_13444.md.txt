### Google 员工承认用户无法在公司的限制下保护自己的隐私

![](/data/attachment/album/202105/31/102110l866byl0qb8l28y8.jpg)

根据 The Verge 的报道，在一起针对 Google 的诉讼中曝光的文件显示，Google 的高管和工程师都清楚，公司让用户保护位置数据隐私存在极大的难度。

> 在昨天的硬核观察中，我们提到 Google 表示不会在功能设置中为自己添加后门，但同时我们也表达了对此的不信任。现在看来，Google 在多个方面依然表现出对个人隐私的渴望，因为这关系到其广告收益。

### 微软为开发者推出 Java 开发文档

![](/data/attachment/album/202105/31/102208xahuuhasdhnsdauz.jpg)

微软发布了其 OpenJDK 的 GA (一般可用版本)，并提供了新的 Java 开发教程，帮助开发者更好地在 Windows 上开发 Java 应用程序。此次发布包括 OpenJDK、VS Code 插件、GitHub 使用指南以及 Java 在 Azure 上的部署教程等资源。

> 作为一门广泛使用的编程语言，Java 终于进入了微软的支持列表，并为开发者提供了一系列使用指南。

### Jim Keller：AMD 员工曾认为无法超越 Intel

![](/data/attachment/album/202105/31/102150lyfygf7fn2b5n7qf.jpg)

在 anandtech 对 Tenstorrent 公司的一次访谈中，该公司 CTO Jim Keller（前 AMD Zen 架构研发团队负责人）谈到了一些关于 AMD Zen 的轶事。他提到，在设计 Zen 时，一小部分人相信 Zen 将会在一代人的时间内追赶上 Intel；一部分人认为如果成功了会非常酷；而更多的人则认为这是不可能的。

> 我们见证了 AMD Zen 架构的成功，这让我想起了一句话：“悲观者往往正确，但乐观者往往成功”。正是像 Jim Keller 这样的乐观者，最终带领工程师们实现了 AMD Zen 架构，使 AMD 成功翻身。

### Mimikatz

**创建日期**: 2020-03-19  
**描述**: Mimikatz 是一种用于凭据转储的工具，能够获取账户和密码信息，这些信息对于攻击者进一步访问系统和企业网络资源非常有用。Mimikatz 具备多种方式来获取凭据信息，包括通过 DCSync/NetSync 获取。(引用: [Deply Mimikatz](https://github.com/gentilkiwi/mimikatz), [GitHub Mimikatz lsadump Module](https://github.com/gentilkiwi/mimikatz/wiki/module-~-lsadump), [Directory Services Internals DPAPI Backup Keys Oct 2015](https://www.dsinternals.com/en/retrieving-dpapi-backup-keys-from-active-directory/), [NCSC Joint Report Public Tools](https://www.ncsc.gov.uk/report/joint-report-on-publicly-available-hacking-tools), [Cobalt Strike Manual 4.3 November 2020](https://web.archive.org/web/20210708035426/https://www.cobaltstrike.com/downloads/csmanual43.pdf))

### Astaroth

**创建日期**: 2019-08-15  
**描述**: Astaroth 恶意软件通过将 XSL 脚本参数 vShow 设置为零来加载其模块，从而以隐藏窗口的形式打开应用程序。(引用: [Cybereason Astaroth Feb 2019](https://www.cybereason.com/blog/information-stealing-malware-targeting-brazil-full-research))

### Bisonal

**创建日期**: 2022-01-26  
**描述**: Bisonal 使用 Windows API 与服务控制管理器通信，以执行线程。(引用: [Talos Bisonal Mar 2020](https://blog.talosintelligence.com/2020/03/bisonal-10-years-of-play.html))

### Gallmaker

**创建日期**: 2019-04-16  
**描述**: Gallmaker 组织使用 WinZip 工具，可能用于在数据外泄前对其进行归档。(引用: [Symantec Gallmaker Oct 2018](https://www.symantec.com/blogs/threat-intelligence/gallmaker-attack-group))

### PowerSploit

**创建日期**: 2020-03-20  
**描述**: PowerSploit 包含一组外泄模块，可以利用 Mimikatz 收集凭据。(引用: [GitHub PowerSploit May 2012](https://github.com/PowerShellMafia/PowerSploit), [PowerSploit Documentation](http://powersploit.readthedocs.io))

### Evilnum

**创建日期**: 2021-01-27  
**描述**: Evilnum 组织可以从受害者处收集电子邮件凭据。(引用: [ESET EvilNum July 2020](https://www.welivesecurity.com/2020/07/09/more-evil-deep-look-evilnum-toolset/))

### MuddyWater

**创建日期**: 2019-01-30  
**描述**: MuddyWater 组织使用的恶意软件可以捕获受害者的屏幕截图。(引用: [Securelist MuddyWater Oct 2018](https://securelist.com/muddywater/88059/))

### OilRig 和 APT34

**创建日期**: 2018-01-16  
**描述**: OilRig 和 APT34 组织的相关活动。(引用: [Palo Alto OilRig May 2016](http://researchcenter.paloaltonetworks.com/2016/05/the-oilrig-campaign-attacks-on-saudi-arabian-organizations-deliver-helminth-backdoor/), [FireEye APT34 Dec 2017](https://www.fireeye.com/blog/threat-research/2017/12/targeted-attack-in-middle-east-by-apt34.html))

### Moses Staff

**创建日期**: 2022-10-11  
**描述**: Moses Staff 组织使用批处理脚本来禁用特定远程机器上的 Windows 防火墙。(引用: [Checkpoint MosesStaff Nov 2021](https://research.checkpoint.com/2021/mosesstaff-targeting-israeli-companies/))

### 监控新构建的 Pod

**创建日期**: 2022-03-30  
**描述**: 监控新构建的 Pod，这些 Pod 可能会部署容器到环境中以促进执行或规避防御。

### Orz

**创建日期**: 2018-04-18  
**描述**: 一些 Orz 版本包含一个名为 MockDll 的嵌入式 DLL，该 DLL 使用 Process Hollowing 和 regsvr32 来执行另一个有效载荷。(引用: [Proofpoint Leviathan Oct 2017](https://www.proofpoint.com/us/threat-insight/post/leviathan-espionage-actor-spearphishes-maritime-and-defense-targets))

### DanBot

**创建日期**: 2022-06-13

### PowerDuke
